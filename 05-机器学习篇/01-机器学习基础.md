## 第一部分：什么是机器学习

### 30.1 机器学习的定义

#### 概念说明

机器学习（Machine Learning）是人工智能的一个分支，它使计算机能够从数据中学习规律，并用这些规律对未知数据进行预测或决策。

**传统编程 vs 机器学习：**

```
┌─────────────────────────────────────────────────────────────┐
│                传统编程 vs 机器学习                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   传统编程：                                                │
│   输入 (数据) + 规则 (程序) → 输出 (结果)                   │
│                                                             │
│   例如：计算平均工资                                          │
│   输入：[5000, 6000, 7000]                                  │
│   规则：sum(data) / len(data)                               │
│   输出：6000                                                │
│                                                             │
│   机器学习：                                                │
│   输入 (数据) + 输出 (标签) → 规则 (模型)                   │
│                                                             │
│   例如：预测房价                                              │
│   输入：[面积，房间数，地段...]                              │
│   输出：[房价]                                              │
│   模型：学习面积、房间数与房价的关系                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**经典定义：**

> "如果一个计算机程序在某项任务上的表现，随着经验的积累而提高，那么我们说这个程序从经验中学习了。" — Tom Mitchell, 1997

### 30.3 机器学习的类型

#### 概念说明

根据学习方式的不同，机器学习主要分为以下几类。

**主要类型：**

```
┌─────────────────────────────────────────────────────────────┐
│                  机器学习类型                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   1. 监督学习 (Supervised Learning)                         │
│   ┌───────────────────────────────────────────────────┐     │
│   │  数据特点：有标签 (输入 + 期望输出)                 │     │
│   │                                                   │     │
│   │  任务类型：                                        │     │
│   │  • 分类 (Classification)：预测类别                 │     │
│   │    例如：垃圾邮件识别 (垃圾/正常)                  │     │
│   │    算法：逻辑回归、决策树、SVM、神经网络          │     │
│   │                                                   │     │
│   │  • 回归 (Regression)：预测连续值                  │     │
│   │    例如：房价预测、销量预测                       │     │
│   │    算法：线性回归、岭回归、随机森林回归           │     │
│   └───────────────────────────────────────────────────┘     │
│                                                             │
│   2. 无监督学习 (Unsupervised Learning)                     │
│   ┌───────────────────────────────────────────────────┐     │
│   │  数据特点：无标签 (只有输入数据)                   │     │
│   │                                                   │     │
│   │  任务类型：                                        │     │

│   │                                                   │     │
│   │  • 降维 (Dimensionality Reduction)：减少特征数     │     │

│   │                                                   │     │
│   │  • 关联规则 (Association)：发现项之间的关系        │     │
│   │    例如：购物篮分析                                │     │
│   │    算法：Apriori、FP-Growth                       │     │
│   └───────────────────────────────────────────────────┘     │
│                                                             │
│   3. 半监督学习 (Semi-supervised Learning)                  │
│   ┌───────────────────────────────────────────────────┐     │
│   │  数据特点：少量有标签 + 大量无标签                  │     │
│   │  应用场景：标签获取成本高                           │     │
│   │  例如：医学图像分析、语音识别                       │     │
│   └───────────────────────────────────────────────────┘     │
│                                                             │
│   4. 强化学习 (Reinforcement Learning)                      │
│   ┌───────────────────────────────────────────────────┐     │
│   │  学习方式：通过与环境交互，根据奖励/惩罚学习        │     │
│   │  应用场景：游戏 AI、机器人控制、自动驾驶            │     │
│   │  算法：Q-Learning、深度强化学习                    │     │
│   └───────────────────────────────────────────────────┘     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 30.4 监督学习 vs 无监督学习

#### 概念说明

这是机器学习最重要的两种类型，理解它们的区别至关重要。

**对比表：**

| 特性 | 监督学习 | 无监督学习 |
|------|---------|-----------|
| **数据标签** | 有标签 | 无标签 |
| **目标** | 预测输出 | 发现模式 |
| **训练数据** | (X, y) | (X) |
| **评估** | 准确率、MSE 等 | 轮廓系数、解释方差 |
| **任务** | 分类、回归 | 聚类、降维 |
| **算法** | 回归、决策树、SVM | K-Means、PCA |
| **难度** | 相对容易 | 相对困难 |

**图示：**


| 3. 数据探索 | 了解数据分布、相关性、异常值 | 15% |
| 4. 数据清洗 | 处理缺失值、异常值、重复值 | 20% |
| 5. 特征工程 | 特征选择、特征构造、特征转换 | 25% |

| 10. 模型部署 | 将模型应用于生产环境 | 5% |

> **重要提示：** 数据预处理和特征工程通常占据 60% 以上的时间，但这对模型性能至关重要！

### 30.6 数据集的组成

#### 概念说明

机器学习数据集通常包含特征矩阵 X 和目标向量 y。

**数据结构：**

```
┌─────────────────────────────────────────────────────────────┐
│                    数据集结构                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   特征矩阵 X (Features)           目标向量 y (Target)       │
│                                                             │
│   ┌─────────────────────┐                                  │
│   │ 特征 1  特征 2  特征 3 │                                  │
│   ├─────────────────────┤                                  │
│   │  5.1    3.5    1.4   │  →  样本 1     │   0 (山鸢尾)   │
│   │  4.9    3.0    1.4   │  →  样本 2     │   0 (山鸢尾)   │
│   │  7.0    3.2    4.7   │  →  样本 3     │   1 (变色鸢尾) │
│   │  6.4    3.2    4.5   │  →  样本 4     │   1 (变色鸢尾) │
│   │  ...    ...    ...   │                                  │
│   └─────────────────────┘                                  │
│                                                             │
│   形状：(n_samples, n_features)     形状：(n_samples,)      │
│   例如：(150, 4)                  例如：(150,)              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**术语解释：**

| 术语 | 英文 | 说明 |
|------|------|------|
| 样本 | Sample | 数据集中的一行，一个观测值 |
| 特征 | Feature | 描述样本的属性，输入变量 |
| 目标 | Target | 要预测的值，输出变量 |
| 特征空间 | Feature Space | 所有特征构成的多维空间 |
| 标签 | Label | 分类问题中的目标值 |

### 30.7 训练集与测试集

#### 概念说明

为了评估模型性能，需要将数据集分为训练集和测试集。

**划分目的：**

```
┌─────────────────────────────────────────────────────────────┐
│              为什么需要训练集/测试集划分                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   训练集 (Training Set)：                                   │
│   • 用途：训练模型，让模型学习规律                           │
│   • 占比：通常 70%-80%                                      │
│   • 模型会"看到"这些数据                                    │
│                                                             │
│   测试集 (Test Set)：                                       │
│   • 用途：评估模型性能                                       │
│   • 占比：通常 20%-30%                                      │
│   • 模型不会"看到"这些数据                                  │
│   • 模拟未知数据的表现                                       │
│                                                             │
│   ⚠️ 重要原则：                                              │
│   • 测试集必须在训练过程中完全"不可见"                       │
│   • 不能用测试集训练模型，否则评估结果会过于乐观             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**划分示例：**

```python
from sklearn.model_selection import train_test_split

# X: 特征矩阵，y: 目标向量
X_train, X_test, y_train, y_test = train_test_split(
    X, y,
    test_size=0.2,        # 测试集占 20%
    random_state=42,      # 随机种子，保证结果可复现
    stratify=y            # 分层抽样，保持类别比例
)

print(f"训练集大小：{len(X_train)}")  # 120
print(f"测试集大小：{len(X_test)}")   # 30
```

### 30.10 第一个机器学习程序

#### 概念说明

使用 scikit-learn 内置的鸢尾花数据集，完成一个完整的分类任务。

**完整代码：**

```python
# 1. 导入必要的库
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

# 2. 加载数据集
iris = load_iris()
X = iris.data      # 特征矩阵 (150, 4)
y = iris.target    # 目标向量 (150,)

print(f"数据集大小：{len(X)}")  # 150
print(f"特征数量：{X.shape[1]}")  # 4
print(f"类别名称：{iris.target_names}")  # ['setosa' 'versicolor' 'virginica']

# 3. 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(
    X, y,
    test_size=0.2,      # 20% 作为测试集
    random_state=42     # 随机种子
)

print(f"训练集：{len(X_train)}, 测试集：{len(X_test)}")

# 4. 创建并训练模型
model = KNeighborsClassifier(n_neighbors=3)  # K=3 的 K 近邻
model.fit(X_train, y_train)  # 训练模型

# 5. 进行预测
y_pred = model.predict(X_test)

# 6. 评估模型
accuracy = accuracy_score(y_test, y_pred)
print(f"准确率：{accuracy:.2%}")  # 例如：100.00%

# 7. 预测新数据
new_sample = [[5.1, 3.5, 1.4, 0.2]]  # 新的鸢尾花样本
prediction = model.predict(new_sample)
print(f"预测类别：{iris.target_names[prediction[0]]}")  # setosa
```

```
预测类别：setosa
```
