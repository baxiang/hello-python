## 核心知识点

### 分类评估指标

- 混淆矩阵：TP、TN、FP、FN
- 准确率、精确率、召回率、F1 分数
- ROC 曲线与 AUC

### 回归评估指标

- MSE、RMSE、MAE、R²

### 超参数调优

- 网格搜索：GridSearchCV
- 随机搜索：RandomizedSearchCV

### 模型保存

- joblib.dump() / joblib.load()
