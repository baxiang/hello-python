## 核心知识点

### 回归算法

- 线性回归：预测连续值
- 正则化：Ridge(L2)、Lasso(L1)

### 分类算法

- 逻辑回归：二分类
- 决策树：可解释性强
- 随机森林：准确率高
- SVM：高维数据有效

### 算法选择

- 线性关系 → 线性回归/逻辑回归
- 非线性关系 → 决策树/随机森林/SVM
- 需要可解释性 → 决策树
- 追求准确率 → 随机森林
