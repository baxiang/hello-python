## 核心知识点

### 聚类算法

- K-Means：最常用，需指定 K
- 层次聚类：树状图可视化
- DBSCAN：任意形状簇，识别噪声

### 降维算法

- PCA：线性降维，特征压缩
- t-SNE：非线性降维，可视化

### K 值选择

- 手肘法：SSE 曲线的拐点
- 轮廓系数法：选择最大值
