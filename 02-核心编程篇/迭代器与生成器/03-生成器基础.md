# 生成器基础

本章讲解 Python 生成器的基础知识，包括 yield 关键字和生成器表达式。

---

## 为什么需要生成器

### 内存效率对比

```python
import sys

# 列表推导式：存储所有元素
list_comp = [x for x in range(1000000)]
print(f"列表大小：{sys.getsizeof(list_comp)} 字节")  # 约 8MB

# 生成器表达式：只存储状态
gen_comp = (x for x in range(1000000))
print(f"生成器大小：{sys.getsizeof(gen_comp)} 字节")  # 约 200 字节
```

**生成器的优势**：
- 惰性求值：只在需要时才计算
- 内存友好：不一次性生成所有数据
- 代码简洁：用 `yield` 代替复杂的迭代器类

## yield 关键字

### return vs yield

```python
def with_return(n):
    """使用 return - 立即返回并结束"""
    for i in range(n):
        return i  # 只返回第一个值

def with_yield(n):
    """使用 yield - 返回值但暂停"""
    for i in range(n):
        yield i  # 返回值，函数暂停

print(with_return(5))      # 0（只返回第一个）
print(list(with_yield(5))) # [0, 1, 2, 3, 4]
```

### 生成器执行流程

```python
def simple_generator():
    print(">>> A: 开始执行")
    yield 1
    print(">>> B: 继续执行")
    yield 2
    print(">>> C: 再次执行")
    yield 3
    print(">>> D: 结束执行")

# 创建生成器（函数不执行！）
gen = simple_generator()
print("生成器已创建")

# 第一次调用 next()
print(f"返回：{next(gen)}")
# >>> A: 开始执行
# 返回：1

# 第二次调用 next()
print(f"返回：{next(gen)}")
# >>> B: 继续执行
# 返回：2
```

**执行流程**：
```
gen = simple_generator()   # 创建生成器，函数不执行！
         │
    next(gen)
         ▼
执行到 yield 1 → 返回 1，暂停
         │
    next(gen)
         ▼
从暂停处继续 → 执行到 yield 2 → 返回 2，暂停
         │
    next(gen)
         ▼
从暂停处继续 → 执行到 yield 3 → 返回 3，暂停
         │
    next(gen)
         ▼
从暂停处继续 → 没有 yield → StopIteration
```

## 生成器函数示例

### 倒数生成器

```python
def countdown(n):
    """倒数生成器"""
    while n > 0:
        yield n
        n -= 1

# 使用
for num in countdown(5):
    print(num)
# 5, 4, 3, 2, 1
```

### 斐波那契生成器

```python
def fibonacci(max_count=None):
    """斐波那契数列生成器"""
    a, b = 0, 1
    count = 0
    while max_count is None or count < max_count:
        yield a
        a, b = b, a + b
        count += 1

# 使用
for num in fibonacci(10):
    print(num)
```

### 大文件逐行读取

```python
def read_large_file(filename):
    """逐行读取大文件，内存友好"""
    with open(filename, "r", encoding="utf-8") as f:
        for line in f:
            yield line.strip()

# 使用
for line in read_large_file("huge_log.txt"):
    if "ERROR" in line:
        print(line)
```

## 生成器表达式

### 语法对比

```python
# 列表推导式：立即生成所有数据（方括号）
squares_list = [x ** 2 for x in range(5)]
print(squares_list)       # [0, 1, 4, 9, 16]

# 生成器表达式：惰性生成（圆括号）
squares_gen = (x ** 2 for x in range(5))
print(list(squares_gen))  # [0, 1, 4, 9, 16]
```

### 实用场景

```python
# 大数值范围计算
total = sum(x for x in range(100000000))

# 多条件过滤
numbers = range(100)
result = (
    x for x in numbers
    if x % 2 == 0      # 偶数
    if x % 3 == 0      # 能被 3 整除
    if x < 50          # 小于 50
)
print(list(result))  # [0, 6, 12, 18, 24, 30, 36, 42, 48]

# 直接作为函数参数
data = [1, 2, 3, 4, 5]
total = sum(x * 2 for x in data)  # 简洁写法
```
