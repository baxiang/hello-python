# 迭代基础

本章讲解 Python 迭代的基本概念，包括可迭代对象、迭代器和 for 循环的工作原理。

---

## 迭代概念关系

### 三者的关系

```
┌─────────────────────────────────────────────────────────────┐
│                    迭代概念关系图                            │
│                                                             │
│   迭代 (Iteration)                                          │
│   ───────────────                                           │
│   重复执行某操作的过程（如 for 循环遍历）                      │
│                                                             │
│         ▼                                                   │
│                                                             │
│   可迭代对象 (Iterable)                                     │
│   ───────────────────                                       │
│   • 可以使用 for...in 遍历的对象                             │
│   • 实现了 __iter__() 方法                                  │
│   • 例如：list, tuple, str, dict, set, range               │
│                                                             │
│         ▼ iter() 获取                                       │
│                                                             │
│   迭代器 (Iterator)                                         │
│   ───────────────                                           │
│   • 可以被 next() 调用的对象                                │
│   • 实现了 __iter__() 和 __next__() 方法                    │
│   • 每次返回一个值，直到耗尽                               │
│   • 只能用一次，用完即弃                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**一句话总结**：
> **可迭代对象** 是"可以遍历的东西"，**迭代器** 是"用来遍历的工具"

## 判断类型

```python
from collections.abc import Iterable, Iterator

# 常见对象类型判断
test_objects = [
    [1, 2, 3],           # list
    "hello",             # str
    {"a": 1},            # dict
    iter([1, 2, 3]),     # 迭代器
]

for obj in test_objects:
    is_iterable = isinstance(obj, Iterable)
    is_iterator = isinstance(obj, Iterator)
    print(f"{type(obj).__name__:15} | 可迭代：{is_iterable} | 迭代器：{is_iterator}")
```

**关键结论**：
- 所有迭代器都是可迭代对象
- 但可迭代对象不一定是迭代器
- list/str/dict 是**可迭代对象**，但不是**迭代器**

## for 循环的工作原理

### 内部机制

你平时写的 for 循环：
```python
numbers = [1, 2, 3, 4, 5]
for num in numbers:
    print(num)
```

**等价于**以下代码：
```python
numbers = [1, 2, 3, 4, 5]

# 第 1 步：通过 iter() 获取迭代器
iterator = iter(numbers)

# 第 2 步：循环调用 next() 获取元素
while True:
    try:
        num = next(iterator)
        print(num)
    except StopIteration:
        # 迭代器耗尽，退出循环
        break
```

### 手动使用迭代器

```python
numbers = [1, 2, 3]
iterator = iter(numbers)  # 获取迭代器

print(next(iterator))  # 1
print(next(iterator))  # 2
print(next(iterator))  # 3

# 再继续调用会抛出 StopIteration
# print(next(iterator))  # ❌ StopIteration

# 带默认值的 next()
print(next(iterator, "耗尽"))  # "耗尽"（不抛异常）
```
