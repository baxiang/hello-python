# 生成器高级特性

本章讲解 Python 生成器的高级特性，包括 send()、throw() 和 close() 方法。

---

## send() 方法

### 双向通信

普通生成器只能"输出"值，使用 `send()` 后可以"输入"值：

```python
def accumulator():
    """累加器生成器"""
    total = 0
    while True:
        # yield 表达式的值是外部 send() 传入的值
        value = yield total
        if value is not None:
            total += value

# 使用
acc = accumulator()

# 第 1 步：启动生成器（必须先用 next() 或 send(None)）
print(next(acc))        # 0

# 第 2 步：发送值
print(acc.send(10))     # 10
print(acc.send(20))     # 30
print(acc.send(5))      # 35
```

### send() 执行流程

```
acc = accumulator()   # 创建生成器
         │
    next(acc)
         ▼
执行到 yield total
返回 0，暂停在 value = yield total
         │
    acc.send(10)
         ▼
value = 10
total = 0 + 10 = 10
继续循环，回到 yield total
返回 10，暂停
```

### 使用注意事项

```
┌─────────────────────────────────────────┐
│         send() 使用注意事项             │
├─────────────────────────────────────────┤
│                                         │
│  1. 必须先启动生成器                    │
│  ─────────────────────                  │
│  gen = generator()                      │
│  next(gen)  # 或 gen.send(None)        │
│                                         │
│  2. yield 是表达式，有返回值            │
│  ─────────────────────                  │
│  value = yield result                   │
│           │      │                      │
│           │      └── 返回给外部的值     │
│           └── send() 传入的值           │
│                                         │
│  3. send(None) 等价于 next()            │
│                                         │
└─────────────────────────────────────────┘
```

## throw() 方法

### 向生成器抛出异常

```python
def my_generator():
    """带异常处理的生成器"""
    try:
        print("生成器开始")
        yield 1
        yield 2
        yield 3
    except ValueError as e:
        print(f"捕获到 ValueError: {e}")
        yield -1
    finally:
        print("生成器结束")

# 使用
gen = my_generator()
print(next(gen))  # 生成器开始 \n 1
print(next(gen))  # 2

# 向生成器抛出异常
gen.throw(ValueError("测试异常"))
# 捕获到 ValueError: 测试异常 \n -1
```

## close() 方法

### 关闭生成器

```python
def infinite_generator():
    """无限生成器"""
    try:
        i = 0
        while True:
            yield i
            i += 1
    finally:
        print("生成器被清理")

gen = infinite_generator()
print(next(gen))  # 0
print(next(gen))  # 1

# 关闭生成器
gen.close()
# 输出：生成器被清理

# 关闭后再调用会抛出 StopIteration
# next(gen)  # ❌ StopIteration
```

## yield from

### 委托生成器

```python
def sub_generator():
    """子生成器"""
    yield 1
    yield 2
    yield 3

def main_generator():
    """主生成器"""
    yield "开始"
    yield from sub_generator()  # 委托给子生成器
    yield "结束"

# 使用
for value in main_generator():
    print(value)
# 开始
# 1
# 2
# 3
# 结束
```

### 展平嵌套列表

```python
def flatten(nested):
    """递归展平嵌套列表"""
    for item in nested:
        if isinstance(item, (list, tuple)):
            yield from flatten(item)
        else:
            yield item

# 使用
nested = [1, [2, 3], [4, [5, 6]], 7]
print(list(flatten(nested)))  # [1, 2, 3, 4, 5, 6, 7]
```
