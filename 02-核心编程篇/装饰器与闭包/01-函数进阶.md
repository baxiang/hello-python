本章讲解 Python 函数的高级特性，包括函数作为对象和嵌套函数。

## 函数是对象

### 一等公民

```
┌─────────────────────────────────────────┐
│       Python 函数是"一等公民"           │
├─────────────────────────────────────────┤
│                                         │
│  一等公民（First-class Citizen）        │
│  ─────────────────────────────          │
│  可以在程序中任意位置使用的对象         │
│                                         │
│  函数可以：                             │
│  ────────────                           │
│  ✅ 赋值给变量                          │
│  ✅ 作为参数传递给其他函数              │
│  ✅ 作为函数的返回值                    │
│  ✅ 存储在数据结构中（列表、字典）      │
│                                         │
└─────────────────────────────────────────┘
```

### 函数的各种用法

```python
# ─────────────────────────────────────
# 1. 赋值给变量
# ─────────────────────────────────────
def greet(name):
    return f"Hello, {name}!"

say_hello = greet
print(say_hello("Alice"))  # Hello, Alice!

# ─────────────────────────────────────
# 2. 作为参数传递
# ─────────────────────────────────────
def execute(func, value):
    """执行传入的函数"""
    return func(value)

result = execute(greet, "David")
print(result)  # Hello, David!

# ─────────────────────────────────────
# 3. 作为返回值
# ─────────────────────────────────────
def get_operation(op):
    """返回不同的操作函数"""
    if op == "add":
        return lambda a, b: a + b
    elif op == "mul":
        return lambda a, b: a * b

add_func = get_operation("add")
print(add_func(3, 5))  # 8

# ─────────────────────────────────────
# 4. 存储在数据结构中
# ─────────────────────────────────────
def add(a, b): return a + b
def sub(a, b): return a - b
def mul(a, b): return a * b

op_dict = {"+": add, "-": sub, "*": mul}
print(op_dict["+"](10, 5))  # 15
```

## 嵌套函数

### 基本概念

```python
def outer():
    """外层函数"""
    print("进入外层函数")

    def inner():
        """内层函数"""
        print("  进入内层函数")
        print("  离开内层函数")

    inner()  # 调用内层函数
    print("离开外层函数")

outer()
# 进入外层函数
#   进入内层函数
#   离开内层函数
# 离开外层函数

# ❌ 内层函数在外部不可见
# inner()  # NameError!
```

## LEGB 作用域规则

### 查找顺序

```
┌─────────────────────────────────────────┐
│         LEGB 作用域查找规则             │
├─────────────────────────────────────────┤
│                                         │
│  L - Local（局部作用域）                │
│      当前函数内部的变量                 │
│                                         │
│  E - Enclosing（封闭作用域）            │
│      外层嵌套函数的变量                 │
│                                         │
│  G - Global（全局作用域）               │
│      模块级别的变量                     │
│                                         │
│  B - Built-in（内置作用域）             │
│      Python 内置的变量和函数            │
│                                         │
│  查找过程：                             │
│  Local → Enclosing → Global → Built-in │
│                                         │
└─────────────────────────────────────────┘
```

### 示例

```python
x = "全局变量"  # Global

def outer():
    y = "外层变量"  # Enclosing

    def inner():
        z = "内层变量"  # Local
        print(f"inner: {z}")    # Local
        print(f"inner: {y}")    # Enclosing
        print(f"inner: {x}")    # Global

    inner()

outer()
```
