## 1. 类型提示简介

### 1.1 什么是类型提示？

类型提示（Type Hints）是 Python 3.5+ 引入的功能，用于为变量、函数参数和返回值添加类型注解。

```python
# 不使用类型提示
def greet(name):
    return f"Hello, {name}"

# 使用类型提示
def greet(name: str) -> str:
    return f"Hello, {name}"
```

### 1.2 为什么使用类型提示？

| 优势 | 说明 |
|------|------|
| **提高可读性** | 明确函数期望的输入输出类型 |
| **静态检查** | 在运行前发现类型错误 |
| **IDE 支持** | 提供更好的代码补全和提示 |
| **文档作用** | 替代部分文档说明 |

### 1.3 类型提示的发展

```python
# Python 3.5 (2015) - 基础类型注解
def foo(x: int) -> int:
    pass

# Python 3.6 (2016) - 变量注解
x: int = 1

# Python 3.8 (2019) - 类型别名和Literal
from typing import Literal

# Python 3.9 (2020) - 内置泛型
list[int]  # 不再需要 from typing import List

# Python 3.10 (2021) - union 语法
def foo(x: int | str) -> int | str:
    pass

# Python 3.11 (2022) - Self 注解、改进的错误信息
# Python 3.12 (2023) - 类型参数语法改进
```

## 2. 基础类型注解

### 2.1 基本类型

```python
# 基础类型
def foo(
    name: str,      # 字符串
    age: int,       # 整数
    height: float, # 浮点数
    is_active: bool # 布尔值
) -> str:          # 返回类型
    return f"{name}, {age}岁"

# None 类型
def log(message: str) -> None:
    print(message)
    # 不需要 return 语句，或 return None

# 无返回（用于生成器等）
def generator() -> str:
    yield "first"
    yield "second"
```

### 2.2 类型别名

```python
# 简单别名
UserId = int
PostId = int

def get_user(user_id: UserId) -> dict:
    return {"id": user_id}

# 复杂别名
from typing import List, Dict

# 用户字典类型
UserDict = Dict[str, str]
# 文章列表类型
ArticleList = List[Dict[str, any]]

def get_users() -> UserDict:
    return {"name": "John"}

def get_articles() -> ArticleList:
    return [{"title": "Hello"}]
```

### 2.3 Any 类型

```python
from typing import Any

# 任意类型 - 关闭类型检查
def process(data: Any) -> Any:
    return data

# 谨慎使用 Any - 它会破坏类型安全
def bad_example(x: Any) -> int:
    return x + 1  # 可能出错，但类型检查器不会报错
```

## 3. 容器类型

### 3.1 列表 List

```python
from typing import List

# 整数列表
def get_numbers() -> List[int]:
    return [1, 2, 3]

# 字符串列表
def process_names(names: List[str]) -> List[str]:
    return [name.upper() for name in names]

# 混合类型列表
from typing import Union
ListOfMixed = List[Union[str, int]]

def process_mixed(items: ListOfMixed) -> None:
    for item in items:
        print(item)
```

### 3.2 字典 Dict

```python
from typing import Dict

# 字符串到整数的映射
def count_words(text: str) -> Dict[str, int]:
    result: Dict[str, int] = {}
    for word in text.split():
        result[word] = result.get(word, 0) + 1
    return result

# 嵌套字典
def get_user_info() -> Dict[str, Dict[str, str]]:
    return {
        "user": {"name": "John", "email": "john@example.com"},
        "profile": {"bio": "Hello"}
    }
```

### 3.3 元组 Tuple

```python
from typing import Tuple

# 固定长度和类型
def get_coords() -> Tuple[float, float]:
    return (10.5, 20.3)

# 混合类型元组
def get_user() -> Tuple[str, int, bool]:
    return ("John", 25, True)

# 可变长元组
from typing import Tuple
def get_scores() -> Tuple[int, ...]:
    return (90, 85, 78, 92)
```

### 3.4 集合 Set

```python
from typing import Set

def unique_items(items: List[str]) -> Set[str]:
    return set(items)

def find_duplicates(items: List[str]) -> Set[str]:
    seen: Set[str] = set()
    duplicates: Set[str] = set()
    for item in items:
        if item in seen:
            duplicates.add(item)
        seen.add(item)
    return duplicates
```

## 4. 可选类型和联合类型

### 4.1 Optional

```python
from typing import Optional

# 可能为 None
def find_user(user_id: int) -> Optional[dict]:
    """查找用户，不存在返回 None"""
    users = {1: {"name": "John"}}
    return users.get(user_id)

# 等价于
def find_user(user_id: int) -> dict | None:
    return {1: {"name": "John"}}.get(user_id)

# 使用示例
user = find_user(1)
if user is not None:
    print(user["name"])  # 类型检查器知道 user 不是 None
```

### 4.2 Union

```python
from typing import Union

# 多种类型之一
def parse(value: str) -> Union[int, float, bool]:
    if value.isdigit():
        return int(value)
    try:
        return float(value)
    except ValueError:
        return value.lower() == "true"

# Python 3.10+ 简化语法
def parse_new(value: str) -> int | float | bool:
    if value.isdigit():
        return int(value)
    try:
        return float(value)
    except ValueError:
        return value.lower() == "true"
```

### 4.3 Literal

```python
from typing import Literal

# 限制为特定值
def get_status(code: Literal[200, 201, 400, 404, 500]) -> str:
    messages = {
        200: "OK",
        201: "Created",
        400: "Bad Request",
        404: "Not Found",
        500: "Internal Error"
    }
    return messages[code]

# 字符串字面量
def set_mode(mode: Literal["read", "write", "append"]) -> None:
    print(f"Mode set to {mode}")

# 使用
get_status(200)    # ✓
get_status(999)    # ✗ 类型错误

set_mode("read")   # ✓
set_mode("delete") # ✗ 类型错误
```

## 5. 函数类型

### 5.1 Callable

```python
from typing import Callable

# 函数作为参数
def apply(func: Callable[[int, int], int], x: int, y: int) -> int:
    """应用二元函数"""
    return func(x, y)

def add(a: int, b: int) -> int:
    return a + b

result = apply(add, 5, 3)  # 8

# 无返回的回调
def on_click(callback: Callable[[], None]) -> None:
    callback()

# 可变参数的回调
def process_all(items: list, callback: Callable[[str], None]) -> None:
    for item in items:
        callback(str(item))
```

### 5.2 匿名函数

```python
from typing import Callable

# 匿名函数的类型
add: Callable[[int, int], int] = lambda x, y: x + y

# 在函数中使用
def math_operation(
    a: int,
    b: int,
    operation: Callable[[int, int], int]
) -> int:
    return operation(a, b)

result = math_operation(10, 5, lambda x, y: x * y)  # 50
```

## 6. 泛型（Generics）

### 6.1 TypeVar

```python
from typing import TypeVar, List

# 定义类型变量
T = TypeVar('T')
U = TypeVar('U')

# 泛型函数
def first(items: List[T]) -> T:
    """返回列表第一个元素"""
    return items[0]

# 使用
numbers: List[int] = [1, 2, 3]
first_number = first(numbers)  # 类型: int

strings: List[str] = ["a", "b", "c"]
first_string = first(strings)  # 类型: str

# 泛型类
class Stack(Generic[T]):
    def __init__(self) -> None:
        self._items: List[T] = []
    
    def push(self, item: T) -> None:
        self._items.append(item)
    
    def pop(self) -> T:
        return self._items.pop()
    
    def peek(self) -> T:
        return self._items[-1]

# 使用
int_stack: Stack[int] = Stack()
int_stack.push(1)
int_stack.push(2)
top: int = int_stack.pop()  # 类型正确
```

### 6.2 Generic

```python
from typing import TypeVar, Generic, List

T = TypeVar('T')

class Container(Generic[T]):
    def __init__(self, items: List[T]) -> None:
        self._items = items
    
    def get_all(self) -> List[T]:
        return self._items
    
    def get_first(self) -> T:
        return self._items[0]

# 使用
string_container = Container(["a", "b", "c"])
items: List[str] = string_container.get_all()
first: str = string_container.get_first()
```

### 6.3 泛型约束

```python
from typing import TypeVar

# 约束类型变量
Number = TypeVar('Number', int, float)

def add(a: Number, b: Number) -> Number:
    return a + b

# 使用
result1 = add(1, 2)      # int
result2 = add(1.5, 2.5)  # float
# add("a", "b")  # 错误 - 不支持 str
```

## 7. 协议（Protocol）

### 7.1 什么是协议？

协议定义了类必须实现的接口方法，用于结构化子类型（静态 duck typing）。

```python
from typing import Protocol

class Drawable(Protocol):
    def draw(self) -> None: ...

class Circle:
    def draw(self) -> None:
        print("Drawing circle")

class Square:
    def draw(self) -> None:
        print("Drawing square")

# 任何有 draw 方法的对象都可以使用
def render(shape: Drawable) -> None:
    shape.draw()

render(Circle())  # ✓
render(Square())  # ✓
```

### 7.2 协议 vs 抽象基类

```python
from abc import ABC, abstractmethod
from typing import Protocol

# 使用 ABC
class Animal(ABC):
    @abstractmethod
    def speak(self) -> str:
        pass

class Dog(Animal):
    def speak(self) -> str:
        return "Woof"

# 使用 Protocol
class Speakable(Protocol):
    def speak(self) -> str: ...

class Cat:
    def speak(self) -> str:
        return "Meow"

# Protocol 不需要继承，更灵活
def make_speak(entity: Speakable) -> str:
    return entity.speak()

make_speak(Dog())  # ✓
make_speak(Cat())  # ✓
```

### 7.3 协议示例

```python
from typing import Protocol, List

class Sized(Protocol):
    def __len__(self) -> int: ...

class Collection:
    def __init__(self, items: List[str]) -> None:
        self.items = items
    
    def __len__(self) -> int:
        return len(self.items)

def print_size(container: Sized) -> None:
    print(f"Size: {len(container)}")

print_size(Collection(["a", "b"]))  # ✓
print_size([1, 2, 3])  # ✓ list 实现了 __len__
```

## 8. 类型守卫

### 8.1 TypeGuard

```python
from typing import TypeGuard, Union

def is_string_list(val: list[object]) -> TypeGuard[list[str]]:
    """类型守卫 - 缩小类型范围"""
    return all(isinstance(x, str) for x in val)

def process(items: list[object]) -> None:
    if is_string_list(items):
        # 类型检查器知道 items 是 list[str]
        print(" ".join(items))  # ✓
    else:
        print("Not all strings")

process(["a", "b", "c"])  # ✓
process([1, 2, "c"])       # ✓
```

### 8.2 isinstance 守卫

```python
from typing import Union

def process(value: int | str) -> None:
    if isinstance(value, int):
        # 类型检查器知道 value 是 int
        result = value * 2
    else:
        # 类型检查器知道 value 是 str
        result = value.upper()
    return result
```

## 9. 特殊类型

### 9.1 NoReturn

```python
from typing import NoReturn

def error(message: str) -> NoReturn:
    """永远抛出异常的函数"""
    raise RuntimeError(message)

def process() -> None:
    result = input("Continue? (y/n): ")
    if result.lower() != "y":
        error("User cancelled")
    # 类型检查器知道 error 不会返回
```

### 9.2 overload

```python
from typing import overload, Union

@overload
def process(value: int) -> int: ...
@overload
def process(value: str) -> str: ...
@overload
def process(value: int | str) -> int | str: ...

def process(value: int | str) -> int | str:
    if isinstance(value, int):
        return value * 2
    return value * 2

# 类型检查器会根据参数类型推断返回类型
x: int = process(5)      # ✓
y: str = process("hi")   # ✓
```

### 9.3 TypedDict

```python
from typing import TypedDict

class UserDict(TypedDict):

# 使用
user: UserDict = {
```

## 11. 实战示例

### 11.1 完整类型提示示例

```python
from typing import TypeVar, Generic, Protocol, Optional, List
from dataclasses import dataclass

# 类型变量
T = TypeVar('T')

# 协议
class Comparable(Protocol):
    def compare_to(self, other: 'Comparable') -> int: ...

# 数据类
@dataclass
class User:
    id: int
    name: str
    email: str
    
    def __str__(self) -> str:
        return f"{self.name} <{self.email}>"

# 泛型类
class Repository(Generic[T]):
    def __init__(self) -> None:
        self._data: List[T] = []
    
    def add(self, item: T) -> None:
        self._data.append(item)
    
    def get(self, index: int) -> Optional[T]:
        return self._data[index] if 0 <= index < len(self._data) else None
    
    def all(self) -> List[T]:
        return self._data.copy()

# 使用
user_repo: Repository[User] = Repository()
user_repo.add(User(1, "John", "john@example.com"))
user_repo.add(User(2, "Jane", "jane@example.com"))

user: Optional[User] = user_repo.get(0)
if user:
    print(user)  # 类型检查器知道 user 不是 None
```

### 11.2 类型提示最佳实践

```python
# ✓ 推荐：使用具体类型
def process_items(items: list[str]) -> list[str]:
    return sorted(items)

# ✗ 避免：过度使用 Any
def bad_process(items: list[Any]) -> Any:
    return items

# ✓ 推荐：使用 Optional 而非 None
def find_user(user_id: int) -> dict | None:
    return {"id": user_id} if user_id > 0 else None

# ✗ 避免：混合风格
def mixed(value: Optional[str] = None) -> Optional[str]:  # 混用风格
    return value

# ✓ 推荐：Python 3.10+ 使用 | 语法
def new_style(value: int | str) -> str:
    return str(value)

# ✓ 推荐：使用 TypedDict
class Config(TypedDict):
    host: str
    port: int
    debug: bool

def load_config() -> Config:
    return {"host": "localhost", "port": 8080, "debug": True}
```
