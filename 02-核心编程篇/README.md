# 核心编程篇

> **本章代码基于 Python 3.11+ 编写**
>
> 学习 Python 核心编程技能，掌握函数、面向对象、异常处理等进阶知识，目标是培养 Python 高级技术专家。

---

## 内容架构：三层深度

每个技术概念按三层深度编写，借鉴 Effective Python 的编写理念：

```
┌─────────────────────────────────────────────────────────────────┐
│                        内容深度金字塔                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                     ┌─────────────┐                             │
│                     │  L3 专家层  │  底层原理、性能、设计动机    │
│                     │   (深入)    │  培养独立思考与判断能力      │
│                     └─────────────┘                             │
│                   ┌─────────────────┐                           │
│                   │    L2 实践层    │  最佳实践、反模式、陷阱    │
│                   │      (用好)     │  培养工程化思维            │
│                   └─────────────────┘                           │
│                 ┌─────────────────────┐                         │
│                 │      L1 理解层      │  语法结构、代码示例      │
│                 │        (会用)       │  培养基础能力            │
│                 └─────────────────────┘                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

| 层级 | 目标读者 | 核心内容 | 必要性 |
|------|---------|---------|--------|
| **L1 理解层** | 初学者 | 语法结构图 + 代码示例 | ✅ 必须有 |
| **L2 实践层** | 进阶者 | 最佳实践 + 反模式对比 | ✅ 核心概念必须有 |
| **L3 专家层** | 高级开发者 | 底层原理 + 性能考量 | ❓ 关键概念必须有 |
