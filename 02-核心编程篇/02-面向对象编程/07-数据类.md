# 数据类 dataclass

> **难度**：⭐⭐ 进阶
> **预计时间**：30分钟
> **前置知识**：面向对象基础（类与对象）
> **引入版本**：Python 3.7+（Python 3.10+ 增强）

本章讲解 `@dataclass` 装饰器，用于快速创建数据容器类。

---

## 为什么需要 dataclass？

### 问题场景

你需要定义一个简单的数据类来存储用户信息。

**传统写法（手动写很多重复代码）：**

```python
class User:
    def __init__(self, name: str, age: int, email: str) -> None:
        self.name = name
        self.age = age
        self.email = email
    
    def __repr__(self) -> str:
        return f"User(name={self.name}, age={self.age}, email={self.email})"
    
    def __eq__(self, other: object) -> bool:
        if not isinstance(other, User):
            return NotImplemented
        return (self.name, self.age, self.email) == (other.name, other.age, other.email)

# 使用
user1 = User("张三", 25, "test@example.com")
user2 = User("张三", 25, "test@example.com")
print(user1)        # User(name=张三, age=25, email=test@example.com)
print(user1 == user2)  # True
```

**问题：**
- 每个类都要写 `__init__`、`__repr__`、`__eq__`
- 代码重复，容易出错
- 字段越多，代码越长

**dataclass 写法（自动生成）：**

```python
from dataclasses import dataclass

@dataclass
class User:
    name: str
    age: int
    email: str

# 一行定义，自动获得 __init__、__repr__、__eq__
user1 = User("张三", 25, "test@example.com")
user2 = User("张三", 25, "test@example.com")
print(user1)        # User(name='张三', age=25, email='test@example.com')
print(user1 == user2)  # True
```

这就是 `@dataclass` 的价值：**自动生成常用方法，减少重复代码**。

## L1 理解层：基础用法

### 最简示例

```python
from dataclasses import dataclass

@dataclass
class Point:
    x: float
    y: float

# 使用
p = Point(3.0, 4.0)
print(p)        # Point(x=3.0, y=4.0)
print(p.x)      # 3.0
```

### 关键代码解释

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `@dataclass` | 装饰器 | 自动生成 `__init__` 等方法 |
| `x: float` | 类型提示字段 | **必须写类型**，否则不识别为字段 |
| `Point(3.0, 4.0)` | 创建实例 | 自动生成的 `__init__` |
| `print(p)` | 打印实例 | 自动生成的 `__repr__` |

---

### 默认值

```python
from dataclasses import dataclass

@dataclass
class Config:
    host: str = "localhost"
    port: int = 8080
    debug: bool = False

# 使用
config1 = Config()                    # 使用默认值
config2 = Config("example.com", 443)  # 自定义值
print(config1)  # Config(host='localhost', port=8080, debug=False)
print(config2)  # Config(host='example.com', port=443, debug=False)
```

**关键规则：有默认值的字段必须在无默认值字段之后。**

```python
# ❌ 错误：有默认值的字段在前
@dataclass
class Wrong:
    name: str = "default"  # 有默认值
    age: int               # 无默认值 ← 报错！

# ✅ 正确：无默认值的字段在前
@dataclass
class Right:
    age: int               # 无默认值
    name: str = "default"  # 有默认值
```

---

### 可变默认值

**陷阱：不能用 `[]` 或 `{}` 作为默认值（与普通类相同问题）。**

```python
# ❌ 错误：可变默认值陷阱
@dataclass
class Wrong:
    items: list[str] = []  # 所有实例共享同一个列表！

# ✅ 正确：用 field(default_factory)
from dataclasses import dataclass, field

@dataclass
class Right:
    items: list[str] = field(default_factory=list)

# 每个实例有独立的列表
r1 = Right()
r2 = Right()
r1.items.append(1)
print(r1.items)  # [1]
print(r2.items)  # []（独立列表）
```

---

### 添加方法

dataclass 可以添加自定义方法：

```python
from dataclasses import dataclass

@dataclass
class Point:
    x: float
    y: float
    
    def distance(self) -> float:
        """计算到原点的距离"""
        return (self.x ** 2 + self.y ** 2) ** 0.5
    
    def move(self, dx: float, dy: float) -> None:
        """移动点"""
        self.x += dx
        self.y += dy

# 使用
p = Point(3.0, 4.0)
print(p.distance())  # 5.0
p.move(1.0, 1.0)
print(p)  # Point(x=4.0, y=5.0)
```

## L2 实践层：进阶用法

### frozen=True 不可变

```python
from dataclasses import dataclass

@dataclass(frozen=True)
class Point:
    x: float
    y: float

p = Point(3.0, 4.0)
print(p)  # Point(x=3.0, y=4.0)

# ❌ 无法修改
# p.x = 5.0  # FrozenInstanceError: cannot assign to field 'x'

# ✅ 可以创建新实例
p2 = Point(5.0, 6.0)
```

**适用场景：**
- 常量/配置类
- 不应该被修改的数据
- 作为字典键（需要不可变）

---

### slots=True 内存优化

Python 3.10+ 支持 `slots=True`，减少内存占用：

```python
from dataclasses import dataclass

@dataclass(slots=True)
class Point:
    x: float
    y: float

# 内存对比
import sys

@dataclass
class PointNormal:
    x: float
    y: float

@dataclass(slots=True)
class PointSlots:
    x: float
    y: float

pn = PointNormal(1.0, 2.0)
ps = PointSlots(1.0, 2.0)

print(sys.getsizeof(pn))  # ~48 bytes
print(sys.getsizeof(ps))  # ~32 bytes（节省 ~33%）
```

**slots 的限制：**
- 不能动态添加新属性
- 类属性访问略慢

---

### kw_only=True 强制关键字参数

Python 3.10+ 支持：

```python
from dataclasses import dataclass

@dataclass(kw_only=True)
class Config:
    host: str
    port: int = 8080

# ✅ 必须使用关键字参数
config = Config(host="localhost")

# ❌ 不能用位置参数
# config = Config("localhost")  # TypeError
```

**适用场景：**
- 参数多，避免位置混淆
- API 更清晰

### 继承

```python
from dataclasses import dataclass

@dataclass
class Animal:
    name: str
    age: int

@dataclass
class Dog(Animal):
    breed: str

# 使用
dog = Dog("Buddy", 3, "Golden Retriever")
print(dog)  # Dog(name='Buddy', age=3, breed='Golden Retriever')
```

**继承规则：**
- 父类字段在前，子类字段在后
- 子类可以添加默认值

### field() 高级配置

```python
from dataclasses import dataclass, field

@dataclass
class User:
    # 不参与 __init__（内部计算）
    id: int = field(init=False)
    
    # 不参与 __repr__（敏感数据）
    password: str = field(repr=False)
    
    # 默认工厂
    tags: list[str] = field(default_factory=list)
    
    # 默认值
    name: str = "Unknown"
    
    def __post_init__(self) -> None:
        """初始化后自动调用"""
        self.id = hash(self.name)

user = User("Alice", "secret123")
print(user)  # User(id=..., name='Alice', tags=[])  ← password 不显示
```

**field() 参数：**

| 参数 | 说明 | 示例 |
|------|------|------|
| `default` | 默认值 | `field(default=0)` |
| `default_factory` | 默认工厂函数 | `field(default_factory=list)` |
| `init` | 是否参与 `__init__` | `field(init=False)` |
| `repr` | 是否参与 `__repr__` | `field(repr=False)` |
| `compare` | 是否参与比较 | `field(compare=False)` |

---

### 推荐做法

| 场景 | 推荐方式 | 原因 |
|------|---------|------|
| 简单数据容器 | `@dataclass` | 代码简洁 |
| 需要不可变 | `@dataclass(frozen=True)` | 防止意外修改 |
| 大量实例 | `@dataclass(slots=True)` | 内存优化 |
| 有默认值列表/字典 | `field(default_factory=list)` | 避免共享陷阱 |
| 需要方法 | 正常添加方法 | dataclass 支持方法 |
| API 数据 | TypedDict 或 Pydantic | 不是 dataclass 场景 |

### 反模式

```python
# ❌ 反模式1：不写类型提示
@dataclass
class Wrong:
    name = "default"  # 不是字段！只是类属性

w = Wrong()
print(w.name)  # "default"（类属性）
# print(w.name) ← 没有 __init__ 参数

# ✅ 正确：写类型提示
@dataclass
class Right:
    name: str = "default"  # 是字段
```

```python
# ❌ 反模式2：用 dataclass 存大量逻辑
@dataclass
class UserService:
    db: Database
    
    def create_user(self, ...):
        # 100 行业务逻辑...
        pass

# ✅ 正确：用普通类存逻辑
class UserService:
    def __init__(self, db: Database):
        self.db = db
    
    def create_user(self, ...):
        # 业务逻辑...
        pass
```

```python
# ❌ 反模式3：混用 dataclass 和复杂继承
@dataclass
class Base:
    x: int

@dataclass  
class Derived(Base):
    y: int
    
    def __init__(self, x, y, z):  # 手动覆盖 __init__
        # 与 dataclass 生成的冲突
        pass

# ✅ 正确：让 dataclass 自动处理，或用普通类
```

## L3 专家层：底层原理

### dataclass 生成了什么

```
┌─────────────────────────────────────────────────────────────┐
│          dataclass 自动生成的方法                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  @dataclass                                                 │
│  class User:                                                │
│      name: str                                              │
│      age: int                                               │
│                                                             │
│  自动生成以下方法：                                            │
│  ─────────────────────────────────────────────              │
│                                                             │
│  1. __init__(self, name: str, age: int)                     │
│     • 自动赋值所有字段                                        │
│     • 处理默认值                                              │
│                                                             │
│  2. __repr__(self)                                          │
│     • 返回 User(name=..., age=...)                          │
│     • 用于打印                                                │
│                                                             │
│  3. __eq__(self, other)                                     │
│     • 比较所有字段                                            │
│     • 用于 == 和 !=                                          │
│                                                             │
│  可选生成（参数控制）：                                         │
│  ─────────────────────────────────────────────              │
│  4. __hash__(self)          ← frozen=True 时生成            │
│  5. __lt__, __le__, __gt__, __ge__  ← order=True 时生成     │
│                                                             │
│  Python 3.10+ 新增：                                          │
│  ─────────────────────────────────────────────              │
│  6. __match_args__          ← 用于 match-case               │
│  7. __slots__               ← slots=True 时生成             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 查看生成的方法

```python
from dataclasses import dataclass

@dataclass
class User:
    name: str
    age: int

# 查看 __init__ 签名
import inspect
print(inspect.signature(User.__init__))
# (self, name: str, age: int) -> None

# 查看 __repr__
user = User("Alice", 25)
print(user.__repr__())
# User(name='Alice', age=25)

# 查看 __eq__
print(User.__eq__)
# <function dataclasses.__create_fn__.<locals>.__eq__ at 0x...>
```

### __post_init__ 钩子

```
┌─────────────────────────────────────────────────────────────┐
│          __post_init__ 执行时机                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  执行顺序：                                                   │
│  ─────────────────────────────────────────────              │
│                                                             │
│  1. User.__init__(name, age)                                 │
│     • self.name = name                                       │
│     • self.age = age                                         │
│                                                             │
│  2. User.__post_init__(self)                                 │
│     • 自动调用                                                │
│     • 用于后续处理                                            │
│                                                             │
│  用途：                                                       │
│  ─────────────────────────────────────────────              │
│  • 计算派生字段                                              │
│  • 验证字段值                                                │
│  • 初始化非字段属性                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**示例：验证和计算**

```python
from dataclasses import dataclass, field

@dataclass
class Rectangle:
    width: float
    height: float
    area: float = field(init=False)  # 不参与 __init__
    
    def __post_init__(self) -> None:
        # 验证
        if self.width <= 0 or self.height <= 0:
            raise ValueError("宽高必须为正数")
        # 计算派生字段
        self.area = self.width * self.height

rect = Rectangle(3.0, 4.0)
print(rect)  # Rectangle(width=3.0, height=4.0, area=12.0)

# rect = Rectangle(-1, 4)  # ValueError
```

### 性能考量

| 方式 | 内存 | 创建速度 | 访问速度 |
|------|------|---------|---------|
| 普通 class | 正常 | 正常 | 正常 |
| `@dataclass` | 正常 | 正常 | 正常 |
| `@dataclass(slots=True)` | -30% | +10% | +15% |
| namedtuple | 最小 | 最快 | 慢（无属性访问） |

**何时用 slots=True：**
- 创建大量实例（百万级）
- 内存敏感场景
- 字段固定的类

---

### 设计动机

| 设计选择 | 原因 | 替代方案对比 |
|----------|------|-------------|
| 类型提示必须 | 明确字段列表 | namedtuple 无类型提示 |
| 自动生成方法 | 减少重复代码 | 普通 class 手动写 |
| 支持默认值 | 配置类常见 | namedtuple 无默认值 |
| frozen 选项 | 不可变需求 | namedtuple 强制不可变 |
| slots 选项 | 性能优化 | 手动定义 __slots__ 复杂 |
