# 属性与方法

> **Python 版本要求：** 3.11+

本章讲解 Python 类的属性和方法，包括实例属性、类属性、实例方法、类方法和静态方法。

---

## 为什么需要不同的属性和方法？

### 问题场景

假设你要开发一个学生管理系统，需要：
1. 统计总学生数量（所有学生共享）
2. 每个学生有自己的姓名和年龄
3. 提供工具方法判断年龄是否成年

**问题代码：**

```python
# 使用全局变量管理共享数据
student_count = 0

class Student:
    def __init__(self, name: str, age: int) -> None:
        self.name = name
        self.age = age
        global student_count
        student_count += 1

# 问题：全局变量容易被意外修改
student_count = 100  # 误操作！

# 每次判断成年都要写判断逻辑
def is_adult(age: int) -> bool:
    return age >= 18

# 判断成年和 Student 类没有关联
print(is_adult(18))  # 与 Student 类无关
```

**问题：**
- 如何让所有学生共享同一个计数器？
- 如何区分每个学生独有的数据？
- 如何创建与实例无关的工具函数？

**解决方案：**

```python
class Student:
    # 类属性：所有实例共享
    count: int = 0
    school: str = "第一中学"
    
    def __init__(self, name: str, age: int) -> None:
        # 实例属性：每个实例独立
        self.name: str = name
        self.age: int = age
        Student.count += 1  # 通过类名修改类属性
    
    @classmethod
    def get_count(cls) -> int:
        """类方法：操作类属性"""
        return cls.count
    
    @staticmethod
    def is_adult(age: int) -> bool:
        """静态方法：工具函数"""
        return age >= 18

# 类属性被保护在类内部
s1 = Student("张三", 18)
s2 = Student("李四", 17)
print(Student.get_count())  # 2

# 静态方法与类关联
print(Student.is_adult(18))  # True
```

**优势：**
- 类属性让所有实例共享数据
- 实例属性让每个实例有独立数据
- 类方法操作类级数据
- 静态方法提供工具函数

## 第一部分：实例属性 vs 类属性

### 1.1 概念动机

在类中定义的属性有两种：实例属性和类属性。理解它们的区别是掌握面向对象的关键。

```
┌─────────────────────────────────────────────────────────────┐
│              实例属性 vs 类属性                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   实例属性：                                                 │
│   ✓ 在 __init__ 中通过 self 定义                            │
│   ✓ 每个实例独立拥有                                        │
│   ✓ 修改只影响当前实例                                      │
│                                                             │
│   类属性：                                                   │
│   ✓ 在类中直接定义                                          │
│   ✓ 所有实例共享                                            │
│   ✓ 修改影响所有实例                                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 最简示例

```python
class Student:
    # 类属性（所有实例共享）
    school: str = "第一中学"
    count: int = 0

    def __init__(self, name: str, age: int) -> None:
        # 实例属性（每个实例独立）
        self.name: str = name
        self.age: int = age
        Student.count += 1  # 修改类属性


# 创建实例
s1 = Student("Alice", 15)
s2 = Student("Bob", 16)

# 访问类属性（推荐通过类名）
print(Student.school)  # 第一中学
print(Student.count)   # 2

# 访问实例属性
print(s1.name)  # Alice
print(s2.name)  # Bob

# 修改类属性（影响所有实例）
Student.school = "实验中学"
print(s1.school)  # 实验中学
print(s2.school)  # 实验中学

# 修改实例属性（只影响当前实例）
s1.name = "Alice New"
print(s1.name)  # Alice New
print(s2.name)  # Bob（不变）
```

### 1.3 详细讲解

**属性对比表：**

| | 实例属性 | 类属性 |
|---|---|---|
| **定义位置** | `__init__`中用`self`定义 | 类中直接定义 |
| **归属** | 每个实例独立拥有 | 所有实例共享 |
| **访问方式** | `self.属性名` | `类名.属性名` |
| **修改影响** | 只影响当前实例 | 影响所有实例 |

**注意事项：**

```python
class Student:
    count: int = 0
    
    def __init__(self, name: str) -> None:
        self.name: str = name
        Student.count += 1

s1 = Student("Alice")
s2 = Student("Bob")

# ❌ 通过实例修改类属性（容易出错）
s1.count = 100  # 这实际上创建了一个实例属性！
print(s1.count)      # 100（实例属性）
print(Student.count) # 2（类属性未改变）

# ✅ 正确方式：通过类名修改
Student.count = 100
print(s1.count)      # 100（如果实例没有同名属性）
print(s2.count)      # 100（所有实例都看到新值）
```

### 1.4 渐进复杂化

类属性的常见应用场景：

```python
class Configuration:
    """系统配置类"""
    
    # 配置项作为类属性
    version: str = "1.0.0"
    debug_mode: bool = False
    max_connections: int = 100
    
    # 类属性可以是复杂类型
    default_settings: dict[str, str | int] = {
        "theme": "dark",
        "language": "zh-CN",
        "timeout": 30,
    }
    
    @classmethod
    def update_setting(cls, key: str, value: str | int) -> None:
        """更新默认配置"""
        cls.default_settings[key] = value
    
    @classmethod
    def get_version(cls) -> str:
        """获取版本"""
        return cls.version

# 所有模块共享同一配置
print(Configuration.version)
Configuration.update_setting("theme", "light")
print(Configuration.default_settings)
```

### 1.5 实际应用

学生管理系统示例：

```python
class Student:
    """学生类"""
    
    # 类属性
    school: str = "第一中学"
    total_count: int = 0
    grade_count: dict[str, int] = {}
    
    def __init__(self, name: str, age: int, grade: str) -> None:
        # 实例属性
        self.name: str = name
        self.age: int = age
        self.grade: str = grade
        
        # 更新类属性
        Student.total_count += 1
        Student.grade_count[grade] = Student.grade_count.get(grade, 0) + 1
    
    def get_info(self) -> dict[str, str | int]:
        """获取学生信息"""
        return {
            "name": self.name,
            "age": self.age,
            "grade": self.grade,
            "school": Student.school,
        }
    
    @classmethod
    def get_statistics(cls) -> dict[str, str | int | dict]:
        """获取统计信息"""
        return {
            "school": cls.school,
            "total": cls.total_count,
            "grade_distribution": cls.grade_count,
        }

# 创建学生
students = [
    Student("张三", 18, "高三"),
    Student("李四", 17, "高二"),
    Student("王五", 16, "高一"),
    Student("赵六", 17, "高二"),
]

# 查看统计
print(Student.get_statistics())
# {'school': '第一中学', 'total': 4, 'grade_distribution': {'高三': 1, '高二': 2, '高一': 1}}
```

## 第二部分：三种方法类型

### 2.1 概念动机

Python 类有三种方法：实例方法、类方法、静态方法。它们有不同的访问能力和使用场景。

```
┌─────────────────────────────────────────────────────────────┐
│              三种方法类型                                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   实例方法：                                                 │
│   ✓ 第一个参数是 self                                       │
│   ✓ 可以访问实例属性和类属性                                │
│   ✓ 用于操作实例数据                                        │
│                                                             │
│   类方法：                                                   │
│   ✓ 使用 @classmethod 装饰器                                │
│   ✓ 第一个参数是 cls                                        │
│   ✓ 只能访问类属性                                          │
│   ✓ 用于工厂方法、操作类数据                                │
│                                                             │
│   静态方法：                                                 │
│   ✓ 使用 @staticmethod 装饰器                               │
│   ✓ 无特殊参数                                              │
│   ✓ 不能访问实例或类属性                                    │
│   ✓ 用于工具函数                                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 最简示例

**实例方法：**

```python
class Student:
    def __init__(self, name: str) -> None:
        self.name: str = name

    # 实例方法：操作实例数据
    def introduce(self) -> None:
        """实例方法：自我介绍"""
        print(f"我是{self.name}")
```

**类方法：**

```python
class Student:
    count: int = 0  # 类属性

    def __init__(self, name: str) -> None:
        self.name: str = name
        Student.count += 1

    # 类方法：操作类数据（使用 @classmethod）
    @classmethod
    def get_count(cls) -> int:
        """类方法：获取学生总数"""
        return cls.count

    @classmethod
    def from_string(cls, data_str: str) -> "Student":
        """类方法：从字符串创建对象（工厂方法）"""
        name = data_str.split(",")[0]
        return cls(name)  # 相当于 Student(name)
```

**静态方法：**

```python
class Student:
    # 静态方法：工具函数（使用 @staticmethod）
    @staticmethod
    def is_adult(age: int) -> bool:
        """静态方法：判断是否成年"""
        return age >= 18
```

### 2.3 详细讲解

**完整示例：**

```python
class Student:
    count: int = 0

    def __init__(self, name: str) -> None:
        self.name: str = name
        Student.count += 1

    def introduce(self) -> None:
        """实例方法"""
        print(f"我是{self.name}")

    @classmethod
    def get_count(cls) -> int:
        """类方法"""
        return cls.count

    @classmethod
    def from_string(cls, data_str: str) -> "Student":
        """类方法：工厂方法"""
        name = data_str.split(",")[0]
        return cls(name)

    @staticmethod
    def is_adult(age: int) -> bool:
        """静态方法"""
        return age >= 18


# 使用
s1 = Student("Alice")
s2 = Student("Bob")

# 实例方法需要通过对象调用
s1.introduce()  # 我是 Alice

# 类方法可以通过类或实例调用
print(Student.get_count())  # 2
print(s1.get_count())       # 2

# 使用类方法创建对象
s3 = Student.from_string("Charlie,18")
print(s3.name)  # Charlie

# 静态方法不需要创建实例
print(Student.is_adult(20))  # True
```

**三种方法对比表：**

| 方法类型 | 装饰器 | 第一个参数 | 访问能力 | 使用场景 |
|----------|--------|-----------|----------|----------|
| **实例方法** | 无 | `self` | 实例属性 + 类属性 | 操作实例数据 |
| **类方法** | `@classmethod` | `cls` | 类属性 | 工厂方法、操作类数据 |
| **静态方法** | `@staticmethod` | 无 | 不能访问实例或类属性 | 工具函数 |

### 2.4 渐进复杂化

工厂方法的多种实现：

```python
class Student:
    """学生类 - 多种创建方式"""
    
    def __init__(self, name: str, age: int, grade: str) -> None:
        self.name: str = name
        self.age: int = age
        self.grade: str = grade
    
    def __str__(self) -> str:
        return f"{self.name}({self.grade}, {self.age}岁)"
    
    @classmethod
    def from_dict(cls, data: dict[str, str | int]) -> "Student":
        """从字典创建"""
        return cls(
            name=str(data.get("name", "")),
            age=int(data.get("age", 0)),
            grade=str(data.get("grade", "")),
        )
    
    @classmethod
    def from_string(cls, data_str: str) -> "Student":
        """从字符串创建"""
        parts = data_str.split(",")
        return cls(parts[0], int(parts[1]), parts[2])
    
    @classmethod
    def from_csv_line(cls, line: str) -> "Student":
        """从 CSV 行创建"""
        name, age, grade = line.strip().split(",")
        return cls(name, int(age), grade)
    
    @staticmethod
    def validate_age(age: int) -> bool:
        """验证年龄"""
        return 0 < age < 150
    
    @staticmethod
    def validate_grade(grade: str) -> bool:
        """验证年级"""
        valid_grades = ["高一", "高二", "高三"]
        return grade in valid_grades

# 多种创建方式
s1 = Student("张三", 18, "高三")
s2 = Student.from_dict({"name": "李四", "age": 17, "grade": "高二"})
s3 = Student.from_string("王五,16,高一")
s4 = Student.from_csv_line("赵六,17,高二\n")

for s in [s1, s2, s3, s4]:
    print(s)
```

### 2.5 实际应用

完整的学生管理系统：

```python
from typing import Optional

class Student:
    """学生类"""
    
    # 类属性
    school: str = "第一中学"
    total_count: int = 0
    
    def __init__(self, name: str, age: int, grade: str, score: int = 0) -> None:
        # 实例属性
        self.name: str = name
        self.age: int = age
        self.grade: str = grade
        self.score: int = score
        
        # 更新统计
        Student.total_count += 1
    
    def update_score(self, new_score: int) -> bool:
        """更新成绩（实例方法）"""
        if 0 <= new_score <= 100:
            self.score = new_score
            return True
        return False
    
    def get_info(self) -> dict[str, str | int]:
        """获取信息（实例方法）"""
        return {
            "name": self.name,
            "age": self.age,
            "grade": self.grade,
            "score": self.score,
            "school": Student.school,
        }
    
    @classmethod
    def change_school(cls, new_school: str) -> None:
        """更换学校（类方法）"""
        cls.school = new_school
    
    @classmethod
    def create_from_json(cls, json_data: dict) -> Optional["Student"]:
        """从 JSON 创建（类方法）"""
        try:
            return cls(
                json_data["name"],
                json_data["age"],
                json_data["grade"],
                json_data.get("score", 0),
            )
        except KeyError:
            return None
    
    @staticmethod
    def calculate_average(scores: list[int]) -> float:
        """计算平均分（静态方法）"""
        if not scores:
            return 0.0
        return sum(scores) / len(scores)
    
    @staticmethod
    def get_grade_level(score: int) -> str:
        """获取等级（静态方法）"""
        if score >= 90:
            return "A"
        elif score >= 80:
            return "B"
        elif score >= 70:
            return "C"
        elif score >= 60:
            return "D"
        else:
            return "F"

# 使用
students = [
    Student("张三", 18, "高三", 85),
    Student("李四", 17, "高二", 92),
    Student("王五", 16, "高一", 78),
]

# 实例方法操作实例数据
students[0].update_score(90)

# 类方法操作类数据
Student.change_school("实验中学")

# 静态方法提供工具功能
scores = [s.score for s in students]
avg = Student.calculate_average(scores)
print(f"平均分: {avg}")
print(f"等级分布: {[Student.get_grade_level(s) for s in scores]}")
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `school: str = "第一中学"` | 类属性，所有实例共享 | 学校信息属于全体学生共有，不应每个实例各存一份 |
| `Student.total_count += 1` | 在 `__init__` 里修改类属性 | 每创建一个实例就累加，通过类名访问确保修改的是类属性而非实例属性 |
| `@classmethod` + `cls` | 类方法第一个参数是类本身 | 操作类属性时用类方法，`cls` 使子类继承时也能正确访问子类属性 |
| `@staticmethod` | 静态方法无 `self`/`cls` 参数 | 工具函数不依赖实例或类状态，用静态方法表明此函数与类的关联只是命名空间 |
| `cls.school = new_school` | 通过 `cls` 修改类属性 | 用 `cls` 而非 `Student` 修改，子类调用时修改的是子类自己的属性 |

## L2 实践层：最佳实践

### 推荐做法

| 做法 | 原因 | 示例 |
|------|------|------|
| **类属性用不可变类型** | 避免实例间意外共享修改 | `MAX_SIZE: int = 100` |
| **通过类名访问类属性** | 避免创建实例属性遮蔽 | `Student.count` 而非 `s.count` |
| **类方法用于工厂方法** | 创建对象的不同方式，支持继承 | `@classmethod def from_dict(cls, data):` |
| **静态方法用于工具函数** | 不依赖实例状态，语义清晰 | `@staticmethod def validate_email(email):` |
| **避免可变类属性** | 防止实例间数据污染 | 用实例属性代替列表/字典类属性 |
| **类属性命名用大写** | 区分常量与普通属性 | `DEFAULT_TIMEOUT = 30` |

### 反模式：不要这样做

```python
# ❌ 通过实例修改类属性（创建实例属性遮蔽）
class Student:
    count: int = 0

    def __init__(self) -> None:
        Student.count += 1

s1 = Student()
s2 = Student()
print(Student.count)  # 2

s1.count = 100  # 创建了实例属性！
print(s1.count)       # 100（实例属性）
print(Student.count)  # 2（类属性未变）
print(s2.count)       # 2（查找顺序：实例→类）

# ❌ 使用可变对象作为类属性
class Config:
    settings: dict[str, str] = {"theme": "dark"}

c1 = Config()
c1.settings["theme"] = "light"  # 修改了共享的类属性！

c2 = Config()
print(c2.settings)  # {'theme': 'light'} - 被污染了！

# ❌ 不该用类方法却用了
class Calculator:
    @classmethod
    def add(cls, a: int, b: int) -> int:  # 不需要 cls
        return a + b

# ❌ 不该用静态方法却用了（依赖实例状态）
class Student:
    def __init__(self, name: str) -> None:
        self.name = name

    @staticmethod
    def get_name(self) -> str:  # 错误！静态方法没有 self
        return self.name  # AttributeError!
```

```python
# ✅ 正确做法

# 通过类名修改类属性
class Student:
    count: int = 0

    def __init__(self) -> None:
        Student.count += 1

    @classmethod
    def get_count(cls) -> int:
        return cls.count

s1 = Student()
Student.count = 100  # 通过类名修改
print(Student.get_count())  # 100

# 可变对象作为实例属性
class Config:
    def __init__(self) -> None:
        self.settings: dict[str, str] = {"theme": "dark"}

c1 = Config()
c1.settings["theme"] = "light"
c2 = Config()
print(c2.settings)  # {'theme': 'dark'} - 未被污染

# 工具函数用静态方法
class Calculator:
    @staticmethod
    def add(a: int, b: int) -> int:
        return a + b

print(Calculator.add(1, 2))  # 3

# 操作实例数据用实例方法
class Student:
    def __init__(self, name: str) -> None:
        self.name = name

    def get_name(self) -> str:
        return self.name
```

### 常见陷阱

| 陷阱 | 问题 | 解决方案 |
|------|------|----------|
| 实例修改类属性 | 创建同名实例属性遮蔽类属性 | 通过类名修改 |
| 可变类属性 | 所有实例共享，修改互相影响 | 改为实例属性 |
| 类方法不需要cls | 不该用类方法却用了 | 改为静态方法 |
| 静态方法使用self | 静态方法没有self参数 | 改为实例方法 |
| 类属性命名混乱 | 与实例属性难以区分 | 类属性用大写命名 |
| 继承时类属性未重写 | 子类可能意外使用父类类属性 | 子类显式重写类属性 |

### 适用场景

| 场景 | 是否推荐 | 原因 |
|------|---------|------|
| 共享常量（版本号、配置） | ✅ 推荐 | 类属性 |
| 计数器、统计信息 | ✅ 推荐 | 类属性 + 类方法 |
| 工厂方法（多种创建方式） | ✅ 推荐 | 类方法 |
| 纯工具函数（不依赖状态） | ✅ 推荐 | 静态方法 |
| 操作实例数据 | ✅ 推荐 | 实例方法 |
| 可变的共享数据 | ❌ 不推荐 | 用实例属性代替 |
| 继承中修改类属性 | ✅ 推荐 | 子类重写类属性 |

## L3 专家层：底层原理

### 属性查找机制

```
属性查找顺序：
┌─────────────────────────────────────────────────────────────┐
│  Python 属性查找流程（MRO + __dict__）                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   查找 obj.attr：                                           │
│                                                             │
│   歌骤 1：数据描述符检查                                     │
│   ┌─────────────────────────────────────┐                   │
│   │ 沿 MRO 查找 __get__ 描述符          │                   │
│   │ 如果找到数据描述符，调用 __get__    │                   │
│   └─────────────────────────────────────┘                   │
│                                                             │
│   歌骤 2：实例属性检查                                       │
│   ┌─────────────────────────────────────┐                   │
│   │ 查找 obj.__dict__['attr']           │                   │
│   │ 如果找到，直接返回                   │                   │
│   └─────────────────────────────────────┘                   │
│                                                             │
│   歌骤 3：非数据描述符检查                                   │
│   ┌─────────────────────────────────────┐                   │
│   │ 沿 MRO 查找类属性                    │                   │
│   │ 如果是非数据描述符，调用 __get__    │                   │
│   └─────────────────────────────────────┘                   │
│                                                             │
│   歌骤 4：类属性检查                                         │
│   ┌─────────────────────────────────────┐                   │
│   │ 沿 MRO 查找 type(obj).__dict__      │                   │
│   │ 如果找到，直接返回                   │                   │
│   └─────────────────────────────────────┘                   │
│                                                             │
│   歌骤 5：AttributeError                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

```python
# 演示属性查找过程
class Parent:
    class_attr: str = "父类属性"

    def __init__(self) -> None:
        self.instance_attr: str = "实例属性"

class Child(Parent):
    class_attr: str = "子类属性"  # 重写类属性

c = Child()

# 查看查找路径
print(Child.__mro__)
# (<class 'Child'>, <class 'Parent'>, <class 'object'>)

# 查找顺序演示
print(c.instance_attr)  # 实例属性（在实例 __dict__ 中）
print(c.class_attr)     # 子类属性（在 Child.__dict__ 中）

# 查看 __dict__
print(c.__dict__)       # {'instance_attr': '实例属性'}
print(Child.__dict__['class_attr'])  # 子类属性
print(Parent.__dict__['class_attr'])  # 父类属性

# 实例属性遮蔽类属性
c.class_attr = "实例的class_attr"  # 创建实例属性
print(c.__dict__)      # {'instance_attr': ..., 'class_attr': '实例的class_attr'}
print(c.class_attr)    # 实例的class_attr（优先实例属性）
print(Child.class_attr)  # 子类属性（类属性未变）
```

### 方法绑定原理

```
方法绑定机制：
┌─────────────────────────────────────────────────────────────┐
│  实例方法、类方法、静态方法的底层差异                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   实例方法：                                                 │
│   ┌─────────────────────────────────────┐                   │
│   │ def method(self):                   │                   │
│   │ • 在类 __dict__ 中存储普通函数      │                   │
│   │ • 访问时自动绑定实例（绑定方法）    │                   │
│   │ • obj.method → bound method(obj)    │                   │
│   │ • Class.method → function           │                   │
│   └─────────────────────────────────────┘                   │
│                                                             │
│   类方法：                                                   │
│   ┌─────────────────────────────────────┐                   │
│   │ @classmethod                        │                   │
│   │ def method(cls):                    │                   │
│   │ • 在类 __dict__ 中存储描述符        │                   │
│   │ • 自动绑定类（绑定方法）            │                   │
│   │ • obj.method → bound method(cls)    │                   │
│   │ • Class.method → bound method(cls)  │                   │
│   └─────────────────────────────────────┘                   │
│                                                             │
│   静态方法：                                                 │
│   ┌─────────────────────────────────────┐                   │
│   │ @staticmethod                       │                   │
│   │ def method():                       │                   │
│   │ • 在类 __dict__ 中存储描述符        │                   │
│   │ • 不绑定任何对象                    │                   │
│   │ • obj.method → function             │                   │
│   │ • Class.method → function           │                   │
│   └─────────────────────────────────────┘                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

```python
# 演示方法绑定差异
class Demo:
    def instance_method(self) -> str:
        return f"实例方法，self={self}"

    @classmethod
    def class_method(cls) -> str:
        return f"类方法，cls={cls}"

    @staticmethod
    def static_method() -> str:
        return "静态方法，无绑定"

d = Demo()

# 实例方法：通过实例访问时自动绑定
print(type(Demo.instance_method))  # function（未绑定）
print(type(d.instance_method))     # method（绑定到 d）
print(d.instance_method())         # 实例方法，self=<Demo object>
print(Demo.instance_method(d))     # 手动传入 self，效果相同

# 类方法：无论通过类还是实例访问，都绑定到类
print(type(Demo.class_method))     # method（绑定到 Demo）
print(type(d.class_method))        # method（绑定到 Demo）
print(d.class_method())            # 类方法，cls=<class 'Demo'>
print(Demo.class_method())         # 类方法，cls=<class 'Demo'>

# 静态方法：永远不绑定
print(type(Demo.static_method))    # function
print(type(d.static_method))       # function
print(d.static_method())           # 静态方法，无绑定
print(Demo.static_method())        # 静态方法，无绑定

# 查看内部存储
print(Demo.__dict__['instance_method'])  # <function>
print(Demo.__dict__['class_method'])     # <classmethod object>
print(Demo.__dict__['static_method'])    # <staticmethod object>
```

### 性能考量

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 实例属性访问 `obj.attr` | O(1) 平均 | __dict__ 字典查找 |
| 类属性访问 `Class.attr` | O(n) | 沿 MRO 查找（n = 继承深度） |
| 方法调用 `obj.method()` | O(n) | 查找方法 + 绑定 |
| 描述符访问 | O(n) | 额外的 __get__ 调用 |

```python
# 性能测试：实例属性 vs 类属性 vs 方法调用
import timeit

class Test:
    class_attr: int = 0
    def __init__(self) -> None:
        self.instance_attr: int = 0
    def instance_method(self) -> int:
        return self.instance_attr
    @classmethod
    def class_method(cls) -> int:
        return cls.class_attr
    @staticmethod
    def static_method() -> int:
        return 0

obj = Test()

print("实例属性:", timeit.timeit("obj.instance_attr", globals=globals(), number=1000000))
print("类属性:", timeit.timeit("Test.class_attr", globals=globals(), number=1000000))
print("实例方法:", timeit.timeit("obj.instance_method()", globals=globals(), number=1000000))
print("类方法:", timeit.timeit("Test.class_method()", globals=globals(), number=1000000))
print("静态方法:", timeit.timeit("Test.static_method()", globals=globals(), number=1000000))

# 结果：实例属性最快，类属性/方法调用需沿 MRO 查找，略慢
```

### 设计动机

| Python 设计选择 | 原因 | 替代方案对比 |
|-----------------|------|-------------|
| `@classmethod`而非直接传类 | 支持继承，子类调用时自动传子类 | Java 无类方法概念 |
| `@staticmethod`放在类中 | 语义关联，命名空间组织 | C 用独立函数 |
| 动态属性查找 | 灵活性，支持运行时添加属性 | Java 需预定义 |
| 描述符协议 | 统一的属性访问控制机制 | C++ 无此概念 |
