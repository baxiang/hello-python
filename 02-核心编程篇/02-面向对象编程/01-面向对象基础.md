# 面向对象基础

> **Python 版本要求：** 3.11+

本章讲解 Python 面向对象编程的基本概念，包括类与对象的关系、类的定义和对象的创建。

## 为什么需要面向对象？

### 问题场景

假设你要开发一个学生管理系统，需要管理大量学生的信息（姓名、年龄、成绩等）。

**使用面向过程的方式：**

```python
# 每个学生需要多个变量
student1_name = "张三"
student1_age = 18
student1_score = 85

student2_name = "李四"
student2_age = 17
student2_score = 92

# 每个学生都需要单独处理
def print_student(name: str, age: int, score: int) -> None:
    print(f"{name}, {age}岁, 成绩:{score}")

print_student(student1_name, student1_age, student1_score)
print_student(student2_name, student2_age, student2_score)
```

**问题：**
- 学生数量增加时，变量数量爆炸
- 相关数据没有组织在一起，容易出错
- 代码难以维护和扩展

**使用面向对象的方式：**

```python
class Student:
    def __init__(self, name: str, age: int, score: int) -> None:
        self.name = name
        self.age = age
        self.score = score
    
    def print_info(self) -> None:
        print(f"{self.name}, {self.age}岁, 成绩:{self.score}")

# 创建学生对象
students = [
    Student("张三", 18, 85),
    Student("李四", 17, 92),
]

# 统一处理
for student in students:
    student.print_info()
```

**优势：**
- 数据和行为封装在一起
- 代码更易维护和扩展
- 可以创建任意数量的学生对象

## 第一部分：面向过程 vs 面向对象

### 1.1 概念动机

面向对象是一种编程思想，它将数据和操作数据的方法封装在一起，形成"对象"。

```
┌─────────────────────────────────────────────────────────────┐
│              面向过程 vs 面向对象                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   面向过程：                                                 │
│   ✓ 关注"怎么做"（步骤）                                   │
│   ✓ 程序 = 函数 + 数据                                     │
│   ✓ 例如：洗菜() → 切菜() → 炒菜()                         │
│                                                             │
│   面向对象：                                                 │
│   ✓ 关注"谁来做"（对象）                                   │
│   ✓ 程序 = 对象 + 消息                                     │
│   ✓ 例如：厨师.洗菜()、厨师.切菜()、厨师.炒菜()            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 最简示例

```python
# ─────────────────────────────────────
# 面向过程：关注"怎么做"（步骤）
# ─────────────────────────────────────

def 洗菜() -> None:
    print("洗菜")

def 切菜() -> None:
    print("切菜")

def 炒菜() -> None:
    print("炒菜")

# 按步骤调用
洗菜()
切菜()
炒菜()


# ─────────────────────────────────────
# 面向对象：关注"谁来做"（对象）
# ─────────────────────────────────────

class 厨师:
    def 洗菜(self) -> None:
        print("洗菜")

    def 切菜(self) -> None:
        print("切菜")

    def 炒菜(self) -> None:
        print("炒菜")

# 创建对象并调用
chef = 厨师()
chef.洗菜()
chef.切菜()
chef.炒菜()
```

**核心理念：**
- **面向过程**：把问题分解为一系列步骤
- **面向对象**：找出问题域中的对象和它们的交互

### 1.3 详细讲解

```
类（Class）：模板/蓝图
│
├── 例如："Dog"类定义了狗的共性
│   - 属性：name, age, breed
│   - 方法：bark(), run()
│
▼
对象（Object）：具体的实例
│
├── my_dog = Dog("Buddy", 3)   → 具体的狗：Buddy，3 岁
├── your_dog = Dog("Max", 5)   → 具体的狗：Max，5 岁
└── his_dog = Dog("Lucy", 2)   → 具体的狗：Lucy，2 岁
```

**类比理解：**
- 类 = 月饼模具，对象 = 用模具做出的月饼
- 类 = 建筑图纸，对象 = 按图纸建好的房子
- 类 = 菜谱，对象 = 按菜谱做好的菜

## 第二部分：定义和创建类

### 2.1 概念动机

类是创建对象的模板，定义了对象应该有哪些属性和方法。

### 2.2 最简示例

**基本语法：**

```python
class ClassName:
    """类的文档字符串 - 说明类的用途"""

    def __init__(self, param1: str, param2: int) -> None:
        """初始化方法（构造函数）"""
        # 实例属性（每个实例独立）
        self.attribute1 = param1
        self.attribute2 = param2

    def method1(self) -> None:
        """实例方法"""
        pass
```

### 2.3 详细讲解

```python
class Dog:
    """表示狗的类"""

    # 类属性（所有实例共享）
    species: str = "Canis familiaris"

    def __init__(self, name: str, age: int, breed: str = "中华田园犬") -> None:
        """
        初始化方法
        参数:
            name: 名字
            age: 年龄
            breed: 品种（默认"中华田园犬"）
        """
        # 实例属性（每个实例独立）
        self.name: str = name
        self.age: int = age
        self.breed: str = breed

    def bark(self) -> None:
        """狗叫"""
        print(f"{self.name} 在叫：汪汪汪！")

    def run(self) -> None:
        """跑步"""
        print(f"{self.name} 正在奔跑！")

    def get_human_age(self) -> int:
        """计算人类年龄"""
        return self.age * 7


# ─────────────────────────────────────
# 创建和使用对象
# ─────────────────────────────────────

# 创建对象（实例化）
my_dog: Dog = Dog("Buddy", 3)
your_dog: Dog = Dog("Max", 5, "金毛")

# 访问属性
print(my_dog.name)      # Buddy
print(my_dog.species)   # Canis familiaris（类属性）

# 调用方法
my_dog.bark()             # Buddy 在叫：汪汪汪！
print(my_dog.get_human_age())  # 21
```

### 2.4 渐进复杂化

当类需要更多功能时，可以添加更多属性和方法：

```python
class Dog:
    """更完整的狗类"""
    
    species: str = "Canis familiaris"
    
    def __init__(self, name: str, age: int, breed: str = "中华田园犬", owner: str = "") -> None:
        self.name: str = name
        self.age: int = age
        self.breed: str = breed
        self.owner: str = owner
        self.energy: int = 100  # 能量值
    
    def bark(self, volume: str = "normal") -> None:
        """不同音量的叫声"""
        if volume == "loud":
            print(f"{self.name} 在大声叫：汪汪汪！")
        elif volume == "quiet":
            print(f"{self.name} 在小声叫：汪...")
        else:
            print(f"{self.name} 在叫：汪汪汪！")
    
    def run(self, distance: int) -> None:
        """跑步消耗能量"""
        if self.energy >= distance:
            self.energy -= distance
            print(f"{self.name} 跑了 {distance}米，剩余能量：{self.energy}")
        else:
            print(f"{self.name} 太累了，需要休息")
    
    def rest(self) -> None:
        """休息恢复能量"""
        self.energy = min(100, self.energy + 20)
        print(f"{self.name} 休息后能量：{self.energy}")
    
    def get_info(self) -> dict[str, str | int]:
        """获取完整信息"""
        return {
            "name": self.name,
            "age": self.age,
            "breed": self.breed,
            "owner": self.owner,
            "energy": self.energy,
        }
```

### 2.5 实际应用

创建多个狗对象并管理：

```python
# 创建狗群
dogs = [
    Dog("Buddy", 3, "金毛", "张三"),
    Dog("Max", 5, "拉布拉多", "李四"),
    Dog("Lucy", 2, "柯基", "王五"),
]

# 让每只狗活动
for dog in dogs:
    dog.bark()
    dog.run(30)
    print(dog.get_info())
    dog.rest()
```

## 第三部分：核心要点总结

### 3.1 核心概念表

| 概念 | 说明 | 示例 |
|------|------|------|
| **类** | 对象的模板 | `class Dog:` |
| **对象** | 类的具体实例 | `my_dog = Dog()` |
| **实例属性** | 每个实例独立 | `self.name` |
| **类属性** | 所有实例共享 | `species = "..."` |
| **`__init__`** | 初始化方法，创建对象时自动调用 | `def __init__(self):` |
| **self** | 指向当前实例对象 | 方法第一个参数 |
