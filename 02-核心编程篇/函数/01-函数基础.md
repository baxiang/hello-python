# 函数基础

本章讲解 Python 函数的基本概念、定义语法和调用方式。

---

## 什么是函数

### 概念说明

**函数（Function）** 是一段有名字的、可以反复调用的代码块。你把一组指令打包起来，给它起个名字，之后只需要喊这个名字，代码就会自动执行。

```
┌─────────────────────────────────────────────┐
│               函数 = 打包好的代码            │
├─────────────────────────────────────────────┤
│                                             │
│   输入（食材）                              │
│       │                                     │
│       ▼                                     │
│  ┌─────────────┐                            │
│  │   函数内部   │  ← 处理逻辑（菜谱步骤）   │
│  └─────────────┘                            │
│       │                                     │
│       ▼                                     │
│   输出（成品菜）                            │
│                                             │
└─────────────────────────────────────────────┘
```

**为什么要使用函数？**

- **避免重复**：同样的代码写一次，调用多次
- **分而治之**：把大问题拆成小函数，各个击破
- **易于修改**：改一处函数，所有调用处自动生效
- **便于测试**：可以单独验证每个函数是否正确

## 函数的定义和调用

### 基本语法

```python
def 函数名(参数列表):
    """文档字符串（可选）"""
    函数体
    return 返回值（可选）
```

### 示例代码

```python
# 定义函数
def greet():
    """打招呼的函数"""
    print("Hello, World!")

# 调用函数（可以多次调用）
greet()  # 输出：Hello, World!
greet()  # 输出：Hello, World!
greet()  # 输出：Hello, World!
```

**语法要点：**

| 元素 | 说明 |
|------|------|
| `def` | 关键字，宣告"我要定义一个函数" |
| 函数名 | 用动词命名，如 `calculate_area`、`get_user` |
| `()` | 括号，即使没有参数也不能省略 |
| `:` | 冒号，不能省略 |
| 缩进 | 函数体必须缩进（通常 4 个空格） |

## 带参数的函数

### 单个参数

```python
def greet(name):
    """向指定的人打招呼"""
    print(f"Hello, {name}!")

greet("Alice")   # Hello, Alice!
greet("Bob")     # Hello, Bob!
```

### 多个参数

```python
def introduce(name, age, city):
    """自我介绍"""
    print(f"我叫 {name}，今年 {age} 岁，来自 {city}")

introduce("张三", 25, "北京")
# 我叫 张三，今年 25 岁，来自 北京
```

## 返回值

### return 语句

```python
# 有返回值的函数
def add(a, b):
    """计算两个数的和"""
    result = a + b
    return result

answer = add(3, 5)
print(answer)  # 8

# 返回值可以直接参与计算
print(add(10, 20) * 2)  # 60
```

### 没有 return 的函数

```python
# 没有 return 的函数返回 None
def say_hello():
    print("Hello!")

result = say_hello()  # 打印 Hello!
print(result)         # None
```

**重要：** Python 中所有函数都有返回值。没有写 `return` 时，自动返回 `None`。

### 多个返回值

```python
def get_info():
    """返回多个值（实际返回元组）"""
    name = "Alice"
    age = 25
    city = "北京"
    return name, age, city

# 接收多个返回值
n, a, c = get_info()
print(n, a, c)  # Alice 25 北京

# 或者作为元组接收
info = get_info()
print(info)     # ('Alice', 25, '北京')
```

## 函数文档字符串

### docstring 的作用

```python
def calculate_area(width, height):
    """
    计算矩形面积
    
    参数:
        width: 宽度（正数）
        height: 高度（正数）
    
    返回:
        面积值
    
    示例:
        >>> calculate_area(5, 3)
        15
    """
    return width * height

# 查看文档
print(calculate_area.__doc__)

# 使用 help() 查看完整文档
help(calculate_area)
```
