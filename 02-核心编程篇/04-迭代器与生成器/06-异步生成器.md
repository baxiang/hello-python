## 为什么需要异步生成器？

### 问题场景

你需要处理大量异步数据流：
- 分页API数据获取
- 实时日志流处理
- 大文件异步逐行读取

**普通生成器：无法处理异步操作**

```python
def fetch_pages(urls: list[str]) -> Generator[dict, None, None]:
    for url in urls:
        response = requests.get(url)  # 同步阻塞！
        yield response.json()
```

**问题：**
- requests.get() 是同步阻塞操作
- 生成器内部无法使用 await
- 无法利用异步并发优势

**异步生成器：结合惰性与异步**

```python
import aiohttp
import asyncio
from collections.abc import AsyncGenerator

async def fetch_pages(urls: list[str]) -> AsyncGenerator[dict, None]:
    async with aiohttp.ClientSession() as session:
        for url in urls:
            async with session.get(url) as response:
                yield await response.json()

async def main() -> None:
    async for page in fetch_pages(["url1", "url2", "url3"]):
        print(page)

asyncio.run(main())
```

这就是异步生成器的价值：**惰性求值 + 异步I/O，高效处理数据流**。

## L1 理解层：异步生成器基础

### 语法结构

```
┌─────────────────────────────────────────────────────────────┐
│  异步生成器语法                                              │
│                                                             │
│  定义：                                                      │
│  async def async_gen() -> AsyncGenerator[T, None]:         │
│      await some_async_operation()                          │
│      yield value                                           │
│                                                             │
│  遍历：                                                      │
│  async for item in async_gen():                            │
│      await process(item)                                   │
│                                                             │
│  类型：                                                      │
│  from collections.abc import AsyncGenerator                │
│                                                             │
│  AsyncGenerator[YieldType, SendType]                       │
│  • YieldType → yield 产生的值类型                           │
│  • SendType → asend() 发送的值类型（通常 None）             │
│                                                             │
│  关键要点：                                                  │
│  1. async def + yield = 异步生成器                         │
│  2. 不能在普通函数中使用 yield                              │
│  3. 必须用 async for 遍历，不能用普通 for                   │
│  4. yield 前可以 await 异步操作                             │
└─────────────────────────────────────────────────────────────┘
```

### 最简示例

```python
import asyncio
from collections.abc import AsyncGenerator

async def async_counter(count: int) -> AsyncGenerator[int, None]:
    for i in range(count):
        await asyncio.sleep(0.1)  # 模拟异步操作
        yield i

async def main() -> None:
    async for num in async_counter(5):
        print(num)

asyncio.run(main())
```

**运行结果：**

```
0
1
2
3
4
```

**关键代码说明：**

| 代码 | 含义 |
|------|------|
| `async def async_counter(...)` | 定义异步生成器函数 |
| `await asyncio.sleep(0.1)` | yield 前可以执行异步操作 |
| `yield i` | 惰性产生值 |
| `async for num in async_counter(5)` | 必须用 async for 遍历 |
| `AsyncGenerator[int, None]` | 类型提示 |

### 详细示例：分页API数据获取

```python
import asyncio
from collections.abc import AsyncGenerator

async def fetch_pages(
    base_url: str,
    max_pages: int = 10,
) -> AsyncGenerator[dict[str, object], None]:
    """模拟分页API数据流"""
    page = 1
    while page <= max_pages:
        await asyncio.sleep(0.05)  # 模拟网络延迟
        yield {
            "page": page,
            "data": [f"item_{page}_{i}" for i in range(5)],
            "has_more": page < max_pages,
        }
        page += 1

async def process_pages() -> list[dict[str, object]]:
    """处理分页数据"""
    all_items: list[str] = []
    async for page_data in fetch_pages("https://api.example.com", max_pages=5):
        print(f"处理第 {page_data['page']} 页")
        all_items.extend(page_data["data"])
    return {"total": len(all_items), "items": all_items}

async def main() -> None:
    result = await process_pages()
    print(f"总计 {result['total']} 条数据")

asyncio.run(main())
```

**运行结果：**

```
处理第 1 页
处理第 2 页
处理第 3 页
处理第 4 页
处理第 5 页
总计 25 条数据
```

## L2 实践层：异步生成器应用

### 实际应用场景

#### 场景1：大文件异步逐行读取

```python
import asyncio
import aiofiles
from collections.abc import AsyncGenerator

async def read_lines_async(filepath: str) -> AsyncGenerator[str, None]:
    """异步逐行读取大文件"""
    async with aiofiles.open(filepath, mode="r") as file:
        async for line in file:
            yield line.strip()

async def process_large_file(filepath: str) -> dict[str, int]:
    """处理大文件，统计行数和词数"""
    line_count: int = 0
    word_count: int = 0
    
    async for line in read_lines_async(filepath):
        line_count += 1
        word_count += len(line.split())
    
    return {"lines": line_count, "words": word_count}

async def main() -> None:
    result = await process_large_file("large_file.txt")
    print(result)

asyncio.run(main())
```

**说明：**
- aiofiles 提供异步文件操作
- 逐行读取，内存占用恒定
- 适合处理超大日志文件

#### 场景2：数据库查询结果流

```python
import asyncio
from collections.abc import AsyncGenerator

async def query_users_batch(batch_size: int = 100) -> AsyncGenerator[dict[str, object], None]:
    """模拟数据库批量查询"""
    total_users: int = 500
    offset: int = 0
    
    while offset < total_users:
        await asyncio.sleep(0.02)  # 模拟数据库查询延迟
        batch: list[dict[str, object]] = [
            {"id": offset + i, "name": f"user_{offset + i}", "active": True}
            for i in range(min(batch_size, total_users - offset))
        ]
        yield {"batch": batch, "offset": offset, "total": total_users}
        offset += batch_size

async def export_users() -> list[dict[str, object]]:
    """导出所有用户"""
    all_users: list[dict[str, object]] = []
    
    async for result in query_users_batch(batch_size=50):
        print(f"获取批次 {result['offset']}/{result['total']}")
        all_users.extend(result["batch"])
    
    return all_users

async def main() -> None:
    users = await export_users()
    print(f"导出 {len(users)} 个用户")

asyncio.run(main())
```

**说明：**
- 批量查询，避免一次性加载所有数据
- 惰性处理，内存可控
- 适合大数据集导出

#### 场景3：实时日志流处理

```python
import asyncio
import random
from collections.abc import AsyncGenerator

async def log_stream(source: str) -> AsyncGenerator[dict[str, str], None]:
    """模拟实时日志流"""
    log_types: list[str] = ["INFO", "WARNING", "ERROR", "DEBUG"]
    
    for i in range(20):
        await asyncio.sleep(random.uniform(0.01, 0.05))
        yield {
            "timestamp": f"2024-01-{i+1:02d} 10:00:00",
            "level": random.choice(log_types),
            "source": source,
            "message": f"Event #{i} from {source}",
        }

async def monitor_logs() -> None:
    """监控多源日志"""
    error_count: int = 0
    
    async for log in log_stream("server-1"):
        if log["level"] == "ERROR":
            error_count += 1
            print(f"[ERROR] {log['timestamp']} {log['message']}")
        
        if error_count >= 3:
            print("错误过多，触发告警")
            break

async def main() -> None:
    await monitor_logs()

asyncio.run(main())
```

**说明：**
- 实时产生日志事件
- 惰性处理，按需过滤
- 可动态中断（break）

### 推荐做法

| 做法 | 原因 | 示例 |
|------|------|------|
| **用 AsyncGenerator 类型提示** | 明确类型，IDE支持更好 | `-> AsyncGenerator[dict, None]` |
| **yield 前做异步操作** | 真正利用异步优势 | `await fetch_data(); yield data` |
| **用 async for 遍历** | 必须的语法要求 | `async for item in agen` |
| **批量获取分页数据** | 平衡性能与内存 | 每批100条，而非逐条 |

### 反模式：不要这样做

#### 错误1：用普通 for 遍历异步生成器

```python
async def async_gen() -> AsyncGenerator[int, None]:
    yield 1

async def main() -> None:
    for item in async_gen():
        print(item)
```

**问题：**
- async_gen() 返回 AsyncGenerator，不是普通 Iterator
- 普通 for 无法处理异步值

```python
async def async_gen() -> AsyncGenerator[int, None]:
    yield 1

async def main() -> None:
    async for item in async_gen():
        print(item)
```

#### 错误2：在同步函数中使用 yield

```python
def sync_func():
    async def inner():
        yield 1
    return inner()
```

**问题：**
- yield 必须在 async def 函数中直接使用
- 不能嵌套在同步函数中定义异步生成器

```python
async def async_gen() -> AsyncGenerator[int, None]:
    yield 1

def get_gen():
    return async_gen()
```

#### 错误3：忘记 await yield 前的异步操作

```python
async def fetch_data() -> dict:
    await asyncio.sleep(0.1)
    return {"data": "value"}

async def async_gen() -> AsyncGenerator[dict, None]:
    data = fetch_data()  # ❌ 没有 await
    yield data
```

**问题：**
- fetch_data() 返回协程对象，不是实际数据
- yield 的是协程，不是期望的数据

```python
async def fetch_data() -> dict:
    await asyncio.sleep(0.1)
    return {"data": "value"}

async def async_gen() -> AsyncGenerator[dict, None]:
    data = await fetch_data()  # ✅ 正确 await
    yield data
```

### 适用场景

| 场景 | 是否推荐 | 原因 |
|------|---------|------|
| 分页API数据获取 | ✅ 推荐 | 惰性加载，内存可控 |
| 大文件逐行处理 | ✅ 推荐 | 异步I/O + 惰性读取 |
| 数据库批量查询 | ✅ 推荐 | 分批获取，避免内存溢出 |
| 实时数据流 | ✅ 推荐 | 按需处理，可中断 |
| 小数据集处理 | ❌ 不推荐 | 普通列表更简单 |
| 同步数据源 | ❌ 不推荐 | 用普通生成器即可 |

## L3 专家层：异步生成器底层原理

### 异步迭代器协议

```
┌─────────────────────────────────────────────────────────────┐
│          异步迭代器协议                                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  异步可迭代对象 (AsyncIterable)：                            │
│  ─────────────────────────────────────                       │
│  • 实现 __aiter__() 方法                                    │
│  • 返回异步迭代器                                           │
│                                                             │
│  异步迭代器 (AsyncIterator)：                                │
│  ─────────────────────────────────────                       │
│  • 实现 __aiter__() 和 __anext__() 方法                     │
│  • __anext__() 是 async 方法，返回 awaitable                │
│  • 迭代结束抛出 StopAsyncIteration                          │
│                                                             │
│  异步生成器 (AsyncGenerator)：                               │
│  ─────────────────────────────────────                       │
│  • async def 函数中使用 yield                               │
│  • 自动实现 __aiter__、__anext__、asend、athrow、aclose     │
│  • 是异步迭代器的特殊实现                                   │
│                                                             │
│  对比：                                                      │
│  ─────────────────────────────────────                       │
│  普通迭代器：                                                │
│  • __iter__() → self                                        │
│  • __next__() → next value                                  │
│  • StopIteration 结束                                       │
│                                                             │
│  异步迭代器：                                                │
│  • __aiter__() → self                                       │
│  • __anext__() → awaitable (async)                         │
│  • StopAsyncIteration 结束                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### async for 执行流程

```python
async for item in async_gen():
    await process(item)
```

**Python 内部执行：**

```
┌─────────────────────────────────────────────────────────────┐
│          async for 执行流程                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 获取异步迭代器                                          │
│     agen = async_gen()                                      │
│     iterator = agen.__aiter__()                             │
│                                                             │
│  2. 循环获取值                                              │
│     while True:                                             │
│         try:                                                │
│             awaitable = iterator.__anext__()                │
│             item = await awaitable                          │
│             await process(item)                             │
│         except StopAsyncIteration:                          │
│             break                                           │
│                                                             │
│  3. 清理                                                    │
│     await iterator.aclose()  # 如果提前退出                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 异步生成器方法

```python
import asyncio
from collections.abc import AsyncGenerator

async def interactive_gen() -> AsyncGenerator[int, int]:
    total: int = 0
    while True:
        received: int = yield total
        if received is None:
            total += 1
        else:
            total = received

async def main() -> None:
    gen = interactive_gen()
    
    value = await gen.asend(None)
    print(value)
    
    value = await gen.asend(10)
    print(value)
    
    value = await gen.asend(None)
    print(value)
    
    await gen.aclose()

asyncio.run(main())
```

**运行结果：**

```
0
10
11
```

**异步生成器方法：**

| 方法 | 说明 | 等价普通生成器 |
|------|------|--------------|
| `agen.asend(value)` | 发送值到生成器 | `gen.send(value)` |
| `agen.asend(None)` | 启动/获取下一个值 | `next(gen)` |
| `agen.athrow(exc)` | 抛出异常 | `gen.throw(exc)` |
| `agen.aclose()` | 关闭生成器 | `gen.close()` |

### 类型系统支持

```python
from collections.abc import AsyncGenerator, AsyncIterator, AsyncIterable

async def my_gen() -> AsyncGenerator[int, None]:
    yield 1

class MyAsyncIterator:
    def __aiter__(self) -> "MyAsyncIterator":
        return self
    
    async def __anext__(self) -> int:
        raise StopAsyncIteration

async def process(items: AsyncIterable[int]) -> list[int]:
    result: list[int] = []
    async for item in items:
        result.append(item)
    return result
```

**类型层次：**

```
┌─────────────────────────────────────────────────────────────┐
│          异步迭代类型层次                                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  AsyncIterable                                              │
│  ───────────────                                            │
│  • 有 __aiter__() 方法                                      │
│                                                             │
│       ↓                                                     │
│                                                             │
│  AsyncIterator                                              │
│  ───────────────                                            │
│  • 有 __aiter__() + __anext__() 方法                        │
│                                                             │
│       ↓                                                     │
│                                                             │
│  AsyncGenerator                                             │
│  ───────────────                                            │
│  • async def + yield                                        │
│  • 自动实现异步迭代器协议                                   │
│  • 还有 asend/athrow/aclose 方法                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 性能考量

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| async for 遍历 | O(n) | n = 值数量 |
| await 每个值 | 取决于异步操作 | 网络I/O、文件I/O等 |
| 内存占用 | O(1) 恒定 | 惰性求值 |
| 并发遍历 | 可并行 | asyncio.gather 多个生成器 |

**性能对比：**

```python
import asyncio
import time
from collections.abc import AsyncGenerator

async def async_batch_fetch(urls: list[str]) -> AsyncGenerator[dict, None]:
    """批量并发获取"""
    batch_size: int = 10
    for i in range(0, len(urls), batch_size):
        batch = urls[i:i+batch_size]
        results = await asyncio.gather(*[fetch(url) for url in batch])
        for result in results:
            yield result

async def fetch(url: str) -> dict:
    await asyncio.sleep(0.01)
    return {"url": url, "data": "result"}

async def sequential_fetch(urls: list[str]) -> list[dict]:
    """顺序获取"""
    results: list[dict] = []
    for url in urls:
        results.append(await fetch(url))
    return results

async def main() -> None:
    urls = [f"url_{i}" for i in range(50)]
    
    start = time.time()
    async for _ in async_batch_fetch(urls):
        pass
    batch_time = time.time() - start
    
    start = time.time()
    await sequential_fetch(urls)
    seq_time = time.time() - start
    
    print(f"批量并发: {batch_time:.3f}s")
    print(f"顺序获取: {seq_time:.3f}s")

asyncio.run(main())
```

**运行结果：**

```
批量并发: 0.06s
顺序获取: 0.52s
```

**说明：**
- 异步生成器 + asyncio.gather 实现批量并发
- 比顺序获取快 8 倍以上

### 设计动机

| 设计选择 | 原因 |
|----------|------|
| async def + yield | 统一异步语法，与 async/await 一致 |
| async for | 必须明确区分同步/异步遍历 |
| StopAsyncIteration | 与 StopIteration 区分，避免混淆 |
| AsyncGenerator 类型 | 类型系统支持，IDE友好 |
