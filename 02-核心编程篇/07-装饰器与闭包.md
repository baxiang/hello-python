### 12.1 函数是对象

#### 理解一等公民

```
┌─────────────────────────────────────────┐
│       Python 函数是"一等公民"           │
├─────────────────────────────────────────┤
│                                         │
│  一等公民（First-class Citizen）        │
│  ─────────────────────────────          │
│  可以在程序中任意位置使用的对象         │
│                                         │
│  函数可以：                             │
│  ────────────                           │
│  ✅ 赋值给变量                          │
│  ✅ 作为参数传递给其他函数              │
│  ✅ 作为函数的返回值                    │
│  ✅ 存储在数据结构中（列表、字典）      │
│                                         │
│  这使函数成为"一等公民"                 │
│                                         │
└─────────────────────────────────────────┘
```

#### 函数的各种用法

```python
# ─────────────────────────────────────
# 1. 赋值给变量
# ─────────────────────────────────────
def greet(name):
    return f"Hello, {name}!"

# 函数赋值给变量
say_hello = greet
print(say_hello("Alice"))  # Hello, Alice!

# 多个变量指向同一函数
func1 = greet
func2 = greet
print(func1("Bob"))   # Hello, Bob!
print(func2("Carol")) # Hello, Carol!

# ─────────────────────────────────────
# 2. 作为参数传递
# ─────────────────────────────────────
def execute(func, value):
    """执行传入的函数"""
    return func(value)

result = execute(greet, "David")
print(result)  # Hello, David!

# 实际应用场景：排序
students = [("Alice", 85), ("Bob", 90), ("Carol", 78)]

# 使用 key 参数传入函数
sorted_students = sorted(students, key=lambda x: x[1])
print(sorted_students)

# ─────────────────────────────────────
# 3. 作为返回值
# ─────────────────────────────────────
def get_operation(op):
    """返回不同的操作函数"""
    if op == "add":
        return lambda a, b: a + b
    elif op == "mul":
        return lambda a, b: a * b
    elif op == "sub":
        return lambda a, b: a - b

add_func = get_operation("add")
print(add_func(3, 5))  # 8

mul_func = get_operation("mul")
print(mul_func(3, 5))  # 15

# ─────────────────────────────────────
# 4. 存储在数据结构中
# ─────────────────────────────────────
def add(a, b): return a + b
def sub(a, b): return a - b
def mul(a, b): return a * b

# 存储在列表中
operations = [add, sub, mul]
for op in operations:
    print(op(10, 5))  # 15, 5, 50

# 存储在字典中
op_dict = {
    "+": add,
    "-": sub,
    "*": mul
}
print(op_dict["+"](10, 5))  # 15
```

### 12.2 嵌套函数与作用域

#### 嵌套函数基础

```python
def outer():
    """外层函数"""
    print("进入外层函数")

    def inner():
        """内层函数"""
        print("  进入内层函数")
        print("  离开内层函数")

    inner()  # 调用内层函数
    print("离开外层函数")

# 使用
outer()
# 输出：
# 进入外层函数
#   进入内层函数
#   离开内层函数
# 离开外层函数

# ❌ 内层函数在外部不可见
# inner()  # NameError!
```

#### Python 作用域规则（LEGB）

```
┌─────────────────────────────────────────┐
│         LEGB 作用域查找规则             │
├─────────────────────────────────────────┤
│                                         │
│  当使用一个变量时，Python 按以下顺序    │
│  查找：                                 │
│                                         │
│  L - Local（局部作用域）                │
│      当前函数内部的变量                 │
│                                         │
│  E - Enclosing（封闭作用域）            │
│      外层嵌套函数的变量                 │
│                                         │
│  G - Global（全局作用域）               │
│      模块级别的变量                     │
│                                         │
│  B - Built-in（内置作用域）             │
│      Python 内置的变量和函数            │
│                                         │
│  ─────────────────────────────────      │
│                                         │
│  查找过程：                             │
│  Local → Enclosing → Global → Built-in │
│                                         │
│  找到即停，找不到则抛异常               │
│                                         │
└─────────────────────────────────────────┘
```

#### 作用域示例

```python
x = "全局变量"  # Global

def outer():
    y = "外层变量"  # Enclosing

    def inner():
        z = "内层变量"  # Local
        print(f"inner: {z}")    # Local
        print(f"inner: {y}")    # Enclosing
        print(f"inner: {x}")    # Global

    inner()
    print(f"outer: {y}")  # Enclosing
    print(f"outer: {x}")  # Global

outer()
print(f"global: {x}")  # Global
```

## 第二部分：闭包详解

### 12.3 什么是闭包

#### 闭包的定义

```
┌─────────────────────────────────────────┐
│              闭包的概念                 │
├─────────────────────────────────────────┤
│                                         │
│  闭包（Closure）                        │
│  ─────────────────                      │
│  内层函数引用了外层函数的变量，         │
│  即使外层函数已执行完毕，               │
│  内层函数仍能访问这些变量。             │
│                                         │
│  ─────────────────────────────────      │
│                                         │
│  构成闭包的条件：                       │
│  1. 必须有嵌套函数                      │
│  2. 内层函数引用外层变量                │
│  3. 外层函数返回内层函数                │
│                                         │
└─────────────────────────────────────────┘
```

#### 闭包示例详解

```python
def make_multiplier(factor):
    """
    创建乘法器工厂函数

    参数:
        factor: 乘数因子
    """
    # factor 是外层函数的局部变量
    def multiply(number):
        # 内层函数引用了外层变量 factor
        return number * factor

    # 返回内层函数
    return multiply

# ─────────────────────────────────────
# 使用闭包
# ─────────────────────────────────────

# 创建不同的乘法器
double = make_multiplier(2)
triple = make_multiplier(3)

# 此时 make_multiplier(2) 已经执行完毕
# 但 factor=2 仍然被 double "记住"

print(double(5))   # 10（5 * 2）
print(double(10))  # 20（10 * 2）

print(triple(5))   # 15（5 * 3）
print(triple(10))  # 30（10 * 3）

# double 和 triple 各自"记住"了不同的 factor 值
```

**闭包工作原理图解：**
```
调用 make_multiplier(2)：

┌─────────────────────────────────┐
│  make_multiplier(factor=2)      │
│  ─────────────────────────      │
│  factor = 2                     │
│                                 │
│  def multiply(number):          │
│      return number * factor  ◄──┼── 引用 factor
│                                 │
│  return multiply ───────────────┼── 返回函数
└─────────────────────────────────│
                                  │
        返回的函数：               │
        double = multiply          │
        factor=2 被"封闭"在其中   │
                                  │
        double(5) → 5 * 2 = 10   │
```

### 12.4 nonlocal 关键字

#### 修改外层变量

```python
# ─────────────────────────────────────
# ❌ 不使用 nonlocal（会出错）
# ─────────────────────────────────────
def counter_wrong():
    count = 0

    def increment():
        # 这里会创建新的局部变量 count
        # 而不是修改外层的 count
        count += 1  # UnboundLocalError!
        return count

    return increment

# ─────────────────────────────────────
# ✅ 使用 nonlocal（正确）
# ─────────────────────────────────────
def counter():
    count = 0

    def increment():
        nonlocal count  # 声明使用外层变量
        count += 1
        return count

    return increment

# 使用
my_counter = counter()
print(my_counter())  # 1
print(my_counter())  # 2
print(my_counter())  # 3

# 每次调用都修改并记住 count 的值
```

#### nonlocal 的使用规则

```
┌─────────────────────────────────────────┐
│          nonlocal 使用规则              │
├─────────────────────────────────────────┤
│                                         │
│  作用：                                 │
│  声明变量来自外层作用域，不是新变量     │
│                                         │
│  ─────────────────────────────────      │
│                                         │
│  使用场景：                             │
│  • 需要修改外层函数的变量              │
│  • 实现状态保持的闭包                   │
│                                         │
│  ─────────────────────────────────      │
│                                         │
│  注意事项：                             │
│  • 不能用于全局变量（用 global）       │
│  • 外层必须有该变量定义                │
│  • 只能嵌套函数中使用                   │
│                                         │
│  ─────────────────────────────────      │
│                                         │
│  global vs nonlocal:                    │
│  global   - 引用全局变量               │
│  nonlocal - 引用外层嵌套变量           │
│                                         │
└─────────────────────────────────────────┘
```

### 12.5 闭包的应用

#### 应用 1：数据隐藏/封装

```python
def create_account(initial_balance):
    """
    创建银行账户（使用闭包隐藏余额）

    余额是私有的，只能通过指定方法访问
    """
    balance = initial_balance  # 私有变量

    def get_balance():
        return balance

    def deposit(amount):
        nonlocal balance
        if amount > 0:
            balance += amount
            return True
        return False

    def withdraw(amount):
        nonlocal balance
        if 0 < amount <= balance:
            balance -= amount
            return True
        return False

    # 返回接口
    return {
        "balance": get_balance,
        "deposit": deposit,
        "withdraw": withdraw
    }

# 使用
account = create_account(1000)

print(account["balance"]())  # 1000
account["deposit"](500)
print(account["balance"]())  # 1500
account["withdraw"](200)
print(account["balance"]())  # 1300

# 无法直接访问 balance
# print(balance)  # NameError!
```

#### 应用 2：缓存（Memoization）

```python
def memoize(func):
    """
    缓存装饰器：记住函数的返回结果

    对于相同的输入，直接返回缓存的结果
    避免重复计算
    """
    cache = {}  # 缓存字典

    def wrapper(*args):
        # 如果参数在缓存中，直接返回
        if args in cache:
            return cache[args]

        # 否则计算并缓存
        result = func(*args)
        cache[args] = result
        return result

    return wrapper

# 使用 memoize 优化斐波那契
@memoize
def fibonacci(n):
    """计算斐波那契数列"""
    if n < 2:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

# 没有 memoize 时，fibonacci(50) 非常慢
# 有了 memoize，瞬间完成
print(fibonacci(50))  # 12586269025
```

**缓存原理图解：**
```
fibonacci(5) 调用过程（有缓存）：

fib(5)
├── fib(4) → 计算并缓存
│   ├── fib(3) → 计算并缓存
│   │   ├── fib(2) → 计算并缓存
│   │   │   ├── fib(1) → 1（缓存）
│   │   │   └── fib(0) → 0（缓存）
│   │   └── fib(1) → 从缓存返回 1 ✓
│   └── fib(2) → 从缓存返回 1 ✓
└── fib(3) → 从缓存返回 2 ✓

没有缓存：重复计算，指数级时间复杂度
有缓存：每个值只计算一次，线性时间复杂度
```

## 第三部分：装饰器基础

### 12.6 什么是装饰器

#### 装饰器的概念

```
┌─────────────────────────────────────────┐
│             装饰器的概念                │
├─────────────────────────────────────────┤
│                                         │
│  装饰器（Decorator）                    │
│  ─────────────────                      │
│  是一个函数，用于修改其他函数的行为    │
│                                         │
│  特点：                                 │
│  • 不修改原函数的代码                  │
│  • 不改变原函数的调用方式              │
│  • 在运行时动态添加功能                │
│                                         │
│  ─────────────────────────────────      │
│                                         │
│  类比：                                 │
│  给房子装修                             │
│  • 房子 = 原函数                        │
│  • 装修 = 装饰器                        │
│  • 装修后房子功能增强，但结构不变       │
│                                         │
└─────────────────────────────────────────┘
```

#### 装饰器基本结构

```python
# ─────────────────────────────────────
# 装饰器的基本结构
# ─────────────────────────────────────
def my_decorator(func):
    """简单的装饰器"""

    def wrapper(*args, **kwargs):
        # 1. 函数执行前的逻辑
        print("函数执行前")

        # 2. 调用原函数
        result = func(*args, **kwargs)

        # 3. 函数执行后的逻辑
        print("函数执行后")

        # 4. 返回结果
        return result

    return wrapper

# ─────────────────────────────────────
# 使用装饰器
# ─────────────────────────────────────

# 方式 1：使用 @ 语法（推荐）
@my_decorator
def say_hello():
    print("Hello!")

say_hello()
# 输出：
# 函数执行前
# Hello!
# 函数执行后

# 方式 2：手动包装（等价）
def say_hello():
    print("Hello!")

say_hello = my_decorator(say_hello)
say_hello()
# 输出相同
```

**装饰器执行流程：**
```
@my_decorator
def greet(name):
    print(f"Hello, {name}!")

greet("Alice")

执行流程：

1. Python 定义 greet 函数
           │
2. 应用装饰器
   greet = my_decorator(greet)
           │
           ▼
   ┌─────────────────┐
   │   wrapper       │
   │  ────────────   │
   │  print("前")    │
   │  greet("Alice") │
   │  print("后")    │
   └─────────────────┘
           │
3. 调用 greet("Alice")
   实际执行 wrapper("Alice")
```

### 12.7 实用装饰器示例

#### 计时装饰器

```python
import time
from functools import wraps

def timer(func):
    """
    计时装饰器：测量函数执行时间
    """
    @wraps(func)
    def wrapper(*args, **kwargs):
        start_time = time.time()
        result = func(*args, **kwargs)
        end_time = time.time()
        elapsed_time = end_time - start_time
        print(f"{func.__name__} 执行时间：{elapsed_time:.4f}秒")
        return result
    return wrapper

# 使用
@timer
def slow_function():
    time.sleep(1)
    print("任务完成")

@timer
def fast_function():
    print("快速任务")

slow_function()
# 任务完成
# slow_function 执行时间：1.0012 秒

fast_function()
# 快速任务
# fast_function 执行时间：0.0001 秒
```

#### 日志装饰器

```python
from functools import wraps

def logger(func):
    """
    日志装饰器：记录函数调用信息
    """
    @wraps(func)
    def wrapper(*args, **kwargs):
        # 记录调用信息
        args_str = ", ".join(map(str, args))
        kwargs_str = ", ".join(f"{k}={v}" for k, v in kwargs.items())
        print(f"[调用] {func.__name__}({args_str}, {kwargs_str})")

        # 执行函数
        result = func(*args, **kwargs)

        # 记录返回结果
        print(f"[返回] {func.__name__} → {result}")
        return result
    return wrapper

# 使用
@logger
def add(a, b):
    return a + b

@logger
def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

add(3, 5)
# [调用] add(3, 5)
# [返回] add → 8

greet("Alice", greeting="Hi")
# [调用] greet(Alice, greeting=Hi)
# [返回] greet → Hi, Alice!
```

#### 权限验证装饰器

```python
from functools import wraps

class User:
    def __init__(self, name, is_admin=False):
        self.name = name
        self.is_authenticated = True
        self.is_admin = is_admin

def require_auth(func):
    """需要登录验证"""
    @wraps(func)
    def wrapper(user, *args, **kwargs):
        if not user.is_authenticated:
            raise PermissionError(f"用户 {user.name} 未登录")
        return func(user, *args, **kwargs)
    return wrapper

def require_admin(func):
    """需要管理员权限"""
    @wraps(func)
    def wrapper(user, *args, **kwargs):
        if not user.is_admin:
            raise PermissionError(f"用户 {user.name} 不是管理员")
        return func(user, *args, **kwargs)
    return wrapper

# 使用
@require_auth
def view_profile(user):
    return f"查看 {user.name} 的个人资料"

@require_auth
@require_admin
def delete_user(user, target_user):
    return f"管理员 {user.name} 删除了 {target_user}"

# 测试
admin = User("Admin", is_admin=True)
normal_user = User("User")

print(view_profile(admin))  # ✅
# print(view_profile(User("Guest")))  # ❌ 未登录

print(delete_user(admin, "BadUser"))  # ✅
# print(delete_user(normal_user, "BadUser"))  # ❌ 不是管理员
```

#### 重试装饰器

```python
from functools import wraps
import time
import random

def retry(times=3, delay=1, exceptions=(Exception,)):
    """
    重试装饰器：失败时自动重试

    参数:
        times: 最大重试次数
        delay: 重试间隔（秒）
        exceptions: 需要捕获的异常类型
    """
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            last_exception = None

            for attempt in range(times):
                try:
                    return func(*args, **kwargs)
                except exceptions as e:
                    last_exception = e
                    print(f"尝试 {attempt + 1}/{times} 失败：{e}")
                    if attempt < times - 1:
                        time.sleep(delay)

            # 所有尝试都失败
            raise last_exception

        return wrapper
    return decorator

# 使用
@retry(times=3, delay=0.5)
def unstable_operation():
    """模拟不稳定操作（50% 失败率）"""
    if random.random() < 0.5:
        raise ConnectionError("网络连接失败")
    return "操作成功"

# 测试
try:
    result = unstable_operation()
    print(result)
except Exception as e:
    print(f"最终失败：{e}")
```

## 第四部分：带参数的装饰器

### 12.8 装饰器工厂

#### 三层嵌套结构

```python
# ─────────────────────────────────────
# 带参数的装饰器结构
# ─────────────────────────────────────
def repeat(times):
    """
    重复执行装饰器

    参数:
        times: 重复次数
    """
    # 第一层：接收装饰器参数
    def decorator(func):
        # 第二层：接收被装饰的函数
        @wraps(func)
        def wrapper(*args, **kwargs):
            # 第三层：接收函数调用参数
            result = None
            for _ in range(times):
                result = func(*args, **kwargs)
            return result
        return wrapper
    return decorator

# ─────────────────────────────────────
# 使用
# ─────────────────────────────────────
@repeat(times=3)
def greet(name):
    print(f"Hello, {name}!")

greet("Alice")
# Hello, Alice!
# Hello, Alice!
# Hello, Alice!

# 等价写法
def greet(name):
    print(f"Hello, {name}!")

greet = repeat(times=3)(greet)
```

**带参数装饰器的执行顺序：**
```
@repeat(times=3)
def greet(name):
    ...

执行过程：

1. 调用 repeat(times=3)
   返回 decorator 函数
          │
2. 用 decorator 包装 greet
   greet = decorator(greet)
   返回 wrapper 函数
          │
3. 调用 greet("Alice")
   实际执行 wrapper("Alice")
```

### 12.9 多层装饰器

#### 装饰器堆叠

```python
from functools import wraps

def decorator1(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        print("[装饰器 1] 前处理")
        result = func(*args, **kwargs)
        print("[装饰器 1] 后处理")
        return result
    return wrapper

def decorator2(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        print("[装饰器 2] 前处理")
        result = func(*args, **kwargs)
        print("[装饰器 2] 后处理")
        return result
    return wrapper

def decorator3(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        print("[装饰器 3] 前处理")
        result = func(*args, **kwargs)
        print("[装饰器 3] 后处理")
        return result
    return wrapper

# 堆叠多个装饰器
@decorator1
@decorator2
@decorator3
def greet(name):
    print(f"  Hello, {name}!")
    return "完成"

# 调用
result = greet("Alice")
```

### 12.10 用类实现装饰器

#### 类装饰器基础

```python
from functools import wraps

class CountCalls:
    """
    统计函数调用次数的装饰器

    使用类而不是函数实现装饰器
    """

    def __init__(self, func):
        # 初始化：保存原函数
        wraps(func)(self)
        self.func = func
        self.count = 0

    def __call__(self, *args, **kwargs):
        # 每次调用函数时执行
        self.count += 1
        print(f"第 {self.count} 次调用 {self.__name__}")
        return self.func(*args, **kwargs)

# 使用
@CountCalls
def say_hello():
    print("Hello!")

@CountCalls
def say_goodbye():
    print("Goodbye!")

say_hello()
# 第 1 次调用 say_hello
# Hello!

say_hello()
# 第 2 次调用 say_hello
# Hello!

say_goodbye()
# 第 1 次调用 say_goodbye
# Goodbye!

# 每个函数有独立的计数器
```

#### 带参数的类装饰器

```python
class Repeat:
    """
    重复执行指定次数的类装饰器
    """

    def __init__(self, times):
        # 接收装饰器参数
        self.times = times

    def __call__(self, func):
        # 接收并装饰函数
        @wraps(func)
        def wrapper(*args, **kwargs):
            result = None
            for _ in range(self.times):
                result = func(*args, **kwargs)
            return result
        return wrapper

# 使用
@Repeat(times=3)
def greet(name):
    print(f"Hello, {name}!")

greet("Alice")
# Hello, Alice!
# Hello, Alice!
# Hello, Alice!
```

### 12.11 functools.wraps 详解

#### 为什么需要 wraps

```python
from functools import wraps

# ─────────────────────────────────────
# ❌ 不使用 wraps（有问题）
# ─────────────────────────────────────
def my_decorator(func):
    def wrapper(*args, **kwargs):
        return func(*args, **kwargs)
    return wrapper

@my_decorator
def greet(name):
    """打招呼函数"""
    print(f"Hello, {name}!")

# 问题：元信息丢失
print(greet.__name__)  # wrapper（不是 greet!）
print(greet.__doc__)   # None（文档丢失!）

# 这会影响：
# • 调试和日志
# • 文档生成工具
# • IDE 的自动补全

# ─────────────────────────────────────
# ✅ 使用 wraps（正确）
# ─────────────────────────────────────
def my_decorator(func):
    @wraps(func)  # 保留原函数的元信息
    def wrapper(*args, **kwargs):
        return func(*args, **kwargs)
    return wrapper

@my_decorator
def greet(name):
    """打招呼函数"""
    print(f"Hello, {name}!")

# 元信息保留
print(greet.__name__)  # greet ✅
print(greet.__doc__)   # 打招呼函数 ✅
```

**wraps 的作用：**
```
@wraps(func) 等价于：

wrapper.__name__ = func.__name__
wrapper.__doc__ = func.__doc__
wrapper.__module__ = func.__module__
wrapper.__qualname__ = func.__qualname__
wrapper.__annotations__ = func.__annotations__
wrapper.__dict__.update(func.__dict__)

保留所有元信息，使装饰后的函数
看起来和原函数一样
```

### 练习 1：闭包计数器

```python
def create_counter(start=0):
    """
    创建一个计数器

    参数:
        start: 初始值

    返回:
        计数器函数
    """
    count = start

    def increment(step=1):
        nonlocal count
        count += step
        return count

    def decrement(step=1):
        nonlocal count
        count -= step
        return count

    def get():
        return count

    def reset():
        nonlocal count
        count = start
        return count

    return {
        "inc": increment,
        "dec": decrement,
        "get": get,
        "reset": reset
    }

# 测试
counter = create_counter(10)
print(counter["get"]())     # 10
print(counter["inc"](5))    # 15
print(counter["dec"](3))    # 12
print(counter["reset"]())   # 10
```

### 练习 2：类型检查装饰器

```python
from functools import wraps

def type_check(*expected_types):
    """
    类型检查装饰器

    参数:
        expected_types: 期望的参数类型
    """
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            # 检查位置参数
            for arg, expected_type in zip(args, expected_types):
                if not isinstance(arg, expected_type):
                    raise TypeError(
                        f"期望 {expected_type.__name__}, "
                        f"得到 {type(arg).__name__}"
                    )

            # 检查关键字参数
            for key, value in kwargs.items():
                if key in func.__annotations__:
                    expected = func.__annotations__[key]
                    if not isinstance(value, expected):
                        raise TypeError(
                            f"参数 {key} 期望 {expected.__name__}, "
                            f"得到 {type(value).__name__}"
                        )

            return func(*args, **kwargs)
        return wrapper
    return decorator

# 使用
@type_check(int, int)
def add(a, b):
    return a + b

print(add(3, 5))        # 8
# print(add("3", 5))    # TypeError!
```

### 练习 3：缓存装饰器（LRU）

```python
from functools import wraps
from collections import OrderedDict

def lru_cache(maxsize=128):
    """
    LRU（最近最少使用）缓存装饰器

    参数:
        maxsize: 最大缓存数量
    """
    cache = OrderedDict()

    def decorator(func):
        @wraps(func)
        def wrapper(*args):
            if args in cache:
                # 移到末尾（标记为最近使用）
                cache.move_to_end(args)
                return cache[args]

            # 计算并缓存
            result = func(*args)
            cache[args] = result
            cache.move_to_end(args)

            # 如果超出限制，删除最旧的
            if len(cache) > maxsize:
                cache.popitem(last=False)

            return result
        return wrapper
    return decorator

# 使用
@lru_cache(maxsize=100)
def fibonacci(n):
    if n < 2:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(50))  # 快速计算
```

### 练习 4：超时装饰器

```python
from functools import wraps
import signal

class TimeoutError(Exception):
    """超时异常"""
    pass

def timeout(seconds):
    """
    超时装饰器：函数执行超过指定时间则抛出异常

    参数:
        seconds: 超时时间（秒）
    """
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            # 定义超时处理函数
            def handler(signum, frame):
                raise TimeoutError(f"函数 {func.__name__} 执行超时（{seconds}秒）")

            # 注册信号处理
            old_handler = signal.signal(signal.SIGALRM, handler)
            signal.alarm(seconds)

            try:
                return func(*args, **kwargs)
            finally:
                # 恢复原处理函数
                signal.alarm(0)
                signal.signal(signal.SIGALRM, old_handler)

        return wrapper
    return decorator

# 使用
@timeout(seconds=2)
def slow_task():
    import time
    time.sleep(3)  # 超过 2 秒
    return "完成"

try:
    slow_task()
except TimeoutError as e:
    print(e)  # 函数 slow_task 执行超时（2 秒）
```

### 练习 5：单例装饰器

```python
from functools import wraps

def singleton(cls):
    """
    单例装饰器：确保类只有一个实例

    参数:
        cls: 要装饰的类

    返回:
        返回单例实例的函数
    """
    instances = {}

    @wraps(cls)
    def get_instance(*args, **kwargs):
        if cls not in instances:
            instances[cls] = cls(*args, **kwargs)
        return instances[cls]

    return get_instance

# 使用
@singleton
class Database:
    def __init__(self):
        print("初始化数据库连接")
        self.connected = True

    def query(self, sql):
        return f"执行 SQL: {sql}"

# 测试
db1 = Database()  # 初始化数据库连接
db2 = Database()  # 不会再次初始化

print(db1 is db2)  # True（同一个实例）
print(db1.query("SELECT * FROM users"))
```
