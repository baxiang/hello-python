# 异常处理

本章讲解 Python 异常处理机制，包括 try-except 语句、else 和 finally 子句。

---

## try-except 语句

### 基本语法

```python
try:
    # 可能出错的代码
    num = int(input("请输入一个数字："))
    result = 10 / num
    print(f"结果：{result}")

except ValueError:
    # 处理 ValueError
    print("输入无效，请输入数字！")

except ZeroDivisionError:
    # 处理 ZeroDivisionError
    print("不能除以零！")
```

## 捕获多个异常

### 分别处理

```python
try:
    num = int(input("请输入数字："))
    result = 10 / num
    print(f"结果：{result}")

except ValueError:
    print("❌ 输入的不是有效数字！")

except ZeroDivisionError:
    print("❌ 不能除以零！")
```

### 统一处理

```python
try:
    num = int(input("请输入数字："))
    result = 10 / num
    print(f"结果：{result}")

except (ValueError, ZeroDivisionError) as e:
    print(f"❌ 发生错误：{e}")
```

#### `as` 关键字的说明

`as` 用于将捕获的异常对象绑定到变量：

```python
try:
    result = 10 / 0
except ZeroDivisionError as e:
    # e 就是捕获到的异常对象
    print(type(e))      # <class 'ZeroDivisionError'>
    print(str(e))       # division by zero
    print(repr(e))      # ZeroDivisionError('division by zero')
```

**注意：** `as` 在 Python 中有多种用途，不限于异常处理：

```
┌─────────────────────────────────────────────────────┐
│              as 关键字的 3 种用途                    │
├─────────────────────────────────────────────────────┤
│                                                     │
│  1. 异常处理                                        │
│     except ValueError as e:                         │
│     → 将异常对象绑定到变量 e                         │
│                                                     │
│  2. 模块导入别名                                    │
│     import numpy as np                              │
│     from math import sqrt as square_root            │
│     → 给模块或函数起别名                             │
│                                                     │
│  3. 上下文管理器                                    │
│     with open("file.txt") as f:                     │
│     → 将上下文管理器的返回值绑定到变量               │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**对比示例：**

```python
# 1. 异常处理中的 as
try:
    risky_operation()
except ValueError as e:
    print(f"错误：{e}")

# 2. 模块导入中的 as
import numpy as np
from math import pi as PI

# 3. 上下文管理器中的 as
with open("file.txt") as f:
    content = f.read()
```

## 获取异常信息

```python
try:
    result = 10 / 0

except Exception as e:
    # e 是异常对象
    print(f"错误类型：{type(e).__name__}")
    print(f"错误信息：{e}")
    print(f"错误详情：{e.args}")

# 输出：
# 错误类型：ZeroDivisionError
# 错误信息：division by zero
# 错误详情：('division by zero',)
```

## else 和 finally 子句

### else 子句

```python
try:
    num = int(input("请输入数字："))
    result = 10 / num

except ValueError:
    print("请输入有效的数字！")

except ZeroDivisionError:
    print("不能除以零！")

else:
    # 只有 try 块没有抛出异常时才执行
    print(f"✅ 计算成功！结果：{result}")
```

### finally 子句

```python
try:
    print("尝试打开文件...")
    file = open("data.txt", "r")
    content = file.read()

except FileNotFoundError:
    print("❌ 文件不存在")

finally:
    # 无论是否发生异常，都会执行
    print("清理资源...")
    if 'file' in locals():
        file.close()
        print("文件已关闭")
```

### finally 的应用场景

```
┌─────────────────────────────────────────┐
│         finally 的应用场景              │
├─────────────────────────────────────────┤
│                                         │
│  📁 文件操作：关闭文件                   │
│  🔌 网络连接：关闭连接                   │
│  🔓 锁：释放锁                          │
│  🗄️ 数据库：关闭连接                     │
│                                         │
│  finally 总是执行，确保资源释放         │
```

## 完整的异常处理结构

```python
def divide_numbers():
    """完整的异常处理示例"""

    try:
        num1 = int(input("请输入被除数："))
        num2 = int(input("请输入除数："))
        result = num1 / num2
        print(f"结果：{result}")

    except ValueError:
        print("❌ 错误：请输入有效的整数！")

    except ZeroDivisionError:
        print("❌ 错误：除数不能为零！")

    except Exception as e:
        print(f"❌ 发生未知错误：{type(e).__name__}: {e}")

    else:
        print("✅ 计算成功完成！")
        return result

    finally:
        print("感谢使用，再见！")

    return None
```
