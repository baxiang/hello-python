# 上下文管理器

本章讲解 Python 上下文管理器，包括 with 语句和自定义上下文管理器。

---

## with 语句

### 基本用法

```python
# 文件操作
with open("file.txt", "r", encoding="utf-8") as f:
    content = f.read()
# 文件自动关闭
```

### 多个上下文管理器

```python
with open("input.txt", "r", encoding="utf-8") as fin, \
     open("output.txt", "w", encoding="utf-8") as fout:
    fout.write(fin.read())
```

## 上下文管理器协议

### 概念说明

上下文管理器需要实现两个方法：
- `__enter__`：进入上下文时调用
- `__exit__`：离开上下文时调用

```python
class ManagedFile:
    """文件上下文管理器"""

    def __init__(self, filename, mode):
        self.filename = filename
        self.mode = mode
        self.file = None

    def __enter__(self):
        """进入上下文时调用"""
        print(f"打开文件：{self.filename}")
        self.file = open(self.filename, self.mode, encoding="utf-8")
        return self.file

    def __exit__(self, exc_type, exc_val, exc_tb):
        """离开上下文时调用"""
        print("关闭文件...")
        if self.file:
            self.file.close()
        return False  # 不吞掉异常

# 使用
with ManagedFile("data.txt", "r") as f:
    content = f.read()
```

## contextlib 模块

### @contextmanager 装饰器

```python
from contextlib import contextmanager

@contextmanager
def managed_file(filename, mode):
    """文件上下文管理器（生成器版本）"""
    f = None
    try:
        print(f"打开文件：{filename}")
        f = open(filename, mode, encoding="utf-8")
        yield f  # 将文件对象传递给 with 块
    finally:
        if f:
            print("关闭文件...")
            f.close()

# 使用
with managed_file("data.txt", "r") as f:
    content = f.read()
```

## 实用示例

### 计时器

```python
import time
from contextlib import contextmanager

@contextmanager
def timer(name="操作"):
    """计时代码执行时间"""
    start = time.time()
    try:
        yield
    finally:
        end = time.time()
        print(f"{name}耗时：{end - start:.4f} 秒")

# 使用
with timer("数据处理"):
    result = [x * 2 for x in range(1000000)]
```

### 临时切换目录

```python
import os
from contextlib import contextmanager

@contextmanager
def working_directory(path):
    """临时切换工作目录"""
    original_dir = os.getcwd()
    try:
        os.chdir(path)
        yield
    finally:
        os.chdir(original_dir)

# 使用
with working_directory("/tmp"):
    print(f"临时目录：{os.getcwd()}")
```

### 抑制异常

```python
from contextlib import suppress

# 抑制指定异常
with suppress(FileNotFoundError):
    os.remove("不存在的文件.txt")
# 不会抛出异常
```
