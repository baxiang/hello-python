# 属性与方法

> **Python 版本要求：** 3.11+

本章讲解 Python 类的属性和方法，包括实例属性、类属性、实例方法、类方法和静态方法。

---

## 为什么需要不同的属性和方法？

### 问题场景

假设你要开发一个学生管理系统，需要：
1. 统计总学生数量（所有学生共享）
2. 每个学生有自己的姓名和年龄
3. 提供工具方法判断年龄是否成年

**问题代码：**

```python
# 使用全局变量管理共享数据
student_count = 0

class Student:
    def __init__(self, name: str, age: int) -> None:
        self.name = name
        self.age = age
        global student_count
        student_count += 1

# 问题：全局变量容易被意外修改
student_count = 100  # 误操作！

# 每次判断成年都要写判断逻辑
def is_adult(age: int) -> bool:
    return age >= 18

# 判断成年和 Student 类没有关联
print(is_adult(18))  # 与 Student 类无关
```

**问题：**
- 如何让所有学生共享同一个计数器？
- 如何区分每个学生独有的数据？
- 如何创建与实例无关的工具函数？

**解决方案：**

```python
class Student:
    # 类属性：所有实例共享
    count: int = 0
    school: str = "第一中学"
    
    def __init__(self, name: str, age: int) -> None:
        # 实例属性：每个实例独立
        self.name: str = name
        self.age: int = age
        Student.count += 1  # 通过类名修改类属性
    
    @classmethod
    def get_count(cls) -> int:
        """类方法：操作类属性"""
        return cls.count
    
    @staticmethod
    def is_adult(age: int) -> bool:
        """静态方法：工具函数"""
        return age >= 18

# 类属性被保护在类内部
s1 = Student("张三", 18)
s2 = Student("李四", 17)
print(Student.get_count())  # 2

# 静态方法与类关联
print(Student.is_adult(18))  # True
```

**优势：**
- 类属性让所有实例共享数据
- 实例属性让每个实例有独立数据
- 类方法操作类级数据
- 静态方法提供工具函数

## 第一部分：实例属性 vs 类属性

### 1.1 概念动机

在类中定义的属性有两种：实例属性和类属性。理解它们的区别是掌握面向对象的关键。

```
┌─────────────────────────────────────────────────────────────┐
│              实例属性 vs 类属性                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   实例属性：                                                 │
│   ✓ 在 __init__ 中通过 self 定义                            │
│   ✓ 每个实例独立拥有                                        │
│   ✓ 修改只影响当前实例                                      │
│                                                             │
│   类属性：                                                   │
│   ✓ 在类中直接定义                                          │
│   ✓ 所有实例共享                                            │
│   ✓ 修改影响所有实例                                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 最简示例

```python
class Student:
    # 类属性（所有实例共享）
    school: str = "第一中学"
    count: int = 0

    def __init__(self, name: str, age: int) -> None:
        # 实例属性（每个实例独立）
        self.name: str = name
        self.age: int = age
        Student.count += 1  # 修改类属性


# 创建实例
s1 = Student("Alice", 15)
s2 = Student("Bob", 16)

# 访问类属性（推荐通过类名）
print(Student.school)  # 第一中学
print(Student.count)   # 2

# 访问实例属性
print(s1.name)  # Alice
print(s2.name)  # Bob

# 修改类属性（影响所有实例）
Student.school = "实验中学"
print(s1.school)  # 实验中学
print(s2.school)  # 实验中学

# 修改实例属性（只影响当前实例）
s1.name = "Alice New"
print(s1.name)  # Alice New
print(s2.name)  # Bob（不变）
```

### 1.3 详细讲解

**属性对比表：**

| | 实例属性 | 类属性 |
|---|---|---|
| **定义位置** | `__init__`中用`self`定义 | 类中直接定义 |
| **归属** | 每个实例独立拥有 | 所有实例共享 |
| **访问方式** | `self.属性名` | `类名.属性名` |
| **修改影响** | 只影响当前实例 | 影响所有实例 |

**注意事项：**

```python
class Student:
    count: int = 0
    
    def __init__(self, name: str) -> None:
        self.name: str = name
        Student.count += 1

s1 = Student("Alice")
s2 = Student("Bob")

# ❌ 通过实例修改类属性（容易出错）
s1.count = 100  # 这实际上创建了一个实例属性！
print(s1.count)      # 100（实例属性）
print(Student.count) # 2（类属性未改变）

# ✅ 正确方式：通过类名修改
Student.count = 100
print(s1.count)      # 100（如果实例没有同名属性）
print(s2.count)      # 100（所有实例都看到新值）
```

### 1.4 渐进复杂化

类属性的常见应用场景：

```python
class Configuration:
    """系统配置类"""
    
    # 配置项作为类属性
    version: str = "1.0.0"
    debug_mode: bool = False
    max_connections: int = 100
    
    # 类属性可以是复杂类型
    default_settings: dict[str, str | int] = {
        "theme": "dark",
        "language": "zh-CN",
        "timeout": 30,
    }
    
    @classmethod
    def update_setting(cls, key: str, value: str | int) -> None:
        """更新默认配置"""
        cls.default_settings[key] = value
    
    @classmethod
    def get_version(cls) -> str:
        """获取版本"""
        return cls.version

# 所有模块共享同一配置
print(Configuration.version)
Configuration.update_setting("theme", "light")
print(Configuration.default_settings)
```

### 1.5 实际应用

学生管理系统示例：

```python
class Student:
    """学生类"""
    
    # 类属性
    school: str = "第一中学"
    total_count: int = 0
    grade_count: dict[str, int] = {}
    
    def __init__(self, name: str, age: int, grade: str) -> None:
        # 实例属性
        self.name: str = name
        self.age: int = age
        self.grade: str = grade
        
        # 更新类属性
        Student.total_count += 1
        Student.grade_count[grade] = Student.grade_count.get(grade, 0) + 1
    
    def get_info(self) -> dict[str, str | int]:
        """获取学生信息"""
        return {
            "name": self.name,
            "age": self.age,
            "grade": self.grade,
            "school": Student.school,
        }
    
    @classmethod
    def get_statistics(cls) -> dict[str, str | int | dict]:
        """获取统计信息"""
        return {
            "school": cls.school,
            "total": cls.total_count,
            "grade_distribution": cls.grade_count,
        }

# 创建学生
students = [
    Student("张三", 18, "高三"),
    Student("李四", 17, "高二"),
    Student("王五", 16, "高一"),
    Student("赵六", 17, "高二"),
]

# 查看统计
print(Student.get_statistics())
# {'school': '第一中学', 'total': 4, 'grade_distribution': {'高三': 1, '高二': 2, '高一': 1}}
```

## 第二部分：三种方法类型

### 2.1 概念动机

Python 类有三种方法：实例方法、类方法、静态方法。它们有不同的访问能力和使用场景。

```
┌─────────────────────────────────────────────────────────────┐
│              三种方法类型                                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   实例方法：                                                 │
│   ✓ 第一个参数是 self                                       │
│   ✓ 可以访问实例属性和类属性                                │
│   ✓ 用于操作实例数据                                        │
│                                                             │
│   类方法：                                                   │
│   ✓ 使用 @classmethod 装饰器                                │
│   ✓ 第一个参数是 cls                                        │
│   ✓ 只能访问类属性                                          │
│   ✓ 用于工厂方法、操作类数据                                │
│                                                             │
│   静态方法：                                                 │
│   ✓ 使用 @staticmethod 装饰器                               │
│   ✓ 无特殊参数                                              │
│   ✓ 不能访问实例或类属性                                    │
│   ✓ 用于工具函数                                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 最简示例

**实例方法：**

```python
class Student:
    def __init__(self, name: str) -> None:
        self.name: str = name

    # 实例方法：操作实例数据
    def introduce(self) -> None:
        """实例方法：自我介绍"""
        print(f"我是{self.name}")
```

**类方法：**

```python
class Student:
    count: int = 0  # 类属性

    def __init__(self, name: str) -> None:
        self.name: str = name
        Student.count += 1

    # 类方法：操作类数据（使用 @classmethod）
    @classmethod
    def get_count(cls) -> int:
        """类方法：获取学生总数"""
        return cls.count

    @classmethod
    def from_string(cls, data_str: str) -> "Student":
        """类方法：从字符串创建对象（工厂方法）"""
        name = data_str.split(",")[0]
        return cls(name)  # 相当于 Student(name)
```

**静态方法：**

```python
class Student:
    # 静态方法：工具函数（使用 @staticmethod）
    @staticmethod
    def is_adult(age: int) -> bool:
        """静态方法：判断是否成年"""
        return age >= 18
```

### 2.3 详细讲解

**完整示例：**

```python
class Student:
    count: int = 0

    def __init__(self, name: str) -> None:
        self.name: str = name
        Student.count += 1

    def introduce(self) -> None:
        """实例方法"""
        print(f"我是{self.name}")

    @classmethod
    def get_count(cls) -> int:
        """类方法"""
        return cls.count

    @classmethod
    def from_string(cls, data_str: str) -> "Student":
        """类方法：工厂方法"""
        name = data_str.split(",")[0]
        return cls(name)

    @staticmethod
    def is_adult(age: int) -> bool:
        """静态方法"""
        return age >= 18


# 使用
s1 = Student("Alice")
s2 = Student("Bob")

# 实例方法需要通过对象调用
s1.introduce()  # 我是 Alice

# 类方法可以通过类或实例调用
print(Student.get_count())  # 2
print(s1.get_count())       # 2

# 使用类方法创建对象
s3 = Student.from_string("Charlie,18")
print(s3.name)  # Charlie

# 静态方法不需要创建实例
print(Student.is_adult(20))  # True
```

**三种方法对比表：**

| 方法类型 | 装饰器 | 第一个参数 | 访问能力 | 使用场景 |
|----------|--------|-----------|----------|----------|
| **实例方法** | 无 | `self` | 实例属性 + 类属性 | 操作实例数据 |
| **类方法** | `@classmethod` | `cls` | 类属性 | 工厂方法、操作类数据 |
| **静态方法** | `@staticmethod` | 无 | 不能访问实例或类属性 | 工具函数 |

### 2.4 渐进复杂化

工厂方法的多种实现：

```python
class Student:
    """学生类 - 多种创建方式"""
    
    def __init__(self, name: str, age: int, grade: str) -> None:
        self.name: str = name
        self.age: int = age
        self.grade: str = grade
    
    def __str__(self) -> str:
        return f"{self.name}({self.grade}, {self.age}岁)"
    
    @classmethod
    def from_dict(cls, data: dict[str, str | int]) -> "Student":
        """从字典创建"""
        return cls(
            name=str(data.get("name", "")),
            age=int(data.get("age", 0)),
            grade=str(data.get("grade", "")),
        )
    
    @classmethod
    def from_string(cls, data_str: str) -> "Student":
        """从字符串创建"""
        parts = data_str.split(",")
        return cls(parts[0], int(parts[1]), parts[2])
    
    @classmethod
    def from_csv_line(cls, line: str) -> "Student":
        """从 CSV 行创建"""
        name, age, grade = line.strip().split(",")
        return cls(name, int(age), grade)
    
    @staticmethod
    def validate_age(age: int) -> bool:
        """验证年龄"""
        return 0 < age < 150
    
    @staticmethod
    def validate_grade(grade: str) -> bool:
        """验证年级"""
        valid_grades = ["高一", "高二", "高三"]
        return grade in valid_grades

# 多种创建方式
s1 = Student("张三", 18, "高三")
s2 = Student.from_dict({"name": "李四", "age": 17, "grade": "高二"})
s3 = Student.from_string("王五,16,高一")
s4 = Student.from_csv_line("赵六,17,高二\n")

for s in [s1, s2, s3, s4]:
    print(s)
```

### 2.5 实际应用

完整的学生管理系统：

```python
from typing import Optional

class Student:
    """学生类"""
    
    # 类属性
    school: str = "第一中学"
    total_count: int = 0
    
    def __init__(self, name: str, age: int, grade: str, score: int = 0) -> None:
        # 实例属性
        self.name: str = name
        self.age: int = age
        self.grade: str = grade
        self.score: int = score
        
        # 更新统计
        Student.total_count += 1
    
    def update_score(self, new_score: int) -> bool:
        """更新成绩（实例方法）"""
        if 0 <= new_score <= 100:
            self.score = new_score
            return True
        return False
    
    def get_info(self) -> dict[str, str | int]:
        """获取信息（实例方法）"""
        return {
            "name": self.name,
            "age": self.age,
            "grade": self.grade,
            "score": self.score,
            "school": Student.school,
        }
    
    @classmethod
    def change_school(cls, new_school: str) -> None:
        """更换学校（类方法）"""
        cls.school = new_school
    
    @classmethod
    def create_from_json(cls, json_data: dict) -> Optional["Student"]:
        """从 JSON 创建（类方法）"""
        try:
            return cls(
                json_data["name"],
                json_data["age"],
                json_data["grade"],
                json_data.get("score", 0),
            )
        except KeyError:
            return None
    
    @staticmethod
    def calculate_average(scores: list[int]) -> float:
        """计算平均分（静态方法）"""
        if not scores:
            return 0.0
        return sum(scores) / len(scores)
    
    @staticmethod
    def get_grade_level(score: int) -> str:
        """获取等级（静态方法）"""
        if score >= 90:
            return "A"
        elif score >= 80:
            return "B"
        elif score >= 70:
            return "C"
        elif score >= 60:
            return "D"
        else:
            return "F"

# 使用
students = [
    Student("张三", 18, "高三", 85),
    Student("李四", 17, "高二", 92),
    Student("王五", 16, "高一", 78),
]

# 实例方法操作实例数据
students[0].update_score(90)

# 类方法操作类数据
Student.change_school("实验中学")

# 静态方法提供工具功能
scores = [s.score for s in students]
avg = Student.calculate_average(scores)
print(f"平均分: {avg}")
print(f"等级分布: {[Student.get_grade_level(s) for s in scores]}")
```
