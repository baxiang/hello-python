# 属性与方法

本章讲解 Python 类的属性和方法，包括实例属性、类属性、实例方法、类方法和静态方法。

---

## 实例属性 vs 类属性

### 概念说明

```python
class Student:
    # 类属性（所有实例共享）
    school = "第一中学"
    count = 0

    def __init__(self, name, age):
        # 实例属性（每个实例独立）
        self.name = name
        self.age = age
        Student.count += 1  # 修改类属性


# 创建实例
s1 = Student("Alice", 15)
s2 = Student("Bob", 16)

# 访问类属性（推荐通过类名）
print(Student.school)  # 第一中学
print(Student.count)   # 2

# 访问实例属性
print(s1.name)  # Alice
print(s2.name)  # Bob

# 修改类属性（影响所有实例）
Student.school = "实验中学"
print(s1.school)  # 实验中学
print(s2.school)  # 实验中学

# 修改实例属性（只影响当前实例）
s1.name = "Alice New"
print(s1.name)  # Alice New
print(s2.name)  # Bob（不变）
```

### 对比

| | 实例属性 | 类属性 |
|---|---|---|
| **定义位置** | `__init__`中用`self`定义 | 类中直接定义 |
| **归属** | 每个实例独立拥有 | 所有实例共享 |
| **访问方式** | `self.属性名` | `类名.属性名` |
| **修改影响** | 只影响当前实例 | 影响所有实例 |

## 三种方法类型

### 实例方法

```python
class Student:
    def __init__(self, name):
        self.name = name

    # 实例方法：操作实例数据
    def introduce(self):
        """实例方法：自我介绍"""
        print(f"我是{self.name}")
```

### 类方法

```python
class Student:
    count = 0  # 类属性

    def __init__(self, name):
        self.name = name
        Student.count += 1

    # 类方法：操作类数据（使用 @classmethod）
    @classmethod
    def get_count(cls):
        """类方法：获取学生总数"""
        return cls.count

    @classmethod
    def from_string(cls, data_str):
        """类方法：从字符串创建对象（工厂方法）"""
        name = data_str.split(",")[0]
        return cls(name)  # 相当于 Student(name)
```

### 静态方法

```python
class Student:
    # 静态方法：工具函数（使用 @staticmethod）
    @staticmethod
    def is_adult(age):
        """静态方法：判断是否成年"""
        return age >= 18
```

### 使用示例

```python
class Student:
    count = 0

    def __init__(self, name):
        self.name = name
        Student.count += 1

    def introduce(self):
        """实例方法"""
        print(f"我是{self.name}")

    @classmethod
    def get_count(cls):
        """类方法"""
        return cls.count

    @classmethod
    def from_string(cls, data_str):
        """类方法：工厂方法"""
        name = data_str.split(",")[0]
        return cls(name)

    @staticmethod
    def is_adult(age):
        """静态方法"""
        return age >= 18


# 使用
s1 = Student("Alice")
s2 = Student("Bob")

# 实例方法需要通过对象调用
s1.introduce()  # 我是 Alice

# 类方法可以通过类或实例调用
print(Student.get_count())  # 2
print(s1.get_count())       # 2

# 使用类方法创建对象
s3 = Student.from_string("Charlie,18")
print(s3.name)  # Charlie

# 静态方法不需要创建实例
print(Student.is_adult(20))  # True
```

---

## 三种方法对比

| 方法类型 | 装饰器 | 第一个参数 | 访问能力 | 使用场景 |
|----------|--------|-----------|----------|----------|
| **实例方法** | 无 | `self` | 实例属性 + 类属性 | 操作实例数据 |
| **类方法** | `@classmethod` | `cls` | 类属性 | 工厂方法、操作类数据 |
| **静态方法** | `@staticmethod` | 无 | 不能访问实例或类属性 | 工具函数 |
