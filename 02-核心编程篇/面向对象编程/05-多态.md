# 多态

本章讲解 Python 面向对象编程中的多态机制，包括多态基础、鸭子类型和抽象基类。

## 什么是多态

### 概念说明

**多态：同一个接口，不同的实现**

```python
class Animal:
    """动物基类"""
    def speak(self):
        raise NotImplementedError("子类必须实现此方法")

class Dog(Animal):
    def speak(self):
        return "汪汪汪"

class Cat(Animal):
    def speak(self):
        return "喵喵喵"

class Duck(Animal):
    def speak(self):
        return "嘎嘎嘎"


# ─────────────────────────────────────
# 多态应用：统一的接口处理不同类型的对象
# ─────────────────────────────────────

def make_animal_speak(animal):
    """让动物叫（不关心具体类型）"""
    print(f"{type(animal).__name__}: {animal.speak()}")

# 创建不同的动物
animals = [Dog(), Cat(), Duck()]

# 统一处理
for animal in animals:
    make_animal_speak(animal)

# 输出：
# Dog: 汪汪汪
# Cat: 喵喵喵
# Duck: 嘎嘎嘎
```

## 鸭子类型

### 概念说明

Python 的多态不依赖于继承，而是依赖于"鸭子类型"。

> "如果它走起来像鸭子，叫起来像鸭子，那它就是鸭子"

```python
class Dog:
    def speak(self):
        return "汪汪"

class Cat:
    def speak(self):
        return "喵喵"

class Robot:
    def speak(self):
        return "滴滴"


# 不需要继承同一个父类
def make_speak(thing):
    """只要对象有 speak() 方法就能用"""
    print(thing.speak())

# 都能正常工作
make_speak(Dog())    # 汪汪
make_speak(Cat())    # 喵喵
make_speak(Robot())  # 滴滴
```

### 鸭子类型的优势

│                                                             │

## 抽象基类（ABC）

### 概念说明

抽象基类定义接口规范，强制子类实现特定方法。

```python
from abc import ABC, abstractmethod

class Shape(ABC):
    """抽象基类：定义所有形状必须实现的方法"""

    @abstractmethod
    def get_area(self):
        """计算面积 - 子类必须实现"""
        pass

    @abstractmethod
    def get_perimeter(self):
        """计算周长 - 子类必须实现"""
        pass


# ❌ 不能实例化抽象基类
# shape = Shape()  # TypeError!


# ✅ 必须实现所有抽象方法
class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def get_area(self):
        return self.width * self.height

    def get_perimeter(self):
        return 2 * (self.width + self.height)


class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def get_area(self):
        return 3.14 * self.radius ** 2

    def get_perimeter(self):
        return 2 * 3.14 * self.radius


# 使用
shapes = [Rectangle(10, 5), Circle(7)]
for shape in shapes:
    area = shape.get_area()
    print(f"{type(shape).__name__}: 面积={area:.2f}")

# 输出：
# Rectangle: 面积=50.00
# Circle: 面积=153.86
```

### 抽象基类的作用

```
┌─────────────────────────────────────────────────────────────┐
│              抽象基类的作用                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   1. 定义接口规范                                           │
│      强制子类实现特定方法                                   │
│                                                             │
│   2. 防止实例化基类                                         │

│   3. 类型检查                                               │
│      isinstance(obj, Shape)                                 │
│                                                             │

```
