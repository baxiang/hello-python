# 继承

本章讲解 Python 面向对象编程中的继承机制，包括继承基础、super() 的使用和多重继承。

---

## 继承基础

### 概念说明

**继承：子类继承父类的特征和行为，可以：**
- 自动拥有父类的属性和方法
- 添加自己特有的属性和方法
- 重写父类的方法

```python
# 父类（基类）
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print("动物在说话")

    def move(self):
        print("动物在移动")


# 子类（派生类）
class Dog(Animal):
    def __init__(self, name, breed):
        # 调用父类的初始化方法
        super().__init__(name)
        self.breed = breed

    # 重写父类方法
    def speak(self):
        print(f"{self.name} 在叫：汪汪汪！")

    # 子类特有方法
    def fetch(self):
        print(f"{self.name} 正在捡球")


# 使用
dog = Dog("Buddy", "金毛")

# 继承的方法
dog.move()   # 动物在移动

# 重写的方法
dog.speak()  # Buddy 在叫：汪汪汪！

# 子类特有方法
dog.fetch()  # Buddy 正在捡球

# 子类特有属性
print(dog.breed)  # 金毛
```

## super() 的使用

### 调用父类方法

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print(f"{self.name} 发出声音")


class Dog(Animal):
    def __init__(self, name, breed):
        # 使用 super() 调用父类方法（推荐）
        super().__init__(name)
        self.breed = breed

    def speak(self):
        # 先调用父类方法，再添加自己的行为
        super().speak()
        print(f"{self.name} 汪汪叫")


class Cat(Animal):
    def __init__(self, name, color):
        super().__init__(name)
        self.color = color

    def speak(self):
        # 完全覆盖父类方法
        print(f"{self.name} 喵喵叫")


# 使用
dog = Dog("Buddy", "金毛")
dog.speak()
# 输出：
# Buddy 发出声音
# Buddy 汪汪叫

cat = Cat("咪咪", "白色")
cat.speak()
# 输出：
# 咪咪 喵喵叫
```

## 多重继承

### 概念说明

Python 支持多重继承，一个子类可以继承多个父类。

```python
class Flyable:
    """会飞的类"""
    def fly(self):
        print("在空中飞翔")


class Swimmable:
    """会游泳的类"""
    def swim(self):
        print("在水中游泳")


# 鸭子：继承多个父类
class Duck(Flyable, Swimmable):
    def __init__(self, name):
        self.name = name

    def introduce(self):
        print(f"我是{self.name}，我会：")
        self.fly()
        self.swim()


# 使用
duck = Duck("唐老鸭")
duck.introduce()
# 我是唐老鸭，我会：
# 在空中飞翔
# 在水中游泳
```

### MRO（方法解析顺序）

```python
class A:
    def method(self):
        print("A")

class B(A):
    def method(self):
        print("B")

class C(A):
    def method(self):
        print("C")

class D(B, C):
    pass

# 查看 MRO
print(D.__mro__)
# (<class 'D'>, <class 'B'>, <class 'C'>, <class 'A'>, <class 'object'>)

# 调用方法时按 MRO 顺序查找
d = D()
d.method()  # B
```
