# 面向对象基础

本章讲解 Python 面向对象编程的基本概念，包括类与对象的关系、类的定义和对象的创建。

---

## 面向过程 vs 面向对象

### 概念说明

```python
# ─────────────────────────────────────
# 面向过程：关注"怎么做"（步骤）
# ─────────────────────────────────────

def 洗菜():
    print("洗菜")

def 切菜():
    print("切菜")

def 炒菜():
    print("炒菜")

# 按步骤调用
洗菜()
切菜()
炒菜()


# ─────────────────────────────────────
# 面向对象：关注"谁来做"（对象）
# ─────────────────────────────────────

class 厨师:
    def 洗菜(self):
        print("洗菜")

    def 切菜(self):
        print("切菜")

    def 炒菜(self):
        print("炒菜")

# 创建对象并调用
chef = 厨师()
chef.洗菜()
chef.切菜()
chef.炒菜()
```

**核心理念：**
- **面向过程**：把问题分解为一系列步骤
- **面向对象**：找出问题域中的对象和它们的交互

---

## 类与对象的关系

### 概念说明

```
类（Class）：模板/蓝图
│
├── 例如："Dog"类定义了狗的共性
│   - 属性：name, age, breed
│   - 方法：bark(), run()
│
▼
对象（Object）：具体的实例
│
├── my_dog = Dog("Buddy", 3)   → 具体的狗：Buddy，3 岁
├── your_dog = Dog("Max", 5)   → 具体的狗：Max，5 岁
└── his_dog = Dog("Lucy", 2)   → 具体的狗：Lucy，2 岁
```

**类比理解：**
- 类 = 月饼模具，对象 = 用模具做出的月饼
- 类 = 建筑图纸，对象 = 按图纸建好的房子
- 类 = 菜谱，对象 = 按菜谱做好的菜

## 定义和创建类

### 基本语法

```python
class ClassName:
    """类的文档字符串 - 说明类的用途"""

    def __init__(self, param1, param2):
        """初始化方法（构造函数）"""
        # 实例属性（每个实例独立）
        self.attribute1 = param1
        self.attribute2 = param2

    def method1(self):
        """实例方法"""
        pass
```

### 完整示例

```python
class Dog:
    """表示狗的类"""

    # 类属性（所有实例共享）
    species = "Canis familiaris"

    def __init__(self, name, age, breed="中华田园犬"):
        """
        初始化方法
        参数:
            name: 名字
            age: 年龄
            breed: 品种（默认"中华田园犬"）
        """
        # 实例属性（每个实例独立）
        self.name = name
        self.age = age
        self.breed = breed

    def bark(self):
        """狗叫"""
        print(f"{self.name} 在叫：汪汪汪！")

    def run(self):
        """跑步"""
        print(f"{self.name} 正在奔跑！")

    def get_human_age(self):
        """计算人类年龄"""
        return self.age * 7


# ─────────────────────────────────────
# 创建和使用对象
# ─────────────────────────────────────

# 创建对象（实例化）
my_dog = Dog("Buddy", 3)
your_dog = Dog("Max", 5, "金毛")

# 访问属性
print(my_dog.name)      # Buddy
print(my_dog.species)   # Canis familiaris（类属性）

# 调用方法
my_dog.bark()             # Buddy 在叫：汪汪汪！
print(my_dog.get_human_age())  # 21
```

## 核心要点

| 概念 | 说明 | 示例 |
|------|------|------|
| **类** | 对象的模板 | `class Dog:` |
| **对象** | 类的具体实例 | `my_dog = Dog()` |
| **实例属性** | 每个实例独立 | `self.name` |
| **类属性** | 所有实例共享 | `species = "..."` |
| **`__init__`** | 初始化方法，创建对象时自动调用 | `def __init__(self):` |
| **self** | 指向当前实例对象 | 方法第一个参数 |
