# 封装

> **Python 版本要求：** 3.11+

本章讲解 Python 面向对象编程中的封装机制，包括私有属性、私有方法和 @property 装饰器。

---

## 为什么需要封装？

### 问题场景

假设你要开发一个银行账户系统：

**问题代码：**

```python
class BankAccount:
    def __init__(self, owner: str, balance: int) -> None:
        self.owner = owner
        self.balance = balance

account = BankAccount("张三", 1000)
# 问题：余额可以被随意修改！
account.balance = -100  # 余额变成负数！
account.balance = "abc"  # 类型错误！
account.balance = 9999999  # 恶意修改！
```

**问题：**
- 数据可以被随意访问和修改
- 无法添加验证逻辑
- 无法保护数据完整性
- 违反业务规则（余额不能为负）

**使用封装后：**

```python
class BankAccount:
    def __init__(self, owner: str, balance: int) -> None:
        self.owner: str = owner
        self.__balance: int = balance  # 私有属性
    
    def deposit(self, amount: int) -> bool:
        """存款（有验证）"""
        if amount > 0:
            self.__balance += amount
            return True
        print("存款金额必须大于 0")
        return False
    
    def withdraw(self, amount: int) -> bool:
        """取款（有验证）"""
        if 0 < amount <= self.__balance:
            self.__balance -= amount
            return True
        print("余额不足")
        return False
    
    def get_balance(self) -> int:
        """查询余额（只读）"""
        return self.__balance

account = BankAccount("张三", 1000)
account.deposit(500)  # 存入 500
account.withdraw(200)  # 取出 200
# account.__balance = -100  # 错误！无法直接访问
# account.balance = -100  # 错误！属性不存在
```

**优势：**
- 数据被保护在类内部
- 只能通过公共接口访问
- 可以添加验证逻辑
- 保证数据完整性

## 第一部分：什么是封装

### 1.1 概念动机

**封装** 是面向对象编程的核心特性之一，它将数据（属性）和操作数据的方法绑定在一起，并隐藏内部实现细节。

```
┌─────────────────────────────────────────────────────────────┐
│                      封装的作用                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   1. 数据隐藏                                               │
│      隐藏内部实现细节，只暴露必要的接口                     │
│                                                             │
│   2. 数据保护                                               │
│      防止外部直接修改内部数据，保证数据安全                 │
│                                                             │
│   3. 降低耦合                                               │
│      内部实现改变不影响外部调用                             │
│                                                             │
│   4. 提高可维护性                                           │
│      修改内部实现无需修改外部代码                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 最简示例

```python
class Temperature:
    """温度类：演示封装"""
    
    def __init__(self, celsius: float) -> None:
        # 私有属性：直接修改可能造成数据不一致
        self.__celsius: float = celsius
    
    # 公共接口：提供安全的访问方式
    def get_celsius(self) -> float:
        """获取摄氏温度"""
        return self.__celsius
    
    def get_fahrenheit(self) -> float:
        """获取华氏温度"""
        return self.__celsius * 9 / 5 + 32
    
    def set_celsius(self, value: float) -> bool:
        """设置摄氏温度（有验证）"""
        if -273.15 <= value:  # 绝对零度验证
            self.__celsius = value
            return True
        print("温度不能低于绝对零度")
        return False

# 使用
temp = Temperature(25)
print(temp.get_celsius())     # 25
print(temp.get_fahrenheit())  # 77.0

temp.set_celsius(30)
print(temp.get_celsius())     # 30

temp.set_celsius(-300)  # 温度不能低于绝对零度
# temp.__celsius = -300  # AttributeError!
```

### 1.3 详细讲解

**封装的三层境界：**

1. **公共（Public）：** 可以直接访问
2. **受保护（Protected）：** 约定不直接访问（单下划线）
3. **私有（Private）：** 名称改写，外部无法直接访问（双下划线）

```python
class Demo:
    """演示三种访问级别"""
    
    def __init__(self) -> None:
        # 公共属性：可以直接访问
        self.public_attr: str = "公共"
        
        # 受保护属性：约定不直接访问
        self._protected_attr: str = "受保护"
        
        # 私有属性：名称改写
        self.__private_attr: str = "私有"
    
    def public_method(self) -> str:
        """公共方法"""
        return self.__private_attr
    
    def _protected_method(self) -> str:
        """受保护方法"""
        return self.__private_attr
    
    def __private_method(self) -> str:
        """私有方法"""
        return self.__private_attr

demo = Demo()

# ✅ 公共属性可以直接访问
print(demo.public_attr)  # 公共

# ⚠️ 受保护属性可以访问，但约定不直接访问
print(demo._protected_attr)  # 受保护

# ❌ 私有属性无法直接访问
# print(demo.__private_attr)  # AttributeError!

# 可以通过名称改写访问（但不推荐）
print(demo._Demo__private_attr)  # 私有
```

**访问级别对比表：**

| 命名方式 | 访问级别 | 说明 | 使用场景 |
|----------|----------|------|----------|
| `name` | 公共 | 可以直接访问 | 公共接口 |
| `_name` | 受保护 | 约定不直接访问 | 子类使用 |
| `__name` | 私有 | 名称改写 | 完全隐藏 |

## 第二部分：私有属性和方法

### 2.1 概念动机

私有属性和方法通过双下划线前缀实现名称改写，防止外部直接访问。

### 2.2 最简示例

```python
class BankAccount:
    def __init__(self, owner: str, balance: int = 0) -> None:
        self.owner: str = owner
        self.__balance: int = balance  # 私有属性（双下划线）
        self.__transaction_count: int = 0

    def deposit(self, amount: int) -> bool:
        """存款 - 公共接口"""
        if amount > 0:
            self.__balance += amount
            self.__transaction_count += 1
            print(f"存入{amount}，余额：{self.__balance}")
            return True
        print("存款金额必须大于 0")
        return False

    def withdraw(self, amount: int) -> bool:
        """取款 - 公共接口"""
        if 0 < amount <= self.__balance:
            self.__balance -= amount
            self.__transaction_count += 1
            print(f"取出{amount}，余额：{self.__balance}")
            return True
        print("余额不足")
        return False

    def get_balance(self) -> int:
        """查询余额 - 公共接口"""
        return self.__balance

    def __validate_amount(self, amount: int) -> bool:
        """私有方法：验证金额"""
        return amount > 0


# 使用
account = BankAccount("张三", 1000)

# ✅ 通过公共接口访问
print(account.get_balance())  # 1000
account.deposit(500)
account.withdraw(200)

# ❌ 无法直接访问私有属性
# print(account.__balance)  # AttributeError!

# 可以通过名称改写访问（但不推荐）
print(account._BankAccount__balance)  # 1300
```

### 2.3 详细讲解

**名称改写机制：**

```python
class MyClass:
    def __init__(self) -> None:
        self.__private = "私有"
    
    def __private_method(self) -> None:
        pass

obj = MyClass()

# Python 自动改写名称
# __private → _MyClass__private
# __private_method → _MyClass__private_method

print(obj._MyClass__private)  # 私有（可以访问，但不推荐）
```

**为什么需要私有属性：**

```python
class Counter:

counter.increment()  # 2
```

### 2.4 渐进复杂化

完整的银行账户系统：

```python
from datetime import datetime
from typing import Optional

class BankAccount:
    """完整的银行账户类"""
    
    # 类属性
    MIN_BALANCE: int = 0
    MAX_TRANSACTION: int = 10000
    
    def __init__(self, owner: str, initial_balance: int = 0) -> None:
        self.owner: str = owner
        self.__balance: int = initial_balance
        self.__transactions: list[dict[str, str | int]] = []
        self.__is_locked: bool = False
        
        # 验证初始余额
        if initial_balance < self.MIN_BALANCE:
            raise ValueError("初始余额不能为负")
    
    def deposit(self, amount: int) -> bool:
        """存款"""
        if self.__is_locked:
            print("账户已锁定")
            return False
        
        if not self.__validate_transaction(amount):
            return False
        
        self.__balance += amount
        self.__record_transaction("deposit", amount)
        return True
    
    def withdraw(self, amount: int) -> bool:
        """取款"""
        if self.__is_locked:
            print("账户已锁定")
            return False
        
        if not self.__validate_transaction(amount):
            return False
        
        if amount > self.__balance:
            print("余额不足")
            return False
        
        self.__balance -= amount
        self.__record_transaction("withdraw", amount)
        return True
    
    def transfer(self, target: "BankAccount", amount: int) -> bool:
        """转账"""
        if self.withdraw(amount):
            if target.deposit(amount):
                self.__record_transaction("transfer", amount, target.owner)
                return True
            # 转账失败，退回金额
            self.__balance += amount
        return False
    
    def get_balance(self) -> int:
        """查询余额"""
        return self.__balance
    
    def get_transaction_history(self, limit: int = 10) -> list[dict]:
        """查询交易历史"""
        return self.__transactions[-limit:]
    
    def lock_account(self) -> None:
        """锁定账户"""
        self.__is_locked = True
    
    def unlock_account(self) -> None:
        """解锁账户"""
        self.__is_locked = False
    
    def __validate_transaction(self, amount: int) -> bool:
        """私有方法：验证交易"""
        if amount <= 0:
            print("交易金额必须大于 0")
            return False
        if amount > self.MAX_TRANSACTION:
            print(f"单笔交易不能超过 {self.MAX_TRANSACTION}")
            return False
        return True
    
    def __record_transaction(
        self,
        type: str,
        amount: int,
        target: Optional[str] = None
    ) -> None:
        """私有方法：记录交易"""
        transaction = {
            "type": type,
            "amount": amount,
            "time": datetime.now().strftime("%Y-%m-%d %H:%M:%S"),
            "balance_after": self.__balance,
        }
        if target:
            transaction["target"] = target
        self.__transactions.append(transaction)

# 使用
acc1 = BankAccount("张三", 1000)
acc2 = BankAccount("李四", 500)

acc1.deposit(500)
acc1.withdraw(200)
acc1.transfer(acc2, 300)

print(f"张三余额: {acc1.get_balance()}")  # 1000
print(f"李四余额: {acc2.get_balance()}")  # 800

print(acc1.get_transaction_history())
```

### 2.5 实际应用

用户管理系统：

```python
import hashlib
from typing import Optional

class User:
    """用户类：演示完整封装"""
    
    def __init__(self, username: str, email: str) -> None:
        self.username: str = username
        self.email: str = email
        
        # 私有属性
        self.__password_hash: Optional[str] = None
        self.__login_count: int = 0
        self.__last_login: Optional[str] = None
        self.__is_active: bool = True
    
    def set_password(self, password: str) -> bool:
        """设置密码"""
        if len(password) < 6:
            print("密码长度至少 6 位")
            return False
        self.__password_hash = self.__hash_password(password)
        return True
    
    def verify_password(self, password: str) -> bool:
        """验证密码"""
        if not self.__is_active:
            print("账户已禁用")
            return False
        if self.__password_hash is None:
            print("未设置密码")
            return False
        
        if self.__hash_password(password) == self.__password_hash:
            self.__record_login()
            return True
        return False
    
    def get_login_info(self) -> dict[str, int | str | None]:
        """获取登录信息"""
        return {
            "username": self.username,
            "login_count": self.__login_count,
            "last_login": self.__last_login,
            "is_active": self.__is_active,
        }
    
    def deactivate(self) -> None:
        """禁用账户"""
        self.__is_active = False
    
    def activate(self) -> None:
        """激活账户"""
        self.__is_active = True
    
    def __hash_password(self, password: str) -> str:
        """私有方法：密码哈希"""
        return hashlib.sha256(password.encode()).hexdigest()
    
    def __record_login(self) -> None:
        """私有方法：记录登录"""
        from datetime import datetime
        self.__login_count += 1
        self.__last_login = datetime.now().strftime("%Y-%m-%d %H:%M:%S")

# 使用
user = User("zhangsan", "zhang@example.com")
user.set_password("password123")

# 正确密码
print(user.verify_password("password123"))  # True
print(user.get_login_info())

# 错误密码
print(user.verify_password("wrong"))  # False

# 禁用账户
user.deactivate()
print(user.verify_password("password123"))  # 账户已禁用
```

## 第三部分：@property 装饰器

### 3.1 概念动机

`@property` 让方法像属性一样调用，可以添加验证逻辑，提供更优雅的访问控制。

```
┌─────────────────────────────────────────────────────────────┐
│              @property 的作用                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   1. 让方法像属性一样调用（不用加括号）                     │
│                                                             │
│   2. 可以添加验证逻辑                                       │
│      在 setter 中验证数据有效性                             │
│                                                             │
│   3. 可以创建只读属性                                       │
│      只定义 getter，不定义 setter                           │
│                                                             │
│   4. 计算属性                                               │
│      动态计算并返回值                                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 最简示例

```python
class Person:
    def __init__(self, age: int) -> None:
        self.__age: int = age  # 私有属性

    @property
    def age(self) -> int:
        """getter 方法：获取 age"""
        return self.__age

    @age.setter
    def age(self, value: int) -> None:
        """setter 方法：设置 age"""
        if 0 < value < 150:
            self.__age = value
        else:
            raise ValueError("年龄必须在 0-150 之间")

    @property
    def age_group(self) -> str:
        """只读属性：年龄段"""
        if self.__age < 18:
            return "未成年"
        elif self.__age < 60:
            return "成年人"
        else:
            return "老年人"


# 使用
p = Person(25)

# 调用 getter（像访问属性一样）
print(p.age)  # 25

# 调用 setter
p.age = 26  # 自动调用 setter 方法

# 调用只读属性
print(p.age_group)  # 成年人

# ❌ 无法设置只读属性
# p.age_group = "青年"  # AttributeError!

# ❌ setter 验证
# p.age = 200  # ValueError!
```

### 3.3 详细讲解

**@property 的完整用法：**

```python
class Circle:
    """圆类：演示 @property"""
    
    def __init__(self, radius: float) -> None:
        self.__radius: float = radius
    
    @property
    def radius(self) -> float:
        """getter：获取半径"""
        return self.__radius
    
    @radius.setter
    def radius(self, value: float) -> None:
        """setter：设置半径（有验证）"""
        if value > 0:
            self.__radius = value
        else:
            raise ValueError("半径必须大于 0")
    
    @property
    def area(self) -> float:
        """只读属性：计算面积"""
        return 3.14 * self.__radius ** 2
    
    @property
    def circumference(self) -> float:
        """只读属性：计算周长"""
        return 2 * 3.14 * self.__radius
    
    @property
    def diameter(self) -> float:
        """只读属性：直径"""
        return 2 * self.__radius


circle = Circle(5)
print(circle.radius)         # 5
print(circle.area)           # 78.5
print(circle.circumference)  # 31.4
print(circle.diameter)       # 10

# 设置 radius 会自动更新计算属性
circle.radius = 10
print(circle.area)           # 314.0
```

**对比 getter/setter 方法：**

```python
# 方式 1：getter/setter 方法（传统）

```

### 3.4 渐进复杂化

温度转换系统：

```python
class Temperature:
    """温度类：多单位转换"""
    
    def __init__(self, celsius: float = 0) -> None:
        self.__celsius: float = celsius
    
    @property
    def celsius(self) -> float:
        """摄氏温度"""
        return self.__celsius
    
    @celsius.setter
    def celsius(self, value: float) -> None:
        """设置摄氏温度"""
        if value >= -273.15:
            self.__celsius = value
        else:
            raise ValueError("温度不能低于绝对零度")
    
    @property
    def fahrenheit(self) -> float:
        """华氏温度（只读）"""
        return self.__celsius * 9 / 5 + 32
    
    @fahrenheit.setter
    def fahrenheit(self, value: float) -> None:
        """通过华氏温度设置"""
        self.celsius = (value - 32) * 5 / 9
    
    @property
    def kelvin(self) -> float:
        """开尔文温度（只读）"""
        return self.__celsius + 273.15
    
    @kelvin.setter
    def kelvin(self, value: float) -> None:
        """通过开尔文温度设置"""
        self.celsius = value - 273.15
    
    @property
    def is_freezing(self) -> bool:
        """是否结冰（只读计算属性）"""
        return self.__celsius <= 0
    
    @property
    def is_boiling(self) -> bool:
        """是否沸腾（只读计算属性）"""
        return self.__celsius >= 100
    
    @property
    def state(self) -> str:
        """物质状态（只读计算属性）"""
        if self.__celsius <= 0:
            return "固态"
        elif self.__celsius >= 100:
            return "气态"
        else:
            return "液态"

# 使用
temp = Temperature(25)

print(temp.celsius)     # 25
print(temp.fahrenheit)  # 77.0
print(temp.kelvin)      # 298.15

# 通过不同单位设置
temp.fahrenheit = 100
print(temp.celsius)     # 37.78

temp.kelvin = 300
print(temp.celsius)     # 26.85

# 计算属性
print(temp.is_freezing)  # False
print(temp.is_boiling)   # False
print(temp.state)        # 液态

temp.celsius = -10
print(temp.state)        # 固态
```

```python
class Product:

    @min_stock.setter
    def min_stock(self, value: int) -> None:
        """设置最低库存"""
        if value > 0:
            self.__min_stock = value
        else:
            raise ValueError("最低库存必须大于 0")
    
    @property
    def is_low_stock(self) -> bool:
        """库存是否不足（只读）"""
        return self.__stock < self.__min_stock
    
    @property
    def stock_status(self) -> str:
        """库存状态（只读）"""
        if self.__stock == 0:
            return "缺货"
        elif self.is_low_stock:
            return "库存不足"
        else:
            return "库存充足"
    
    @property
    def value(self) -> float:
        """库存价值（只读计算属性）"""
        return self.__price * self.__stock
    
    def add_stock(self, amount: int) -> bool:
        """增加库存"""
        new_stock = self.__stock + amount
        if new_stock <= self.__max_stock:
            self.__stock = new_stock
            return True
        print(f"库存不能超过 {self.__max_stock}")
        return False
    
    def remove_stock(self, amount: int) -> bool:
        """减少库存"""
        if amount <= self.__stock:
            self.__stock -= amount
            return True
        print("库存不足")
        return False

# 使用
```
