## 为什么需要闭包？

### 问题场景：请求计数器

继续 Web API 系统，需要统计每个路由的请求次数：

```python
# ❌ 方案一：全局变量（容易被篡改）
request_count: int = 0

def handle_request(req: Request) -> Response:
    global request_count
    request_count += 1
    return Response(200, {"count": request_count})

request_count = 100  # 外部意外修改！
handle_request(req)  # count = 101（错误）

# ✅ 方案二：闭包（私有变量，外部无法修改）
def create_counter() -> Callable[[], int]:
    count: int = 0  # 私有变量，外部无法访问
    def increment() -> int:
        nonlocal count
        count += 1
        return count
    return increment

counter = create_counter()
print(counter())  # 1
print(counter())  # 2
# 无法从外部修改 count！
```

**问题：如何让函数"记住"自己的状态？闭包是什么？**

## 生活类比：背包旅行

把闭包想象成背包旅行：

```
┌─────────────────────────────────────────────────────────────┐
│  闭包 = 背包旅行                                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  出发地 = 外层函数                                            │
│  • 准备行李（定义变量）                                       │
│  • 打包背包（定义内层函数）                                   │
│  • 旅行者出发（返回内层函数）                                 │
│                                                             │
│  背包 = 闭包                                                  │
│  • 装着出发地的物品（捕获的变量）                             │
│  • 随旅行者到处移动                                          │
│  • 外部无法打开（私有变量）                                   │
│                                                             │
│  旅行者 = 内层函数                                            │
│  • 背着背包旅行                                              │
│  • 可以随时使用背包里的物品                                   │
│  • 记住出发地的状态                                           │
│                                                             │
│  示例：                                                      │
│  ─────────────────────                                      │
│  def create_counter():      # 出发地                        │
│      count = 0              # 打包：装进背包                 │
│      def increment():       # 旅行者                        │
│          nonlocal count                                     │
│          count += 1         # 使用背包里的物品               │
│          return count                                        │
│      return increment        # 旅行者出发                   │
│                                                             │
│  counter = create_counter()  # 创建一个旅行者               │
│  counter()  # 旅行者使用背包，返回 1                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**一句话：**
> **闭包** = 内层函数 + 被捕获的外层变量（像背包带着物品）

## 闭包的构成

### 三个必要条件

```
┌─────────────────────────────────────────────────────────────┐
│  闭包的三个条件                                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 嵌套函数                                                 │
│  ─────────────────                                          │
│  def outer():                                               │
│      def inner():  # 内层函数                               │
│          ...                                                 │
│                                                             │
│  2. 内层函数引用外层变量                                      │
│  ─────────────────                                          │
│  def outer():                                               │
│      x = 10                                                  │
│      def inner():                                           │
│          return x  # 引用外层变量                           │
│                                                             │
│  3. 外层函数返回内层函数                                      │
│  ─────────────────                                          │
│  def outer():                                               │
│      x = 10                                                  │
│      def inner():                                           │
│          return x                                            │
│      return inner  # 返回内层函数                           │
│                                                             │
│  缺一不可：                                                  │
│  ─────────────────                                          │
│  ❌ 没有嵌套 → 不是闭包                                      │
│  ❌ 不引用外层变量 → 没捕获变量                              │
│  ❌ 不返回内层函数 → 无法形成闭包                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 最简示例

```python
from typing import Callable

def make_multiplier(factor: int) -> Callable[[int], int]:
    """创建乘法器"""
    def multiply(number: int) -> int:
        # 内层函数引用外层变量 factor
        return number * factor
    return multiply

# 创建不同的乘法器
double: Callable[[int], int] = make_multiplier(2)
triple: Callable[[int], int] = make_multiplier(3)

print(double(5))   # 10（factor=2 被"记住"）
print(triple(5))   # 15（factor=3 被"记住"）

# 验证闭包确实捕获了变量
print(double.__code__.co_freevars)  # ('factor',)
```

## 闭包执行时间线

```
┌─────────────────────────────────────────────────────────────┐
│  闭包创建与执行时间线                                          │
│                                                             │
│  代码：                                                       │
│  def make_multiplier(factor):                               │
│      def multiply(number):                                  │
│          return number * factor                             │
│      return multiply                                        │
│                                                             │
│  double = make_multiplier(2)                                │
│  double(5)                                                   │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 时间线                                               │   │
│  │                                                      │   │
│  │  1. 调用 make_multiplier(2)                         │   │
│  │     • 创建外层函数栈帧                               │   │
│  │     • factor = 2                                    │   │
│  │     • 定义内层函数 multiply                          │   │
│  │     • multiply 捕获 factor（记在 __closure__）      │   │
│  │        │                                             │   │
│  │        ↓                                             │   │
│  │  2. 返回 multiply                                    │   │
│  │     • 外层函数栈帧销毁                               │   │
│  │     • factor 本该消失，但被闭包捕获保留              │   │
│  │     • double = multiply                             │   │
│  │        │                                             │   │
│  │        ↓                                             │   │
│  │  3. 调用 double(5)                                   │   │
│  │     • 创建内层函数栈帧                               │   │
│  │     • number = 5                                     │   │
│  │     • 从 __closure__ 取出 factor = 2                │   │
│  │     • 返回 5 * 2 = 10                                │   │
│  │        │                                             │   │
│  │        ↓                                             │   │
│  │  4. 结果：10                                          │   │
│  │                                                      │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  ⚠️ 关键：外层函数结束后，factor 不消失                        │
│  ⚠️ 关键：闭包把 factor "打包带走"                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 贯穿实战：请求计数器

### 场景说明

Web API 系统需要为每个路由独立统计请求次数：

```python
from typing import Callable
from dataclasses import dataclass

@dataclass
class Request:
    path: str
    method: str
    data: dict

@dataclass
class Response:
    status: int
    data: dict

def create_route_counter(
    route_name: str, 
    start: int = 0
) -> Callable[[], dict[str, int | str]]:
    """创建路由请求计数器
    
    贯穿项目：Web API 请求处理系统
    核心功能：闭包管理私有计数状态
    """
    count: int = start  # 私有变量，外部无法访问
    
    def increment() -> dict[str, int | str]:
        """增加计数并返回当前值"""
        nonlocal count
        count += 1
        return {"route": route_name, "count": count}
    
    def reset() -> dict[str, int | str]:
        """重置计数"""
        nonlocal count
        count = 0
        return {"route": route_name, "count": count}
    
    # 返回包含多个操作的字典
    return lambda: increment()

# 为不同路由创建独立计数器
users_counter: Callable[[], dict] = create_route_counter("/users")
products_counter: Callable[[], dict] = create_route_counter("/products")

# 各路由独立计数
print(users_counter())    # {'route': '/users', 'count': 1}
print(users_counter())    # {'route': '/users', 'count': 2}
print(products_counter()) # {'route': '/products', 'count': 1}
print(users_counter())    # {'route': '/users', 'count': 3}

# 计数器状态互不影响
```

### 带状态的闭包（使用 nonlocal）

```python
from typing import Callable

def create_counter_with_ops(start: int = 0) -> dict[str, Callable]:
    """创建带多种操作的计数器"""
    count: int = start
    
    def increment() -> int:
        nonlocal count
        count += 1
        return count
    
    def decrement() -> int:
        nonlocal count
        count -= 1
        return count
    
    def get_value() -> int:
        return count
    
    def reset() -> int:
        nonlocal count
        count = 0
        return count
    
    return {
        "increment": increment,
        "decrement": decrement,
        "get": get_value,
        "reset": reset
    }

# 使用
counter: dict[str, Callable] = create_counter_with_ops(10)
print(counter["increment"]())  # 11
print(counter["increment"]())  # 12
print(counter["decrement"]())  # 11
print(counter["get"]())        # 11
print(counter["reset"]())      # 0
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `nonlocal count` | 声明使用外层变量 | Python 默认创建新局部变量，需要声明 |
| `count = start` | 外层函数定义变量 | 被闭包捕获的私有变量 |
| `return {...}` | 返回多个操作函数 | 提供多种操作方式 |
| `create_counter()` | 创建闭包实例 | 每次调用创建独立状态 |

## nonlocal 关键字

### 为什么需要 nonlocal

```python
# ❌ 错误：没有 nonlocal，创建新局部变量
def counter_wrong() -> Callable[[], int]:
    count: int = 0
    def increment() -> int:
        count += 1  # 创建新局部变量，报错！
        return count
    return increment

# UnboundLocalError: local variable 'count' referenced before assignment

# ✅ 正确：使用 nonlocal 声明
def counter_correct() -> Callable[[], int]:
    count: int = 0
    def increment() -> int:
        nonlocal count  # 声明使用外层的 count
        count += 1      # 修改外层变量
        return count
    return increment
```

### nonlocal vs global

```
┌─────────────────────────────────────────────────────────────┐
│  nonlocal vs global                                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  nonlocal                                                    │
│  ─────────────────                                          │
│  • 引用外层函数的变量（E层）                                  │
│  • 用于闭包修改捕获的变量                                    │
│  • 作用范围：当前函数的外层                                   │
│                                                             │
│  global                                                      │
│  ─────────────────                                          │
│  • 引用模块级别的变量（G层）                                  │
│  • 用于修改全局变量                                          │
│  • 作用范围：整个模块                                         │
│                                                             │
│  选择：                                                      │
│  ─────────────────                                          │
│  • 闭包内修改外层变量 → nonlocal                             │
│  • 修改全局变量 → global                                     │
│  • 只读取不修改 → 无需声明                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 推荐做法

| 做法 | 原因 | 示例 |
|------|------|------|
| **用闭包替代全局变量** | 避免状态污染 | `create_counter()` |
| **用 nonlocal 修改外层变量** | 必需语法 | `nonlocal count` |
| **返回多个操作函数** | 灵活操作 | `{"get": get, "reset": reset}` |
| **验证闭包变量** | 确保捕获 | `func.__code__.co_freevars` |

5. 避免全局污染、状态隔离、更安全
