## 为什么需要进阶装饰器？

### 问题场景：可配置的装饰器

继续 Web API 系统，需要：
- 权限验证：不同角色有不同权限
- 失败重试：可配置重试次数和延迟

```python
# ❌ 基础装饰器：无法配置参数
@auth_required  # 固定权限检查，无法区分角色
def delete_user(req):
    pass

# ✅ 带参数装饰器：可以配置
@auth_required(role="admin")  # 只允许 admin
def delete_user(req):
    pass

@retry(times=3, delay=1.0)  # 重试3次，间隔1秒
def fetch_api(url):
    pass
```

**问题：如何让装饰器接收参数？**

## 生活类比：定制包装盒

把带参数装饰器想象成定制包装盒：

```
┌─────────────────────────────────────────────────────────────┐
│  带参数装饰器 = 定制包装盒                                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  基础装饰器 = 标准包装纸                                       │
│  • 只有一种样式                                              │
│  • 不能调整大小/颜色                                         │
│                                                             │
│  带参数装饰器 = 定制包装盒                                     │
│  • 可以指定尺寸（参数 times=3）                              │
│  • 可以指定颜色（参数 role="admin"）                         │
│  • 按需求定制                                               │
│                                                             │
│  三层结构：                                                  │
│  ─────────────────                                          │
│  最外层：接收参数（包装盒工厂）                               │
│  def retry(times):                                          │
│      # 根据参数定制                                          │
│                                                             │
│  中间层：接收函数（包装盒）                                   │
│      def decorator(func):                                   │
│          # 保存函数                                          │
│                                                             │
│  最内层：包装逻辑（盒子里的功能）                             │
│          def wrapper(*args):                                │
│              for _ in range(times):                         │
│                  func()                                     │
│                                                             │
│  示例：                                                      │
│  @retry(times=3)                                            │
│  def fetch_api():                                           │
│      pass                                                   │
│                                                             │
│  等价于：                                                    │
│  fetch_api = retry(times=3)(fetch_api)                      │
│                                                             │
│  1. repeat(times=3) 返回 decorator                          │
│  2. decorator(fetch_api) 返回 wrapper                       │
│  3. fetch_api = wrapper                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**一句话：**
> **带参数装饰器** = 三层函数（参数层 → 装饰器层 → 包装层）

## 带参数装饰器结构

### 三层嵌套

```
┌─────────────────────────────────────────────────────────────┐
│  带参数装饰器三层结构                                          │
│                                                             │
│  def repeat(times):          # 第一层：接收参数              │
│      def decorator(func):    # 第二层：接收函数              │
│          def wrapper(*args): # 第三层：包装逻辑              │
│              for _ in range(times):                         │
│                  result = func(*args)                       │
│              return result                                  │
│          return wrapper                                     │
│      return decorator                                       │
│                                                             │
│  @repeat(times=3)                                           │
│  def greet(name):                                           │
│      print(f"Hello, {name}")                                │
│                                                             │
│  执行过程：                                                  │
│  ─────────────────                                          │
│  1. repeat(times=3) → 返回 decorator                        │
│  2. decorator(greet) → 返回 wrapper                         │
│  3. greet = wrapper                                         │
│                                                             │
│  调用 greet("Alice") 时：                                    │
│  ─────────────────                                          │
│  执行 wrapper("Alice")                                      │
│  wrapper 内部循环3次调用原 greet                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 最简示例：@repeat

```python
from typing import Callable, TypeVar
from functools import wraps

T = TypeVar('T')

def repeat(times: int) -> Callable[[Callable[..., T]], Callable[..., T]]:
    """重复执行装饰器
    
    参数：
        times: 重复执行的次数
    """
    def decorator(func: Callable[..., T]) -> Callable[..., T]:
        @wraps(func)
        def wrapper(*args: tuple, **kwargs: dict) -> T:
            result: T | None = None
            for _ in range(times):
                result = func(*args, **kwargs)
            return result  # type: ignore
        return wrapper
    return decorator

@repeat(times=3)
def greet(name: str) -> None:
    print(f"Hello, {name}!")

greet("Alice")
# Hello, Alice!
# Hello, Alice!
# Hello, Alice!
```

## 贯穿实战：权限验证装饰器

### @auth_required 装饰器

```python
from typing import Callable, TypeVar, Any
from functools import wraps
from dataclasses import dataclass

T = TypeVar('T')

@dataclass
class Request:
    path: str
    method: str
    user: dict[str, Any]  # 包含用户信息

@dataclass
class Response:
    status: int
    data: dict[str, Any]

def auth_required(role: str = "user") -> Callable[[Callable[..., Response]], Callable[..., Response]]:
    """权限验证装饰器
    
    贬穿项目：Web API 请求处理系统
    功能：验证用户是否有指定角色权限
    
    参数：
        role: 要求的角色（如 "admin", "user"）
    """
    def decorator(func: Callable[..., Response]) -> Callable[..., Response]:
        @wraps(func)
        def wrapper(req: Request, *args: tuple, **kwargs: dict) -> Response:
            # 检查用户角色
            user_role: str = req.user.get("role", "guest")
            
            if user_role != role:
                return Response(403, {
                    "error": "权限不足",
                    "required": role,
                    "actual": user_role
                })
            
            # 有权限，执行原函数
            return func(req, *args, **kwargs)
        return wrapper
    return decorator

# 使用
@auth_required(role="admin")
def delete_user(req: Request) -> Response:
    return Response(200, {"deleted": True})

@auth_required(role="user")
def view_profile(req: Request) -> Response:
    return Response(200, {"profile": req.user})

# 测试
admin_req = Request("/users", "DELETE", {"id": 1, "role": "admin"})
guest_req = Request("/users", "DELETE", {"id": 1, "role": "guest"})

print(delete_user(admin_req))  # Response(200, {'deleted': True})
print(delete_user(guest_req))  # Response(403, {'error': '权限不足'})
```

## 贯穿实战：重试装饰器

### @retry 装饰器

```python
import time
from typing import Callable, TypeVar, Any
from functools import wraps

T = TypeVar('T')

def retry(
    times: int = 3,
    delay: float = 1.0,
    exceptions: tuple[type, ...] = (Exception,)
) -> Callable[[Callable[..., T]], Callable[..., T]]:
    """重试装饰器
    
    贯穿项目：Web API 请求处理系统
    功能：失败时自动重试
    
    参数：
        times: 重试次数
        delay: 重试间隔（秒）
        exceptions: 捕获的异常类型
    """
    def decorator(func: Callable[..., T]) -> Callable[..., T]:
        @wraps(func)
        def wrapper(*args: tuple, **kwargs: dict) -> T:
            last_exception: Exception | None = None
            
            for attempt in range(times + 1):  # 包括首次尝试
                try:
                    return func(*args, **kwargs)
                except exceptions as e:
                    last_exception = e
                    if attempt < times:
                        print(f"[RETRY] {func.__name__} 第 {attempt + 1} 次失败，{delay}s 后重试")
                        time.sleep(delay)
            
            # 所有重试都失败
            raise last_exception  # type: ignore
        return wrapper
    return decorator

# 使用：模拟不稳定 API
call_count: int = 0

@retry(times=3, delay=0.5, exceptions=(ConnectionError,))
def fetch_api(url: str) -> dict[str, Any]:
    """模拟不稳定 API"""
    global call_count
    call_count += 1
    
    if call_count < 3:  # 前2次失败
        raise ConnectionError("连接失败")
    
    return {"status": "success", "url": url}

# 测试
try:
    result = fetch_api("https://api.example.com")
    print(result)  # {'status': 'success', 'url': '...'}
except ConnectionError as e:
    print(f"最终失败: {e}")
```

## 类装饰器

### 用类实现装饰器

```python
from typing import Callable, Any
from functools import wraps

class CountCalls:
    """计数装饰器（类实现）"""
    
    def __init__(self, func: Callable[..., Any]) -> None:
        self.func: Callable[..., Any] = func
        self.count: int = 0
        wraps(func)(self)  # 保留元信息
    
    def __call__(self, *args: tuple, **kwargs: dict) -> Any:
        self.count += 1
        print(f"{self.func.__name__} 已调用 {self.count} 次")
        return self.func(*args, **kwargs)

@CountCalls
def say_hello(name: str) -> str:
    return f"Hello, {name}!"

print(say_hello("Alice"))  # say_hello 已调用 1 次
print(say_hello("Bob"))    # say_hello 已调用 2 次
print(f"总调用次数: {say_hello.count}")  # 2
```

## 推荐做法

| 做法 | 原因 | 示例 |
|------|------|------|
| **三层函数实现带参数** | 标准模式 | `def retry(times): def decorator(func): def wrapper():` |
| **参数设默认值** | 支持无参数调用 | `times: int = 3` |
| **用 functools.wraps** | 保留元信息 | `@wraps(func)` |
| **类装饰器管理状态** | 状态可访问 | `@CountCalls` |
| **捕获特定异常** | 精确控制 | `exceptions=(ConnectionError,)` |
