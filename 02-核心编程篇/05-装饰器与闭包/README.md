## 核心概念预览

### 函数是一等公民

```python
def route(path: str, handler: Callable) -> None:
    handler(request)  # 函数作为参数传递
```

### 闭包

```python
def create_counter():
    count = 0  # 私有变量
    def increment():
        nonlocal count
        count += 1
        return count
    return increment
```

### 装饰器

```python
@timer
@logger
def handle_request(request):
    return process(request)
```
