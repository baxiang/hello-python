#### 面向对象四大特性

```
┌─────────────────────────────────────────┐
│         面向对象四大特性                │
├─────────────────────────────────────────┤
│                                         │
│  1️⃣ 抽象（Abstraction）                 │
│     ────────────────                    │
│     提取共同特征，忽略无关细节          │
│                                         │
│     例如："动物"类提取所有动物的共同点   │
│     - 有名字                            │
│     - 会吃东西                          │
│     - 会发出声音                        │
│                                         │
│     class 动物：                        │
│         def __init__(self, name): ...   │
│         def eat(self): ...              │
│         def speak(self): ...            │
│                                         │
│  ─────────────────────────────────      │
│                                         │
│  2️⃣ 封装（Encapsulation）               │
│     ─────────────────                   │
│     隐藏内部实现，只暴露接口            │
│                                         │
│     例如：手机内部电路被封装            │
│     - 外部看不到内部电路（私有）        │
│     - 只能通过按钮操作（公共接口）      │
│                                         │
│     class 手机：                        │
│         def __init__(self):             │
│             self.__电路 = ...  # 私有    │
│         def 按电源键 (self): ...  # 公共 │
│                                         │
│  ─────────────────────────────────      │
│                                         │
│  3️⃣ 继承（Inheritance）                 │
│     ───────────────                     │
│     子类继承父类的特征和行为            │
│                                         │
│     例如："狗"继承"动物"的特征           │
│     - 自动拥有 name 属性和 eat() 方法     │
│     - 可以添加自己特有的属性（breed）   │
│     - 可以重写方法（speak）             │
│                                         │
│     class 狗 (动物):                    │
│         def speak(self):                │
│             return "汪汪"               │
│                                         │
│  ─────────────────────────────────      │
│                                         │
│  4️⃣ 多态（Polymorphism）                │
│     ────────────────                    │
│     同一接口，不同实现                  │
│                                         │
│     例如：animal.speak()                │
│     - 狗 → "汪汪"                       │
│     - 猫 → "喵喵"                       │
│     - 鸟 → "啾啾"                       │
│                                         │
│     def make_speak(animal):             │
│         print(animal.speak())  # 多态   │
│                                         │
└─────────────────────────────────────────┘
```

#### 完整示例：Dog 类

```python
class Dog:
    """
    表示狗的类

    属性:
        name: 狗的名字
        age: 狗的年龄（岁）
        breed: 狗的品种

    方法:
        bark(): 狗叫
        run(): 跑步
        get_human_age(): 转换为人类年龄
    """

    # 类属性 - 所有狗共享
    species = "Canis familiaris"  # 生物学名
    max_age = 20  # 最大寿命

    def __init__(self, name, age, breed="中华田园犬"):
        """
        初始化方法

        参数:
            name: 名字
            age: 年龄
            breed: 品种（默认"中华田园犬"）
        """
        # 实例属性 - 每只狗独立
        self.name = name
        self.age = age
        self.breed = breed
        print(f"🐶 {name} 来到了这个世界！")

    def bark(self):
        """狗叫"""
        print(f"{self.name} 在叫：汪汪汪！")

    def run(self):
        """跑步"""
        print(f"{self.name} 正在快乐地奔跑！")

    def get_human_age(self):
        """计算人类年龄"""
        return self.age * 7

    def introduce(self):
        """自我介绍"""
        print(f"我是 {self.name}，今年{self.age}岁，"
              f"品种是{self.breed}，"
              f"人类年龄{self.get_human_age()}岁")


# ─────────────────────────────────────
# 创建和使用对象
# ─────────────────────────────────────

# 创建对象（实例化）
my_dog = Dog("Buddy", 3)
# 输出：🐶 Buddy 来到了这个世界！

your_dog = Dog("Max", 5, "金毛")
# 输出：🐶 Max 来到了这个世界！

# 访问属性
print(my_dog.name)      # Buddy
print(my_dog.age)       # 3
print(my_dog.breed)     # 中华田园犬
print(my_dog.species)   # Canis familiaris（类属性）

print(your_dog.name)    # Max
print(your_dog.breed)   # 金毛

# 调用方法
my_dog.bark()             # Buddy 在叫：汪汪汪！
your_dog.run()            # Max 正在快乐地奔跑！
print(my_dog.get_human_age())  # 21

# 调用自定义方法
my_dog.introduce()
# 我是 Buddy，今年 3 岁，品种是中华田园犬，人类年龄 21 岁

# 动态添加属性（Python 特性，但不推荐）
my_dog.color = "棕色"
print(my_dog.color)  # 棕色
```

### 9.3 `__init__` 方法详解

#### 构造函数的作用

```python
class Person:
    def __init__(self, name, age=0):
        """
        初始化方法（构造函数）

        在创建对象时自动调用，用于：
        1. 初始化属性
        2. 执行必要的设置
        3. 验证参数
        """
        self.name = name
        self.age = age
        print(f"创建了 Person 对象：{self.name}")


# 创建对象时自动调用 __init__

        self.age = age
        self.grade = grade
        self.school = school
```

#### 带参数验证的构造函数

```python
class BankAccount:
    def __init__(self, owner, initial_balance=0):
        """
        银行账户构造函数

        参数:
            owner: 账户持有人姓名
            initial_balance: 初始余额

        异常:
            ValueError: 当姓名无效或余额为负数时

        # 验证姓名
        if not owner or not owner.strip():
            raise ValueError("姓名不能为空")

        # 验证初始余额
        if initial_balance < 0:
            raise ValueError("初始余额不能为负数")

        self.owner = owner
        self.balance = initial_balance
        print(f"✅ 账户创建成功：{owner}, 余额：{initial_balance}")


# 正常使用
account1 = BankAccount("张三", 1000)
```

### 9.4 self 参数详解

#### self 的作用

```python
class MyClass:
    def __init__(self, value):
        # self 指向当前实例对象
        self.value = value

    def show(self):
        # self 让我们能访问实例的属性
        print(f"值是：{self.value}")

    def compare(self, other):
        # self 可以与其他对象比较
        print(f"我的值：{self.value}, 你的值：{other.value}")


obj1 = MyClass(10)
obj2 = MyClass(20)

obj1.show()  # 值是：10（self = obj1）
obj2.show()  # 值是：20（self = obj2）

obj1.compare(obj2)  # 我的值：10, 你的值：20
```

#### self 的常见误区

```python
class MyClass:
    # ❌ 错误：忘记 self
    def wrong_method():
        print("没有 self 参数")

    # ✅ 正确
    def correct_method(self):
        print("有 self 参数")

    # ❌ 错误：self 不是第一个参数
    def wrong_order(x, self):
        pass

    # ✅ 正确：self 必须是第一个
    def correct_order(self, x):
        pass


obj = MyClass()
# obj.wrong_method()  # ❌ TypeError!
obj.correct_method()  # ✅ 正确
```

**重要规则：**

```
┌─────────────────────────────────────────┐
│              self 规则                   │
├─────────────────────────────────────────┤
│                                         │
│  1. self 必须是实例方法的第一个参数      │
│  2. self 名称可以自定义，但约定用 self  │
│  3. 调用方法时不需要传递 self            │
│     Python 自动传递                      │
│  4. 类方法和静态方法不需要 self          │
│  5. __init__ 中 self 指向正在创建的对象  │
│                                         │
│  为什么需要 self？                       │
│  ─────────────────                      │
│  Python 需要知道方法属于哪个对象         │
│  这与 JavaScript 的 this 类似            │
│  但 Python 要求显式声明 self             │
│                                         │
└─────────────────────────────────────────┘
```

### 9.5 实例属性 vs 类属性

#### 两种属性的区别

```python
class Student:
    # 类属性（所有实例共享）
    school = "第一中学"
    count = 0  # 用于统计学生数量

    def __init__(self, name, age):
        # 实例属性（每个实例独立）
        self.name = name
        self.age = age

        # 修改类属性
        Student.count += 1


# 创建实例
s1 = Student("Alice", 15)
s2 = Student("Bob", 16)
s3 = Student("Charlie", 17)

# ─────────────────────────────────────
# 访问属性
# ─────────────────────────────────────

# 访问类属性（通过类名）- 推荐
print(Student.school)  # 第一中学
print(Student.count)   # 3

# 访问类属性（通过实例）- 不推荐，容易混淆
print(s1.school)  # 第一中学
print(s2.school)  # 第一中学

# 访问实例属性
print(s1.name)  # Alice
print(s2.name)  # Bob

# ─────────────────────────────────────
# 修改属性
# ─────────────────────────────────────

# 修改类属性（影响所有实例）
Student.school = "实验中学"
print(s1.school)  # 实验中学
print(s2.school)  # 实验中学

# 修改实例属性（只影响当前实例）
s1.name = "Alice New"
print(s1.name)  # Alice New
print(s2.name)  # Bob（不变）

# ⚠️ 陷阱：通过实例"修改"类属性
s1.school = "私立学校"  # 这会给 s1 添加实例属性 school
print(s1.school)        # 私立学校（实例属性）
print(s2.school)        # 实验中学（类属性）
print(Student.school)   # 实验中学（类属性）

# 查看属性位置
print(s1.__dict__)   # {'name': 'Alice New', 'age': 15, 'school': '私立学校'}
print(Student.__dict__)  # 包含 'school', 'count' 等
```

### 9.6 三种方法类型

#### 实例方法（Instance Method）

```python
class Circle:
    def __init__(self, radius):
        self.radius = radius  # 实例属性

    def get_area(self):
        """实例方法：计算面积"""
        return 3.14159 * self.radius ** 2

    def get_circumference(self):
        """实例方法：计算周长"""
        return 2 * 3.14159 * self.radius

    def scale(self, factor):
        """实例方法：缩放"""
        self.radius *= factor
        print(f"半径缩放为原来的 {factor} 倍")


# 使用
c = Circle(5)
print(c.get_area())           # 78.54
print(c.get_circumference())  # 31.42
c.scale(2)
print(c.get_area())           # 314.16

# 实例方法需要通过对象调用
# Circle.get_area()  # ❌ TypeError: 缺少 self
```

#### 类方法（Class Method）

```python
class Student:
    count = 0  # 类属性

    def __init__(self, name):
        self.name = name
        Student.count += 1

    @classmethod
    def get_count(cls):
        """类方法：获取学生总数"""
        # cls 指向 Student 类
        return cls.count

    @classmethod
    def from_string(cls, data_str):
        """类方法：从字符串创建对象（工厂方法）"""
        # cls 指向 Student 类，可以用来创建对象
        parts = data_str.split(",")
        name = parts[0]
        return cls(name)  # 相当于 Student(name)


# 使用
print(Student.get_count())  # 0

s1 = Student("Alice")
s2 = Student("Bob")

print(Student.get_count())  # 2

# 使用类方法创建对象
s3 = Student.from_string("Charlie,18,一年级")
print(s3.name)  # Charlie
```

**类方法的典型应用：工厂方法**

```python
from datetime import date

class Date:
    def __init__(self, year, month, day):
        self.year = year
        self.month = month
        self.day = day

    @classmethod
    def from_string(cls, date_str):
        """从 'YYYY-MM-DD' 格式创建日期"""
        year, month, day = map(int, date_str.split("-"))
        return cls(year, month, day)

    @classmethod
    def today(cls):
        """获取今天的日期（工厂方法）"""
        today = date.today()
        return cls(today.year, today.month, today.day)

    @classmethod
    def from_timestamp(cls, timestamp):
        """从时间戳创建日期"""
        dt = date.fromtimestamp(timestamp)
        return cls(dt.year, dt.month, dt.day)

    def __str__(self):
        return f"{self.year}-{self.month:02d}-{self.day:02d}"


# 使用
d1 = Date.from_string("2024-01-15")
print(d1)  # 2024-01-15

d2 = Date.today()
print(f"今天：{d2}")

d3 = Date.from_timestamp(1704067200)  # 2024-01-01
print(d3)
```

```python
        """静态方法：加法"""

    def is_even(n):

        if not args:
            return 0
        return sum(args) / len(args)

    @staticmethod
    def clamp(value, min_val, max_val):
        """静态方法：将值限制在范围内"""
        return max(min_val, min(value, max_val))
```

#### 私有属性和方法

```python
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self.__balance = balance  # 私有属性（双下划线）
        self.__transaction_count = 0  # 私有属性

    def deposit(self, amount):
        """存款 - 公共接口"""
        if amount > 0:
            self.__balance += amount
            self.__transaction_count += 1
            print(f"存入{amount}，余额：{self.__balance}")
            return True
        print("存款金额必须大于 0")
        return False

    def withdraw(self, amount):
        """取款 - 公共接口"""
        if 0 < amount <= self.__balance:
            self.__balance -= amount
            self.__transaction_count += 1
            print(f"取出{amount}，余额：{self.__balance}")
            return True
        print("余额不足")
        return False

    def get_balance(self):
        """查询余额 - 公共接口"""
        return self.__balance

    def __validate_amount(self, amount):
        """私有方法：验证金额"""
        return amount > 0

    def __log_transaction(self, action, amount):
        """私有方法：记录交易"""
        print(f"[日志] {action}: {amount}")


# 使用
account = BankAccount("张三", 1000)

# ✅ 通过公共接口访问
print(account.get_balance())  # 1000
account.deposit(500)
account.withdraw(200)

# ❌ 无法直接访问私有属性
# print(account.__balance)  # AttributeError!
```

```python
        self._width = width
        self._height = height

    def width(self):
        return self._width

    @width.setter
    def width(self, value):

        self._width = value

    @property
    def height(self):
        return self._height

    @height.setter
    def height(self, value):
        if value <= 0:
            raise ValueError("高度必须大于 0")
        self._height = value

    def area(self):

    @property
    def perimeter(self):
        """计算周长 - 只读属性"""
        return 2 * (self._width + self._height)

    @property
    def is_square(self):
        """判断是否为正方形 - 只读属性"""
        return self._width == self._height
```

```python
# 图形类示例
class Shape:
    """图形基类"""
    def get_area(self):
        raise NotImplementedError("子类必须实现")

    def get_perimeter(self):
        raise NotImplementedError("子类必须实现")

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def get_area(self):
        return self.width * self.height

    def get_perimeter(self):
        return 2 * (self.width + self.height)

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def get_area(self):
        return 3.14159 * self.radius ** 2

    def get_perimeter(self):
        return 2 * 3.14159 * self.radius

class Triangle(Shape):
    def __init__(self, a, b, c):
        self.a, self.b, self.c = a, b, c

    def get_area(self):
        # 海伦公式
        s = (self.a + self.b + self.c) / 2
        return (s * (s-self.a) * (s-self.b) * (s-self.c)) ** 0.5

    def get_perimeter(self):
        return self.a + self.b + self.c


# ─────────────────────────────────────
# 多态应用：统一处理不同图形
# ─────────────────────────────────────

shapes = [
    Rectangle(10, 5),
    Circle(7),
    Triangle(3, 4, 5)
]

total_area = 0
total_perimeter = 0

for shape in shapes:
    area = shape.get_area()
    perimeter = shape.get_perimeter()
    total_area += area
    total_perimeter += perimeter
    print(f"{type(shape).__name__}: 面积={area:.2f}, 周长={perimeter:.2f}")

print(f"\n总面积：{total_area:.2f}")
print(f"总周长：{total_perimeter:.2f}")
```

### 9.10 组合（Composition）

#### 组合 vs 继承

```
┌─────────────────────────────────────────┐
│         组合 vs 继承                    │
├─────────────────────────────────────────┤
│                                         │
│  继承（Inheritance）                    │
│  ─────────────────                      │
│  关系："is-a"（是一个）                 │
│  例：狗 is-a 动物                       │
│  代码：class Dog(Animal): ...           │
│                                         │
│  组合（Composition）                    │
│  ─────────────────                      │
│  关系："has-a"（有一个）                │
│  例：汽车 has-a 发动机                  │
│  代码：                                 │
│       class Car:                        │
│           def __init__(self):           │
│               self.engine = Engine()    │
│                                         │
│  设计原则：                             │
│  ────────────                           │
│  "组合优于继承"                         │
│  组合更灵活，耦合度更低                 │
│                                         │
└─────────────────────────────────────────┘
```

#### 组合示例

```python
class Engine:
    """发动机类"""
    def __init__(self, horsepower):
        self.horsepower = horsepower

    def start(self):
        return "发动机启动"

    def stop(self):
        return "发动机停止"


class Wheel:
    """轮子类"""
    def __init__(self, size):
        self.size = size

    def rotate(self):
        return "轮子转动"


class Car:
    """汽车类 - 使用组合"""
    def __init__(self, brand, engine_hp, wheel_size):
        self.brand = brand
        # 组合：汽车"有"发动机和轮子
        self.engine = Engine(engine_hp)
        self.wheels = [Wheel(wheel_size) for _ in range(4)]

    def drive(self):
        """开车"""
        result = [f"{self.brand} 汽车："]
        result.append(self.engine.start())
        for i, wheel in enumerate(self.wheels):
            result.append(f"  轮子{i+1}: {wheel.rotate()}")
        return "\n".join(result)

    def park(self):
```

```python
class Vector:
    """向量类"""
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other):
        """加法：v1 + v2"""
        return Vector(self.x + other.x, self.y + other.y)

    def __sub__(self, other):
        """减法：v1 - v2"""
        return Vector(self.x - other.x, self.y - other.y)

    def __mul__(self, other):
        """乘法：v * n（数乘）"""
        if isinstance(other, (int, float)):
            return Vector(self.x * other, self.y * other)
        return NotImplemented

    def __truediv__(self, other):
        """除法：v / n"""
        if isinstance(other, (int, float)):
            return Vector(self.x / other, self.y / other)
        return NotImplemented

    def __eq__(self, other):
        """相等比较：v1 == v2"""
        return self.x == other.x and self.y == other.y

    def __lt__(self, other):
        """小于比较：v1 < v2（比较模长）"""
        return (self.x ** 2 + self.y ** 2) < (other.x ** 2 + other.y ** 2)

    def __abs__(self):
        """绝对值：模长"""
        return (self.x ** 2 + self.y ** 2) ** 0.5

    def __str__(self):
        return f"Vector({self.x}, {self.y})"
```

```python
from abc import ABC, abstractmethod

class Shape(ABC):
    """抽象基类：定义所有形状必须实现的方法"""

    @abstractmethod
    def get_area(self):
        """计算面积 - 子类必须实现"""
        pass

    @abstractmethod
    def get_perimeter(self):
        """计算周长 - 子类必须实现"""
        pass

    def describe(self):
        """普通方法 - 子类可以继承"""
        return f"这是一个{self.__class__.__name__}"


# ❌ 不能实例化抽象基类
# shape = Shape()  # TypeError!

# ✅ 必须实现所有抽象方法
class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def get_area(self):
        return self.width * self.height

    def get_perimeter(self):
        return 2 * (self.width + self.height)


class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def get_area(self):
        return 3.14159 * self.radius ** 2

    def get_perimeter(self):
        return 2 * 3.14159 * self.radius


# 使用
shapes = [Rectangle(10, 5), Circle(7)]
for shape in shapes:
    print(f"{shape.describe()}: 面积={shape.get_area():.2f}")

# 输出：
# 这是一个 Rectangle: 面积=50.00
# 这是一个 Circle: 面积=153.94
```

```python
class Product:
    """商品类"""
    def __init__(self, product_id, name, price, stock):
        self.product_id = product_id
        self.name = name
        self.price = price
        self.stock = stock

    def add_stock(self, quantity):
        """入库"""
        if quantity <= 0:
            raise ValueError("入库数量必须大于 0")
        self.stock += quantity

    def remove_stock(self, quantity):
        """出库"""
        if quantity <= 0:
            raise ValueError("出库数量必须大于 0")
        if quantity > self.stock:
            raise ValueError("库存不足")
        self.stock -= quantity

    def __str__(self):
        return f"{self.name} - ¥{self.price} (库存：{self.stock})"


class ShoppingCart:
    """购物车类"""
    def __init__(self):
        self.items = {}  # {product_id: (product, quantity)}

    def add_item(self, product, quantity=1):
        """添加商品"""
        if quantity <= 0:
            raise ValueError("数量必须大于 0")

        if product.product_id in self.items:
            existing_product, existing_qty = self.items[product.product_id]
            self.items[product.product_id] = (existing_product, existing_qty + quantity)
        else:
            self.items[product.product_id] = (product, quantity)

    def remove_item(self, product_id):
        """移除商品"""
        if product_id in self.items:
            del self.items[product_id]

    def update_quantity(self, product_id, quantity):
        """更新数量"""
        if quantity <= 0:
            self.remove_item(product_id)
        else:
            product, _ = self.items[product_id]
            self.items[product_id] = (product, quantity)

    def get_total(self):
        """计算总价"""
        total = 0
        for product, quantity in self.items.values():
            total += product.price * quantity
        return total

    def show_cart(self):
        """显示购物车"""
        if not self.items:
            print("购物车为空")
            return

        print("=== 购物车 ===")
        for product_id, (product, quantity) in self.items.items():
            subtotal = product.price * quantity
            print(f"{product.name} x {quantity} = ¥{subtotal:.2f}")
        print(f"总计：¥{self.get_total():.2f}")


class Order:

        self.cart = cart
        self.customer_name = customer_name

        # 扣减库存
        for product, quantity in self.cart.items.values():
            product.remove_stock(quantity)

        self.status = "completed"
        return self

    def __str__(self):
        return f"订单{self.order_id} - {self.customer_name} - {self.status}"
```

```
│  • 常量：全大写 + 下划线                 │
│                                         │
│  ✅ 使用继承的条件：                    │
│  ─────────────────                      │
│  • 确实是"is-a"关系                     │
│  • 避免过度继承（建议不超过 3 层）      │
│  • 考虑使用组合代替继承                 │
│                                         │
│  ✅ 文档字符串：                        │
│  ────────────                           │
│  • 类和方法都要写文档字符串            │
│  • 说明参数、返回值、功能               │
│                                         │
│  ❌ 避免：                              │
│  ────────────                           │
│  • 上帝类（一个类做太多事）            │
│  • 滥用继承（能用组合就用组合）        │
│  • 直接访问私有属性                      │
│  • 在__init__之外调用实例方法           │
│                                         │
└─────────────────────────────────────────┘
```

### 常见错误

```python
# ❌ 错误 1：忘记 self
class MyClass:
    def wrong_method():
        pass

# ✅ 正确
class MyClass:
    def correct_method(self):
        pass


# ❌ 错误 2：使用可变对象作为类属性
class Dog:
    tricks = []  # 所有狗共享同一个列表！

    def add_trick(self, trick):
        self.tricks.append(trick)

# ✅ 正确
class Dog:
    def __init__(self):
        self.tricks = []  # 每只狗独立的列表


# ❌ 错误 3：在__init__ 中返回
class Person:
    def __init__(self, name):
        self.name = name
        return self  # __init__ 不能有返回值

# ✅ 正确
class Person:
    def __init__(self, name):
        self.name = name


# ❌ 错误 4：过度使用继承
class Bird:
    def fly(self):
        pass

class Penguin(Bird):  # 企鹅不会飞！
    def fly(self):
        raise NotImplementedError

# ✅ 正确：使用组合或更细的抽象
class Bird:
    pass

class Flyable:
    def fly(self):
        pass

class Penguin(Bird):  # 企鹅是鸟，但不会飞
    pass

class Sparrow(Bird, Flyable):  # 麻雀是鸟且会飞
    pass
```
