## 一、面向对象基础

### 9.1 什么是面向对象

#### 面向过程 vs 面向对象

```python
# ─────────────────────────────────────
# 面向过程：关注"怎么做"（步骤）
# ─────────────────────────────────────

def 洗菜 ():
    print("洗菜")

def 切菜 ():
    print("切菜")

def 炒菜 ():
    print("炒菜")

# 按步骤调用
洗菜 ()
切菜 ()
炒菜 ()


# ─────────────────────────────────────
# 面向对象：关注"谁来做"（对象）
# ─────────────────────────────────────

class 厨师:
    def 洗菜 (self):
        print("洗菜")

    def 切菜 (self):
        print("切菜")

    def 炒菜 (self):
        print("炒菜")

# 创建对象并调用
chef = 厨师 ()
chef.洗菜 ()
chef.切菜 ()
chef.炒菜 ()
```

**核心理念：**
- **面向过程**：把问题分解为一系列步骤
- **面向对象**：找出问题域中的对象和它们的交互

### 9.2 类与对象的关系

```
类（Class）：模板/蓝图
│
├── 例如："Dog"类定义了狗的共性
│   - 属性：name, age, breed
│   - 方法：bark(), run()
│
▼
对象（Object）：具体的实例
│
├── my_dog = Dog("Buddy", 3)   → 具体的狗：Buddy，3 岁
├── your_dog = Dog("Max", 5)   → 具体的狗：Max，5 岁
└── his_dog = Dog("Lucy", 2)   → 具体的狗：Lucy，2 岁
```

**类比理解：**
- 类 = 月饼模具，对象 = 用模具做出的月饼
- 类 = 建筑图纸，对象 = 按图纸建好的房子
- 类 = 菜谱，对象 = 按菜谱做好的菜

### 9.3 定义和创建类

#### 基本语法

```python
class ClassName:
    """类的文档字符串 - 说明类的用途"""

    def __init__(self, param1, param2):
        """初始化方法（构造函数）"""
        # 实例属性（每个实例独立）
        self.attribute1 = param1
        self.attribute2 = param2

    def method1(self):
        """实例方法"""
        pass
```

#### 完整示例

```python
class Dog:
    """表示狗的类"""

    # 类属性（所有实例共享）
    species = "Canis familiaris"

    def __init__(self, name, age, breed="中华田园犬"):
        """
        初始化方法
        参数:
            name: 名字
            age: 年龄
            breed: 品种（默认"中华田园犬"）
        """
        # 实例属性（每个实例独立）
        self.name = name
        self.age = age
        self.breed = breed

    def bark(self):
        """狗叫"""
        print(f"{self.name} 在叫：汪汪汪！")

    def run(self):
        """跑步"""
        print(f"{self.name} 正在奔跑！")

    def get_human_age(self):
        """计算人类年龄"""
        return self.age * 7


# ─────────────────────────────────────
# 创建和使用对象
# ─────────────────────────────────────

# 创建对象（实例化）
my_dog = Dog("Buddy", 3)
your_dog = Dog("Max", 5, "金毛")

# 访问属性
print(my_dog.name)      # Buddy
print(my_dog.species)   # Canis familiaris（类属性）

# 调用方法
my_dog.bark()             # Buddy 在叫：汪汪汪！
print(my_dog.get_human_age())  # 21
```

#### 核心要点

| 概念 | 说明 | 示例 |
|------|------|------|
| **类** | 对象的模板 | `class Dog:` |
| **对象** | 类的具体实例 | `my_dog = Dog()` |
| **实例属性** | 每个实例独立 | `self.name` |
| **类属性** | 所有实例共享 | `species = "..."` |
| **`__init__`** | 初始化方法，创建对象时自动调用 | `def __init__(self):` |
| **self** | 指向当前实例对象 | 方法第一个参数 |

## 二、属性和方法

### 9.4 实例属性 vs 类属性

```python
class Student:
    # 类属性（所有实例共享）
    school = "第一中学"
    count = 0

    def __init__(self, name, age):
        # 实例属性（每个实例独立）
        self.name = name
        self.age = age
        Student.count += 1  # 修改类属性


# 创建实例
s1 = Student("Alice", 15)
s2 = Student("Bob", 16)

# 访问类属性（推荐通过类名）
print(Student.school)  # 第一中学
print(Student.count)   # 2

# 访问实例属性
print(s1.name)  # Alice
print(s2.name)  # Bob

# 修改类属性（影响所有实例）
Student.school = "实验中学"
print(s1.school)  # 实验中学
print(s2.school)  # 实验中学

# 修改实例属性（只影响当前实例）
s1.name = "Alice New"
print(s1.name)  # Alice New
print(s2.name)  # Bob（不变）
```

**重要区别：**

| | 实例属性 | 类属性 |
|---|---|---|
| **定义位置** | `__init__`中用`self`定义 | 类中直接定义 |
| **归属** | 每个实例独立拥有 | 所有实例共享 |
| **访问方式** | `self.属性名` | `类名。属性名` 或 `实例。属性名` |
| **修改影响** | 只影响当前实例 | 影响所有实例 |

### 9.5 三种方法类型

```python
class Student:
    count = 0  # 类属性

    def __init__(self, name):
        self.name = name
        Student.count += 1

    # ─────────────────────────────────
    # 1. 实例方法：操作实例数据
    # ─────────────────────────────────
    def introduce(self):
        """实例方法：自我介绍"""
        print(f"我是{self.name}")

    # ─────────────────────────────────
    # 2. 类方法：操作类数据（使用 @classmethod）
    # ─────────────────────────────────
    @classmethod
    def get_count(cls):
        """类方法：获取学生总数"""
        return cls.count

    @classmethod
    def from_string(cls, data_str):
        """类方法：从字符串创建对象（工厂方法）"""
        name = data_str.split(",")[0]
        return cls(name)  # 相当于 Student(name)

    # ─────────────────────────────────
    # 3. 静态方法：工具函数（使用 @staticmethod）
    # ─────────────────────────────────
    @staticmethod
    def is_adult(age):
        """静态方法：判断是否成年"""
        return age >= 18


# 使用
print(Student.get_count())  # 0

s1 = Student("Alice")
s2 = Student("Bob")

print(Student.get_count())  # 2

# 实例方法需要通过对象调用
s1.introduce()  # 我是 Alice

# 类方法可以通过类或实例调用
print(Student.get_count())  # 2
print(s1.get_count())       # 2

# 使用类方法创建对象
s3 = Student.from_string("Charlie,18")
print(s3.name)  # Charlie

# 静态方法不需要创建实例
print(Student.is_adult(20))  # True
```

**三种方法对比：**

| 方法类型 | 装饰器 | 第一个参数 | 访问能力 | 使用场景 |
|----------|--------|-----------|----------|----------|
| **实例方法** | 无 | `self` | 实例属性 + 类属性 | 操作实例数据 |
| **类方法** | `@classmethod` | `cls` | 类属性 | 工厂方法、操作类数据 |
| **静态方法** | `@staticmethod` | 无 | 不能访问实例或类属性 | 工具函数 |

## 三、继承

### 9.6 继承基础

**继承：子类继承父类的特征和行为，可以：**
- 自动拥有父类的属性和方法
- 添加自己特有的属性和方法
- 重写父类的方法

```python
# 父类（基类）
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print("动物在说话")

    def move(self):
        print("动物在移动")


# 子类（派生类）
class Dog(Animal):
    def __init__(self, name, breed):
        # 调用父类的初始化方法
        super().__init__(name)
        self.breed = breed

    # 重写父类方法
    def speak(self):
        print(f"{self.name} 在叫：汪汪汪！")

    # 子类特有方法
    def fetch(self):
        print(f"{self.name} 正在捡球")


# 使用
dog = Dog("Buddy", "金毛")

# 继承的方法
dog.move()   # 动物在移动

# 重写的方法
dog.speak()  # Buddy 在叫：汪汪汪！

# 子类特有方法
dog.fetch()  # Buddy 正在捡球

# 子类特有属性
print(dog.breed)  # 金毛
```

### 9.7 super() 的使用

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print(f"{self.name} 发出声音")


class Dog(Animal):
    def __init__(self, name, breed):
        # 使用 super() 调用父类方法（推荐）
        super().__init__(name)
        self.breed = breed

    def speak(self):
        # 先调用父类方法，再添加自己的行为
        super().speak()
        print(f"{self.name} 汪汪叫")


class Cat(Animal):
    def __init__(self, name, color):
        super().__init__(name)
        self.color = color

    def speak(self):
        # 完全覆盖父类方法
        print(f"{self.name} 喵喵叫")


# 使用
dog = Dog("Buddy", "金毛")
dog.speak()
# 输出：
# Buddy 发出声音
# Buddy 汪汪叫

cat = Cat("咪咪", "白色")
cat.speak()
# 输出：
# 咪咪 喵喵叫
```

## 四、封装

### 9.9 私有属性和方法

```python
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self.__balance = balance  # 私有属性（双下划线）
        self.__transaction_count = 0

    def deposit(self, amount):
        """存款 - 公共接口"""
        if amount > 0:
            self.__balance += amount
            self.__transaction_count += 1
            print(f"存入{amount}，余额：{self.__balance}")
            return True
        print("存款金额必须大于 0")
        return False

    def withdraw(self, amount):
        """取款 - 公共接口"""
        if 0 < amount <= self.__balance:
            self.__balance -= amount
            self.__transaction_count += 1
            print(f"取出{amount}，余额：{self.__balance}")
            return True
        print("余额不足")
        return False

    def get_balance(self):
        """查询余额 - 公共接口"""
        return self.__balance

    def __validate_amount(self, amount):
        """私有方法：验证金额"""
        return amount > 0


# 使用
account = BankAccount("张三", 1000)

# ✅ 通过公共接口访问
print(account.get_balance())  # 1000
account.deposit(500)
account.withdraw(200)

# ❌ 无法直接访问私有属性
# print(account.__balance)  # AttributeError!

# 可以通过名称改写访问（但不推荐）
print(account._BankAccount__balance)  # 1300
```

**封装的核心：**
- **隐藏内部实现**：私有属性（`__xxx`）外部不能直接访问
- **暴露公共接口**：通过公共方法（`deposit`、`withdraw`）操作数据
- **好处**：数据安全、代码易维护、降低耦合

### 9.10 @property 装饰器

```python
class Person:
    def __init__(self, age):
        self.__age = age  # 私有属性

    @property
    def age(self):
        """getter 方法：获取 age"""
        return self.__age

    @age.setter
    def age(self, value):
        """setter 方法：设置 age"""
        if 0 < value < 150:
            self.__age = value
        else:
            raise ValueError("年龄必须在 0-150 之间")

    @property
    def age_group(self):
        """只读属性：年龄段"""
        if self.__age < 18:
            return "未成年"
        elif self.__age < 60:
            return "成年人"
        else:
            return "老年人"


# 使用
p = Person(25)

# 调用 getter（像访问属性一样）
print(p.age)  # 25

# 调用 setter
p.age = 26  # 自动调用 setter 方法

# 调用只读属性
print(p.age_group)  # 成年人

# ❌ 无法设置只读属性
# p.age_group = "青年"  # AttributeError!
```

**@property 的作用：**
- 让方法像属性一样调用（不用加括号）
- 可以添加验证逻辑
- 可以创建只读属性

### 9.13 抽象基类（ABC）

```python
from abc import ABC, abstractmethod

class Shape(ABC):
    """抽象基类：定义所有形状必须实现的方法"""

    @abstractmethod
    def get_area(self):
        """计算面积 - 子类必须实现"""
        pass

    @abstractmethod
    def get_perimeter(self):
        """计算周长 - 子类必须实现"""
        pass


# ❌ 不能实例化抽象基类
# shape = Shape()  # TypeError!


# ✅ 必须实现所有抽象方法
class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def get_area(self):
        return self.width * self.height

    def get_perimeter(self):
        return 2 * (self.width + self.height)


class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def get_area(self):
        return 3.14 * self.radius ** 2

    def get_perimeter(self):
        return 2 * 3.14 * self.radius


# 使用
shapes = [Rectangle(10, 5), Circle(7)]
for shape in shapes:
    area = shape.get_area()
    print(f"{type(shape).__name__}: 面积={area:.2f}")

# 输出：
# Rectangle: 面积=50.00
# Circle: 面积=153.86
```

**抽象基类的作用：**
- 定义接口规范：强制子类实现特定方法
- 防止实例化基类：只能通过具体子类创建对象
- 类型检查：`isinstance(obj, Shape)`

## 六、组合 vs 继承

### 9.14 组合（Composition）

**继承与组合的区别：**

| | 继承 | 组合 |
|---|---|---|
| **关系** | "is-a"（是一个） | "has-a"（有一个） |
| **示例** | 狗 is-a 动物 | 汽车 has-a 发动机 |
| **代码** | `class Dog(Animal):` | `self.engine = Engine()` |
| **耦合度** | 高 | 低 |

**设计原则：组合优于继承**

```python
class Engine:
    """发动机类"""
    def start(self):
        return "发动机启动"

class Wheel:
    """轮子类"""
    def rotate(self):
        return "轮子转动"

class Car:
    """汽车类 - 使用组合"""
    def __init__(self, brand):
        self.brand = brand
        # 组合：汽车"有"发动机和轮子
        self.engine = Engine()
        self.wheels = [Wheel() for _ in range(4)]

    def drive(self):
        result = [f"{self.brand} 汽车："]
        result.append(self.engine.start())
        for wheel in self.wheels:
            result.append(wheel.rotate())
        return "\n".join(result)


# 使用
car = Car("特斯拉")
print(car.drive())
# 特斯拉 汽车：
# 发动机启动
# 轮子转动
# 轮子转动
# 轮子转动
# 轮子转动
```

### 练习 1：图书馆管理系统

```python
class Book:
    """图书类"""
    def __init__(self, isbn, title, author):
        self.isbn = isbn
        self.title = title
        self.author = author
        self.is_borrowed = False

    def borrow(self):
        """借书"""
        if self.is_borrowed:
            return False, "书籍已被借出"
        self.is_borrowed = True
        return True, "借阅成功"

    def return_book(self):
        """还书"""
        if not self.is_borrowed:
            return False, "书籍未被借出"
        self.is_borrowed = False
        return True, "还书成功"

    def __str__(self):
        status = "已借出" if self.is_borrowed else "可借阅"
        return f"《{self.title}》by {self.author} [{status}]"


class Library:
    """图书馆类"""
    def __init__(self, name):
        self.name = name
        self.books = {}

    def add_book(self, book):
        """添加图书"""
        self.books[book.isbn] = book
        print(f"添加了：《{book.title}》")

    def search_by_title(self, keyword):
        """按书名搜索"""
        results = [book for book in self.books.values()
                   if keyword.lower() in book.title.lower()]
        return results

    def show_all_books(self):
        """显示所有图书"""
        print(f"\n=== {self.name} 的藏书 ===")
        for book in self.books.values():
            print(book)


# 测试
library = Library("市立图书馆")

book1 = Book("001", "Python 编程", "张三")
book2 = Book("002", "JavaScript 高级", "李四")

library.add_book(book1)
library.add_book(book2)

# 借书
success, msg = book1.borrow()
print(msg)  # 借阅成功

# 搜索
results = library.search_by_title("Python")
print(f"搜索结果：{results}")

# 显示
library.show_all_books()
```

### 练习 2：员工管理系统

```python
from abc import ABC, abstractmethod

class Employee(ABC):
    """员工抽象基类"""
    def __init__(self, emp_id, name, department):
        self.emp_id = emp_id
        self.name = name
        self.department = department

    @abstractmethod
    def calculate_salary(self):
        """计算工资 - 子类必须实现"""
        pass

    def get_info(self):
        return f"{self.emp_id}: {self.name} - {self.department}"


class Manager(Employee):
    """经理类"""
    def __init__(self, emp_id, name, department, base_salary, bonus):
        super().__init__(emp_id, name, department)
        self.base_salary = base_salary
        self.bonus = bonus

    def calculate_salary(self):
        return self.base_salary + self.bonus


class Developer(Employee):
    """开发工程师类"""
    def __init__(self, emp_id, name, department, base_salary, hours, rate):
        super().__init__(emp_id, name, department)
        self.base_salary = base_salary
        self.hours = hours
        self.rate = rate

    def calculate_salary(self):
        return self.base_salary + (self.hours * self.rate)


# 测试
employees = [
    Manager("M001", "张经理", "技术部", 20000, 5000),
    Developer("D001", "李开发", "技术部", 15000, 20, 200),
]

print("=== 员工工资单 ===")
for emp in employees:
    salary = emp.calculate_salary()
    print(f"{emp.get_info()}: {salary}元")
```

## 本章总结

### 知识点回顾

| 概念 | 说明 | 关键语法 |
|------|------|----------|
| **类** | 对象的模板 | `class ClassName:` |
| **对象** | 类的实例 | `obj = ClassName()` |
| **构造函数** | 初始化对象 | `__init__(self, ...)` |
| **self** | 指向实例本身 | 方法第一个参数 |
| **实例属性** | 每个实例独立 | `self.attr = value` |
| **类属性** | 所有实例共享 | `class_attr = value` |
| **实例方法** | 操作实例 | `def method(self):` |
| **类方法** | 操作类 | `@classmethod` |
| **静态方法** | 工具函数 | `@staticmethod` |
| **继承** | 子类继承父类 | `class Child(Parent):` |
| **封装** | 隐藏内部 | `__private`, `@property` |
| **多态** | 同接口不同实现 | 重写父类方法 |
| **组合** | has-a 关系 | `self.component = Component()` |
| **抽象基类** | 定义接口 | `ABC`, `@abstractmethod` |

### 面向对象四大特性

```
┌─────────────────────────────────────────┐
│       面向对象四大特性                  │
├─────────────────────────────────────────┤
│                                         │
│  1️⃣ 抽象（Abstraction）                 │
│     提取共同特征，忽略无关细节          │
│     例如：Animal 类提取所有动物的共性     │
│                                         │
│  2️⃣ 封装（Encapsulation）               │
│     隐藏内部实现，只暴露接口            │
│     例如：__balance 私有属性             │
│                                         │
│  3️⃣ 继承（Inheritance）                 │
│     子类继承父类的特征和行为            │
│     例如：Dog(Animal)                   │
│                                         │
│  4️⃣ 多态（Polymorphism）                │
│     同一接口，不同实现                  │
│     例如：animal.speak() 不同动物叫不同  │
│                                         │
└─────────────────────────────────────────┘
```

### 最佳实践

```
✅ 推荐：
────────
• 类名用大驼峰：MyClassName
• 方法和属性用小写 + 下划线：my_method
• 私有属性用双下划线：__private
• 优先使用组合而非继承
• 写文档字符串说明类的用途

❌ 避免：
────────
• 上帝类（一个类做太多事）
• 过度继承（建议不超过 3 层）
• 直接访问私有属性
• 使用可变对象作为类属性
```
