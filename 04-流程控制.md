## 第一部分：流程控制基础

### 4.1 什么是流程控制

#### 概念说明

**流程控制（Control Flow）** 是指决定程序中代码执行顺序的机制。通过流程控制，我们可以让程序根据不同的条件执行不同的代码，或者重复执行某段代码。

```
┌─────────────────────────────────────────┐
│          程序执行流程示意图              │
├─────────────────────────────────────────┤
│                                         │
│  顺序执行（默认）：                      │
│  ┌─────┐                               │
│  │ 第 1 行 │                               │
│  └──┬──┘                               │
│     ▼                                  │
│  ┌─────┐                               │
│  │ 第 2 行 │                               │
│  └──┬──┘                               │
│     ▼                                  │
│  ┌─────┐                               │
│  │ 第 3 行 │                               │
│  └─────┘                               │
│                                         │
└─────────────────────────────────────────┘
```

#### 流程控制的类型

```
┌─────────────────────────────────────────┐
│         Python 流程控制类型              │
├─────────────────────────────────────────┤
│                                         │
│  1. 条件分支（选择结构）                │
│     • if 语句                           │
│     • if-else 语句                      │
│     • if-elif-else 语句                 │
│                                         │
│  2. 循环结构（重复执行）                │
│     • for 循环                          │
│     • while 循环                        │
│                                         │
│  3. 循环控制                            │
│     • break（跳出循环）                 │
│     • continue（跳过本次）              │
│                                         │
└─────────────────────────────────────────┘
```

## 第二部分：条件语句详解

### 4.2 if 语句基础

#### 基本语法

```
┌─────────────────────────────────────────┐
│            if 语句语法                   │
├─────────────────────────────────────────┤
│                                         │
│   if 条件:                               │
│       条件成立时执行的代码             │
│                                         │
│   语法说明：                            │
│   ─────────────                         │
│   • if      → 关键字，表示条件判断     │
│   • 条件    → 返回 True 或 False 的表达式 │
│   • :       → 冒号，不能省略！          │
│   • 缩进    → 代码块必须缩进（4 空格）   │
│                                         │
└─────────────────────────────────────────┘
```

#### 什么是"条件"

```python
# 条件是一个返回 True 或 False 的表达式

# 比较表达式作为条件
5 > 3      # True
10 == 10   # True
age < 18   # 可能是 True 或 False

# 布尔变量作为条件
is_raining = True
if is_raining:  # 条件为 True
    print("带伞")

# 逻辑运算作为条件
age = 20
has_money = True
if age >= 18 and has_money:  # 两个条件都满足
    print("可以购买")
```

#### 示例：基本的 if 语句

```python
# 示例 1：判断是否为正数

number = 5

if number > 0:
    print(f"{number} 是正数")

print("程序结束")
```

**代码详解：**
```
if number > 0:
│   │      │  │
│   │      │  └── 冒号（必须！）
│   │      └── 条件：number 大于 0 吗？
│   └── 比较运算符
└── if 关键字

    print(...)
    └── 缩进的代码（条件成立时才执行）
```

**执行流程图：**
```
        ┌─────────────┐
        │   开始      │
        └──────┬──────┘
               │
               ▼
        ┌─────────────┐
        │ number = 5  │
        └──────┬──────┘
               │
               ▼
        ┌─────────────┐
        │ number > 0? │  条件判断
        └──────┬──────┘
               │
       ┌───────┴───────┐
       │               │
      Yes             No
       │               │
       ▼               ▼
  ┌─────────┐     ┌─────────┐
  │ 打印    │     │ 跳过    │
  │ "是正数"│     │ if 块   │
  └────┬────┘     └────┬────┘
       │               │
       └───────┬───────┘
               │
               ▼
        ┌─────────────┐
        │ 程序结束    │
        └─────────────┘
```

### 4.3 if-else 语句

#### 基本语法

```
┌─────────────────────────────────────────┐
│          if-else 语句语法                │
├─────────────────────────────────────────┤
│                                         │
│   if 条件:                               │
│       条件成立时执行的代码             │
│   else:                                  │
│       条件不成立时执行的代码           │
│                                         │
│   语法说明：                            │
│   ─────────────                         │
│   • else    → 关键字，不需要条件       │
│   • :       → else 后面也要冒号          │
│   • 缩进    → if 和 else 的代码块都要缩进 │
│                                         │
└─────────────────────────────────────────┘
```

#### 示例：if-else 语句

```python
# 示例 1：判断奇偶数

number = 7

if number % 2 == 0:
    print(f"{number} 是偶数")
else:
    print(f"{number} 是奇数")

# 输出：7 是奇数
```

**代码执行过程：**
```
number = 7

if number % 2 == 0:
│   │
│   └── 7 % 2 = 1
│       1 == 0 ？→ False
│
└── 条件为 False，执行 else 块

else:
    print(f"{number} 是奇数")
    └── 输出：7 是奇数
```

#### 实际应用示例

```python
# 示例 2：用户登录验证

# 预设的正确密码
correct_password = "123456"

# 获取用户输入的密码
user_password = input("请输入密码：")

# 验证密码
if user_password == correct_password:
    print("✓ 密码正确，登录成功！")
else:
    print("✗ 密码错误，登录失败！")
```

**注意：**
```python
# ❌ 错误：使用 = 而不是 ==
if user_password = correct_password:  # 语法错误！
    pass

# ✅ 正确：使用 == 进行比较
if user_password == correct_password:
    pass
```

### 4.4 if-elif-else 语句

#### 基本语法

```
┌─────────────────────────────────────────┐
│       if-elif-else 语句语法              │
├─────────────────────────────────────────┤
│                                         │
│   if 条件 1:                             │
│       条件 1 成立时执行                  │
│   elif 条件 2:                           │
│       条件 2 成立时执行                  │
│   elif 条件 3:                           │
│       条件 3 成立时执行                  │
│   ...                                    │
│   else:                                  │
│       以上条件都不成立时执行            │
│                                         │
│   语法说明：                            │
│   ─────────────                         │
│   • elif    → "else if" 的缩写           │
│   • 可以有多个 elif                     │
│   • else 是可选的，最多只能有一个       │
│                                         │
└─────────────────────────────────────────┘
```

#### 执行流程

```
        ┌─────────────┐
        │  检查条件 1   │
        └──────┬──────┘
               │
       ┌───────┴───────┐
       │               │
      True           False
       │               │
       ▼               ▼
  ┌─────────┐     ┌─────────────┐
  │ 执行 1   │     │ 检查条件 2   │
  │ 退出    │     └──────┬──────┘
  └─────────┘            │
                  ┌──────┴──────┐
                  │             │
                 True         False
                  │             │
                  ▼             ▼
             ┌─────────┐   ┌──────────┐
             │ 执行 2   │   │ 继续检查 │
             │ 退出    │   │ ...      │
             └─────────┘   └──────────┘
```

#### 示例：成绩等级判断

```python
# 示例：根据分数判断等级

score = 85

if score >= 90:
    grade = "A"
    comment = "优秀"
elif score >= 80:
    grade = "B"
    comment = "良好"
elif score >= 70:
    grade = "C"
    comment = "中等"
elif score >= 60:
    grade = "D"
    comment = "及格"
else:
    grade = "F"
    comment = "不及格"

print(f"分数：{score}, 等级：{grade}, {comment}")
# 输出：分数：85, 等级：B, 良好
```

**执行过程分析：**
```python
score = 85

# 第 1 步：检查 score >= 90
if 85 >= 90:  # False
    # 不执行，继续检查下一个条件

# 第 2 步：检查 score >= 80
elif 85 >= 80:  # True
    # 条件成立，执行此块
    grade = "B"
    comment = "良好"
    # 执行完毕，退出整个 if-elif-else 结构

# 后面的 elif 和 else 不再检查
```

**重要提示：**
```
┌─────────────────────────────────────────┐
│              ⭐ 重要提示 ⭐               │
├─────────────────────────────────────────┤
│                                         │
│  if-elif-else 结构中，一旦某个条件     │
│  满足，就会执行对应的代码块，然后      │
│  立即退出整个结构，不再检查后续条件！  │
│                                         │
│  所以条件的顺序很重要！                 │
│                                         │
└─────────────────────────────────────────┘
```

### 4.5 嵌套 if 语句

#### 基本结构

```python
# 在 if 语句内部再使用 if 语句

age = 20
has_ticket = True

if age >= 18:  # 外层条件
    print("已成年")

    if has_ticket:  # 内层条件
        print("可以入场")
    else:
        print("需要买票")
else:
    print("未成年，不能入场")
```

**执行流程：**
```
外层：age >= 18?
  │
  ├── Yes → 打印"已成年"
  │         │
  │         ▼
  │     内层：has_ticket?
  │         │
  │         ├── Yes → 打印"可以入场"
  │         │
  │         └── No → 打印"需要买票"
  │
  └── No → 打印"未成年，不能入场"
```

## 第三部分：三元运算符

### 4.6 条件表达式

#### 基本语法

```
┌─────────────────────────────────────────┐
│         三元运算符语法                   │
├─────────────────────────────────────────┤
│                                         │
│   值 1 if 条件 else 值 2                  │
│                                         │
│   含义：                                │
│   • 如果条件为 True，返回值 1            │
│   • 如果条件为 False，返回值 2          │
│                                         │
│   等价于：                              │
│   if 条件:                              │
│       结果 = 值 1                         │
│   else:                                 │
│       结果 = 值 2                         │
│                                         │
└─────────────────────────────────────────┘
```

#### 示例对比

```python
# 传统写法
age = 20
if age >= 18:
    status = "成年"
else:
    status = "未成年"

# 三元运算符写法（更简洁）
age = 20
status = "成年" if age >= 18 else "未成年"

# 两种方式效果相同
print(status)  # 输出：成年
```

#### 实际应用

```python
# 示例 1：判断正负
num = -5
result = "正数" if num > 0 else "非正数"

# 示例 2：设置默认值
username = ""  # 空字符串
display_name = username if username else "匿名用户"
print(display_name)  # 输出：匿名用户

# 示例 3：最大值
a, b = 10, 20
max_value = a if a > b else b
print(max_value)  # 输出：20
```

## 第四部分：for 循环详解

### 4.7 for 循环基础

#### 什么是 for 循环

**for 循环** 用于遍历序列（如列表、元组、字符串）或其他可迭代对象中的每个元素。

#### 基本语法

```
┌─────────────────────────────────────────┐
│            for 循环语法                  │
├─────────────────────────────────────────┤
│                                         │
│   for 变量 in 可迭代对象:                │
│       循环体（重复执行的代码）          │
│                                         │
│   语法说明：                            │
│   ─────────────                         │
│   • for       → 关键字                  │
│   • 变量      → 每次循环的当前元素      │
│   • in        → 关键字                  │
│   • 可迭代对象 → 要遍历的数据           │
│   • :         → 冒号，不能省略          │
│   • 缩进      → 循环体必须缩进          │
│                                         │
└─────────────────────────────────────────┘
```

#### 示例：遍历列表

```python
# 示例 1：遍历水果列表

fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(fruit)

# 输出：
# apple
# banana
# orange
```

**执行过程图解：**
```
fruits = ["apple", "banana", "orange"]
          │
          ▼
      第 1 次循环
          │
          ▼
      fruit = "apple"
      print("apple")
          │
          ▼
      第 2 次循环
          │
          ▼
      fruit = "banana"
      print("banana")
          │
          ▼
      第 3 次循环
          │
          ▼
      fruit = "orange"
      print("orange")
          │
          ▼
      列表结束，退出循环
```

### 4.8 range() 函数

#### 什么是 range()

**range()** 函数生成一个数字序列，常用于 for 循环中指定循环次数。

#### 三种用法

```
┌─────────────────────────────────────────┐
│          range() 函数用法                │
├─────────────────────────────────────────┤
│                                         │
│  1. range(stop)                         │
│     生成 0 到 stop-1 的数字              │
│     range(5) → 0, 1, 2, 3, 4           │
│                                         │
│  2. range(start, stop)                  │
│     生成 start 到 stop-1 的数字          │
│     range(2, 5) → 2, 3, 4              │
│                                         │
│  3. range(start, stop, step)            │
│     生成 start 到 stop-1，步长为 step   │
│     range(0, 10, 2) → 0, 2, 4, 6, 8   │
│                                         │
└─────────────────────────────────────────┘
```

#### 示例详解

```python
# 用法 1：range(stop)
print("range(5):")
for i in range(5):
    print(i, end=" ")
# 输出：0 1 2 3 4

# 用法 2：range(start, stop)
print("\nrange(2, 6):")
for i in range(2, 6):
    print(i, end=" ")
# 输出：2 3 4 5

# 用法 3：range(start, stop, step)
print("\nrange(0, 10, 2):")
for i in range(0, 10, 2):
    print(i, end=" ")
# 输出：0 2 4 6 8

# 倒序循环
print("\nrange(5, 0, -1):")
for i in range(5, 0, -1):
    print(i, end=" ")
# 输出：5 4 3 2 1
```

**重要提示：**
```
┌─────────────────────────────────────────┐
│              ⭐ 重要提示 ⭐               │
├─────────────────────────────────────────┤
│                                         │
│  range() 生成的数字序列是"包头不包尾"   │
│                                         │
│  range(1, 5) 生成：1, 2, 3, 4          │
│  不包含 5！                            │
│                                         │
│  记忆方法：从 1 开始，到 5 之前结束      │
│                                         │
└─────────────────────────────────────────┘
```

### 4.9 for 循环实际应用

#### 示例 1：计算累加和

```python
# 计算 1+2+3+...+100 的和

total = 0  # 初始化总和为 0

for i in range(1, 101):  # 1 到 100
    total = total + i    # 累加

print(f"1 到 100 的和是：{total}")
# 输出：1 到 100 的和是：5050
```

**执行过程：**
```
total = 0

第 1 次循环：i = 1
  total = 0 + 1 = 1

第 2 次循环：i = 2
  total = 1 + 2 = 3

第 3 次循环：i = 3
  total = 3 + 3 = 6

...

第 100 次循环：i = 100
  total = 4950 + 100 = 5050

循环结束，输出结果
```

#### 示例 2：打印乘法表

```python
# 打印 99 乘法表

for i in range(1, 10):      # 外层循环：1-9
    for j in range(1, 10):  # 内层循环：1-9
        print(f"{j}×{i}={i*j}", end="\t")
    print()  # 换行
```

**输出：**
```
1×1=1	2×1=2	3×1=3	...	9×1=9
1×2=2	2×2=4	3×2=6	...	9×2=18
...
1×9=9	2×9=18	3×9=27	...	9×9=81
```

#### 示例 3：遍历字符串

```python
# 遍历字符串中的每个字符

word = "Python"

for char in word:
    print(char)

# 输出：
# P
# y
# t
# h
# o
# n
```

## 第五部分：while 循环详解

### 4.10 while 循环基础

#### 基本语法

```
┌─────────────────────────────────────────┐
│          while 循环语法                  │
├─────────────────────────────────────────┤
│                                         │
│   while 条件:                            │
│       循环体                             │
│                                         │
│   语法说明：                            │
│   ─────────────                         │
│   • while   → 关键字                    │
│   • 条件    → 每次循环前检查的条件      │
│   • :       → 冒号，不能省略            │
│   • 缩进    → 循环体必须缩进            │
│                                         │
│   ⚠️ 注意：必须在循环体内修改条件，    │
│   否则会变成无限循环！                  │
│                                         │
└─────────────────────────────────────────┘
```

#### 示例：基本 while 循环

```python
# 示例：从 1 数到 5

count = 1  # 初始化计数器

while count <= 5:  # 条件：count 小于等于 5
    print(count)
    count = count + 1  # 增加计数器

print("循环结束")
```

**执行过程：**
```
count = 1

第 1 次检查：count <= 5? → 1 <= 5 → True
  执行循环体：
  print(1)
  count = 1 + 1 = 2

第 2 次检查：count <= 5? → 2 <= 5 → True
  执行循环体：
  print(2)
  count = 2 + 1 = 3

...

第 5 次检查：count <= 5? → 5 <= 5 → True
  执行循环体：
  print(5)
  count = 5 + 1 = 6

第 6 次检查：count <= 5? → 6 <= 5 → False
  不执行循环体，退出循环

print("循环结束")
```

#### for vs while 对比

```
┌─────────────────┬─────────────────┬─────────────────┐
│     特点        │     for 循环     │    while 循环    │
├─────────────────┼─────────────────┼─────────────────┤
│ 使用场景        │ 遍历已知序列    │ 条件循环        │
│ 循环次数        │ 通常已知        │ 可能未知        │
│ 计数器管理      │ 自动            │ 手动            │
│ 无限循环风险    │ 低              │ 高              │
└─────────────────┴─────────────────┴─────────────────┘
```

```python
# 用 for 实现：计算 1 到 100 的和
total = 0
for i in range(1, 101):
    total += i

# 用 while 实现：计算 1 到 100 的和
total = 0
i = 1
while i <= 100:
    total += i
    i += 1

# 两种方法结果相同
print(total)  # 5050
```

## 第六部分：循环控制语句

### 4.11 break 语句

#### 功能说明

**break** 用于立即跳出（终止）整个循环。

```
┌─────────────────────────────────────────┐
│            break 语句作用                │
├─────────────────────────────────────────┤
│                                         │
│  for/while 循环:                        │
│      循环体                            │
│      if 条件:                          │
│          break    ← 立即跳出整个循环   │
│      循环体                            │
│                                         │
│  break 后的代码不再执行                 │
│                                         │
└─────────────────────────────────────────┘
```

#### 示例：查找第一个满足条件的数

```python
# 找到第一个能被 7 整除的数（1-100 之间）

for i in range(1, 101):
    if i % 7 == 0:
        print(f"找到了：{i}")
        break  # 找到后跳出循环

# 输出：找到了：7
# （只输出一次，然后循环结束）
```

**执行流程：**
```
i = 1:  1 % 7 = 1 ≠ 0  → 继续
i = 2:  2 % 7 = 2 ≠ 0  → 继续
...
i = 6:  6 % 7 = 6 ≠ 0  → 继续
i = 7:  7 % 7 = 0  → 条件满足！
       打印"找到了：7"
       break → 跳出循环

循环结束，i=8,9,10...不再执行
```

### 4.12 continue 语句

#### 功能说明

**continue** 用于跳过本次循环的剩余代码，直接进入下一次循环。

```
┌─────────────────────────────────────────┐
│          continue 语句作用               │
├─────────────────────────────────────────┤
│                                         │
│  for/while 循环:                        │
│      循环体                            │
│      if 条件:                          │
│          continue ← 跳过本次，继续下次 │
│      循环体（这行在条件满足时不执行）  │
│                                         │
└─────────────────────────────────────────┘
```

#### 示例：只打印奇数

```python
# 打印 1-10 之间的所有奇数

for i in range(1, 11):
    if i % 2 == 0:  # 如果是偶数
        continue    # 跳过本次循环

    print(i, end=" ")
# 输出：1 3 5 7 9
```

**执行过程：**
```
i = 1:  1 % 2 = 1 ≠ 0  → 不执行 continue  → 打印 1
i = 2:  2 % 2 = 0  → 执行 continue → 跳过打印，进入下次
i = 3:  3 % 2 = 1 ≠ 0  → 不执行 continue  → 打印 3
i = 4:  4 % 2 = 0  → 执行 continue → 跳过打印，进入下次
...
```

### 4.13 break vs continue 对比

```python
# break 示例
print("break 示例:")
for i in range(1, 6):
    if i == 3:
        break
    print(i)
# 输出：1 2
# （i=3 时跳出整个循环，4 和 5 不执行）

# continue 示例
print("continue 示例:")
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
# 输出：1 2 4 5
# （i=3 时跳过本次，4 和 5 继续执行）
```

**图解对比：**
```
break:                    continue:

i=1 → 打印 1               i=1 → 打印 1
i=2 → 打印 2               i=2 → 打印 2
i=3 → break → 退出        i=3 → continue → 跳过
         ⨯ 不打印              i=4 → 打印 4
         ⨯ 不打印              i=5 → 打印 5
```

## 第七部分：循环的 else 子句

### 4.14 for/while + else

#### 基本语法

```python
# for ... else
for 变量 in 可迭代对象:
    循环体
else:
    循环正常结束时执行（没有被 break 打断）

# while ... else
while 条件:
    循环体
else:
    循环正常结束时执行（没有被 break 打断）
```

#### 示例说明

```python
# 示例 1：正常结束的循环

for i in range(5):
    print(i)
else:
    print("循环正常结束")

# 输出：
# 0
# 1
# 2
# 3
# 4
# 循环正常结束
```

```python
# 示例 2：被 break 打断的循环

for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("这句话不会打印")

# 输出：
# 0
# 1
# 2
# （else 块不会执行，因为被 break 打断了）
```

#### 实际应用：查找元素

```python
# 在列表中查找元素

numbers = [1, 3, 5, 7, 9]
target = 7

for num in numbers:
    if num == target:
        print(f"找到了：{target}")
        break
else:
    # 如果循环正常结束（没找到）
    print(f"没找到：{target}")

# 输出：找到了：7
```

## 本章总结

### 知识点回顾

| 知识点 | 语法 | 说明 |
|--------|------|------|
| if 语句 | `if 条件:` | 条件成立时执行 |
| if-else | `if 条件: ... else:` | 二选一执行 |
| if-elif-else | `if...elif...else` | 多选一执行 |
| for 循环 | `for 变量 in 序列:` | 遍历序列 |
| range() | `range(start, stop, step)` | 生成数字序列 |
| while 循环 | `while 条件:` | 条件为真时循环 |
| break | `break` | 跳出整个循环 |
| continue | `continue` | 跳过本次循环 |
| else 子句 | `for/while ... else` | 循环正常结束时执行 |

### 流程图速查

```
if 语句：              for 循环：
  条件判断                遍历序列
     │                      │
  ┌──┴──┐                  │
  │     │                  ▼
 Yes   No              执行循环体

while 循环：           break/continue：
  条件判断                break → 跳出
     │                    continue → 跳过本次
     ▼
  执行循环体
     │
     └──→ 返回条件
```

### 常见错误提醒

```
❌ 忘记冒号
if x > 0  # 缺少冒号
    pass

✅ 正确
if x > 0:
    pass

❌ 缩进错误
if x > 0:
print(x)  # 缺少缩进

✅ 正确
if x > 0:
    print(x)

❌ while 循环忘记更新条件
while x < 10:
    print(x)
    # 忘记 x += 1 → 无限循环！

✅ 正确
while x < 10:
    print(x)
    x += 1
```
