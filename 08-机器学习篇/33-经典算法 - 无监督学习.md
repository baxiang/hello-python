## 第一部分：聚类算法

### 33.1 K-Means 聚类

#### 概念说明

K-Means 是最常用的聚类算法，通过迭代将数据分为 K 个簇。

**算法流程：**

```
┌─────────────────────────────────────────────────────────────┐
│              K-Means 算法流程                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   1. 初始化：随机选择 K 个点作为初始质心                      │
│                                                             │
│   2. 分配：将每个样本点分配到最近的质心                     │
│      形成 K 个簇                                              │
│                                                             │
│   3. 更新：计算每个簇的新质心（均值）                       │
│                                                             │
│   4. 重复：重复步骤 2-3，直到质心不再变化                    │
│                                                             │
│   5. 输出：K 个簇及其质心                                    │
│                                                             │
│   目标函数：最小化簇内平方和 (SSE)                           │
│   SSE = Σ Σ ||x - μ||²                                      │
│         i  x∈Ci                                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**代码示例：**

```python
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs
import matplotlib.pyplot as plt

# 创建聚类数据
X, y = make_blobs(n_samples=300, centers=4, random_state=42)

# 创建 K-Means 模型
kmeans = KMeans(
    n_clusters=4,       # 簇的数量
    init='k-means++',   # 初始化方法
    n_init=10,          # 运行次数
    max_iter=300,       # 最大迭代次数
    random_state=42
)

# 训练
kmeans.fit(X)

# 预测
labels = kmeans.labels_
centers = kmeans.cluster_centers_

# 可视化
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis', alpha=0.5)
plt.scatter(centers[:, 0], centers[:, 1], c='red', s=200, marker='X')
plt.title(f'K-Means 聚类结果 (K={kmeans.n_clusters})')
plt.show()

# 评估：轮廓系数
from sklearn.metrics import silhouette_score
score = silhouette_score(X, labels)
print(f"轮廓系数：{score:.4f}")
```

### 33.2 选择 K 值

#### 概念说明

K-Means 需要预先指定 K 值，选择合适的 K 值很重要。

**方法 1：手肘法**

```python
inertias = []
K_range = range(1, 11)

for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42)
    kmeans.fit(X)
    inertias.append(kmeans.inertia_)  # SSE

# 绘制手肘图
plt.plot(K_range, inertias, 'bo-')
plt.xlabel('K 值')
plt.ylabel('SSE')
plt.title('手肘法选择 K 值')
plt.show()

# 选择手肘位置的 K 值
```

**方法 2：轮廓系数法**

```python
from sklearn.metrics import silhouette_score

silhouette_scores = []
K_range = range(2, 11)  # K=1 时轮廓系数无定义

for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42)
    labels = kmeans.fit_predict(X)
    score = silhouette_score(X, labels)
    silhouette_scores.append(score)

# 绘制轮廓系数图
plt.plot(K_range, silhouette_scores, 'ro-')
plt.xlabel('K 值')
plt.ylabel('轮廓系数')
plt.title('轮廓系数法选择 K 值')
plt.show()

# 选择轮廓系数最大的 K 值
best_k = K_range[silhouette_scores.index(max(silhouette_scores))]
print(f"最佳 K 值：{best_k}")
```

### 33.3 层次聚类

#### 概念说明

层次聚类通过计算样本间的距离，构建一个树状的层次结构。

**聚类策略：**

```
┌─────────────────────────────────────────────────────────────┐
│              层次聚类策略                                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   1. 凝聚式 (Agglomerative) - 自底向上                      │
│   ┌───────────────────────────────────────────────────┐     │
│   │  步骤：                                            │     │
│   │  1. 每个样本作为一个簇                             │     │
│   │  2. 合并最相似的两个簇                             │     │
│   │  3. 重复合并，直到所有样本在一个簇中               │     │
│   │                                                   │     │
│   │  例如：A,B,C,D,E → (A,B),C,D,E → (A,B),(C,D),E    │     │
│   │        → (A,B,C,D),E → (A,B,C,D,E)                │     │
│   └───────────────────────────────────────────────────┘     │
│                                                             │
│   2. 分裂式 (Divisive) - 自顶向下                           │
│   ┌───────────────────────────────────────────────────┐     │
│   │  步骤：                                            │     │
│   │  1. 所有样本作为一个簇                             │     │
│   │  2. 分裂成两个簇                                   │     │
│   │  3. 重复分裂，直到每个样本一个簇                   │     │
│   └───────────────────────────────────────────────────┘     │
│                                                             │
│   距离度量：                                                │
│   • 欧氏距离：最常用                                        │
│   • 曼哈顿距离：城市街区距离                                │
│   • 余弦相似度：文本数据                                    │
│                                                             │
│   链接方式：                                                │
│   • 单链接：最近邻                                          │
│   • 全链接：最远邻                                          │
│   • 平均链接：平均距离                                      │
│   • Ward 法：最小化方差                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**代码示例：**

```python
from sklearn.cluster import AgglomerativeClustering
from scipy.cluster.hierarchy import dendrogram, linkage
import matplotlib.pyplot as plt

# 创建层次聚类模型
agg = AgglomerativeClustering(
    n_clusters=3,           # 簇的数量
    affinity='euclidean',   # 距离度量
    linkage='ward'          # 链接方式
)

# 训练并预测
labels = agg.fit_predict(X)

# 可视化：树状图
linkage_matrix = linkage(X, method='ward')
plt.figure(figsize=(10, 6))
dendrogram(linkage_matrix)
plt.title('层次聚类树状图')
plt.xlabel('样本索引')
plt.ylabel('距离')
plt.show()
```

### 33.4 DBSCAN 密度聚类

#### 概念说明

DBSCAN 是一种基于密度的聚类算法，可以发现任意形状的簇。

**核心概念：**

```
┌─────────────────────────────────────────────────────────────┐
│              DBSCAN 核心概念                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   参数：                                                     │
│   • eps (ε)：邻域半径                                       │
│   • min_samples：核心点所需最小样本数                       │
│                                                             │
│   点的类型：                                                 │
│   ┌───────────────────────────────────────────────────┐     │
│   │  核心点 (Core Point)：                              │     │
│   │  eps 邻域内至少有 min_samples 个点                   │     │
│   │                                                   │     │
│   │  边界点 (Border Point)：                            │     │
│   │  eps 邻域内点数少于 min_samples，但在核心点邻域内  │     │
│   │                                                   │     │
│   │  噪声点 (Noise Point)：                             │     │
│   │  既不是核心点也不是边界点                          │     │
│   └───────────────────────────────────────────────────┘     │
│                                                             │
│   聚类过程：                                                 │
│   1. 从任意点开始，如果为核心点，则形成一个簇              │
│   2. 找到该核心点邻域内的所有点，加入簇                    │
│   3. 对新加入的核心点重复步骤 2                             │
│   4. 继续处理下一个未访问的点                              │
│                                                             │
│   优点：                                                     │
│   ✓ 可以发现任意形状的簇                                    │
│   ✓ 可以识别噪声点                                          │
│   ✓ 不需要预先指定簇的数量                                  │
│                                                             │
│   缺点：                                                     │
│   ✗ 对参数 eps 和 min_samples 敏感                            │
│   ✗ 密度差异大的数据效果不好                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**代码示例：**

```python
from sklearn.cluster import DBSCAN

# 创建 DBSCAN 模型
dbscan = DBSCAN(
    eps=0.5,            # 邻域半径
    min_samples=5,      # 核心点最小样本数
    metric='euclidean'  # 距离度量
)

# 训练并预测
labels = dbscan.fit_predict(X)

# 统计结果
n_clusters = len(set(labels)) - (1 if -1 in labels else 0)
n_noise = list(labels).count(-1)

print(f"发现簇的数量：{n_clusters}")
print(f"噪声点数量：{n_noise}")

# 可视化
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis', alpha=0.5)
plt.title(f'DBSCAN 聚类结果 (簇数={n_clusters}, 噪声={n_noise})')
plt.show()
```

## 第二部分：降维算法

### 33.5 PCA 主成分分析

#### 概念说明

PCA 通过线性变换将数据投影到方差最大的方向，实现降维。

**核心思想：**

```
┌─────────────────────────────────────────────────────────────┐
│                  PCA 主成分分析                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   目标：                                                     │
│   • 找到数据中方差最大的方向（主成分）                      │
│   • 将数据投影到低维空间                                    │
│   • 保留尽可能多的信息（方差）                              │
│                                                             │
│   步骤：                                                     │
│   1. 数据标准化（均值为 0，方差为 1）                        │
│   2. 计算协方差矩阵                                         │
│   3. 计算特征值和特征向量                                   │
│   4. 选择前 k 个最大的特征值对应的特征向量                   │
│   5. 构建投影矩阵，将数据投影到新空间                       │
│                                                             │
│   主成分：                                                   │
│   • PC1：第一主成分，方差最大方向                           │
│   • PC2：第二主成分，与 PC1 正交，方差次大                   │
│   • ...                                                     │
│                                                             │
│   解释方差比：                                               │
│   • 每个主成分保留的方差比例                                │
│   • 累计解释方差比：前 k 个主成分的方差和占比               │
│                                                             │
│   应用场景：                                                 │
│   • 数据可视化（降到 2D/3D）                                │
│   • 特征压缩                                                │
│   • 去除多重共线性                                          │
│   • 数据去噪                                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**代码示例：**

```python
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler
import matplotlib.pyplot as plt

# 标准化数据
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 创建 PCA 模型
pca = PCA(n_components=2)  # 降到 2 维
X_pca = pca.fit_transform(X_scaled)

# 解释方差比
print(f"各主成分解释方差比：{pca.explained_variance_ratio_}")
print(f"累计解释方差比：{sum(pca.explained_variance_ratio_):.4f}")

# 可视化
plt.figure(figsize=(10, 4))

# 原始数据
plt.subplot(1, 2, 1)
plt.scatter(X[:, 0], X[:, 1], alpha=0.5)
plt.title('原始数据')

# PCA 降维后
plt.subplot(1, 2, 2)
plt.scatter(X_pca[:, 0], X_pca[:, 1], alpha=0.5)
plt.title(f'PCA 降维后\n(解释方差比={sum(pca.explained_variance_ratio_):.2%})')

plt.tight_layout()
plt.show()

# 选择主成分数量
pca_full = PCA()
pca_full.fit(X_scaled)

plt.plot(range(1, len(pca_full.explained_variance_ratio_)+1),
         pca_full.explained_variance_ratio_.cumulative(), 'bo-')
plt.axhline(y=0.95, color='r', linestyle='--', label='95% 方差')
plt.xlabel('主成分数量')
plt.ylabel('累计解释方差比')
plt.title('选择主成分数量')
plt.legend()
plt.show()
```

### 33.6 t-SNE 非线性降维

#### 概念说明

t-SNE 是一种非线性降维方法，特别适合高维数据的可视化。

**PCA vs t-SNE：**

| 特性 | PCA | t-SNE |
|------|-----|-------|
| 类型 | 线性 | 非线性 |
| 速度 | 快 | 慢 |
| 全局结构 | 保留 | 不保留 |
| 局部结构 | 不保留 | 保留 |
| 适用场景 | 特征压缩、去噪 | 可视化 |

**代码示例：**

```python
from sklearn.manifold import TSNE

# 创建 t-SNE 模型
tsne = TSNE(
    n_components=2,     # 降到 2 维
    perplexity=30,      # 困惑度
    learning_rate=100,
    n_iter=1000,
    random_state=42
)

X_tsne = tsne.fit_transform(X)

# 可视化
plt.scatter(X_tsne[:, 0], X_tsne[:, 1], alpha=0.5)
plt.title('t-SNE 降维结果')
plt.show()
```
