## 环境准备

开始前请确保已安装以下依赖：

```bash
# 创建项目
uv init ml-learning
cd ml-learning

# 核心库
uv add scikit-learn numpy pandas matplotlib seaborn

# 数据处理
uv add jupyter notebook

# 可选：深度学习库（第九篇使用）
# uv add torch torchvision
```

**推荐使用 Jupyter Notebook 学习：**

```bash
uv run jupyter notebook
```

## 学完本篇你能做什么？

完成本篇章学习后，你将能够：

✅ **理解机器学习核心概念**
   - 监督学习 vs 无监督学习
   - 训练集/测试集划分
   - 过拟合与欠拟合

✅ **掌握数据预处理技能**
   - 缺失值处理
   - 特征编码（独热编码、标签编码）
   - 特征缩放（标准化、归一化）

✅ **使用经典算法解决问题**
   - 线性回归/逻辑回归
   - 决策树/随机森林
   - SVM 支持向量机
   - K-Means 聚类
   - PCA 降维

✅ **评估和优化模型**
   - 准确率、精确率、召回率、F1
   - ROC 曲线与 AUC
   - 交叉验证
   - 网格搜索调参

✅ **独立完成 ML 项目**
   - 数据探索与清洗
   - 特征工程
   - 模型训练与评估
   - 模型保存与部署
