# 09.04.03 - 上下文工程

> 理解 LangChain 的三层上下文模型：State、Store、Context

## 上下文工程概述

Agent 在执行过程中需要管理多种类型的上下文：

```
上下文架构：
┌──────────────────────────────────────┐
│  Model Context                       │
│  ├─ prompt: 系统提示词               │
│  ├─ messages: 对话消息列表           │
│  ├─ tools: 可用工具列表              │
│  ├─ model: 当前使用的模型            │
│  └─ response_format: 输出格式        │
│                                       │
│  Tool Context                        │
│  ├─ reads: 工具可读取的状态          │
│  └─ writes: 工具可写入的状态         │
│                                       │
│  Life-cycle Context                  │
│  ├─ Runtime: 运行时信息              │
│  ├─ Store: 长期记忆                  │
│  └─ Config: 配置参数                 │
└──────────────────────────────────────┘
```

---

## Model Context

Model Context 是传递给大语言模型的所有信息：

### 上下文组件对比

```
State / Store / Context 对比：
┌──────────────────────────────────────┐
│  State（短期状态）                    │
│  ├─ 生命周期：单次 Agent 执行         │
│  ├─ 作用域：当前对话                  │
│  ├─ 持久化：Checkpointer（按 thread） │
│  └─ 用途：对话消息、临时变量         │
│                                       │
│  Store（长期存储）                    │
│  ├─ 生命周期：持久化                  │
│  ├─ 作用域：跨对话、跨用户            │
│  ├─ 持久化：Store 后端               │
│  └─ 用途：用户偏好、知识库           │
│                                       │
│  Context（运行时上下文）              │
│  ├─ 生命周期：单次请求               │
│  ├─ 作用域：当前请求                 │
│  ├─ 持久化：不持久化                  │
│  └─ 用途：用户 ID、权限、配置        │
└──────────────────────────────────────┘
```

### 各维度对比表

| 维度 | State | Store | Context |
|------|-------|-------|---------|
| 生命周期 | 单次执行 | 持久化 | 单次请求 |
| 作用域 | 当前对话 | 全局 | 当前请求 |
| 持久化 | Checkpointer | Store 后端 | 无 |
| 访问方式 | `state["key"]` | `store.get/put` | `runtime.context` |
| 典型内容 | 消息、变量 | 用户偏好、文档 | 用户 ID、角色 |
| 并发安全 | 单线程 | 多线程安全 | 只读 |

## State：短期状态

### State 结构

```python
from pydantic import BaseModel
from typing import Annotated
from langgraph.graph.message import add_messages

class AgentState(BaseModel):
    """Agent 状态定义"""
    messages: Annotated[list, add_messages]  # 对话消息
    context: str = ""                         # 额外上下文
    metadata: dict = {}                       # 元数据
    user_id: str = ""                         # 用户 ID
    conversation_stage: str = "initial"       # 对话阶段
```

### State 在中间件中的使用

```python
from langchain.agents.middleware import before_agent, wrap_model_call

@before_agent
def read_state(state, runtime):
    """读取 State"""
    message_count = len(state["messages"])
    user_id = state.get("user_id", "unknown")
    stage = state.get("conversation_stage", "initial")
    
    print(f"用户 {user_id} 在第 {stage} 阶段，已有 {message_count} 条消息")
    return None

@before_agent
def update_state(state, runtime):
    """更新 State"""
    return {
        "metadata": {
            **state.get("metadata", {}),
            "last_accessed": "2024-01-01T00:00:00",
        },
        "conversation_stage": "active",
    }
```

### State 更新策略

```
State 更新策略：
┌──────────────────────────────────────┐
│  策略1：追加（Append）                │
│  new_state = {**old_state, **updates} │
│                                       │
│  策略2：覆盖（Replace）               │
│  new_state[key] = new_value           │
│                                       │
│  策略3：合并（Merge）                 │
│  new_state[key] = {**old, **new}     │
│                                       │
│  策略4：减少（Reduce）                │
│  new_state[key] = reducer(old, new)   │
└──────────────────────────────────────┘
```

### 消息状态管理

```python
from langchain.agents.middleware import before_agent
from langchain.messages import SystemMessage, trim_messages

@before_agent
def manage_context_window(state, runtime):
    """管理上下文窗口大小"""
    messages = state["messages"]
    
    # 如果消息过多，进行裁剪
    if len(messages) > 20:
        # 保留系统消息和最近 10 条消息
        system_messages = [m for m in messages if m.type == "system"]
        recent_messages = messages[-10:]
        
        # 添加摘要
        summary_msg = SystemMessage(
            content=f"[之前的 {len(messages) - 10 - len(system_messages)} 条消息已压缩]"
        )
        
        return {
            "messages": [*system_messages, summary_msg, *recent_messages]
        }
    
    return None
```

## Store：长期记忆

### Store 结构

```
Store 命名空间结构：
┌──────────────────────────────────────┐
│  namespace: tuple[str, ...]          │
│    类似文件夹路径                     │
│    示例: ("user", "123", "prefs")    │
│                                       │
│  key: str                            │
│    类似文件名                         │
│    示例: "profile"                   │
│                                       │
│  value: dict                         │
│    JSON 可序列化的值                 │
│    示例: {"name": "张三", ...}       │
└──────────────────────────────────────┘
```

### Store 在中间件中的使用

```python
from langchain.agents.middleware import before_agent, after_agent

@before_agent
def load_user_context(state, runtime):
    """从 Store 加载用户上下文"""
    user_id = runtime.context.get("user_id")
    if not user_id:
        return None
    
    # 加载用户偏好
    prefs = runtime.store.get(("user", user_id), "preferences")
    
    # 加载用户历史摘要
    summary = runtime.store.get(("user", user_id, "history"), "summary")
    
    context_parts = []
    if prefs:
        context_parts.append(f"用户偏好: {prefs.value}")
    if summary:
        context_parts.append(f"历史摘要: {summary.value}")
    
    if context_parts:
        return {"context": "\n".join(context_parts)}
    
    return None

@after_agent
def save_interaction(state, runtime):
    """保存交互到 Store"""
    user_id = runtime.context.get("user_id")
    if not user_id:
        return None
    
    last_user_msg = next(
        (m for m in reversed(state["messages"]) if m.type == "human"),
        None
    )
    last_ai_msg = next(
        (m for m in reversed(state["messages"]) if m.type == "ai"),
        None
    )
    
    if last_user_msg and last_ai_msg:
        import uuid
        interaction_id = str(uuid.uuid4())
        
        runtime.store.put(
            ("user", user_id, "interactions"),
            interaction_id,
            {
                "user_input": last_user_msg.content,
                "ai_output": last_ai_msg.content[:500],
                "timestamp": "2024-01-01T00:00:00",
            }
        )
```

### Store 搜索

```python
@before_agent
def search_relevant_context(state, runtime):
    """搜索相关上下文"""
    user_id = runtime.context.get("user_id")
    if not user_id:
        return None
    
    last_msg = state["messages"][-1]
    
    # 语义搜索相关记忆
    memories = runtime.store.search(
        ("user", user_id, "memories"),
        query=last_msg.content,
        limit=3,
    )
    
    if memories:
        context = "相关记忆:\n" + "\n".join(
            f"- {m.value.get('fact', '')}" for m in memories
        )
        return {"context": context}
    
    return None
```

## Context：运行时上下文

### Context 定义

```python
from dataclasses import dataclass

@dataclass
class AppContext:
    """应用运行时上下文"""
    user_id: str
    user_role: str = "guest"
    session_id: str = ""
    locale: str = "zh-CN"
    timezone: str = "Asia/Shanghai"
```

### Context 在中间件中的使用

```python
from langchain.agents.middleware import before_agent, wrap_model_call

@before_agent
def apply_locale(state, runtime):
    """根据 locale 调整系统提示"""
    locale = runtime.context.locale
    
    locale_prompts = {
        "zh-CN": "你是一个友好的中文助手。",
        "en-US": "You are a friendly English assistant.",
        "ja-JP": "あなたはフレンドリーな日本語アシスタントです。",
    }
    
    system_prompt = locale_prompts.get(locale, locale_prompts["en-US"])
    
    return {"system_prompt": system_prompt}

@wrap_model_call
def apply_timezone(request: ModelRequest, handler) -> ModelResponse:
    """在请求中注入时区信息"""
    timezone = request.context.timezone
    
    # 修改系统提示加入时区
    timezone_msg = f"当前时区: {timezone}"
    
    # 这里可以修改 request.messages 来加入时区信息
    # 为了简单，我们只打印
    print(f"时区上下文: {timezone_msg}")
    
    return handler(request)
```

### Context 权限控制

```python
from langchain.agents.middleware import wrap_model_call, ModelRequest, ModelResponse

# 工具权限映射
TOOL_PERMISSIONS = {
    "guest": ["search", "weather"],
    "user": ["search", "weather", "email"],
    "admin": ["search", "weather", "email", "delete", "config"],
}

@wrap_model_call
def enforce_permissions(request: ModelRequest, handler) -> ModelResponse:
    """根据用户角色 enforce 工具权限"""
    user_role = request.context.user_role
    allowed_tools = TOOL_PERMISSIONS.get(user_role, [])
    
    # 过滤工具
    filtered = [t for t in request.tools if t.name in allowed_tools]
    
    print(f"角色: {user_role}")
    print(f"允许工具: {allowed_tools}")
    print(f"实际工具: {[t.name for t in filtered]}")
    
    return handler(request.override(tools=filtered))
```

## Tool Context

工具可以读取和写入 State：

### 工具读取 State

```python
from langchain.tools import tool

@tool
def get_user_context(state: dict) -> str:
    """获取当前用户上下文"""
    user_id = state.get("user_id", "unknown")
    stage = state.get("conversation_stage", "unknown")
    
    return f"用户ID: {user_id}, 对话阶段: {stage}"
```

### 工具写入 State

```python
from langchain.tools import tool
from langchain.messages import ToolMessage

@tool
def update_user_preference(preference: str, value: str) -> str:
    """更新用户偏好"""
    # 工具内部逻辑
    return f"已更新偏好: {preference} = {value}"
```

### 工具读写模式对比

| 模式 | 读取 State | 写入 State | 示例 |
|------|-----------|-----------|------|
| 纯读取 | ✅ | ❌ | 查询用户信息 |
| 读 + 写 | ✅ | ✅ | 更新用户偏好 |
| 纯写入 | ❌ | ✅ | 记录日志 |
| 无状态 | ❌ | ❌ | 计算工具 |

## Life-cycle Context

### 完整生命周期

```
Agent 生命周期：
┌──────────────────────────────────────┐
│  1. 初始化                            │
│     ├─ 创建 Agent                     │
│     ├─ 配置中间件                     │
│     └─ 设置 Checkpointer             │
│                                       │
│  2. 请求处理                          │
│     ├─ 创建 Context                   │
│     ├─ 加载 State（从 Checkpointer）  │
│     ├─ 加载 Store 数据                │
│     └─ 执行 Agent Loop               │
│                                       │
│  3. Agent Loop                       │
│     ├─ before_agent                   │
│     ├─ wrap_model_call                │
│     ├─ model.invoke()                 │
│     ├─ wrap_tool_call                 │
│     └─ 循环直到完成                   │
│                                       │
│  4. 完成处理                          │
│     ├─ after_agent                    │
│     ├─ 保存 State（Checkpointer）     │
│     └─ 更新 Store                     │
│                                       │
│  5. 响应返回                          │
│     └─ 返回最终消息                   │
└──────────────────────────────────────┘
```

### Context 初始化

```python
from langchain.agents import create_agent
from langgraph.checkpoint.memory import InMemorySaver
from langgraph.store.memory import InMemoryStore
from dataclasses import dataclass

@dataclass
class AppContext:
    user_id: str
    user_role: str

# 1. 创建组件
checkpointer = InMemorySaver()
store = InMemoryStore()

# 2. 创建 Agent
agent = create_agent(
    model="moonshot:moonshot-v1-8k",
    tools=[search_tool, email_tool],
    middleware=[
        logging_middleware,
        permission_middleware,
    ],
    checkpointer=checkpointer,
    store=store,
)

# 3. 调用时传入 Context
config = {"configurable": {"thread_id": "session-123"}}
context = AppContext(user_id="user-1", user_role="admin")

result = agent.invoke(
    {"messages": [{"role": "user", "content": "你好"}]},
    config=config,
    context=context,
)
```

### Context 传播

```
Context 传播路径：
┌──────────────────────────────────────┐
│  agent.invoke(context=ctx)           │
│       ↓                              │
│  runtime.context                     │
│       ↓                              │
│  @before_agent(state, runtime)       │
│  @after_agent(state, runtime)        │
│  @wrap_model_call(request, handler)  │
│    → request.context                 │
│  @wrap_tool_call(request, handler)   │
│    → request.context                 │
└──────────────────────────────────────┘
```

## 上下文工程实战模式

### 模式1：上下文注入

```python
from langchain.agents.middleware import before_agent

@before_agent
def inject_full_context(state, runtime):
    """注入完整上下文"""
    user_id = runtime.context.user_id
    
    # 1. 从 Store 加载用户信息
    user_info = runtime.store.get(("user", user_id), "profile")
    
    # 2. 从 Store 加载相关记忆
    last_msg = state["messages"][-1]
    memories = runtime.store.search(
        ("user", user_id, "memories"),
        query=last_msg.content,
        limit=5,
    )
    
    # 3. 构建上下文
    context_parts = []
    
    if user_info:
        context_parts.append(f"用户信息: {user_info.value}")
    
    if memories:
        context_parts.append("相关记忆:")
        for m in memories:
            context_parts.append(f"- {m.value.get('fact', '')}")
    
    if context_parts:
        return {"context": "\n".join(context_parts)}
    
    return None
```

### 模式2：上下文压缩

```python
from langchain.agents.middleware import before_agent

@before_agent
def compress_context(state, runtime):
    """压缩上下文到关键信息"""
    context = state.get("context", "")
    
    if len(context) > 1000:
        # 简单压缩：保留前 500 字
        compressed = context[:500] + "...[已压缩]"
        return {"context": compressed}
    
    return None
```

### 模式3：上下文分层

```python
from langchain.agents.middleware import before_agent

@before_agent
def layered_context(state, runtime):
    """分层上下文管理"""
    # 第一层：系统级上下文（不变）
    system_context = "你是一个专业的 AI 助手。"
    
    # 第二层：用户级上下文（持久）
    user_id = runtime.context.user_id
    user_prefs = runtime.store.get(("user", user_id), "preferences")
    user_context = f"用户偏好: {user_prefs.value}" if user_prefs else ""
    
    # 第三层：会话级上下文（短期）
    session_context = state.get("context", "")
    
    # 第四层：消息级上下文（当前）
    last_msg = state["messages"][-1]
    
    # 组合所有层
    full_context = "\n".join(filter(None, [
        system_context,
        user_context,
        session_context,
    ]))
    
    return {"full_context": full_context}
```

from langchain.agents.middleware import wrap_model_call, ModelRequest, ModelResponse

@wrap_model_call
def debug_context(request: ModelRequest, handler) -> ModelResponse:

### 上下文变更追踪

```python
from langchain.agents.middleware import before_agent, after_agent

context_changes = []

@before_agent
def track_before(state, runtime):
    """记录 before 状态"""
    context_changes.append({
        "phase": "before",
        "state_keys": list(state.keys()),
        "message_count": len(state.get("messages", [])),
    })

@after_agent
def track_after(state, runtime):
    """记录 after 状态"""
    context_changes.append({
        "phase": "after",
        "state_keys": list(state.keys()),
        "message_count": len(state.get("messages", [])),
    })
    
    # 打印变更
    for change in context_changes:
        print(f"[{change['phase']}] keys={change['state_keys']}, messages={change['message_count']}")
    
    context_changes.clear()
```

## 小结

| 要点 | 说明 |
|------|------|
| 三层上下文 | State（短期）、Store（长期）、Context（运行时） |
| State | 对话消息、临时变量，按 thread 持久化 |
| Store | 用户偏好、知识库，跨对话共享 |
| Context | 用户 ID、角色、配置，单次请求有效 |
| Model Context | prompt + messages + tools + model + response_format |
| Tool Context | 工具可读取/写入 State |
| Life-cycle | 初始化 → 请求 → Loop → 完成 → 响应 |
