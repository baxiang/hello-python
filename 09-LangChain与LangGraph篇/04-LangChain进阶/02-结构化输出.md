# 09.04.02 - 结构化输出

**结构化输出：** 让 LLM 返回特定格式的数据（JSON、Pydantic 对象等）。

```
结构化输出：
┌──────────────────────────────────────┐
│  输入: "分析这篇评论"                  │
│                                       │
│  输出: {                              │
│    "rating": 5,                       │
│    "sentiment": "positive",           │
│    "key_points": ["快", "好用"]       │
│  }                                    │
└──────────────────────────────────────┘
```

### 为什么用它？

- API 接口需要固定格式
- 数据库存储需要结构化
- 避免解析自然语言

---

### 两种策略

```
输出策略：
┌──────────────────────────────────────┐
│  ProviderStrategy → 提供商原生支持    │
│  ToolStrategy     → 工具调用实现      │
│  自动选择        → 根据模型能力决定   │
└──────────────────────────────────────┘
```

**策略对比：**

| 策略 | 适用模型 | 优点 | 缺点 |
|------|----------|------|------|
| ProviderStrategy | OpenAI、Anthropic、xAI | 原生支持，可靠性高 | 仅部分模型支持 |
| ToolStrategy | 所有支持工具的模型 | 兼容性好 | 通过工具调用实现 |
| 自动选择 | 任意模型 | 自动选择最佳策略 | 需 langchain>=1.1 |

---

### Schema 类型支持

| Schema 类型 | 返回类型 | 特点 |
|------------|---------|------|
| Pydantic BaseModel | Pydantic 实例 | 自动验证，字段描述 |
| TypedDict | dict | 简单，无运行时验证 |
| Dataclass | dict | 支持默认值 |
| JSON Schema | dict | 最大控制，跨平台兼容 |

---

### 最简示例：Pydantic 模型

```python
from pydantic import BaseModel, Field
from langchain_openai import ChatOpenAI

class ProductReview(BaseModel):
    rating: int = Field(description="评分 1-5")
    sentiment: str = Field(description="情感：positive/negative")
    key_points: list[str] = Field(description="关键点")

model = ChatOpenAI(
    model="moonshot-v1-8k",
    openai_api_base="https://api.moonshot.cn/v1",
)
structured_model = model.with_structured_output(ProductReview)

result = structured_model.invoke("分析：'产品很好用，发货快，五星推荐！'")
print(result)
# ProductReview(rating=5, sentiment='positive', key_points=['好用', '发货快'])
```

### TypedDict 示例

```python
from typing import Literal
from typing_extensions import TypedDict

class ContactInfo(TypedDict):
    name: str
    email: str
    phone: str

model = ChatOpenAI(
    model="moonshot-v1-8k",
    openai_api_base="https://api.moonshot.cn/v1",
)
structured_model = model.with_structured_output(ContactInfo)

result = structured_model.invoke("提取联系方式：张三，zhang@example.com，13800138000")
# {'name': '张三', 'email': 'zhang@example.com', 'phone': '13800138000'}
```

### 使用 Agent 的结构化输出

```python
from pydantic import BaseModel, Field
from langchain.agents import create_agent
from langchain.agents.structured_output import ToolStrategy

class MeetingAction(BaseModel):
    task: str = Field(description="任务内容")
    assignee: str = Field(description="负责人")
    priority: Literal["low", "medium", "high"]

agent = create_agent(
    model="moonshot:moonshot-v1-8k",
    response_format=ToolStrategy(MeetingAction),
)

result = agent.invoke({
    "messages": [{"role": "user", "content": "下周 Sarah 需要完成项目文档"}]
})
print(result["structured_response"])
```

---

### ProviderStrategy 示例

```python
from langchain.agents.structured_output import ProviderStrategy

agent = create_agent(
    model="moonshot:moonshot-v1-8k",
    response_format=ProviderStrategy(ContactInfo),
)
```

**注意：** ProviderStrategy 需要模型原生支持结构化输出。

### Union 类型：多 Schema 选择

```python
from typing import Union

class ProductReview(BaseModel):
    rating: int
    sentiment: str

class CustomerComplaint(BaseModel):
    issue_type: str
    severity: str

agent = create_agent(
    model="moonshot:moonshot-v1-8k",
    response_format=ToolStrategy(Union[ProductReview, CustomerComplaint]),
)
```

### 错误处理

**默认行为：** 自动重试验证错误。

```python
from langchain.agents.structured_output import ToolStrategy

agent = create_agent(
    model="moonshot:moonshot-v1-8k",
    response_format=ToolStrategy(
        schema=ProductReview,
        handle_errors=True,  # 默认值，自动重试
    ),
)
```

**错误处理策略：**

| handle_errors 值 | 行为 |
|-----------------|------|
| `True` | 捕获所有错误，默认消息 |
| `"str"` | 使用自定义消息 |
| `ValueError` | 仅捕获该异常类型 |
| `(ValueError, TypeError)` | 捕获多个异常类型 |
| `False` | 不重试，抛出异常 |

**自定义错误处理函数：**

```python
def custom_error_handler(error: Exception) -> str:
    if isinstance(error, StructuredOutputValidationError):
        return "格式错误，请重试。"
    return f"错误: {str(error)}"

agent = create_agent(
    model="moonshot:moonshot-v1-8k",
    response_format=ToolStrategy(
        schema=ProductReview,
        handle_errors=custom_error_handler,
    ),
)
```

---

### strict 参数

**ProviderStrategy 支持 strict 模式：**

```python
agent = create_agent(
    model="moonshot:moonshot-v1-8k",
    response_format=ProviderStrategy(ContactInfo, strict=True),
)
```

**strict=True 要求：** 模型必须严格遵循 Schema，不允许额外字段。

### 嵌套结构

```python
from pydantic import BaseModel

class Actor(BaseModel):
    name: str
    role: str

class Movie(BaseModel):
    title: str
    year: int
    cast: list[Actor]
    genres: list[str]

model = ChatOpenAI(
    model="moonshot-v1-8k",
    openai_api_base="https://api.moonshot.cn/v1",
)
structured_model = model.with_structured_output(Movie)

result = structured_model.invoke("介绍电影《盗梦空间》")
```

### 为什么用它？

```python
import json

response = model.invoke("分析评论")
text = response.content
data = json.loads(text)  # 可能失败，格式不稳定

structured_model = model.with_structured_output(ProductReview)
data = structured_model.invoke("分析评论")  # 稳定的 Pydantic 对象
```

| 传统方式 | 结构化输出 |
|----------|------------|
| 解析 JSON 字符串 | 直接返回对象 |
| 格式不稳定 | Schema 保证格式 |
| 需手动验证 | Pydantic 自动验证 |
