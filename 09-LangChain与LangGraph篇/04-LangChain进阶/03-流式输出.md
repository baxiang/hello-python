# 09.04.03 - 流式输出

**流式输出：** 实时返回 LLM 生成的文本，而非等待完整响应。

```
流式输出：
┌──────────────────────────────────────┐
│  传统 invoke:                         │
│  用户 → 等待 → 完整响应               │
│                                       │
│  流式 stream:                         │
│  用户 → "你" → "好" → "！" → 完成     │
└──────────────────────────────────────┘
```

### 为什么用它？

- 减少用户等待时间
- 实时展示生成过程
- 更好的用户体验

---

### stream_mode 类型

```
stream_mode：
┌──────────────────────────────────────┐
│  updates  → 每步状态更新              │
│  messages → LLM token 流              │
│  custom   → 自定义数据                │
└──────────────────────────────────────┘
```

| Mode | 描述 | 用途 |
|------|------|------|
| `updates` | 每步状态更新 | 监控 Agent 进度 |
| `messages` | LLM token 流 | 实时展示文本 |
| `custom` | 自定义数据 | 工具内部推送 |

---

### 最简示例：LLM 流式

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(model="gpt-4o-mini")

for chunk in model.stream("介绍 Python"):
    print(chunk.content, end="", flush=True)
```

---

### Agent 流式：updates 模式

```python
from langchain.agents import create_agent

agent = create_agent(
    model="openai:gpt-4o-mini",
    tools=[get_weather],
)

for chunk in agent.stream(
    {"messages": [{"role": "user", "content": "北京天气？"}]},
    stream_mode="updates",
    version="v2",
):
    if chunk["type"] == "updates":
        for step, data in chunk["data"].items():
            print(f"步骤: {step}")
            if "messages" in data:
                msg = data["messages"][-1]
                for block in msg.content_blocks:
                    print(f"  {block}")
```

---

### Agent 流式：messages 模式

```python
for chunk in agent.stream(
    {"messages": [{"role": "user", "content": "北京天气？"}]},
    stream_mode="messages",
    version="v2",
):
    if chunk["type"] == "messages":
        token, metadata = chunk["data"]
        node = metadata["langgraph_node"]
        if token.text:
            print(f"[{node}] {token.text}")
```

### 自定义流式数据

**在工具中发送自定义数据：**

```python
from langchain.tools import tool
from langgraph.config import get_stream_writer

@tool
def search(query: str) -> str:
    writer = get_stream_writer()
    writer(f"搜索: {query}")
    writer(f"找到 10 条结果")
    return "搜索完成"
```

**接收自定义数据：**

```python
for chunk in agent.stream(
    {"messages": [{"role": "user", "content": "搜索新闻"}]},
    stream_mode="custom",
    version="v2",
):
    if chunk["type"] == "custom":
        print(chunk["data"])
```

---

### 多模式流式

**同时接收多种模式：**

```python
for chunk in agent.stream(
    {"messages": [{"role": "user", "content": "北京天气？"}]},
    stream_mode=["updates", "custom", "messages"],
    version="v2",
):
    if chunk["type"] == "updates":
        print(f"状态更新: {chunk['data']}")
    elif chunk["type"] == "messages":
        token, _ = chunk["data"]
        print(token.text, end="")
    elif chunk["type"] == "custom":
        print(f"自定义: {chunk['data']}")
```

---

### v2 格式

**LangGraph >= 1.1 支持 v2 格式：**

```python
for chunk in agent.stream(
    {"messages": [{"role": "user", "content": "北京天气？"}]},
    stream_mode="updates",
    version="v2",
):
    print(chunk["type"])   # "updates"
    print(chunk["data"])   # payload
```

**v2 优势：**

| 特性 | v1 | v2 |
|------|----|----|
| 格式 | `(mode, data)` 元组 | `{"type", "data"}` dict |
| invoke 返回 | dict | `.value` + `.interrupts` |
| 类型安全 | 无 | 类型推断 |

### 流式推理输出

**支持推理模型（如 Claude）：**

```python
from langchain_anthropic import ChatAnthropic

model = ChatAnthropic(
    model="claude-sonnet-4-6",
    thinking={"type": "enabled", "budget_tokens": 5000},
)

for token, _ in agent.stream(
    {"messages": [{"role": "user", "content": "解释量子计算"}]},
    stream_mode="messages",
):
    reasoning = [b for b in token.content_blocks if b["type"] == "reasoning"]
    text = [b for b in token.content_blocks if b["type"] == "text"]
    if reasoning:
        print(f"[推理] {reasoning[0]['reasoning']}", end="")
    if text:
        print(text[0]["text"], end="")
```

### 流式工具调用

**同时流式工具调用和执行结果：**

```python
def get_weather(city: str) -> str:
    return f"{city}: sunny"

agent = create_agent("openai:gpt-4o-mini", tools=[get_weather])

for chunk in agent.stream(
    {"messages": [{"role": "user", "content": "北京天气"}]},
    stream_mode=["messages", "updates"],
    version="v2",
):
    if chunk["type"] == "messages":
        token, _ = chunk["data"]
        if token.tool_call_chunks:
            print(f"工具调用片段: {token.tool_call_chunks}")
    elif chunk["type"] == "updates":
        for source, update in chunk["data"].items():
            if source == "model":
                print(f"完整工具调用: {update['messages'][-1].tool_calls}")
            if source == "tools":
                print(f"工具结果: {update['messages'][-1].content}")
```

### 消息块聚合

**流式消息聚合成完整 AIMessage：**

```python
full_message = None
for chunk in agent.stream(
    {"messages": [{"role": "user", "content": "你好"}]},
    stream_mode="messages",
):
    token, _ = chunk["data"]
    full_message = token if full_message is None else full_message + token

print(full_message.content)  # 完整内容
print(full_message.tool_calls)  # 完整工具调用
```

### 流式 vs invoke

| 方式 | 返回时机 | 用途 |
|------|----------|------|
| invoke | 完整响应后 | 批处理、API |
| stream | 逐 token | 实时 UI、聊天 |

```python
result = agent.invoke({"messages": [...]})  # 等待完整响应

for chunk in agent.stream({"messages": [...]}):  # 实时更新
    print(chunk)
```

### Human-in-the-loop 流式

**处理中断流式：**

```python
from langgraph.checkpoint.memory import InMemorySaver
from langgraph.types import Command

agent = create_agent(
    "openai:gpt-4o-mini",
    tools=[get_weather],
    middleware=[HumanInTheLoopMiddleware(interrupt_on={"get_weather": True})],
    checkpointer=InMemorySaver(),
)

config = {"configurable": {"thread_id": "id1"}}
interrupts = []

for chunk in agent.stream({"messages": [...]}, config=config, stream_mode="updates"):
    if chunk["type"] == "updates":
        for source, update in chunk["data"].items():
            if source == "__interrupt__":
                interrupts.extend(update)

# 响应中断
agent.stream(Command(resume={"decisions": [{"type": "approve"}]}), config=config)
```

---

### 流式总结

| 要点 | 说明 |
|------|------|
| stream_mode | updates/messages/custom |
| version="v2" | 统一格式，推荐使用 |
| get_stream_writer | 工具内发送自定义数据 |
| content_blocks | 标准化内容块格式 |
| 消息聚合 | 累加 token 成完整消息 |
