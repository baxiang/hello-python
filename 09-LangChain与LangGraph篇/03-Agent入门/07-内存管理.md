## 自定义状态

### 添加自定义字段

通过 middleware 和 state_schema 扩展状态：

```python
from langchain.agents import create_agent
from langchain.tools import tool, ToolRuntime

@tool
def save_preference(key: str, value: str, runtime: ToolRuntime) -> str:
    """保存用户偏好"""
    # 直接更新状态
    runtime.state[key] = value
    return f"偏好 '{key}' 已保存为 '{value}'"

@tool
def get_preference(key: str, runtime: ToolRuntime) -> str:
    """获取用户偏好"""
    value = runtime.state.get(key)
    return value if value else f"未找到偏好 '{key}'"

agent = create_agent(
    model="moonshot:moonshot-v1-8k",
    tools=[save_preference, get_preference],
)

# 使用
result = agent.invoke({
    "messages": [{"role": "user", "content": "保存语言偏好为中文"}]
})
```

### 状态持久化

```python
@tool
def get_all_state(runtime: ToolRuntime) -> str:
    """获取所有状态"""
    state = runtime.state
    items = []
    for key, value in state.items():
        if key != "messages":  # 跳过消息列表
            items.append(f"{key}: {value}")
    return "\n".join(items) if items else "无自定义状态"
```

## 短期记忆模式

### 对话历史截断

防止上下文过长：

```python
from langchain.agents.middleware import wrap_model_call

@wrap_model_call
def truncate_history(request, handler):
    """截断对话历史，保留最近 N 条消息"""
    max_messages = 10
    messages = request.state["messages"]
    
    if len(messages) > max_messages:
        # 保留最近的消息
        request = request.override(
            state={**request.state, "messages": messages[-max_messages:]}
        )
    
    return handler(request)
```

### 摘要记忆

将历史对话压缩为摘要：

```python
from langchain.chat_models import init_chat_model

summarizer = init_chat_model("moonshot:moonshot-v1-8k")

@wrap_model_call
def summarize_history(request, handler):
    """当消息过多时生成摘要"""
    messages = request.state["messages"]
    
    if len(messages) > 20:
        # 保留系统提示词和最近 5 条消息
        system_msgs = [m for m in messages if m.type == "system"]
        recent_msgs = messages[-5:]
        
        # 生成历史摘要
        old_messages = messages[len(system_msgs):-5]
        if old_messages:
            summary_prompt = (
                "用一句话总结以下对话历史：\n"
                + "\n".join(m.content for m in old_messages if m.content)
            )
            summary = summarizer.invoke(summary_prompt)
            
            # 用摘要替换旧消息
            from langchain.messages import SystemMessage
            new_messages = system_msgs + [
                SystemMessage(content=f"历史对话摘要: {summary.content}")
            ] + recent_msgs
            
            request = request.override(
                state={**request.state, "messages": new_messages}
            )
    
    return handler(request)
```

## 长期记忆

### 使用 Store 持久化

```python
from langchain.tools import tool, ToolRuntime

@tool
def save_long_term_memory(key: str, value: str, runtime: ToolRuntime) -> str:
    """保存长期记忆"""
    store = runtime.store
    user_id = runtime.context.user_id if runtime.context else "anonymous"
    
    store.put(("memories", user_id), key, value)
    return f"记忆 '{key}' 已保存"

@tool
def recall_memory(key: str, runtime: ToolRuntime) -> str:
    """回忆长期记忆"""
    store = runtime.store
    user_id = runtime.context.user_id if runtime.context else "anonymous"
    
    result = store.get(("memories", user_id), key)
    return result.value if result else f"未找到记忆 '{key}'"
```

## 记忆模式对比

| 模式 | 存储位置 | 生命周期 | 适用场景 |
|------|----------|----------|----------|
| 消息历史 | state["messages"] | 会话内 | 短期对话上下文 |
| 自定义状态 | state[key] | 会话内 | 会话级配置 |
| 摘要 | state["messages"] | 会话内 | 长对话压缩 |
| Store | 外部存储 | 持久化 | 用户偏好、历史 |
| 数据库 | 外部系统 | 持久化 | 业务数据 |

## 常见错误

### 错误1：状态过大

```python
# ❌ 错误：在状态中存储大量数据
runtime.state["search_results"] = huge_data  # 可能超过上下文限制

# ✅ 正确：只存储关键信息或引用
runtime.state["search_results_count"] = len(results)
runtime.state["search_results_ref"] = result_id  # 引用
```

---

### 错误2：忘记初始化

```python
# ❌ 错误：假设状态字段存在
@tool
def my_tool(runtime: ToolRuntime) -> str:
    count = runtime.state["counter"]  # KeyError: 'counter'

# ✅ 正确：使用 get 或初始化
@tool
def my_tool(runtime: ToolRuntime) -> str:
    count = runtime.state.get("counter", 0)
```

---

### 错误3：消息历史无限增长

```python
# ❌ 错误：不限制消息历史
# 长时间对话会导致上下文超出模型限制

# ✅ 正确：使用中间件限制
@wrap_model_call
def limit_messages(request, handler):
    messages = request.state["messages"]
    if len(messages) > 20:
        request = request.override(
            state={**request.state, "messages": messages[-20:]}
        )
    return handler(request)
```

## 小结

| 要点 | 说明 |
|------|------|
| AgentState | 默认包含 messages 字段 |
| 自定义状态 | 通过 runtime.state 读写 |
| 消息类型 | HumanMessage, AIMessage, ToolMessage 等 |

| 摘要记忆 | 压缩历史对话为摘要 |
