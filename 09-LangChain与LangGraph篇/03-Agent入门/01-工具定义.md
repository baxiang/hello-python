# 09.03.01 - 工具定义

**工具：** Agent 可以调用的函数，用于执行具体操作（搜索、计算、API调用等）。

```
工具定义：
┌──────────────────────────────────────┐
│  @tool                                │
│  def my_tool(param: str) -> str:     │
│      """工具描述"""                   │
│      return result                   │
└──────────────────────────────────────┘
```

### 最简示例：定义工具

```python
from langchain.tools import tool

@tool
def get_weather(city: str) -> str:
    """获取城市天气"""
    return f"{city} 今天天气晴朗"

# 使用工具
result = get_weather.invoke({"city": "北京"})
print(result)
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `@tool` | 工具装饰器 | 将函数转为 LangChain 工具 |
| `"""获取城市天气"""` | 工具描述 | LLM 根据描述选择工具 |
| `invoke({"city": "北京"})` | 调用工具 | 传入参数执行 |

### 工具参数验证

使用 Pydantic 验证参数：

```python
from langchain.tools import tool
from pydantic import BaseModel, Field

class WeatherInput(BaseModel):
    city: str = Field(description="城市名称")
    unit: str = Field(default="celsius", description="温度单位")

@tool(args_schema=WeatherInput)
def get_weather(city: str, unit: str = "celsius") -> str:
    """获取城市天气"""
    return f"{city} 温度: 25{unit}"
```

### 详细示例：多工具定义

```python
from langchain.tools import tool
import datetime

@tool
def get_current_time() -> str:
    """获取当前时间"""
    return datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S")

@tool
def calculate(expression: str) -> str:
    """计算数学表达式"""
    try:
        result = eval(expression)
        return str(result)
    except Exception as e:
        return f"计算错误: {e}"

@tool
def search_web(query: str) -> str:
    """搜索网络（模拟）"""
    return f"搜索结果: {query}"
```

### bind_tools：绑定工具到模型

将工具绑定到 LLM，让模型可以调用：

```python
from langchain.chat_models import init_chat_model
from langchain.tools import tool

@tool
def get_weather(city: str) -> str:
    """获取城市天气"""
    return f"{city} 天气晴朗"

model = init_chat_model("openai:gpt-4o-mini")
model_with_tools = model.bind_tools([get_weather])

response = model_with_tools.invoke("北京天气怎么样？")

if response.tool_calls:
    for tool_call in response.tool_calls:
        print(f"调用工具: {tool_call['name']}")
        print(f"参数: {tool_call['args']}")
```
