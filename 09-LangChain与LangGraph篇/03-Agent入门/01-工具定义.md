# 09.03.01 - 工具定义

**工具：** Agent 可以调用的函数，用于执行具体操作（搜索、计算、API调用等）。

```
工具定义：
┌──────────────────────────────────────┐
│  @tool                                │
│  def my_tool(param: str) -> str:     │
│      """工具描述（模型选择依据）"""   │
│      return result                   │
└──────────────────────────────────────┘
```

---

### 工具的作用

工具扩展 Agent 能力：
- 获取实时数据（天气、股票）
- 执行计算（数学运算）
- 查询数据库
- 调用外部 API
- 操作文件系统

---

## 创建工具

### @tool 装饰器

最简单的创建方式：

```python
from langchain.tools import tool

@tool
def get_weather(city: str) -> str:
    """获取城市天气信息"""
    return f"{city} 天气晴朗，25°C"
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `@tool` | 工具装饰器 | 将函数转为 LangChain 工具 |
| `"""获取城市天气"""` | 工具描述 | LLM 根据描述选择工具 |
| `city: str` | 类型提示 | 定义输入 schema（必须） |

---

### 自定义工具名称

```python
@tool("web_search")  # 自定义名称
def search(query: str) -> str:
    """搜索网络信息"""
    return f"搜索结果: {query}"

print(search.name)  # web_search
```

**命名建议：** 使用 `snake_case`（如 `web_search`），避免空格和特殊字符。

---

### 自定义描述

```python
@tool("calculator", description="执行数学计算。用于任何数学问题。")
def calc(expression: str) -> str:
    """计算数学表达式"""
    return str(eval(expression))
```

### Pydantic 参数验证

复杂参数使用 Pydantic 模型：

```python
from pydantic import BaseModel, Field
from typing import Literal
from langchain.tools import tool

class WeatherInput(BaseModel):
    """天气查询输入"""
    location: str = Field(description="城市名称或坐标")
    units: Literal["celsius", "fahrenheit"] = Field(
        default="celsius",
        description="温度单位"
    )
    include_forecast: bool = Field(
        default=False,
        description="是否包含5天预报"
    )

@tool(args_schema=WeatherInput)
def get_weather(
    location: str,
    units: str = "celsius",
    include_forecast: bool = False
) -> str:
    """获取当前天气和可选预报"""
    temp = 22 if units == "celsius" else 72
    result = f"{location} 当前温度: {temp}°{units[0].upper()}"
    if include_forecast:
        result += "\n未来5天: 晴天"
    return result
```

## ToolRuntime：访问上下文

工具可以通过 `ToolRuntime` 访问运行时信息：

```
ToolRuntime 提供：
┌──────────────────────────────────────┐
│  State       → 短期记忆               │
│  Context     → 运行时配置             │
│  Store       → 长期记忆               │
│  StreamWriter → 流式输出              │
│  ExecutionInfo → 执行信息             │
└──────────────────────────────────────┘
```

### 访问 State

```python
from langchain.tools import tool, ToolRuntime
from langchain.messages import HumanMessage

@tool
def get_last_user_message(runtime: ToolRuntime) -> str:
    """获取用户最后一条消息"""
    messages = runtime.state["messages"]
    
    for message in reversed(messages):
        if isinstance(message, HumanMessage):
            return message.content
    
    return "未找到用户消息"
```

### 访问 Store

```python
@tool
def get_user_info(user_id: str, runtime: ToolRuntime) -> str:
    """查询用户信息"""
    store = runtime.store
    user_info = store.get(("users",), user_id)
    return str(user_info.value) if user_info else "未知用户"
```

### 流式输出

```python
@tool
def get_weather(city: str, runtime: ToolRuntime) -> str:
    """获取城市天气"""
    writer = runtime.stream_writer
    
    writer(f"正在查询: {city}")  # 实时反馈
    writer(f"获取数据成功")
    
    return f"{city} 天气晴朗"
```

## ToolNode：工具执行节点

`ToolNode` 是预构建的工具执行节点，自动处理并行执行和错误：

```python
from langgraph.prebuilt import ToolNode
from langgraph.graph import StateGraph, MessagesState, START, END

@tool
def search(query: str) -> str:
    """搜索信息"""
    return f"结果: {query}"

@tool
def calculate(expr: str) -> str:
    """计算表达式"""
    return str(eval(expr))

tool_node = ToolNode([search, calculate])

builder = StateGraph(MessagesState)
builder.add_node("tools", tool_node)
```

---

### 错误处理

```python
from langgraph.prebuilt import ToolNode

# 默认：捕获调用错误，重抛执行错误
tool_node = ToolNode(tools)

# 捕获所有错误，返回错误消息给 LLM
tool_node = ToolNode(tools, handle_tool_errors=True)

# 自定义错误消息
tool_node = ToolNode(tools, handle_tool_errors="出错了，请重试")

# 自定义错误处理器
def handle_error(e: ValueError) -> str:
    return f"输入无效: {e}"

tool_node = ToolNode(tools, handle_tool_errors=handle_error)

# 捕获特定异常类型
tool_node = ToolNode(tools, handle_tool_errors=(ValueError, TypeError))
```

### 工具返回值类型

工具可以返回三种类型：

**返回字符串：**

```python
@tool
def get_weather(city: str) -> str:
    return f"{city} 天气晴朗"
```

**返回对象：**

```python
@tool
def get_weather_data(city: str) -> dict:
    return {
        "city": city,
        "temperature": 22,
        "conditions": "sunny",
    }
```

**返回 Command：**

```python
from langgraph.types import Command
from langchain.messages import ToolMessage

@tool
def set_language(language: str, runtime: ToolRuntime) -> Command:
    """设置语言偏好"""
    return Command(
        update={
            "preferred_language": language,
            "messages": [
                ToolMessage(
                    content=f"语言设置为 {language}",
                    tool_call_id=runtime.tool_call_id,
                )
            ],
        }
    )
```

## 绑定工具到模型

使用 `bind_tools` 让模型可以调用工具：

```python
from langchain_openai import ChatOpenAI
from langchain.tools import tool

@tool
def get_weather(city: str) -> str:
    """获取城市天气"""
    return f"{city} 天气晴朗"

model = ChatOpenAI(model="gpt-4o-mini")
model_with_tools = model.bind_tools([get_weather])

response = model_with_tools.invoke("北京天气怎么样？")

# 检查工具调用
for tool_call in response.tool_calls:
    print(f"工具: {tool_call['name']}")
    print(f"参数: {tool_call['args']}")
    print(f"ID: {tool_call['id']}")
```

---

### 工具执行循环

```python
# 第一步：模型生成工具调用
messages = [{"role": "user", "content": "北京天气？"}]
ai_msg = model_with_tools.invoke(messages)
messages.append(ai_msg)

# 第二步：执行工具
for tool_call in ai_msg.tool_calls:
    result = get_weather.invoke(tool_call)
    messages.append(result)

# 第三步：模型处理结果
final_response = model_with_tools.invoke(messages)
print(final_response.content)
```

---

### 强制工具调用

```python
# 强制使用任意工具
model.bind_tools([tool_1], tool_choice="any")

# 强制使用特定工具
model.bind_tools([tool_1], tool_choice="tool_1")
```

---

### 并行工具调用

模型可以同时调用多个工具：

```python
response = model_with_tools.invoke("北京和上海的天气？")

# 可能返回多个工具调用
print(response.tool_calls)
# [
#   {'name': 'get_weather', 'args': {'city': '北京'}, 'id': 'call_1'},
#   {'name': 'get_weather', 'args': {'city': '上海'}, 'id': 'call_2'},
# ]
```

---

### 流式工具调用

```python
for chunk in model_with_tools.stream("北京天气？"):
    for tool_chunk in chunk.tool_call_chunks:
        if name := tool_chunk.get("name"):
            print(f"工具: {name}")
        if args := tool_chunk.get("args"):
            print(f"参数片段: {args}")

# 累积完整工具调用
gathered = None
for chunk in model_with_tools.stream("北京天气？"):
    gathered = chunk if gathered is None else gathered + chunk
print(gathered.tool_calls)  # 完整工具调用列表
```
