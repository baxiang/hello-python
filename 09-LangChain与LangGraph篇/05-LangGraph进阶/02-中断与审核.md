# 09.05.02 - 中断与审核

**Interrupt：** 暂停图执行，等待外部输入（Human-in-the-loop）。

```
中断流程：
┌──────────────────────────────────────┐
│  执行 → interrupt() → 暂停           │
│           ↓                          │
│  外部输入 → Command(resume) → 继续   │
└──────────────────────────────────────┘
```

### 为什么用它？

- 审批关键操作（转账、删除）
- 人工纠正 AI 输出
- 确认敏感操作

---

### interrupt() 函数

```python
from langgraph.types import interrupt

def approval_node(state: State):
    answer = interrupt("是否批准此操作？")
    
    if answer:
        return {"status": "approved"}
    return {"status": "rejected"}
```

### 最简示例：审批流程

```python
from typing import TypedDict
from langgraph.graph import StateGraph, START, END
from langgraph.types import interrupt, Command
from langgraph.checkpoint.memory import MemorySaver

class State(TypedDict):
    action: str
    status: str

def request_node(state: State):
    return state

def approval_node(state: State):
    answer = interrupt({
        "question": "是否批准？",
        "action": state["action"],
    })
    
    return {"status": "approved" if answer else "rejected"}

builder = StateGraph(State)
builder.add_node("request", request_node)
builder.add_node("approval", approval_node)
builder.add_edge(START, "request")
builder.add_edge("request", "approval")
builder.add_edge("approval", END)

checkpointer = MemorySaver()
graph = builder.compile(checkpointer=checkpointer)

# 第一次调用：触发中断
config = {"configurable": {"thread_id": "123"}}
result = graph.invoke({"action": "转账100元", "status": ""}, config)
print(result["__interrupt__"])

# 恢复执行：传入回答
result = graph.invoke(Command(resume=True), config)
print(result["status"])
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `interrupt()` | 暂停执行 | 等待人工输入 |
| `Command(resume=True)` | 恢复执行 | 传入回答继续 |
| `checkpointer` | 状态保存 | 中断时保存状态 |
| `thread_id` | 会话标识 | 区分不同对话 |

### 审批流程

```python
def approval_node(state: State) -> Command:
    answer = interrupt("批准或拒绝？")
    
    if answer:
        return Command(goto="proceed")
    return Command(goto="cancel")
```

### 多中断处理

并行节点可能同时中断：

```python
# 处理多个中断
for interrupt_info in result["__interrupt__"]:
    resume_map[interrupt_info.id] = get_answer(interrupt_info.value)

# 批量恢复
result = graph.invoke(Command(resume=resume_map), config)
```

### 工具中的中断

```python
from langchain.tools import tool
from langgraph.types import interrupt

@tool
def send_email(to: str, subject: str, body: str):
    """发送邮件"""
    response = interrupt({
        "action": "send_email",
        "to": to,
        "message": "批准发送？",
    })
    
    if response.get("approve"):
        return f"邮件已发送给 {to}"
    return "邮件已取消"
```
