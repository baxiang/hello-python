# 09.05.01 - 记忆管理

**记忆：** Agent 执行过程中的上下文保存，分为短期记忆和长期记忆。

```
记忆类型：
┌──────────────────────────────────────┐
│  短期记忆 (State) → 当前对话上下文    │
│  长期记忆 (Store) → 范久化存储        │
│                                       │
│  semantic    → 语义记忆（事实）       │
│  episodic    → 事件记忆（经历）       │
│  procedural  → 程序记忆（技能）       │
└──────────────────────────────────────┘
```

### 短期记忆 (State)

State 是图执行过程中的共享状态，用于存储当前对话上下文。

```python
from typing import Annotated, TypedDict
from langgraph.graph import MessagesState
from operator import add

class State(MessagesState):
    documents: list[str]  # 累积文档
    summary: str          # 当前摘要
```

---

### MessagesState

LangGraph 提供预置的 `MessagesState`，自动处理消息追加：

```python
from langgraph.graph import MessagesState

class State(MessagesState):
    user_id: str
    documents: list[str]
```

**MessagesState 内置：**
- `messages: Annotated[list[AnyMessage], add_messages]`
- 自动处理消息 ID 追踪
- 支持消息更新和追加

---

### 长期记忆 (Store)

Store 用于持久化存储，跨对话保持上下文。

```python
from langgraph.store.memory import InMemoryStore

store = InMemoryStore()

# 存储数据
store.put(namespace=["user", "123"], key="profile", value={"name": "张三"})

# 读取数据
profile = store.get(namespace=["user", "123"], key="profile")
print(profile.value)
```

---

### 详细示例：带记忆的对话

```python
from langgraph.graph import StateGraph, START, END, MessagesState
from langgraph.store.memory import InMemoryStore
from langchain.messages import HumanMessage, AIMessage
from langchain_openai import ChatOpenAI

class State(MessagesState):
    user_id: str

def chat_node(state: State, store: InMemoryStore):
    # 第一步：获取用户历史
    history = store.get(namespace=["history", state["user_id"]], key="context")
    
    # 第二步：构建上下文
    messages = state["messages"]
    if history:
        messages = history.value + messages
    
    # 第三步：调用 LLM
    model = ChatOpenAI(model="gpt-4o-mini")
    response = model.invoke(messages)
    
    # 第四步：保存历史
    store.put(namespace=["history", state["user_id"]], key="context", value=messages)
    
    return {"messages": [response]}

store = InMemoryStore()
builder = StateGraph(State)
builder.add_node("chat", chat_node)
builder.add_edge(START, "chat")
builder.add_edge("chat", END)

graph = builder.compile(store=store)
```

### 记忆类型

```
记忆分类：
┌──────────────────────────────────────┐
│  semantic    → 事实知识               │
│              例："北京是首都"         │
│                                       │
│  episodic    → 事件经历               │
│              例："昨天问了天气"       │
│                                       │
│  procedural → 操作技能                │
│              例："如何计算"           │
└──────────────────────────────────────┘
```
