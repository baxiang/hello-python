# 09.05.05 - 持久化

> MemorySaver、SqliteSaver、PostgresSaver、异步 checkpointers、加密序列化

## 持久化概述

持久化使用 Checkpointer 保存图状态，支持恢复、重放和跨线程记忆：

```
持久化架构：
┌──────────────────────────────────────┐
│  Checkpointer                         │
│                                       │
│  thread_id   → 会话标识               │
│  checkpoint  → 状态快照               │
│  checkpoint_id → 特定快照ID           │
│  Store       → 跨线程存储             │
└──────────────────────────────────────┘
```

### 为什么需要持久化？

| 需求 | 无持久化 | 有持久化 |
|------|---------|---------|
| 多轮对话 | 每次独立 | 保持上下文 |
| 中断恢复 | 无法恢复 | 从断点继续 |
| 调试回溯 | 无法回溯 | Time Travel |
| 故障恢复 | 丢失状态 | 自动恢复 |
| 跨会话记忆 | 无 | Store 支持 |

---

## Checkpointer 类型

```
Checkpointer 类型：
┌──────────────────────────────────────┐
│  MemorySaver      → 内存（开发用）    │
│  SqliteSaver      → SQLite           │
│  PostgresSaver    → PostgreSQL       │
│  AsyncSqliteSaver → 异步SQLite       │
│  AsyncPostgresSaver → 异步PostgreSQL │
└──────────────────────────────────────┘
```

| Checkpointer | 用途 | 安装 | 说明 |
|--------------|------|------|------|
| MemorySaver | 开发测试 | 内置 | 数据不持久，重启丢失 |
| SqliteSaver | 本地应用 | `uv add langgraph-checkpoint-sqlite` | 单文件数据库 |
| PostgresSaver | 生产环境 | `uv add langgraph-checkpoint-postgres` | LangSmith 默认 |

## MemorySaver

### 基本用法

```python
from langgraph.checkpoint.memory import MemorySaver

checkpointer = MemorySaver()
graph = builder.compile(checkpointer=checkpointer)

config = {"configurable": {"thread_id": "session-123"}}
result = graph.invoke({"input": "test"}, config)
```

### Thread：会话

```python
config = {"configurable": {"thread_id": "user-456"}}

# 同一个 thread_id 保持上下文
result1 = graph.invoke({"messages": ["你好"]}, config)
result2 = graph.invoke({"messages": ["再见"]}, config)

# 不同 thread_id 是新会话
config2 = {"configurable": {"thread_id": "user-789"}}
result3 = graph.invoke({"messages": ["你好"]}, config2)
```

## Checkpoint：状态快照

### 执行流程

```
执行流程：
START → node_a → node_b → END

Checkpoint:
1. 空 checkpoint，next=START
2. node_a 输出，next=node_b
3. node_b 输出，next=()
```

### 获取状态

```python
config = {"configurable": {"thread_id": "1"}}
state = graph.get_state(config)

print(state.values)       # {'foo': 'b', 'bar': ['a', 'b']}
print(state.next)         # () 表示完成
print(state.config)       # {'thread_id': '1', 'checkpoint_id': '...'}
print(state.metadata)     # {'step': 2, 'source': 'loop'}
```

### 获取特定 checkpoint

```python
config = {"configurable": {
    "thread_id": "1",
    "checkpoint_id": "1ef663ba-...",
}}
graph.get_state(config)
```

### StateSnapshot 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| values | dict | State 值 |
| next | tuple | 下一个要执行的节点 |
| config | dict | thread_id + checkpoint_id |
| metadata | dict | step, source, writes |
| created_at | str | ISO 时间戳 |
| parent_config | dict | 父 checkpoint 配置 |
| tasks | tuple | 待执行任务 |

## 状态历史

### 获取历史

```python
history = list(graph.get_state_history(config))

for state in history:
    print(f"step: {state.metadata['step']}")
    print(f"values: {state.values}")
    print(f"next: {state.next}")

# 查找特定 checkpoint
before_node_b = next(s for s in history if s.next == ("node_b",))
```

### 重放

```python
config = {"configurable": {
    "thread_id": "1",
    "checkpoint_id": "before_node_b",
}}

# 从该 checkpoint 重新执行
graph.invoke(None, config)
```

## 更新状态

```python
config = {"configurable": {"thread_id": "1"}}

# 更新状态
graph.update_state(config, {"foo": "new_value"})

# 指定来源节点
graph.update_state(config, {"foo": "value"}, as_node="node_a")
```

## SqliteSaver

### 安装

```bash
uv add langgraph-checkpoint-sqlite
```

### 使用

```python
import sqlite3
from langgraph.checkpoint.sqlite import SqliteSaver

conn = sqlite3.connect("checkpoint.db")
checkpointer = SqliteSaver(conn)

graph = builder.compile(checkpointer=checkpointer)

config = {"configurable": {"thread_id": "1"}}
result = graph.invoke({"messages": ["你好"]}, config)

# 关闭连接
conn.close()
```

### 上下文管理器

```python
from langgraph.checkpoint.sqlite import SqliteSaver

with sqlite3.connect("checkpoint.db") as conn:
    checkpointer = SqliteSaver(conn)
    graph = builder.compile(checkpointer=checkpointer)
    
    config = {"configurable": {"thread_id": "1"}}
    result = graph.invoke({"messages": ["你好"]}, config)
```

## PostgresSaver

### 安装

```bash
uv add langgraph-checkpoint-postgres
```

### 同步版本

```python
from langgraph.checkpoint.postgres import PostgresSaver

conn_string = "postgresql://user:pass@host:5432/db"
checkpointer = PostgresSaver.from_conn_string(conn_string)
checkpointer.setup()

graph = builder.compile(checkpointer=checkpointer)
```

### 异步版本

```python
from langgraph.checkpoint.postgres.aio import AsyncPostgresSaver

checkpointer = AsyncPostgresSaver.from_conn_string(conn_string)
await checkpointer.setup()

graph = builder.compile(checkpointer=checkpointer)

# 异步调用
result = await graph.ainvoke({"messages": ["你好"]}, config)
```

### 生产配置

```python
from langgraph.checkpoint.postgres import PostgresSaver

checkpointer = PostgresSaver.from_conn_string(
    "postgresql://user:pass@host:5432/db",
    # 连接池配置
    # 根据需要配置
)
checkpointer.setup()
```

## 加密

### 启用状态加密

```bash
export LANGGRAPH_AES_KEY="your-32-byte-key!!"
```

```python
from langgraph.checkpoint.serde.encrypted import EncryptedSerializer

# 读取 LANGGRAPH_AES_KEY 环境变量
serde = EncryptedSerializer.from_pycryptodome_aes()

# 用于 PostgresSaver
checkpointer = PostgresSaver.from_conn_string(
    "postgresql://user:pass@host:5432/db",
    serde=serde,
)
```

### 加密 Checkpointer 对比

| 特性 | 未加密 | 加密 |
|------|--------|------|
| 数据可见 | 明文 | AES 加密 |
| 性能 | 快 | 稍慢 |
| 安全 | 低 | 高 |
| 适用 | 开发 | 生产 |

## Checkpoint Namespace

## 异步 Checkpointer

### AsyncSqliteSaver

```python
from langgraph.checkpoint.sqlite.aio import AsyncSqliteSaver
import aiosqlite

async with aiosqlite.connect("checkpoint.db") as conn:
    checkpointer = AsyncSqliteSaver(conn)
    graph = builder.compile(checkpointer=checkpointer)
    
    config = {"configurable": {"thread_id": "1"}}
    result = await graph.ainvoke({"messages": ["你好"]}, config)
```

### 异步操作

```python
# 异步获取状态
state = await graph.aget_state(config)

# 异步获取历史
history = [s async for s in graph.aget_state_history(config)]

# 异步更新状态
await graph.aupdate_state(config, {"foo": "value"})
```

## 持久化最佳实践

### 1. 开发环境

```python
from langgraph.checkpoint.memory import MemorySaver

checkpointer = MemorySaver()  # 简单，无需配置
```

### 2. 本地应用

```python
from langgraph.checkpoint.sqlite import SqliteSaver
import sqlite3

conn = sqlite3.connect("checkpoint.db", check_same_thread=False)
checkpointer = SqliteSaver(conn)
```

### 3. 生产环境

```python
from langgraph.checkpoint.postgres import PostgresSaver

checkpointer = PostgresSaver.from_conn_string(DATABASE_URL)
checkpointer.setup()
```

### 4. 加密生产

```python
from langgraph.checkpoint.postgres import PostgresSaver
from langgraph.checkpoint.serde.encrypted import EncryptedSerializer

serde = EncryptedSerializer.from_pycryptodome_aes()
checkpointer = PostgresSaver.from_conn_string(DATABASE_URL, serde=serde)
checkpointer.setup()
```

## Checkpointer vs Store

| 特性 | Checkpointer | Store |
|------|-------------|-------|
| 用途 | 状态持久化 | 长期记忆 |
| 作用域 | 按 thread | 全局 |
| 结构 | checkpoint 快照 | namespace/key/value |
| 搜索 | 无 | 语义搜索 |
| 必需 | 中断/多轮必需 | 可选 |

---

## 小结

| 要点 | 说明 |
|------|------|
| Checkpointer | 保存每步状态快照 |
| thread_id | 会话标识，必需 |
| MemorySaver | 开发用，数据不持久 |
| SqliteSaver | 本地持久化，单文件 |
| PostgresSaver | 生产环境，支持异步 |
| 加密 | EncryptedSerializer + AES |
| 状态管理 | get_state / get_state_history / update_state |
| 重放 | 从 checkpoint_id 重新执行 |
