# 09.05.02 - 长期记忆

> BaseStore、InMemoryStore、PostgresStore、namespace 模式、跨会话持久化

## 长期记忆概述

长期记忆跨对话共享，存储在自定义 namespace 中：

```
长期记忆结构：
┌──────────────────────────────────────┐
│  Store.put(namespace, key, value)    │
│  Store.get(namespace, key)           │
│  Store.search(namespace, query)      │
│  Store.delete(namespace, key)        │
│                                       │
│  namespace → 类似文件夹路径           │
│  key      → 类似文件名                │
│  value    → JSON 可序列化的值         │
└──────────────────────────────────────┘
```

### 短期 vs 长期记忆

| 特性 | 短期记忆（State） | 长期记忆（Store） |
|------|------------------|------------------|
| 生命周期 | 单次执行 | 持久化 |
| 作用域 | 当前 thread | 全局/跨 thread |
| 持久化 | Checkpointer | Store 后端 |
| 数据结构 | TypedDict | namespace/key/value |
| 搜索 | 无 | 语义搜索 |
| 适用 | 对话上下文 | 用户偏好、知识库 |

## Store 基础

### InMemoryStore

```python
from langgraph.store.memory import InMemoryStore

store = InMemoryStore()

# 存储
namespace = ("user", "123", "preferences")
store.put(namespace, "profile", {
    "name": "张三",
    "language": "zh-CN",
    "interests": ["Python", "AI"],
})

# 读取
profile = store.get(namespace, "profile")
print(profile.value)  # {"name": "张三", ...}

# 搜索
results = store.search(("user", "123"))
for r in results:
    print(r.namespace, r.key, r.value)

# 删除
store.delete(namespace, "profile")
```

### Namespace 模式

```
Namespace 结构：
┌──────────────────────────────────────┐
│  ("user", "123", "preferences")      │
│    ↓                                  │
│  user/                               │
│    123/                              │
│      preferences/                    │
│        profile                       │
│        settings                      │
│                                       │
│  优势：                               │
│  - 层次化管理                         │
│  - 批量搜索                           │
│  - 权限隔离                           │
└──────────────────────────────────────┘
```

### Namespace 搜索

```python
# 搜索特定 namespace 下所有内容
results = store.search(("user", "123"))

# 搜索子 namespace
results = store.search(("user", "123", "preferences"))

# 跨用户搜索
results = store.search(("user",))
```

## 语义搜索

### 配置向量索引

```python
from langgraph.store.memory import InMemoryStore
from langchain.embeddings import init_embeddings

store = InMemoryStore(
    index={
        "embed": init_embeddings("zhipuai:embedding-2"),
        "dims": 1024,
        "fields": ["$"],  # 嵌入所有字段
    }
)

# 存储文档
docs = [
    {"title": "Python 基础", "content": "Python 是一种编程语言..."},
    {"title": "Python 函数", "content": "函数是可重复使用的代码块..."},
    {"title": "JavaScript 基础", "content": "JavaScript 是网页脚本..."},
]

for i, doc in enumerate(docs):
    store.put(("docs",), f"doc_{i}", doc)

# 语义搜索
results = store.search(
    ("docs",),
    query="什么是函数",  # 语义相似度
    limit=2,
)

for r in results:
    print(f"标题: {r.value['title']}")
    print(f"相似度: {r.score}")
```

### 搜索选项

| 参数 | 类型 | 说明 |
|------|------|------|
| `query` | `str` | 语义搜索查询 |
| `filter` | `dict` | 字段过滤条件 |
| `limit` | `int` | 返回结果数 |
| `offset` | `int` | 偏移量 |

### 组合搜索

```python
# 语义 + 过滤
results = store.search(
    ("user", "123", "memories"),
    query="编程偏好",
    filter={"type": "fact"},
    limit=5,
)
```

## 记忆类型

### Semantic Memory：语义记忆

存储事实和知识：

```python
# Profile 模式：单一文档持续更新
def update_user_profile(store, user_id, updates):
    """更新用户画像"""
    namespace = ("user", user_id)
    
    # 获取旧数据
    old = store.get(namespace, "profile")
    
    if old:
        # 合并
        new_profile = {**old.value, **updates}
    else:
        new_profile = updates
    
    store.put(namespace, "profile", new_profile)

# 使用
update_user_profile(store, "123", {"name": "张三"})
update_user_profile(store, "123", {"language": "zh-CN"})
# 结果: {"name": "张三", "language": "zh-CN"}
```

```python
# Collection 模式：多文档集合
def add_memory(store, user_id, fact):
    """添加记忆"""
    import uuid
    namespace = ("user", user_id, "memories")
    memory_id = str(uuid.uuid4())
    
    store.put(namespace, memory_id, {
        "fact": fact,
        "timestamp": "2024-01-01T00:00:00",
    })

# 使用
add_memory(store, "123", "用户喜欢 Python")
add_memory(store, "123", "用户使用 macOS")

# 搜索相关记忆
results = store.search(
    ("user", "123", "memories"),
    query="编程偏好",
    limit=3,
)
```

### Episodic Memory：事件记忆

存储过往经验和行动：

```python
# 存储任务执行历史
def store_task_history(store, task_input, task_output):
    """存储任务历史"""
    namespace = ("agent", "tasks", "successful")
    
    import uuid
    task_id = str(uuid.uuid4())
    
    store.put(namespace, task_id, {
        "input": task_input,
        "output": task_output,
        "timestamp": "2024-01-01T00:00:00",
    })

# 作为 few-shot 示例
def get_similar_tasks(store, query):
    """获取相似任务作为示例"""
    namespace = ("agent", "tasks", "successful")
    
    results = store.search(namespace, query=query, limit=3)
    
    examples = []
    for r in results:
        examples.append({
            "input": r.value["input"],
            "output": r.value["output"],
        })
    
    return examples
```

### Procedural Memory：程序记忆

存储执行规则和指令：

```python
def update_system_instructions(store, feedback):
    """根据反馈更新系统指令"""
    from langchain.chat_models import init_chat_model
    
    namespace = ("agent", "instructions")
    
    # 获取当前指令
    current = store.get(namespace, "system_prompt")
    
    if current:
        model = init_chat_model("moonshot:moonshot-v1-8k")
        
        # 优化指令
        new_prompt = model.invoke(f"""
        当前指令: {current.value['instructions']}
        用户反馈: {feedback}
        请优化指令使其更好地响应用户需求。
        """)
        
        store.put(namespace, "system_prompt", {
            "instructions": new_prompt.content
        })
```

## 在 Agent 中使用 Store

### 编译时配置 Store

```python
from langgraph.graph import StateGraph, MessagesState, START, END
from langgraph.store.memory import InMemoryStore
from langgraph.checkpoint.memory import MemorySaver
from langchain.chat_models import init_chat_model
from dataclasses import dataclass

@dataclass
class Context:
    user_id: str

store = InMemoryStore()

def chat_node(state: MessagesState, store: InMemoryStore):
    """带长期记忆的对话节点"""
    # 获取用户 ID
    user_id = state.get("user_id", "default")
    
    # 加载用户偏好
    prefs = store.get(("user", user_id), "preferences")
    
    # 构建上下文
    context = ""
    if prefs:
        context = f"用户偏好: {prefs.value}"
    
    # 调用模型
    model = init_chat_model("moonshot:moonshot-v1-8k")
    messages = state["messages"]
    if context:
        from langchain.messages import SystemMessage
        messages = [SystemMessage(context)] + messages
    
    response = model.invoke(messages)
    
    # 保存交互（可选）
    if "我喜欢" in state["messages"][-1].content:
        store.put(
            ("user", user_id, "memories"),
            f"mem_{len(state['messages'])}",
            {"fact": state["messages"][-1].content}
        )
    
    return {"messages": [response]}

# 构建图
builder = StateGraph(MessagesState, context_schema=Context)
builder.add_node("chat", chat_node)
builder.add_edge(START, "chat")
builder.add_edge("chat", END)

checkpointer = MemorySaver()
graph = builder.compile(checkpointer=checkpointer, store=store)

# 调用
config = {"configurable": {"thread_id": "session-1"}}
context = Context(user_id="user-123")

result = graph.invoke(
    {"messages": ["你好"]},
    config=config,
    context=context,
)
```

### Runtime 中访问 Store

```python
from langgraph.runtime import Runtime
from dataclasses import dataclass

@dataclass
class AppContext:
    user_id: str

async def save_memory(state: MessagesState, runtime: Runtime[AppContext]):
    """保存记忆"""
    user_id = runtime.context.user_id
    namespace = (user_id, "memories")
    
    import uuid
    memory_id = str(uuid.uuid4())
    
    await runtime.store.aput(
        namespace,
        memory_id,
        {"memory": state["messages"][-1].content}
    )

async def load_memory(state: MessagesState, runtime: Runtime[AppContext]):
    """加载记忆"""
    user_id = runtime.context.user_id
    namespace = (user_id, "memories")
    
    memories = await runtime.store.asearch(
        namespace,
        query="用户偏好",
        limit=3,
    )
    
    if memories:
        context = "用户记忆:\n" + "\n".join(
            f"- {m.value.get('memory', '')}" for m in memories
        )
        return {"context": context}
    
    return None
```

## Store vs Checkpointer

| 特性 | Store | Checkpointer |
|------|-------|-------------|
| 用途 | 长期记忆 | 短期状态持久化 |
| 作用域 | 全局 | 按 thread |
| 数据结构 | namespace/key/value | checkpoint 快照 |
| 搜索 | 支持语义搜索 | 不支持 |
| 跨会话 | 是 | 否（仅同 thread） |
| 典型内容 | 用户偏好、知识库 | 对话状态、中断点 |

### 组合使用

```python
from langgraph.checkpoint.memory import MemorySaver
from langgraph.store.memory import InMemoryStore

checkpointer = MemorySaver()  # 短期记忆
store = InMemoryStore()       # 长期记忆

graph = builder.compile(
    checkpointer=checkpointer,
    store=store,
)
```

## 最佳实践

### 1. Namespace 设计

```python
# 推荐模式
("user", user_id, "preferences")   # 用户偏好
("user", user_id, "memories")      # 用户记忆
("user", user_id, "history")       # 用户历史
("agent", "tasks", "successful")   # 成功任务
("agent", "instructions")          # 系统指令

# 避免
("data", ...)  # 太泛
("user_123_prefs")  # 不利用层次
```

### 2. 写入策略

```python
# 热路径：实时写入
def hot_path_write(state, store, user_id):
    """响应用户时写入"""
    if is_important(state["messages"][-1]):
        store.put(
            ("user", user_id, "memories"),
            f"mem_{time.time()}",
            extract_fact(state),
        )

# 后台：异步写入
async def background_write(store, conversations):
    """后台任务定期写入"""
    for conv in conversations:
        summary = summarize(conv)
        await store.aput(
            ("memories", conv.user_id),
            f"ep_{conv.id}",
            summary,
        )
```

### 3. 搜索优化

```python
# 限制搜索结果
results = store.search(namespace, query=query, limit=5)

# 使用过滤减少搜索范围
results = store.search(
    namespace,
    query=query,
    filter={"type": "fact"},
    limit=5,
)
```

## 小结

| 要点 | 说明 |
|------|------|
| Store | 长期记忆，跨对话共享 |
| Namespace | 层次化组织，类似文件夹路径 |
| 搜索 | 语义搜索 + 字段过滤 |
| 记忆类型 | Semantic（事实）、Episodic（事件）、Procedural（规则） |
| 后端 | InMemoryStore（开发）、PostgresStore（生产） |
| 写入策略 | 热路径（实时）、后台（异步） |
| 组合使用 | Store（长期）+ Checkpointer（短期） |
