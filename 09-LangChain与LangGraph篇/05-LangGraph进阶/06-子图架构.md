# 09.05.06 - 子图架构

> 嵌套 StateGraph、子图调用、父子通信、Command.PARENT

## 子图概述

子图允许将复杂图分解为可复用的子模块：

```
子图架构：
┌──────────────────────────────────────┐
│  父图 (Supervisor)                    │
│  ┌────────────────────────────────┐  │
│  │  子图 A (Research)              │  │
│  │  ├─ search_node                │  │
│  │  └─ analyze_node               │  │
│  └────────────────────────────────┘  │
│  ┌────────────────────────────────┐  │
│  │  子图 B (Writer)                │  │
│  │  ├─ draft_node                 │  │
│  │  └─ review_node                │  │
│  └────────────────────────────────┘  │
│                                       │
│  父图协调子图执行                      │
└──────────────────────────────────────┘
```

### 为什么用子图？

| 优势 | 说明 |
|------|------|
| 模块化 | 独立开发和测试子图 |
| 复用 | 多个父图共享子图 |
| 隔离 | 子图有自己的 State |
| 层次 | 清晰的调用关系 |

## 嵌套 StateGraph

### 基本子图

```python
from typing import TypedDict
from langgraph.graph import StateGraph, START, END

# 子图状态
class ResearchState(TypedDict):
    query: str
    results: list[str]

# 子图节点
def search_node(state: ResearchState):
    return {"results": [f"搜索结果: {state['query']}"]}

def analyze_node(state: ResearchState):
    return {"results": [*state["results"], "分析完成"]}

# 构建子图
research_builder = StateGraph(ResearchState)
research_builder.add_node("search", search_node)
research_builder.add_node("analyze", analyze_node)
research_builder.add_edge(START, "search")
research_builder.add_edge("search", "analyze")
research_builder.add_edge("analyze", END)

research_graph = research_builder.compile()
```

### 在父图中调用子图

```python
from typing import TypedDict
from langgraph.graph import StateGraph, START, END

# 父图状态
class ParentState(TypedDict):
    query: str
    research_results: list[str]
    final_output: str

# 调用子图
def research_step(state: ParentState):
    """调用研究子图"""
    # 子图输入
    subgraph_input = {"query": state["query"]}
    
    # 调用子图
    result = research_graph.invoke(subgraph_input)
    
    # 返回父图状态更新
    return {"research_results": result["results"]}

def write_step(state: ParentState):
    """写作步骤"""
    return {"final_output": f"基于 {state['research_results']} 生成报告"}

# 构建父图
builder = StateGraph(ParentState)
builder.add_node("research", research_step)
builder.add_node("write", write_step)
builder.add_edge(START, "research")
builder.add_edge("research", "write")
builder.add_edge("write", END)

graph = builder.compile()

# 调用
result = graph.invoke({"query": "Python 异步编程"})
print(result["final_output"])
```

## 子图作为节点

### 直接添加子图为节点

```python
from langgraph.graph import StateGraph, START, END

# 父图状态（需要包含子图状态）
class CombinedState(TypedDict):
    query: str
    results: list[str]
    final_output: str

# 构建父图
builder = StateGraph(CombinedState)

# 直接添加子图为节点
builder.add_node("research", research_graph)
builder.add_node("write", write_step)

builder.add_edge(START, "research")
builder.add_edge("research", "write")
builder.add_edge("write", END)

graph = builder.compile()

# 调用
result = graph.invoke({"query": "Python 异步编程", "results": []})
```

### 状态映射

当子图和父图状态不完全匹配时，需要映射：

```python
def research_wrapper(state: ParentState):
    """子图包装器：映射状态"""
    # 父图 → 子图
    subgraph_input = {
        "query": state["query"],
        "results": [],  # 子图初始值
    }
    
    # 调用子图
    result = research_graph.invoke(subgraph_input)
    
    # 子图 → 父图
    return {"research_results": result["results"]}
```

## 父子通信

### Command.GOTO 子图

```python
from langgraph.types import Command
from typing import Literal

def supervisor(state: ParentState) -> Command[Literal["research", "write", END]]:
    """监督者节点"""
    if not state.get("research_results"):
        return Command(goto="research")
    elif not state.get("final_output"):
        return Command(goto="write")
    return Command(goto=END)
```

### Command.PARENT

子图返回父图：

```python
from langgraph.types import Command

def child_node(state: ResearchState) -> Command[Literal["__end__"]]:
    """子图节点返回父图"""
    # 处理完成后返回父图
    return Command(goto="__end__")
```

### 子图向父图传递数据

```python
def child_result_node(state: ResearchState):
    """子图结果节点"""
    # 返回结果，父图可以访问
    return {
        "results": state["results"],
        "summary": "研究完成",
    }
```

## 多层嵌套

```python
# 子子图
class SubSubState(TypedDict):
    data: str
    processed: str

sub_sub_builder = StateGraph(SubSubState)
sub_sub_builder.add_node("process", lambda s: {"processed": f"处理: {s['data']}"})
sub_sub_builder.add_edge(START, "process")
sub_sub_builder.add_edge("process", END)
sub_sub_graph = sub_sub_builder.compile()

# 子图
class SubState(TypedDict):
    input: str
    sub_result: str

def call_sub_sub(state: SubState):
    result = sub_sub_graph.invoke({"data": state["input"]})
    return {"sub_result": result["processed"]}

sub_builder = StateGraph(SubState)
sub_builder.add_node("process", call_sub_sub)
sub_builder.add_edge(START, "process")
sub_builder.add_edge("process", END)
sub_graph = sub_builder.compile()

# 父图
class ParentState(TypedDict):
    query: str
    final_result: str

def call_sub(state: ParentState):
    result = sub_graph.invoke({"input": state["query"]})
    return {"final_result": result["sub_result"]}

builder = StateGraph(ParentState)
builder.add_node("process", call_sub)
builder.add_edge(START, "process")
builder.add_edge("process", END)

graph = builder.compile()
result = graph.invoke({"query": "测试数据"})
print(result["final_result"])  # "处理: 测试数据"
```

## 子图持久化

### 子图独立 Checkpointer

```python
from langgraph.checkpoint.memory import MemorySaver

# 子图带持久化
research_checkpointer = MemorySaver()
research_graph = research_builder.compile(
    checkpointer=research_checkpointer
)

# 父图带持久化
parent_checkpointer = MemorySaver()
graph = builder.compile(
    checkpointer=parent_checkpointer
)
```

### 共享 Checkpointer

```python
# 共享同一个 checkpointer
shared_checkpointer = MemorySaver()

research_graph = research_builder.compile(
    checkpointer=shared_checkpointer
)

graph = builder.compile(
    checkpointer=shared_checkpointer
)
```

## 子图流式输出

### 识别子图输出

```python
for chunk in graph.stream(
    {"query": "Python 异步编程"},
    stream_mode=["messages", "updates"],
    subgraphs=True,
    version="v2",
):
    if chunk["type"] == "updates":
        node, update = chunk["data"]
        
        # 判断来源
        if chunk.get("metadata", {}).get("checkpoint_ns"):
            # 子图输出
            print(f"[子图] {node}: {update}")
        else:
            # 父图输出
            print(f"[父图] {node}: {update}")
```

## 子图最佳实践

### 1. 清晰的状态边界

```python
# ✅ 好的做法：子图状态独立
class ResearchState(TypedDict):
    query: str
    results: list[str]

# ❌ 不好的做法：子图依赖父图状态
class BadResearchState(TypedDict):
    query: str
    results: list[str]
    parent_field: str  # 不应该知道父图状态
```

### 2. 最小化接口

```python
# ✅ 好的做法：最小输入输出
class ResearchInput(TypedDict):
    query: str

class ResearchOutput(TypedDict):
    results: list[str]

# ❌ 不好的做法：暴露内部细节
class BadResearchState(TypedDict):
    query: str
    intermediate_step1: str
    intermediate_step2: str
    results: list[str]
```

### 3. 可测试性

```python
# 子图可以独立测试
def test_research_graph():
    result = research_graph.invoke({"query": "测试查询"})
    assert "results" in result
    assert len(result["results"]) > 0
```

## 小结

| 要点 | 说明 |
|------|------|
| 子图 | 嵌套 StateGraph，模块化设计 |
| 调用方式 | `subgraph.invoke(input)` 或直接添加为节点 |
| 状态映射 | 父图 ↔ 子图状态转换 |
| 通信 | Command(goto=...) 控制流 |
| 多层嵌套 | 子图可以包含子图 |
| 持久化 | 独立或共享 Checkpointer |
| 流式输出 | subgraphs=True 显示子图输出 |
| 最佳实践 | 清晰边界、最小接口、可测试 |
