# 09.05.03 - 持久化

**持久化：** 使用 Checkpointer 保存图状态，支持恢复、重放和跨线程记忆。

```
持久化：
┌──────────────────────────────────────┐
│  Checkpointer                         │
│                                       │
│  thread_id   → 会话标识               │
│  checkpoint  → 状态快照               │
│  checkpoint_id → 特定快照ID           │
│  Store       → 跨线程存储             │
└──────────────────────────────────────┘
```

### 为什么需要持久化？

- 中断后恢复执行
- 多轮对话保持上下文
- 调试和回溯（Time Travel）
- 跨线程共享信息（长期记忆）
- 故障恢复

---

### Checkpointer 类型

```
Checkpointer：
┌──────────────────────────────────────┐
│  InMemorySaver  → 内存（开发用）      │
│  SqliteSaver    → SQLite              │
│  PostgresSaver  → PostgreSQL          │
│  AsyncSqliteSaver → 异步SQLite        │
│  AsyncPostgresSaver → 异步PostgreSQL  │
└──────────────────────────────────────┘
```

| Checkpointer | 用途 | 说明 |
|--------------|------|------|
| InMemorySaver | 开发测试 | 数据不持久，重启丢失 |
| SqliteSaver | 本地应用 | 单文件数据库 |
| PostgresSaver | 生产环境 | LangSmith 默认使用 |

---

### 最简示例：配置 Checkpointer

```python
from langgraph.checkpoint.memory import InMemorySaver
from langgraph.graph import StateGraph

checkpointer = InMemorySaver()
graph = builder.compile(checkpointer=checkpointer)

config = {"configurable": {"thread_id": "session-123"}}
result = graph.invoke({"input": "test"}, config)
```

---

### Thread：会话

**thread_id 是持久化的指针：**

```python
config = {"configurable": {"thread_id": "user-456"}}

# 同一个 thread_id 保持上下文
result1 = graph.invoke({"messages": ["你好"]}, config)
result2 = graph.invoke({"messages": ["再见"]}, config)

# 不同 thread_id 是新会话
config2 = {"configurable": {"thread_id": "user-789"}}
result3 = graph.invoke({"messages": ["你好"]}, config2)
```

---

### Checkpoint：状态快照

**每个 super-step 保存一个 checkpoint：**

```
执行流程：
START → node_a → node_b → END

Checkpoint:
1. 空 checkpoint，next=START
2. node_a 输出，next=node_b
3. node_b 输出，next=()
```

---

### 获取状态

**获取最新状态：**

```python
config = {"configurable": {"thread_id": "1"}}
state = graph.get_state(config)

print(state.values)       # {'foo': 'b', 'bar': ['a', 'b']}
print(state.next)         # () 表示完成
print(state.config)       # {'thread_id': '1', 'checkpoint_id': '...'}
print(state.metadata)     # {'step': 2, 'source': 'loop'}
```

**获取特定 checkpoint：**

```python
config = {"configurable": {
    "thread_id": "1",
    "checkpoint_id": "1ef663ba-...",
}}
graph.get_state(config)
```

---

### 状态历史

```python
history = list(graph.get_state_history(config))

for state in history:
    print(f"step: {state.metadata['step']}")
    print(f"values: {state.values}")
    print(f"next: {state.next}")

# 查找特定 checkpoint
before_node_b = next(s for s in history if s.next == ("node_b",))
```

### StateSnapshot 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| values | dict | State 值 |
| next | tuple | 下一个要执行的节点 |
| config | dict | thread_id + checkpoint_id |
| metadata | dict | step, source, writes |
| created_at | str | ISO 时间戳 |
| parent_config | dict | 父 checkpoint 配置 |
| tasks | tuple | 待执行任务 |

### Replay：重放

**从特定 checkpoint 重放：**

```python
config = {"configurable": {
    "thread_id": "1",
    "checkpoint_id": "before_node_b",
}}

# 从该 checkpoint 重新执行
graph.invoke(None, config)
```

---

### Update State：更新状态

```python
graph.update_state(config, {"foo": "new_value"})

# 指定来源节点
graph.update_state(config, {"foo": "value"}, as_node="node_a")
```

### Store：跨线程存储

**Store 用于跨线程共享信息（长期记忆）：**

```python
from langgraph.store.memory import InMemoryStore

store = InMemoryStore()

# 存储用户记忆
user_id = "1"
namespace = (user_id, "memories")
memory_id = str(uuid.uuid4())

store.put(namespace, memory_id, {"food_preference": "喜欢披萨"})

# 搜索记忆
memories = store.search(namespace)
print(memories[-1].value)
```

---

### 语义搜索

**Store 支持向量搜索：**

```python
from langchain.embeddings import init_embeddings

store = InMemoryStore(
    index={
        "embed": init_embeddings("openai:text-embedding-3-small"),
        "dims": 1536,
        "fields": ["$"],  # 嵌入所有字段
    }
)

# 自然语言搜索
memories = store.search(
    namespace,
    query="用户喜欢吃什么？",
    limit=3,
)
```

### 在节点中使用 Store

```python
from dataclasses import dataclass
from langgraph.runtime import Runtime

@dataclass
class Context:
    user_id: str

async def save_memory(state: State, runtime: Runtime[Context]):
    user_id = runtime.context.user_id
    namespace = (user_id, "memories")
    
    await runtime.store.aput(namespace, str(uuid.uuid4()), {"memory": "..."})

async def load_memory(state: State, runtime: Runtime[Context]):
    user_id = runtime.context.user_id
    namespace = (user_id, "memories")
    
    memories = await runtime.store.asearch(namespace, query="...", limit=3)
```

### 编译时配置 Store

```python
@dataclass
class Context:
    user_id: str

checkpointer = InMemorySaver()
store = InMemoryStore()

builder = StateGraph(MessagesState, context_schema=Context)
# ... 添加节点和边 ...

graph = builder.compile(checkpointer=checkpointer, store=store)

# 使用时传入 context
graph.invoke(
    {"messages": ["你好"]},
    config={"configurable": {"thread_id": "1"}},
    context=Context(user_id="user-123"),
)
```

### Checkpointer 库

| 库 | 安装 | Checkpointer |
|----|------|-------------|
| langgraph-checkpoint | 内置 | InMemorySaver |
| langgraph-checkpoint-sqlite | pip install | SqliteSaver |
| langgraph-checkpoint-postgres | pip install | PostgresSaver |

**SqliteSaver 示例：**

```python
import sqlite3
from langgraph.checkpoint.sqlite import SqliteSaver

conn = sqlite3.connect("checkpoint.db")
checkpointer = SqliteSaver(conn)
graph = builder.compile(checkpointer=checkpointer)
```

**AsyncPostgresSaver 示例：**

```python
from langgraph.checkpoint.postgres.aio import AsyncPostgresSaver

checkpointer = AsyncPostgresSaver.from_conn_string("postgresql://...")
await checkpointer.setup()
graph = builder.compile(checkpointer=checkpointer)
```

### 持久化总结

| 要点 | 说明 |
|------|------|
| Checkpointer | 保存每步状态快照 |
| thread_id | 会话标识，必需 |
| Store | 跨线程长期记忆 |
| get_state | 获取当前状态 |
| get_state_history | 获取历史状态 |
| update_state | 人工修改状态 |
| Replay | 从 checkpoint 重放 |
