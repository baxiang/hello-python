# 09.05.03 - 持久化

**持久化：** 使用 Checkpointer 保存图状态，支持恢复和重放。

```
持久化：
┌──────────────────────────────────────┐
│  Checkpointer                         │
│                                       │
│  thread_id → 会话标识                 │
│  checkpoint → 状态快照                │
│  resume → 从断点恢复                  │
└──────────────────────────────────────┘
```

### 为什么需要持久化？

- 中断后恢复执行
- 多轮对话保持上下文
- 调试和回溯

---

### Checkpointer 类型

```
Checkpointer：
┌──────────────────────────────────────┐
│  MemorySaver      → 内存（开发用）    │
│  SqliteSaver      → SQLite            │
│  PostgresSaver    → PostgreSQL        │
│  RedisSaver       → Redis             │
└──────────────────────────────────────┘
```

---

### 最简示例：配置 Checkpointer

```python
from langgraph.checkpoint.memory import MemorySaver
from langgraph.graph import StateGraph

checkpointer = MemorySaver()
graph = builder.compile(checkpointer=checkpointer)

config = {"configurable": {"thread_id": "session-123"}}
result = graph.invoke({"input": "test"}, config)
```

---

### thread_id：会话标识

`thread_id` 用于区分不同对话会话：

```python
config = {"configurable": {"thread_id": "user-456"}}

# 同一个 thread_id 保持上下文
result1 = graph.invoke({"messages": ["你好"]}, config)
result2 = graph.invoke({"messages": ["再见"]}, config)
```

---

### 详细示例：多轮对话

```python
from langgraph.checkpoint.memory import MemorySaver
from langgraph.graph import StateGraph, MessagesState, START, END
from langchain_openai import ChatOpenAI

def chat_node(state: MessagesState):
    model = ChatOpenAI(model="gpt-4o-mini")
    response = model.invoke(state["messages"])
    return {"messages": [response]}

builder = StateGraph(MessagesState)
builder.add_node("chat", chat_node)
builder.add_edge(START, "chat")
builder.add_edge("chat", END)

checkpointer = MemorySaver()
graph = builder.compile(checkpointer=checkpointer)

# 同一会话多轮对话
config = {"configurable": {"thread_id": "conv-1"}}

result1 = graph.invoke({"messages": ["你好"]}, config)
print(result1["messages"][-1].content)

result2 = graph.invoke({"messages": ["你是谁？"]}, config)
print(result2["messages"][-1].content)
```

---

### 查看历史状态

```python
# 获取所有 checkpoint
checkpoints = graph.get_state_history(config)

for checkpoint in checkpoints:
    print(checkpoint.config)
    print(checkpoint.values)
```

---

### 从特定状态恢复

```python
# 获取特定 checkpoint
state = graph.get_state(config)

# 从该状态恢复
graph.invoke(None, config)
```

### GraphRecursionError

防止无限循环：

```python
result = graph.invoke(
    {"messages": ["问题"]},
    config={"recursion_limit": 10}
)
```
