# 09.02.01 - 图基础概念

**LangGraph：** 基于 Pregel 模型的状态图框架，用于构建复杂 Agent 工作流。

```
图架构：
┌──────────────────────────────────────┐
│  StateGraph                           │
│                                       │
│  ┌─────┐      ┌─────┐      ┌─────┐   │
│  │Node │ ───→ │Node │ ───→ │Node │   │
│  │ A   │      │ B   │      │ C   │   │
│  └─────┐      └─────┐      └─────┐   │
│                                       │
│  State ← 节点间共享状态               │
│  Edges ← 控制执行流向                 │
└──────────────────────────────────────┘
```

### 为什么用它？

LangChain 的 Agent 底层是 LangGraph。理解 LangGraph 有助于：
- 自定义 Agent 行为
- 构建复杂工作流
- 实现多 Agent 协作

### 三要素

```
三要素：
┌──────────────────────────────────────┐
│  State   → 共享数据结构               │
│  Nodes   → 执行逻辑的函数             │
│  Edges   → 控制流向的路由             │
└──────────────────────────────────────┘
```

**Nodes：** 节点函数，接收当前 State，执行计算，返回更新。
**Edges：** 边，决定下一个执行哪个节点。
**State：** 状态，节点间共享的数据结构。

### 最简示例：Hello World

```python
from typing import TypedDict
from langgraph.graph import StateGraph, START, END

# 第一步：定义 State
class State(TypedDict):
    message: str

# 第二步：定义节点函数
def hello_node(state: State) -> dict:
    return {"message": "Hello, LangGraph!"}

# 第三步：构建图
builder = StateGraph(State)
builder.add_node("hello", hello_node)
builder.add_edge(START, "hello")
builder.add_edge("hello", END)

# 第四步：编译和执行
graph = builder.compile()
result = graph.invoke({"message": ""})

print(result["message"])
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `TypedDict` | 定义 State 类型 | 类型提示，IDE 支持 |
| `add_node()` | 添加节点 | 定义执行单元 |
| `add_edge(START, "hello")` | 入口边 | 指定第一个执行的节点 |
| `add_edge("hello", END)` | 结束边 | 指定执行结束 |
| `builder.compile()` | 编译图 | 检查结构完整性 |
| `graph.invoke()` | 执行图 | 运行整个流程 |

### 详细示例：多节点流程

```python
from typing import TypedDict
from langgraph.graph import StateGraph, START, END

class State(TypedDict):
    input: str
    processed: str
    output: str

def process_node(state: State) -> dict:
    return {"processed": state["input"].upper()}

def format_node(state: State) -> dict:
    return {"output": f"结果: {state['processed']}"}

builder = StateGraph(State)
builder.add_node("process", process_node)
builder.add_node("format", format_node)

builder.add_edge(START, "process")
builder.add_edge("process", "format")
builder.add_edge("format", END)

graph = builder.compile()
result = graph.invoke({"input": "hello", "processed": "", "output": ""})

print(result["output"])
```
