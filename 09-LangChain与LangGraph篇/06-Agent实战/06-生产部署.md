生产部署需要可靠的基础设施支持：

```
生产部署架构：
┌──────────────────────────────────────┐
│  负载均衡器                           │
│    ↓                                 │
│  FastAPI 实例 × N                     │
│    ↓                                 │
│  PostgreSQL (Checkpointer + Store)   │
│    ↓                                 │
│  Redis (缓存)                        │
│    ↓                                 │
│  监控 (Prometheus + Grafana)         │
│    ↓                                 │
│  日志 (ELK / Loki)                   │
└──────────────────────────────────────┘
```

## PostgreSQL 持久化

### 安装和配置

```bash
uv add langgraph-checkpoint-postgres
```

### 同步 Checkpointer

```python
from langgraph.checkpoint.postgres import PostgresSaver

DATABASE_URL = "postgresql://user:password@host:5432/dbname"

checkpointer = PostgresSaver.from_conn_string(DATABASE_URL)
checkpointer.setup()

graph = builder.compile(checkpointer=checkpointer)
```

### 异步 Checkpointer

```python
from langgraph.checkpoint.postgres.aio import AsyncPostgresSaver

checkpointer = AsyncPostgresSaver.from_conn_string(DATABASE_URL)
await checkpointer.setup()

graph = builder.compile(checkpointer=checkpointer)

# 异步调用
result = await graph.ainvoke(inputs, config=config)
```

### PostgresStore

```python
from langgraph.store.postgres import PostgresStore
from langchain.embeddings import init_embeddings

store = PostgresStore.from_conn_string(DATABASE_URL)
store.setup()

# 带向量索引
store = PostgresStore(
    conn_string=DATABASE_URL,
    index={
        "embed": init_embeddings("zhipuai:embedding-2"),
        "dims": 1024,
        "fields": ["$"],
    },
)
store.setup()
```

## Docker 部署

### Dockerfile

```dockerfile
FROM python:3.11-slim

WORKDIR /app

# 安装 uv
RUN pip install uv

# 复制依赖文件
COPY pyproject.toml uv.lock ./

# 安装依赖
RUN uv sync --frozen

# 复制应用代码
COPY app ./app

# 暴露端口
EXPOSE 8000

# 启动命令
CMD ["uv", "run", "uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000"]
```

### docker-compose.yml

```yaml
version: '3.8'

services:
  app:
    build: .
    ports:
      - "8000:8000"
    environment:
      - DATABASE_URL=postgresql://user:password@db:5432/agent_db
      - MOONSHOT_API_KEY=${MOONSHOT_API_KEY}
      - DEEPSEEK_API_KEY=${DEEPSEEK_API_KEY}
      - LANGSMITH_API_KEY=${LANGSMITH_API_KEY}
      - LANGSMITH_TRACING=true
      - LANGSMITH_PROJECT=production
    depends_on:
      db:
        condition: service_healthy
    restart: unless-stopped

  db:
    image: postgres:16
    environment:
      - POSTGRES_USER=user
      - POSTGRES_PASSWORD=password
      - POSTGRES_DB=agent_db
    volumes:
      - postgres_data:/var/lib/postgresql/data
    ports:
      - "5432:5432"
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U user -d agent_db"]
      interval: 5s
      timeout: 5s
      retries: 5
    restart: unless-stopped

volumes:
  postgres_data:
```

### 构建和运行

```bash
# 构建

# 运行
docker compose up -d

# 查看日志
docker compose logs -f app

# 停止
docker compose down
```

## 健康检查

### 基本健康端点

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/health")
async def health():
    """基本健康检查"""
    return {
        "status": "healthy",
        "version": "1.0.0",
        "timestamp": "2024-01-01T00:00:00",
    }

@app.get("/health/ready")
async def readiness():
    """就绪检查"""
    # 检查数据库连接
    try:
        checkpointer.get_state({"configurable": {"thread_id": "health_check"}})
        db_status = "ok"
    except Exception as e:
        db_status = f"error: {e}"
    
    return {
        "status": "ready" if db_status == "ok" else "not_ready",
        "database": db_status,
    }

@app.get("/health/live")
async def liveness():
    """存活检查"""
    return {"status": "alive"}
```

### 完整健康检查

```python
from fastapi import FastAPI
import time

app = FastAPI()
start_time = time.time()

@app.get("/health/detailed")
async def detailed_health():
    """详细健康检查"""
    checks = {}
    
    # 1. 数据库
    try:
        checkpointer.get_state({"configurable": {"thread_id": "health"}})
        checks["database"] = "ok"
    except Exception as e:
        checks["database"] = f"error: {e}"
    
    # 2. Store
    try:
        store.search(("health",))
        checks["store"] = "ok"
    except Exception as e:
        checks["store"] = f"error: {e}"
    
    # 3. 内存
    import psutil
    process = psutil.Process()
    checks["memory_mb"] = round(process.memory_info().rss / 1024 / 1024, 2)
    
    # 4. 运行时间
    checks["uptime_seconds"] = round(time.time() - start_time, 2)
    
    all_ok = all(v == "ok" for k, v in checks.items() if isinstance(v, str))
    
    return {
        "status": "healthy" if all_ok else "degraded",
        "checks": checks,
    }
```

## 监控指标

### Prometheus 指标

```bash
uv add prometheus-client
```

```python
from fastapi import FastAPI
from prometheus_client import Counter, Histogram, generate_latest, CONTENT_TYPE_LATEST
import time

app = FastAPI()

# 定义指标
REQUEST_COUNT = Counter(
    'agent_requests_total',
    'Total agent requests',
    ['status', 'model'],
)

REQUEST_DURATION = Histogram(
    'agent_request_duration_seconds',
    'Agent request duration',
    ['model'],
)

TOKEN_COUNT = Counter(
    'agent_tokens_total',
    'Total tokens used',
    ['type'],  # prompt or completion
)

@app.get("/metrics")
async def metrics():
    """Prometheus 指标"""
    return Response(generate_latest(), media_type=CONTENT_TYPE_LATEST)

# 在请求中使用
@app.post("/chat")
async def chat(request: ChatRequest):
    start_time = time.time()
    
    try:
        result = agent.invoke(inputs, config=config)
        
        duration = time.time() - start_time
        REQUEST_COUNT.labels(status="success", model="moonshot").inc()
        REQUEST_DURATION.labels(model="moonshot").observe(duration)
        
        return result
    except Exception as e:
        REQUEST_COUNT.labels(status="error", model="moonshot").inc()
        raise
```

### 关键指标

| 指标 | 说明 | 告警阈值 |
|------|------|---------|
| 请求延迟 | 平均响应时间 | > 10s |
| 错误率 | 失败请求比例 | > 5% |
| Token 使用 | 总 token 消耗 | 预算 80% |
| 内存使用 | 进程内存 | > 80% |
| 数据库连接 | 活跃连接数 | > 80% |

## 日志配置

### 结构化日志

```python
import logging
import json
import sys

class JSONFormatter(logging.Formatter):
    def format(self, record):
        log_entry = {
            "timestamp": self.formatTime(record),
            "level": record.levelname,
            "message": record.getMessage(),
            "module": record.module,
            "function": record.funcName,
            "line": record.lineno,
        }
        
        if record.exc_info:
            log_entry["exception"] = self.formatException(record.exc_info)
        
        return json.dumps(log_entry, ensure_ascii=False)

# 配置日志
logger = logging.getLogger("agent")
logger.setLevel(logging.INFO)

handler = logging.StreamHandler(sys.stdout)
handler.setFormatter(JSONFormatter())
logger.addHandler(handler)
```

## 错误处理

### 全局错误处理

```python
from fastapi import FastAPI, Request
from fastapi.responses import JSONResponse
import logging

app = FastAPI()
logger = logging.getLogger("agent")

@app.exception_handler(Exception)
async def global_exception_handler(request: Request, exc: Exception):
    """全局异常处理"""
    logger.error(f"Unhandled exception: {exc}", exc_info=True)
    
    return JSONResponse(
        status_code=500,
        content={
            "error": "Internal server error",
            "detail": str(exc) if app.debug else None,
        },
    )

@app.exception_handler(TimeoutError)
async def timeout_handler(request: Request, exc: TimeoutError):
    """超时处理"""
    return JSONResponse(
        status_code=504,
        content={"error": "Request timeout"},
    )
```

### GraphRecursionError

from langgraph.errors import GraphRecursionError
from langchain.messages import AIMessage

try:
    result = graph.invoke(inputs, config={"recursion_limit": 10})
