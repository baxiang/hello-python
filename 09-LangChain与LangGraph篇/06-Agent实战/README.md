# 第 09.06 章 - Agent 实战

本章通过 4 个完整项目，实践 LangChain 和 LangGraph 的综合应用。

- 构建完整 AI 应用
- 掌握多 Agent 协作
- 理解生产部署流程

## 章节内容

| 序号 | 标题 | 内容 |
|------|------|------|
| 01 | 多Agent协作 | Subgraph、Handoffs、并行执行 |
