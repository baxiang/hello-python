# 09.06.03 - 生产部署

**LangGraph Server：** 生产级 Agent 服务部署方案。

```
部署架构：
┌──────────────────────────────────────┐
│  LangGraph Server                     │
│                                       │
│  API Endpoint → /invoke               │
│               → /stream               │
│               → /threads              │
│                                       │
│  Checkpointer → 持久化                │
│  Store       → 长期记忆               │
│  LangSmith  → 可观测性                │
└──────────────────────────────────────┘
```

| 方式 | 说明 | 适用场景 |

### FastAPI 部署

```python
from fastapi import FastAPI
from pydantic import BaseModel
from langgraph.checkpoint.memory import MemorySaver

app = FastAPI()
checkpointer = MemorySaver()
graph = builder.compile(checkpointer=checkpointer)

class ChatRequest(BaseModel):
    thread_id: str
    message: str

@app.post("/chat")
async def chat(request: ChatRequest):
    config = {"configurable": {"thread_id": request.thread_id}}
    result = graph.invoke({"messages": [{"role": "user", "content": request.message}]}, config)
    return {"response": result["messages"][-1].content}

@app.post("/stream")
async def stream(request: ChatRequest):
    from fastapi.responses import StreamingResponse
    
    config = {"configurable": {"thread_id": request.thread_id}}
    
    async def generate():
        async for chunk in graph.astream({"messages": [{"role": "user", "content": request.message}]}, config):
            yield f"data: {chunk}\n\n"
    
    return StreamingResponse(generate(), media_type="text/event-stream")

@app.get("/history/{thread_id}")
async def history(thread_id: str):
    config = {"configurable": {"thread_id": thread_id}}
    history = list(graph.get_state_history(config))
    return {"history": [s.values for s in history]}
```

### Agent Server API

**主要端点：**

| 端点 | 方法 | 用途 |
|------|------|------|
| /threads | POST | 创建会话 |
| /threads/{id}/runs | POST | 执行 Agent |
| /threads/{id}/runs/stream | POST | 流式执行 |
| /threads/{id}/state | GET | 获取状态 |
| /threads/{id}/history | GET | 获取历史 |

**示例：**

```bash
# 创建会话
curl -X POST http://localhost:8000/threads \
  -H "Content-Type: application/json" \
  -d '{"thread_id": "123"}'

# 执行 Agent
curl -X POST http://localhost:8000/threads/123/runs \
  -H "Content-Type: application/json" \
  -d '{"input": {"messages": [{"role": "user", "content": "你好"}]}}'
```

### 生产 Checkpointer

**PostgreSQL（推荐生产使用）：**

```python
from langgraph.checkpoint.postgres import PostgresSaver

conn_string = "postgresql://user:pass@host:5432/db"
checkpointer = PostgresSaver.from_conn_string(conn_string)
checkpointer.setup()

graph = builder.compile(checkpointer=checkpointer)
```

**异步版本：**

```python
from langgraph.checkpoint.postgres.aio import AsyncPostgresSaver

checkpointer = AsyncPostgresSaver.from_conn_string(conn_string)
await checkpointer.setup()
```

### 错误处理

**GraphRecursionError：**

```python
from langgraph.errors import GraphRecursionError

try:
    result = graph.invoke({"messages": [...]}, config={"recursion_limit": 10})
except GraphRecursionError:
    result = {"messages": [AIMessage("执行超时，请简化请求")]}
```

### 速率限制

**使用 Middleware 实现：**

```python
from langchain.agents.middleware import wrap_model_call
from datetime import datetime, timedelta

rate_limit_cache = {}

@wrap_model_call
def rate_limit(request: ModelRequest, handler) -> ModelResponse:
    user_id = request.runtime.context.user_id
    now = datetime.now()
    
    if user_id in rate_limit_cache:
        last_call = rate_limit_cache[user_id]
        if now - last_call < timedelta(seconds=1):
            raise Exception("请求过于频繁，请稍后再试")
    
    rate_limit_cache[user_id] = now
    return handler(request)
```
