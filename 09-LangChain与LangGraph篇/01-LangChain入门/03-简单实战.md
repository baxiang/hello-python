# 09.01.03 - 简单实战：问答对话

本节整合前两节内容，构建一个简单的问答对话应用。

---

### 完整示例：多轮问答

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, AIMessage, SystemMessage

model = init_chat_model("openai:gpt-4o-mini")
messages = [
    SystemMessage("你是一个友好的 Python 学习助手。回答要简洁易懂。"),
]

def chat(user_input: str) -> str:
    messages.append(HumanMessage(user_input))
    response = model.invoke(messages)
    messages.append(response)
    return response.content

print(chat("什么是 Python？"))
print(chat("它有什么特点？"))
print(chat("适合什么人学习？"))

print("\n=== 对话历史 ===")
for m in messages:
    role = m.__class__.__name__.replace("Message", "")
    print(f"{role}: {m.content[:50]}...")
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `messages.append(HumanMessage(...))` | 记录用户输入 | 保持对话上下文 |
| `messages.append(response)` | 记录 AI 响应 | 构建完整对话历史 |
| `model.invoke(messages)` | 传入完整历史 | AI 根据上下文回答 |

### 流式对话示例

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, SystemMessage

model = init_chat_model("openai:gpt-4o-mini")
messages = [SystemMessage("你是一个友好的 Python 学习助手。")]

def chat_stream(user_input: str):
    messages.append(HumanMessage(user_input))
    
    print("\n助手: ", end="")
    full_response = None
    for chunk in model.stream(messages):
        print(chunk.text, end="", flush=True)
        full_response = chunk if full_response is None else full_response + chunk
    
    messages.append(full_response)
    print()

chat_stream("请解释什么是列表推导式")
chat_stream("给我一个具体的例子")
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `for chunk in model.stream(messages)` | 流式接收响应 | 实时展示，用户体验更好 |
| `full_response = chunk if ... else full_response + chunk` | 聚合完整消息 | 流式消息需要累加成完整 AIMessage |

### 带参数控制的对话

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, SystemMessage

model = init_chat_model(
    "openai:gpt-4o-mini",
    temperature=0.3,
    max_tokens=500,
)

messages = [
    SystemMessage("你是一个技术文档翻译助手。翻译要准确、专业。"),
]

def translate(text: str) -> str:
    messages.append(HumanMessage(f"请翻译以下内容：{text}"))
    response = model.invoke(messages)
    messages.append(response)
    return response.content

result = translate("Python is a high-level programming language.")
print(result)
```

**关键代码说明：**

| 参数 | 设置 | 为什么这样写 |
|------|------|-------------|
| `temperature=0.3` | 低随机性 | 翻译需要确定性输出 |
| `max_tokens=500` | 输出限制 | 翻译结果通常较短 |

### Token 统计监控

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, SystemMessage

model = init_chat_model("openai:gpt-4o-mini")
messages = [SystemMessage("你是一个 Python 老师。")]

def chat_with_stats(user_input: str) -> dict:
    messages.append(HumanMessage(user_input))
    response = model.invoke(messages)
    messages.append(response)
    
    stats = {
        "response": response.content,
        "tokens": response.usage_metadata,
    }
    return stats

result = chat_with_stats("什么是函数？")
print(f"回答: {result['response']}")
print(f"Token 统计: {result['tokens']}")

total_input = sum(
    m.usage_metadata.get("input_tokens", 0)
    for m in messages if hasattr(m, "usage_metadata")
)
total_output = sum(
    m.usage_metadata.get("output_tokens", 0)
    for m in messages if hasattr(m, "usage_metadata")
)
print(f"累计消耗: 输入 {total_input} tokens, 输出 {total_output} tokens")
```

### 批量问答示例

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, SystemMessage

model = init_chat_model("openai:gpt-4o-mini")

questions = [
    "什么是变量？",
    "什么是函数？",
    "什么是类？",
    "什么是模块？",
    "什么是包？",
]

batch_messages = [
    [SystemMessage("用一句话简洁回答"), HumanMessage(q)]
    for q in questions
]

responses = model.batch(batch_messages, config={"max_concurrency": 3})

for q, r in zip(questions, responses):
    print(f"Q: {q}")
    print(f"A: {r.content}\n")
```

### 实战要点总结

| 要点 | 说明 |
|------|------|
| 对话历史管理 | 每次 invoke 后 append AI 响应到 messages |
| 流式输出 | 使用 stream() 实现实时响应 |
| 参数控制 | 根据任务调整 temperature 和 max_tokens |
| Token 监控 | 使用 usage_metadata 追踪消耗 |
| 批量处理 | 使用 batch() 提高效率 |
