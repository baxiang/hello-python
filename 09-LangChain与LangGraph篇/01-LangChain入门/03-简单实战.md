# 09.01.03 - 简单实战：问答对话

本节整合前两节内容，构建一个简单的问答对话应用。

### 完整示例：多轮问答

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, AIMessage, SystemMessage

# 第一步：初始化模型和对话历史
model = init_chat_model("openai:gpt-4o-mini")
messages = [
    SystemMessage("你是一个友好的 Python 学习助手。回答要简洁易懂。"),
]

# 第二步：对话循环
def chat(user_input: str) -> str:
    messages.append(HumanMessage(user_input))
    response = model.invoke(messages)
    messages.append(response)
    return response.content

# 第三步：测试对话
print(chat("什么是 Python？"))
print(chat("它有什么特点？"))
print(chat("适合什么人学习？"))

# 第四步：查看完整对话历史
print("\n=== 对话历史 ===")
for m in messages:
    role = m.__class__.__name__.replace("Message", "")
    print(f"{role}: {m.content[:50]}...")
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `messages.append(HumanMessage(...))` | 记录用户输入 | 保持对话上下文 |
| `messages.append(response)` | 记录 AI 响应 | 构建完整对话历史 |
| `model.invoke(messages)` | 传入完整历史 | AI 根据上下文回答 |
