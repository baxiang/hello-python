# 09.01.01 - 模型与消息

**LangChain 模型：** 统一的 LLM 调用接口，支持多个 AI 提供商。

```
语法结构：
┌──────────────────────────────────────┐
│  init_chat_model("provider:model")   │
│                                       │
│  "openai:gpt-4"     → OpenAI         │
│  "anthropic:claude" → Anthropic      │
└──────────────────────────────────────┘
```

### 最简示例

```python
from langchain.chat_models import init_chat_model

model = init_chat_model("openai:gpt-4o-mini")
response = model.invoke("你好")
print(response.content)
```

### 详细示例：多提供商

```python
from langchain.chat_models import init_chat_model

# OpenAI
openai_model = init_chat_model("openai:gpt-4o-mini")

# Anthropic
anthropic_model = init_chat_model("anthropic:claude-3-sonnet")

# 本地模型 (Ollama)
local_model = init_chat_model("ollama:llama3")

# 统一调用方式
response = openai_model.invoke("什么是 Python?")
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `init_chat_model()` | 初始化模型 | 统一接口，无需学习各 SDK |
| `"openai:gpt-4o-mini"` | provider:model 格式 | 标准命名，易于切换 |
| `.invoke()` | 同步调用 | 最简单的调用方式 |

### 消息类型

**消息：** 与 LLM 对话的基本单位，不同角色对应不同消息类型。

```
消息类型：
┌──────────────────────────────────────┐
│  HumanMessage    → 用户输入           │
│  AIMessage      → AI 响应             │
│  SystemMessage  → 系统指令            │
│  ToolMessage    → 工具返回            │
└──────────────────────────────────────┘
```

### 最简示例：多消息对话

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, AIMessage, SystemMessage

model = init_chat_model("openai:gpt-4o-mini")

messages = [
    SystemMessage("你是一个 Python 老师"),
    HumanMessage("什么是列表？"),
]

response = model.invoke(messages)
print(response.content)
```

### 详细示例：多轮对话

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, AIMessage, SystemMessage

model = init_chat_model("openai:gpt-4o-mini")

# 第一步：创建初始对话
messages = [
    SystemMessage("你是一个 Python 老师"),
    HumanMessage("什么是列表？"),
]

# 第二步：获取 AI 响应
response = model.invoke(messages)
messages.append(response)  # 添加 AI 响应到对话历史

# 第三步：继续对话
messages.append(HumanMessage("如何创建列表？"))
response = model.invoke(messages)

print(response.content)
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `SystemMessage` | 系统指令 | 设定 AI 角色和行为 |
| `messages.append(response)` | 添加 AI 响应 | 保持对话上下文 |
| `invoke(messages)` | 传入消息列表 | LLM 根据历史生成响应 |

### 为什么用它？

```python
# 没有 LangChain：每个 SDK 不同
import openai
import anthropic

openai.chat.completions.create(...)   # OpenAI 格式
anthropic.messages.create(...)        # Anthropic 格式

# 有 LangChain：统一接口
init_chat_model("openai:gpt-4").invoke(...)
init_chat_model("anthropic:claude").invoke(...)
```
