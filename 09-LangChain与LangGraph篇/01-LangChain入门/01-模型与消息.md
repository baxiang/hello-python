# 09.01.01 - 模型与消息

**LangChain 模型：** 统一的 LLM 调用接口，支持多个 AI 提供商，提供文本生成、工具调用、结构化输出等功能。

```
模型能力：
┌──────────────────────────────────────┐
│  文本生成   → invoke/stream/batch     │
│  工具调用   → bind_tools               │
│  结构化输出 → with_structured_output  │
│  多模态     → 图像/音频/视频           │
│  推理能力   → reasoning content        │
└──────────────────────────────────────┘
```

---

### 初始化模型

**init_chat_model：** 统一的模型初始化函数，支持所有主要提供商。

```
语法结构：
┌──────────────────────────────────────┐
│  init_chat_model("provider:model")   │
│                                       │
│  "moonshot:moonshot-v1-8k"            │
│  "deepseek:deepseek-chat"             │
│  "google_genai:gemini-2.5-flash"     │
│  "azure_openai:gpt-4"                │
│  "bedrock:anthropic.claude-3"        │
└──────────────────────────────────────┘
```

### 最简示例

```python
from langchain.chat_models import init_chat_model

model = init_chat_model("moonshot:moonshot-v1-8k")
response = model.invoke("你好")
print(response.content)
```

### 详细示例：不同提供商

```python
from langchain.chat_models import init_chat_model

# 月之暗面 Kimi
openai_model = init_chat_model("moonshot:moonshot-v1-8k")

# DeepSeek
deepseek_model = init_chat_model("deepseek:deepseek-chat")

# Google Gemini
gemini_model = init_chat_model("google_genai:gemini-2.5-flash-lite")

# Azure OpenAI
azure_model = init_chat_model("azure_openai:gpt-4")

# AWS Bedrock
bedrock_model = init_chat_model(
    "anthropic.claude-3-5-sonnet-20240620-v1:0",
    model_provider="bedrock_converse",
)

# 本地模型 (Ollama)
local_model = init_chat_model("ollama:llama3")

# 统一调用方式
response = openai_model.invoke("什么是 Python？")
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `init_chat_model()` | 初始化模型 | 统一接口，无需学习各 SDK |
| `"moonshot:moonshot-v1-8k"` | provider:model 格式 | 标准命名，易于切换提供商 |
| `.invoke()` | 同步调用 | 最简单的调用方式 |

### 模型参数

模型支持多种配置参数，影响输出行为：

```
模型参数：
┌──────────────────────────────────────┐
│  model        → 模型名称              │
│  temperature  → 输出随机性 (0-2)      │
│  max_tokens   → 最大输出长度          │
│  timeout      → 请求超时时间          │
│  max_retries  → 失败重试次数          │
└──────────────────────────────────────┘
```

### 详细示例：配置参数

```python
from langchain.chat_models import init_chat_model

model = init_chat_model(
    "moonshot:moonshot-v1-32k",
    temperature=0.7,      # 输出随机性：越高越创意
    max_tokens=1000,      # 最大输出 1000 tokens
    timeout=30,           # 超时 30 秒
    max_retries=6,        # 默认重试次数，网络不稳定可增至 10-15
)

response = model.invoke("写一首诗")
```

**关键代码说明：**

| 参数 | 含义 | 推荐值 |
|------|------|--------|
| `temperature=0` | 确定性输出 | 需要精确答案时 |
| `temperature=0.7` | 平衡创意 | 日常对话 |
| `temperature=1.5` | 高创意 | 创意写作 |
| `max_tokens` | 输出长度上限 | 根据任务需求 |

### 调用方式

**三种主要调用方式：**

```
调用方式：
┌──────────────────────────────────────┐
│  invoke   → 同步单次调用              │
│  stream   → 流式实时输出              │
│  batch    → 批量并行调用              │
└──────────────────────────────────────┘
```

### invoke：同步调用

最直接的调用方式，等待模型完整响应：

```python
from langchain.chat_models import init_chat_model

model = init_chat_model("moonshot:moonshot-v1-8k")

response = model.invoke("你好")
print(response.content)
print(response.usage_metadata)
# {'input_tokens': 8, 'output_tokens': 304, 'total_tokens': 312}
```

### stream：流式输出

实时返回生成内容，适合长文本和实时展示：

```python
from langchain.chat_models import init_chat_model

model = init_chat_model("moonshot:moonshot-v1-8k")

for chunk in model.stream("请介绍 Python"):
    print(chunk.text, end="", flush=True)

# 累积完整消息
full = None
for chunk in model.stream("介绍 Python"):
    full = chunk if full is None else full + chunk

print(full.content_blocks)  # 完整内容块
```

**关键代码说明：**

| 代码 | 含义 | 为什么这样写 |
|------|------|-------------|
| `chunk.text` | 当前文本片段 | 实时显示生成进度 |
| `full + chunk` | 累积消息 | 拼接成完整 AIMessage |
| `content_blocks` | 标准化内容 | 跨提供商统一格式 |

### batch：批量调用

并行处理多个请求，提高效率：

```python
from langchain.chat_models import init_chat_model

model = init_chat_model("moonshot:moonshot-v1-8k")

responses = model.batch([
    "什么是变量？",
    "什么是函数？",
    "什么是类？",
])

for r in responses:
    print(r.content)

# 控制并发数
responses = model.batch(
    list_of_inputs,
    config={"max_concurrency": 5}  # 最大 5 个并行调用
)
```

## 消息类型

**消息：** 与 LLM 对话的基本单位，包含角色、内容和元数据。

```
消息组成：
┌──────────────────────────────────────┐
│  Role     → 角色 (system/user/ai)    │
│  Content  → 内容 (文本/图像/音频)     │
│  Metadata → 元数据 (ID/token计数)    │
└──────────────────────────────────────┘
```

### 消息类型分类

```
消息类型：
┌──────────────────────────────────────┐
│  SystemMessage  → 系统指令            │
│  HumanMessage  → 用户输入             │
│  AIMessage     → AI 响应              │
│  ToolMessage   → 工具返回             │
└──────────────────────────────────────┘
```

### SystemMessage：系统指令

设定模型行为和角色：

```python
from langchain.messages import SystemMessage

# 简单指令
system_msg = SystemMessage("你是一个有帮助的助手")

# 详细角色设定
system_msg = SystemMessage("""
你是一个资深 Python 开发者。
- 擅长 Web 框架
- 提供代码示例
- 解释要简洁但有深度
""")
```

### HumanMessage：用户输入

表示用户发送的消息：

```python
from langchain.messages import HumanMessage

# 纯文本
human_msg = HumanMessage("你好")

# 带元数据
human_msg = HumanMessage(
    content="你好",
    name="alice",    # 标识不同用户
    id="msg_123",    # 唯一标识符
)
```

### AIMessage：AI 响应

模型返回的消息，包含内容、工具调用和元数据：

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage

model = init_chat_model("moonshot:moonshot-v1-8k")
response = model.invoke([HumanMessage("你好")])

print(response.text)           # 文本内容
print(response.content_blocks) # 标准化内容块
print(response.tool_calls)     # 工具调用列表
print(response.usage_metadata) # token 使用情况
```

**AIMessage 属性：**

| 属性 | 含义 | 用途 |
|------|------|------|
| `text` | 文本内容 | 简单文本响应 |
| `content_blocks` | 标准化内容块 | 跨提供商统一格式 |
| `tool_calls` | 工具调用列表 | Agent 工具执行 |
| `usage_metadata` | token 计数 | 成本追踪 |

### ToolMessage：工具返回

工具执行结果，必须匹配 tool_call_id：

```python
from langchain.messages import AIMessage, ToolMessage

# AI 请求调用工具
ai_msg = AIMessage(
    content=[],
    tool_calls=[{
        "name": "get_weather",
        "args": {"location": "北京"},
        "id": "call_123"
    }]
)

# 执行工具后返回结果
tool_msg = ToolMessage(
    content="北京今天晴，25°C",
    tool_call_id="call_123",  # 必须匹配
    name="get_weather",
)

# 继续对话
messages = [ai_msg, tool_msg]
response = model.invoke(messages)
```

### 多轮对话示例

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, AIMessage, SystemMessage

model = init_chat_model("moonshot:moonshot-v1-8k")

# 第一步：创建对话历史
messages = [
    SystemMessage("你是一个 Python 老师"),
    HumanMessage("什么是列表？"),
]

# 第二步：获取 AI 响应
response = model.invoke(messages)
messages.append(response)  # 添加 AI 响应到历史

# 第三步：继续对话
messages.append(HumanMessage("如何创建列表？"))
response = model.invoke(messages)

print(response.content)
print(f"Token 使用: {response.usage_metadata}")
```

## Content Blocks：标准化内容

LangChain 提供跨提供商的标准化内容格式：

```python
from langchain.messages import AIMessage

response = model.invoke("你好")

# 访问标准化内容块
for block in response.content_blocks:
    if block["type"] == "text":
        print(block["text"])
    elif block["type"] == "reasoning":
        print(f"推理: {block['reasoning']}")
    elif block["type"] == "tool_call":
        print(f"工具: {block['name']}")
```

**Content Block 类型：**

| 类型 | 含义 | 用途 |
|------|------|------|
| `"text"` | 文本内容 | 普通响应 |
| `"reasoning"` | 推理步骤 | 深度思考模型 |
| `"tool_call"` | 工具调用 | Agent 执行 |
| `"image"` | 图像数据 | 多模态 |
| `"audio"` | 音频数据 | 多模态 |

### 为什么用它？

```python
# 没有 LangChain：每个 SDK 不同
import openai
import anthropic

openai.chat.completions.create(...)   # OpenAI 格式
anthropic.messages.create(...)        # Anthropic 格式

# 有 LangChain：统一接口
init_chat_model("moonshot:moonshot-v1-8k").invoke(...)
init_chat_model("deepseek:deepseek-chat").invoke(...)
init_chat_model("google_genai:gemini").invoke(...)
```
