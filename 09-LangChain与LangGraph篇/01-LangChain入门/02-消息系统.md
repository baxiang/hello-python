# 09.01.02 - 消息系统

> LangChain 消息系统是 LLM 交互的核心，每种消息类型都有明确的语义和用途

## 为什么需要消息系统

LLM 本质上是无状态的，每次请求都需要传递完整的上下文：

```
请求 1:  "你好"                              →  AI: "你好！有什么可以帮您？"
请求 2:  "你好" + "你好有什么可以帮您？" + "我叫小明" → AI: "你好小明！"
请求 3:  请求2的全部内容 + "我今年25岁"              → AI: "好的小明，你25岁。"
```

消息系统通过类型化的消息对象，清晰地表达了对话中每一部分的角色：

```
┌─────────────────────────────────────────────────┐
│                  消息类型体系                      │
├──────────────┬──────────────────────────────────┤
│ 角色          │ 消息类型                          │
├──────────────┼──────────────────────────────────┤
│ 系统指令      │ SystemMessage                    │
│ 用户输入      │ HumanMessage                     │
│ AI 回复       │ AIMessage                        │
│ 工具调用      │ AIMessage (tool_calls)           │
│ 工具结果      │ ToolMessage                      │
│ 函数返回      │ FunctionMessage (已废弃)          │
└──────────────┴──────────────────────────────────┘
```

## 核心消息类型

### SystemMessage（系统消息）

设定 AI 的行为和角色，通常放在对话开头：

```python
from langchain.messages import SystemMessage

# 简单文本
msg = SystemMessage("你是一个专业的 Python 编程助手。")

# 带额外信息
msg = SystemMessage(
    content="你是一个专业的 Python 编程助手。",
    name="system",  # 可选的标识符
)
```

```python
from langchain.chat_models import init_chat_model
from langchain.messages import SystemMessage, HumanMessage

model = init_chat_model("moonshot:moonshot-v1-8k")

response = model.invoke([
    SystemMessage("你是一个严谨的数学老师，用简单的方式解释概念。"),
    HumanMessage("什么是质数？"),
])
print(response.content)
```

**重要**: 不是所有模型都支持多个 SystemMessage。OpenAI 和 Moonshot 支持在对话中插入系统消息，但 Anthropic 只接受一个系统提示。

### HumanMessage（用户消息）

代表用户的输入，是最常见的消息类型：

```python
from langchain.messages import HumanMessage

# 简单字符串
msg = HumanMessage("你好，请帮我写一个排序算法")

# 多模态内容（文本 + 图片）
msg = HumanMessage(content=[
    {"type": "text", "text": "这张图片里有什么？"},
    {
        "type": "image_url",
        "image_url": {"url": "https://example.com/image.jpg"},
    },
])

# 带名字（多用户场景）
msg = HumanMessage(content="我觉得方案 A 更好", name="张三")
```

### AIMessage（AI 回复消息）

代表模型的回复，包含内容和可选的工具调用：

```python
from langchain.messages import AIMessage

# 简单回复
msg = AIMessage("好的，我来帮你写一个快速排序算法。")

# 带工具调用
msg = AIMessage(
    content="",
    tool_calls=[
        {
            "name": "get_weather",
            "args": {"city": "北京"},
            "id": "call_abc123",
        }
    ],
)

# 带响应元数据
msg = AIMessage(
    content="你好！",
    response_metadata={
        "finish_reason": "stop",
        "model_name": "moonshot-v1-8k",
    },
)
```

### ToolMessage（工具执行结果）

当 AI 调用工具后，用 ToolMessage 返回工具的执行结果：

```python
from langchain.messages import ToolMessage

# 成功的工具结果
msg = ToolMessage(
    content="北京今天天气：晴，25°C",
    tool_call_id="call_abc123",  # 必须匹配 AIMessage 中的 tool_call id
)

# 错误的工具结果
msg = ToolMessage(
    content="错误：无法获取天气信息，城市不存在",
    tool_call_id="call_abc123",
    status="error",
)
```

## 消息类型完整对照表

| 消息类型 | 角色 | 用途 | content 类型 |
|---------|------|------|-------------|
| `SystemMessage` | `system` | 设定 AI 行为和角色 | `str` 或 `list[dict]` |
| `HumanMessage` | `user` | 用户输入 | `str` 或 `list[dict]` |
| `AIMessage` | `assistant` | AI 回复 | `str` 或 `list[dict]` |
| `ToolMessage` | `tool` | 工具执行结果 | `str` |

## 消息内容格式

### 纯文本内容

最简单的形式，直接传入字符串：

```python
from langchain.messages import HumanMessage

msg = HumanMessage("请解释 Python 中的装饰器")
```

### 内容块（Content Blocks）

复杂内容使用列表格式，支持多模态：

```python
msg = HumanMessage(content=[
    {"type": "text", "text": "请分析这张图片"},
    {
        "type": "image_url",
        "image_url": {
            "url": "https://example.com/photo.jpg",
            "detail": "high",  # low / high / auto
        },
    },
])
```

**支持的 content block 类型**：

| 类型 | 说明 | 示例 |
|------|------|------|
| `text` | 纯文本 | `{"type": "text", "text": "你好"}` |
| `image_url` | 图片 URL 或 base64 | `{"type": "image_url", "image_url": {"url": "..."}}` |
| `video_url` | 视频 URL | `{"type": "video_url", "video_url": {"url": "..."}}` |
| `audio_url` | 音频 URL | `{"type": "audio_url", "audio_url": {"url": "..."}}` |
| `tool_use` | 工具调用块 | `{"type": "tool_use", "id": "...", "name": "...", "input": {...}}` |
| `tool_result` | 工具结果块 | `{"type": "tool_result", "tool_use_id": "...", "content": "..."}` |

### Base64 图片

对于本地图片，可以转为 base64 编码：

```python
import base64
from pathlib import Path
from langchain.messages import HumanMessage

# 读取本地图片并编码
image_path = Path("photo.jpg")
encoded = base64.b64encode(image_path.read_bytes()).decode()

msg = HumanMessage(content=[
    {"type": "text", "text": "描述这张图片"},
    {
        "type": "image_url",
        "image_url": {
            "url": f"data:image/jpeg;base64,{encoded}",
        },
    },
])
```

## Token 使用与统计

### 获取 Token 信息

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage

model = init_chat_model("moonshot:moonshot-v1-8k")

response = model.invoke([HumanMessage("你好，请简单介绍一下自己")])

# 使用统计
print(f"输入 tokens: {response.usage_metadata['input_tokens']}")
print(f"输出 tokens: {response.usage_metadata['output_tokens']}")
print(f"总 tokens: {response.usage_metadata['total_tokens']}")
```

### 不同模型的 Token 估算

| 模型 | 中文 ≈ 字/token | 英文 ≈ 词/token |
|------|----------------|----------------|
| moonshot-v1-8k | ~1.5 字 | ~0.75 词 |
| moonshot-v1-32k | ~1.5 字 | ~0.75 词 |
| moonshot-v1-128k | ~1.5 字 | ~0.75 词 |
| deepseek-chat | ~1.2 字 | ~0.8 词 |

```python
# 粗略估算函数
def estimate_tokens(text: str, is_chinese: bool = True) -> int:
    """估算文本的 token 数量"""
    if is_chinese:
        return max(1, len(text) // 1.5)
    return max(1, len(text.split()) // 0.75)

text = "装饰器是 Python 中一种强大的功能"
print(f"估算 token 数: {estimate_tokens(text)}")
```

## 消息工具方法

### 消息序列化

```python
from langchain.messages import HumanMessage, AIMessage, SystemMessage, to_dict, from_dict

msg = HumanMessage("你好")

# 消息转字典
msg_dict = to_dict(msg)
print(msg_dict)
# {'type': 'human', 'data': {'content': '你好', 'additional_kwargs': {}}}

# 字典转消息
restored_msg = from_dict(msg_dict)
```

### 消息过滤与转换

```python
from langchain.messages import (
    HumanMessage,
    AIMessage,
    SystemMessage,
    filter_messages,
)

messages = [
    SystemMessage("你是助手"),
    HumanMessage("你好"),
    AIMessage("你好！"),
    HumanMessage("再问一个"),
]

# 只保留人类消息
human_only = filter_messages(messages, include_types=[HumanMessage])
print(len(human_only))  # 2

# 排除系统消息
no_system = filter_messages(messages, exclude_types=[SystemMessage])
print(len(no_system))  # 3
```

### 消息裁剪（控制上下文长度）

```python
from langchain.messages import (
    HumanMessage,
    AIMessage,
    SystemMessage,
    trim_messages,
)

messages = [
    SystemMessage("你是助手"),
    HumanMessage("第一轮问题"),
    AIMessage("第一轮回答"),
    HumanMessage("第二轮问题"),
    AIMessage("第二轮回答"),
    HumanMessage("第三轮问题"),
]

# 裁剪到最多 4 条消息，保留系统消息
trimmed = trim_messages(
    messages,
    max_tokens=1000,      # 按 token 数裁剪
    strategy="last",      # 保留最后的消息
    include_system=True,  # 始终保留系统消息
    allow_partial=False,  # 不允许截断单条消息
)

for msg in trimmed:
    print(f"{type(msg).__name__}: {msg.content}")
```

```
裁剪结果：
┌─────────────────────────────────────────┐
│  SystemMessage: 你是助手     (保留)       │
│  HumanMessage: 第二轮问题    (保留)       │
│  AIMessage: 第二轮回答       (保留)       │
│  HumanMessage: 第三轮问题    (保留)       │
│                                         │
│  注：第一轮对话被移除以控制上下文长度       │
└─────────────────────────────────────────┘
```

## 完整对话示例

### 简单对话

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, AIMessage

model = init_chat_model("deepseek:deepseek-chat")

messages = []

# 第一轮
messages.append(HumanMessage("Python 中 list 和 tuple 有什么区别？"))
response1 = model.invoke(messages)
print(f"AI: {response1.content}")
messages.append(response1)

# 第二轮
messages.append(HumanMessage("那 dict 呢？"))
response2 = model.invoke(messages)
print(f"AI: {response2.content}")
messages.append(response2)
```

### 带工具调用的完整对话

```python
from langchain.chat_models import init_chat_model
from langchain.messages import HumanMessage, AIMessage, ToolMessage
from langchain.tools import tool

@tool
def get_weather(city: str) -> str:
    """获取指定城市的天气信息"""
    weather_data = {
        "北京": "晴，25°C",
        "上海": "多云，22°C",
        "广州": "雷阵雨，30°C",
    }
    return weather_data.get(city, f"无法获取 {city} 的天气信息")

# 初始化模型并绑定工具
model = init_chat_model("moonshot:moonshot-v1-8k")
model_with_tools = model.bind_tools([get_weather])

# 用户提问
messages = [HumanMessage("北京今天天气怎么样？")]

# AI 决定调用工具
response = model_with_tools.invoke(messages)
messages.append(response)

# 执行工具调用
if response.tool_calls:
    for tool_call in response.tool_calls:
        tool_name = tool_call["name"]
        tool_args = tool_call["args"]
        tool_id = tool_call["id"]

        # 执行工具
        if tool_name == "get_weather":
            result = get_weather.invoke(tool_args)
        else:
            result = f"未知工具: {tool_name}"

        # 添加工具结果
        messages.append(ToolMessage(content=result, tool_call_id=tool_id))

    # AI 根据工具结果生成最终回复
    final_response = model_with_tools.invoke(messages)
    print(f"最终回复: {final_response.content}")
```

### 多轮工具对话

```python
def chat_with_tools(query: str, max_iterations: int = 5):
    """带工具调用的对话循环"""
    messages = [HumanMessage(query)]

    for _ in range(max_iterations):
        response = model_with_tools.invoke(messages)
        messages.append(response)

        # 如果没有工具调用，说明对话完成
        if not response.tool_calls:
            return response.content

        # 执行所有工具调用
        for tool_call in response.tool_calls:
            result = get_weather.invoke(tool_call["args"])
            messages.append(ToolMessage(
                content=result,
                tool_call_id=tool_call["id"],
            ))

    return "对话达到最大迭代次数"

# 使用
print(chat_with_tools("北京和上海今天天气哪个更好？"))
```

## 消息最佳实践

### 1. 始终保留系统消息

```python
# 好的做法
messages = [
    SystemMessage("你是一个有帮助的助手，回答要简洁。"),
    HumanMessage("你好"),
]

# 裁剪时保留系统消息
trimmed = trim_messages(
    messages,
    max_tokens=500,
    include_system=True,  # 始终保留
)
```

### 2. 使用消息历史管理对话

```python
from langchain_core.chat_history import BaseChatMessageHistory, InMemoryChatMessageHistory

history = InMemoryChatMessageHistory()

# 添加消息
history.add_user_message("你好")
history.add_ai_message("你好！有什么可以帮您？")
history.add_user_message("我想学习 Python")

# 获取完整历史
messages = history.messages
for msg in messages:
    print(f"{type(msg).__name__}: {msg.content}")
```

### 3. 避免消息列表过长

```python
# 上下文窗口限制
model_context = {
    "moonshot-v1-8k": 8192,
    "moonshot-v1-32k": 32768,
    "moonshot-v1-128k": 128000,
    "deepseek-chat": 64000,
}

# 定期检查 token 使用
def check_context_length(messages: list, max_tokens: int = 7000) -> bool:
    """检查消息列表是否超过安全阈值"""
    total = sum(len(m.content) for m in messages if hasattr(m, 'content'))
    estimated = total // 1.5  # 中文粗略估算
    return estimated < max_tokens
```

### 4. 工具调用 ID 必须匹配

```python
# 错误：tool_call_id 不匹配
ai_msg = AIMessage(
    content="",
    tool_calls=[{"name": "weather", "args": {"city": "北京"}, "id": "call_123"}],
)
tool_msg = ToolMessage(content="晴", tool_call_id="call_456")  # ID 不匹配！

# 正确：ID 必须一致
tool_msg = ToolMessage(content="晴", tool_call_id="call_123")  # OK
```

## 小结

| 要点 | 说明 |
|------|------|
| 消息类型 | SystemMessage / HumanMessage / AIMessage / ToolMessage 各司其职 |
| 内容格式 | 支持纯文本和 content blocks（多模态） |
| Token 统计 | `response.usage_metadata` 获取精确计数 |
| 消息裁剪 | `trim_messages` 控制上下文长度，防止超出模型窗口 |
| 工具调用 | `tool_call_id` 必须在 AIMessage 和 ToolMessage 之间匹配 |
| 多模态 | `image_url` content block 支持 URL 和 base64 编码 |
