# 09.01.01 - 模型基础

> LangChain 统一模型接口，一个接口对接所有 LLM 提供商

## 为什么需要统一模型接口

不同 LLM 提供商的 API 差异巨大：

```
OpenAI API          Anthropic API        Google API
────────────        ─────────────        ────────────
messages=[]         messages=[]          contents=[]
role: "user"        role: "user"         role: "user"
response.choices    response.content     response.candidates
usage.prompt_tokens usage.usage.input    usage_metadata
```

LangChain 通过 `init_chat_model` 屏蔽这些差异：

```python
# 一行代码切换任意提供商
model = init_chat_model("moonshot:moonshot-v1-8k")
model = init_chat_model("deepseek:deepseek-chat")
model = init_chat_model("openai:gpt-4o")
```

## 安装

```bash

uv add "langchain[openai]"
```

## 初始化模型

### 方式1：init_chat_model（推荐）

```python
from langchain.chat_models import init_chat_model

# 月之暗面 Kimi
model = init_chat_model("moonshot:moonshot-v1-8k")

# DeepSeek
model = init_chat_model("deepseek:deepseek-chat")

# 通用格式: "provider:model_name"
model = init_chat_model("provider:model_name")
```

### 方式2：直接使用模型类

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(
    model="moonshot-v1-8k",
    openai_api_base="https://api.moonshot.cn/v1",
    api_key="${MOONSHOT_API_KEY}",
)
```

### 方式3：Azure / Bedrock 等

```python
# Azure OpenAI
from langchain_openai import AzureChatOpenAI

model = AzureChatOpenAI(
    model="gpt-4o",
    azure_deployment="my-deployment",
)

# AWS Bedrock
from langchain_aws import ChatBedrock

model = ChatBedrock(model="anthropic.claude-3-5-sonnet-20240620-v1:0")
```

## 模型参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `model` | `str` | 必需 | 模型标识，可用 `provider:model` 格式 |
| `temperature` | `float` | 模型默认 | 输出随机性（0=确定，1=创意） |
| `max_tokens` | `int` | 模型默认 | 最大输出 token 数 |
| `timeout` | `int` | 无限制 | 请求超时秒数 |
| `max_retries` | `int` | 6 | 失败重试次数（指数退避） |
| `api_key` | `str` | 环境变量 | API 密钥 |

```python
model = init_chat_model(
    "moonshot:moonshot-v1-8k",
    temperature=0.1,      # 确定性输出
    max_tokens=4000,      # 最大输出长度
    timeout=120,          # 超时 120 秒
    max_retries=10,       # 网络不稳定时增加
)
```

## 调用模型

### invoke（单次调用）

```python
model = init_chat_model("moonshot:moonshot-v1-8k")

# 简单字符串输入
response = model.invoke("你好，请介绍一下自己")
print(response.content)

# 消息列表输入
from langchain.messages import HumanMessage, SystemMessage

response = model.invoke([
    SystemMessage("你是一个 Python 老师。"),
    HumanMessage("解释一下什么是装饰器"),
])
print(response.content)
```

### stream（流式调用）

```python
for chunk in model.stream("请介绍 Python 的优点"):
    print(chunk.text, end="", flush=True)
```

### batch（批量调用）

```python
responses = model.batch([
    "什么是列表推导式？",
    "什么是生成器？",
    "什么是装饰器？",
])
for r in responses:
    print(r.content)
```

### batch_as_completed（异步完成）

```python
for idx, response in model.batch_as_completed([
    "问题1",
    "问题2",
    "问题3",
]):
    print(f"索引 {idx}: {response.content}")
```

## 多轮对话

```python
from langchain.messages import HumanMessage, AIMessage

messages = [
    HumanMessage("Python 中 list 和 tuple 有什么区别？"),
]

# 第一轮
response1 = model.invoke(messages)
messages.append(response1)

# 第二轮
messages.append(HumanMessage("那 dict 呢？"))
response2 = model.invoke(messages)
messages.append(response2)
```

## 模型配置最佳实践

| 场景 | temperature | max_tokens | 推荐模型 |
|------|------------|------------|---------|
| 代码生成 | 0.0-0.1 | 2000-4000 | moonshot-v1-8k |
| 数据分析 | 0.1-0.3 | 2000-4000 | deepseek-chat |
| 创意写作 | 0.7-1.0 | 1000-2000 | moonshot-v1-32k |
| 摘要总结 | 0.3-0.5 | 500-1000 | moonshot-v1-8k |

## 小结

| 要点 | 说明 |
|------|------|
| 统一接口 | `init_chat_model("provider:model")` 一行切换 |
| 调用方式 | invoke / stream / batch / batch_as_completed |
| 消息格式 | 字符串或消息列表（SystemMessage / HumanMessage / AIMessage） |
| 重试机制 | 自动重试网络错误、429、5xx（默认 6 次） |
