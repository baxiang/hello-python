# 09.01.06 - 提示工程

> 提示词是人与 AI 的沟通桥梁，好的提示词能让 AI 发挥最大效能

## 提示词的核心组成

```
┌──────────────────────────────────────────────────────────────┐
│                      提示词结构                                │
│                                                              │
│  ┌────────────────────────────────────────────────────┐      │
│  │ SystemMessage                                       │      │
│  │ ┌──────────────────────────────────────────────┐   │      │
│  │ │ 角色设定: "你是一个资深的 Python 工程师"       │   │      │
│  │ │ 行为准则: "回答要简洁，包含代码示例"           │   │      │
│  │ │ 格式要求: "使用 Markdown 格式输出"             │   │      │
│  │ └──────────────────────────────────────────────┘   │      │
│  └────────────────────────────────────────────────────┘      │
│  ┌────────────────────────────────────────────────────┐      │
│  │ HumanMessage                                        │      │
│  │ ┌──────────────────────────────────────────────┐   │      │
│  │ │ 用户问题: "解释一下 Python 中的装饰器"         │   │      │
│  │ │ 上下文信息: "我已经学过函数和类"               │   │      │
│  │ └──────────────────────────────────────────────┘   │      │
│  └────────────────────────────────────────────────────┘      │
└──────────────────────────────────────────────────────────────┘
```

## System Prompt 设计

### 基础系统提示

```python
from langchain.chat_models import init_chat_model
from langchain.messages import SystemMessage, HumanMessage

model = init_chat_model("moonshot:moonshot-v1-8k")

# 简单系统提示
response = model.invoke([
    SystemMessage("你是一个专业的编程助手，专注于回答 Python 相关问题。"),
    HumanMessage("什么是装饰器？"),
])
print(response.content)
```

### 结构化系统提示

```python
system_prompt = """你是一个资深的 Python 编程老师，教学风格如下：

1. 用简单的语言解释复杂概念
2. 每个概念都附带代码示例
3. 指出初学者常见的错误
4. 鼓励学生思考和实践

请始终保持友好、耐心的态度，用中文回答问题。"""

response = model.invoke([
    SystemMessage(system_prompt),
    HumanMessage("解释一下 Python 中的 *args 和 **kwargs"),
])
print(response.content)
```

### 多角色系统提示

```python
# 不同场景使用不同角色设定
roles = {
    "code_reviewer": """你是一个严格的代码审查员。
- 指出代码中的问题和潜在 bug
- 给出改进建议
- 关注性能、可读性和安全性""",

    "teacher": """你是一个耐心的编程老师。
- 用简单易懂的方式解释概念
- 给出示例代码
- 指出常见错误""",

    "architect": """你是一个系统架构师。
- 关注整体设计
- 评估技术选型
- 考虑可扩展性和维护性""",
}

def get_response(role: str, question: str):
    system_prompt = roles.get(role, roles["teacher"])
    return model.invoke([
        SystemMessage(system_prompt),
        HumanMessage(question),
    ]).content

# 同一问题，不同角色
question = "如何处理 Python 项目中的依赖管理？"

print("=== 代码审查员视角 ===")
print(get_response("code_reviewer", question))

print("\n=== 老师视角 ===")
print(get_response("teacher", question))

print("\n=== 架构师视角 ===")
print(get_response("architect", question))
```

## Prompt 模板

### 基础模板

```python
from langchain_core.prompts import PromptTemplate

# 定义模板
template = PromptTemplate(
    template="你是一个{role}。请回答以下问题: {question}",
    input_variables=["role", "question"],
)

# 填充变量
prompt = template.format(role="Python 老师", question="什么是装饰器？")
print(prompt)
# 输出: 你是一个Python 老师。请回答以下问题: 什么是装饰器？
```

### Chat Prompt 模板

```python
from langchain_core.prompts import ChatPromptTemplate

# 定义聊天模板
chat_template = ChatPromptTemplate.from_messages([
    ("system", "你是一个{role}。"),
    ("human", "{question}"),
])

# 填充并转换为消息
messages = chat_template.format_messages(
    role="Python 老师",
    question="什么是装饰器？"
)

# 直接调用
model = init_chat_model("moonshot:moonshot-v1-8k")
response = model.invoke(messages)
print(response.content)
```

### 带上下文的模板

```python
from langchain_core.prompts import ChatPromptTemplate

template = ChatPromptTemplate.from_messages([
    ("system", """你是一个编程助手。
当前上下文:
- 语言: {language}
- 难度: {level}
- 用户背景: {background}
"""),
    ("human", "{question}"),
])

messages = template.format_messages(
    language="Python",
    level="初级",
    background="学过基础语法",
    question="什么是列表推导式？",
)

model = init_chat_model("deepseek:deepseek-chat")

```

### 模板链

```python
from langchain_core.prompts import ChatPromptTemplate

# 分析模板
analysis_template = ChatPromptTemplate.from_messages([
    ("system", "你是一个代码分析专家。"),
    ("human", "分析以下代码的问题:\n```python\n{code}\n```"),
])

# 建议模板
suggestion_template = ChatPromptTemplate.from_messages([
    ("system", "你是一个代码优化专家。"),
    ("human", "基于以下分析，给出优化建议:\n{analysis}"),
])

code = """
def get_users():
    users = []
    for i in range(1000):
        users.append({"id": i, "name": f"user_{i}"})
    return users
"""

# 第一步：分析
model = init_chat_model("moonshot:moonshot-v1-8k")
analysis_messages = analysis_template.format_messages(code=code)
analysis = model.invoke(analysis_messages).content

print("=== 代码分析 ===")
print(analysis)

# 第二步：建议
suggestion_messages = suggestion_template.format_messages(analysis=analysis)
suggestion = model.invoke(suggestion_messages).content

print("\n=== 优化建议 ===")
print(suggestion)
```

### 图片 + 结构化输出

```python
from pydantic import BaseModel, Field

class ImageDescription(BaseModel):
    """图片描述"""
    main_subject: str = Field(description="图片中的主要对象")
    setting: str = Field(description="场景/环境")
    colors: list[str] = Field(description="主要颜色")
    mood: str = Field(description="氛围/情绪")

model = init_chat_model("moonshot:moonshot-v1-8k")
structured_model = model.with_structured_output(ImageDescription)

# 流式输出
for chunk in structured_model.stream([
    HumanMessage(content=[
        {"type": "text", "text": "分析这张图片"},
        {"type": "image_url", "image_url": {"url": "https://example.com/photo.jpg"}},
    ]),
]):
    print(chunk)
```

## Prompt 缓存

### 缓存原理

```
┌───────────────────────────────────────────────────────┐
│                  Prompt 缓存机制                       │
│                                                       │
│  请求 1:  ┌────────────────────────────────┐          │
│           │ System: 你是一个Python老师      │          │
│           │ Human: 什么是装饰器？           │          │
│           └────────────────────────────────┘          │
│                      ↓ 缓存                          │
│  请求 2:  ┌────────────────────────────────┐          │
│           │ System: 你是一个Python老师      │ ← 命中缓存 │
│           │ Human: 什么是列表推导式？       │          │
│           └────────────────────────────────┘          │
│                                                       │
│  System prompt 部分被缓存，不需要重新处理               │
└───────────────────────────────────────────────────────┘
```

### 利用缓存优化成本

```python
from langchain.chat_models import init_chat_model
from langchain.messages import SystemMessage, HumanMessage

model = init_chat_model("moonshot:moonshot-v1-8k")

# 固定系统提示，多次提问
system = SystemMessage("""你是一个 Python 编程老师。
请遵循以下原则：
1. 用简单的方式解释
2. 提供代码示例
3. 指出常见错误
""")

questions = [
    "什么是装饰器？",
    "什么是列表推导式？",
    "什么是上下文管理器？",
    "什么是生成器？",
]

for q in questions:
    response = model.invoke([system, HumanMessage(q)])
    print(f"Q: {q}")
    print(f"A: {response.content[:100]}...")
    print(f"Tokens: {response.usage_metadata}")
    print()
```

### 缓存命中策略

```python
# 好的缓存策略：系统提示保持一致
system = SystemMessage("你是一个翻译助手，将英文翻译成中文。")

# 每次只改变用户输入，系统提示不变 → 缓存命中率高
for text in ["Hello", "World", "Python"]:
    response = model.invoke([
        system,
        HumanMessage(f"翻译: {text}"),
    ])

# 不好的缓存策略：系统提示每次都变化
for text in ["Hello", "World", "Python"]:
    response = model.invoke([
        SystemMessage(f"你是一个翻译助手，翻译'{text}'"),  # 每次都不同
        HumanMessage(text),
    ])
```

system_prompt = """

帮助用户理解机器学习概念，并提供实用的代码示例。

- 必须包含代码示例

### 模式3: 少样本提示（Few-Shot）

```python
from langchain.messages import HumanMessage, AIMessage, SystemMessage

    SystemMessage("你是一个简洁的 Python 老师，用最简短的话回答问题。"),

## Prompt 优化技巧

### 技巧1: 具体明确

```python
# 不好：模糊
prompt = "解释一下 Python"

# 好：具体
prompt = "解释 Python 中的 GIL 是什么，它对多线程编程有什么影响"
```

### 技巧2: 提供上下文

```python
# 不好：没有背景
prompt = "怎么写更好？"

# 好：提供完整上下文
prompt = """我正在写一个 Web API，使用 FastAPI。
当前的用户认证代码如下：

```python
@app.post("/login")
def login(username: str, password: str):
    user = db.query(username)
    if user.password == password:
        return {"token": "xxx"}
```

这段代码有什么安全问题？如何改进？"""
```

### 技巧3: 分解复杂问题

```python
# 不好：一个大问题
prompt = "教我 Python 从入门到精通"

# 好：分解成小问题
topics = [
    "Python 的基本数据类型有哪些？",
    "如何定义和使用函数？",
    "类和对象是什么关系？",
    "如何处理文件读写？",
    "什么是装饰器，如何使用？",
]

for topic in topics:
    response = model.invoke([
        SystemMessage("你是一个循序渐进的 Python 老师。"),
        HumanMessage(topic),
    ])
    print(f"主题: {topic}")
    print(f"回答: {response.content[:200]}...\n")
```

### 技巧4: 设置边界

```python
system_prompt = """你是一个代码审查助手。
请注意：
- 只关注代码本身，不要评价代码作者
- 指出的问题必须有具体依据
- 建议要可行且具体
- 不要过度批评
"""
```

### 技巧5: 迭代优化

```python
def iterative_refine(question: str, max_iterations: int = 3):
    """迭代优化回答"""
    messages = [
        SystemMessage("你是一个追求准确的专家。如果不确定的事情，请说明。"),
        HumanMessage(question),
    ]

    for i in range(max_iterations):
        response = model.invoke(messages)
        messages.append(response)

        # 检查回答是否完整
        follow_up = HumanMessage(
            "你的回答是否完整准确？如果有不确定的地方，请补充。"
        )
        messages.append(follow_up)

    return response.content

result = iterative_refine("Python 3.12 有什么新特性？")
print(result)
```

## 常见错误

### 错误1: 提示词过长

```python
# 不好的做法：塞入过多无关信息
system_prompt = """你是一个助手。
Python 是一种编程语言，由 Guido van Rossum 创建，
于 1991 年发布。Python 的设计哲学是代码可读性，
它支持多种编程范式，包括面向对象、命令式、函数式...
（省略 2000 字 Python 百科内容）

请回答：什么是装饰器？"""

# 好的做法：精简到核心
system_prompt = """你是一个 Python 老师。
简洁地解释概念，提供代码示例。"""
```

### 错误2: 矛盾指令

```python
# 矛盾
system_prompt = """
请非常详细地解释这个概念。（要求详细）
但回答不要超过 100 字。（要求简短）
"""

# 一致
system_prompt = """
用 2-3 句话解释概念，并附带一个代码示例。
"""
```

### 错误3: 依赖模型训练数据外的信息

```python
# 模型不知道的信息
response = model.invoke([
    SystemMessage("你是一个助手"),
    HumanMessage("我们公司内部的 Python 编码规范是什么？"),
])
# → 模型不知道，会编造答案（幻觉）

# 好的做法：提供上下文
response = model.invoke([
    SystemMessage("你是一个助手。基于以下编码规范回答问题。"),
    HumanMessage("""
    编码规范：
    1. 函数名使用 snake_case
    2. 类名使用 PascalCase
    3. 每行不超过 88 字符
    4. 使用类型注解

    问题：我们项目要求函数名用 camelCase 吗？
    """),
])
```

## 小结

| 要点 | 说明 |
|------|------|
| 系统提示 | 设定角色、行为准则、输出格式 |
| 模板 | `ChatPromptTemplate` 实现可复用提示词 |
| 多模态 | content blocks 支持文本+图片+视频混合提示 |
| 缓存 | 固定系统提示可提高缓存命中率，降低成本 |
| 设计模式 | 角色-任务-约束、思维链、少样本、输出格式控制 |
| 优化技巧 | 具体明确、提供上下文、分解问题、设置边界 |
| 常见错误 | 提示词过长、矛盾指令、依赖训练外信息 |
