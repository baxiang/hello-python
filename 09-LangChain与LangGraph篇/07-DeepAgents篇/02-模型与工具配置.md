## 模型配置

### 支持的模型

Deep Agents 需要支持**工具调用**（Tool Calling）的模型。所有 LangChain 集成的聊天模型均可使用。

| 提供商 | 模型标识 | 安装 |
|--------|---------|------|
| Moonshot Kimi | `moonshot:moonshot-v1-8k` | `uv add langchain[openai]` |
| DeepSeek | `deepseek:deepseek-chat` | `uv add langchain[openai]` |
| OpenAI | `openai:gpt-4o` | `uv add langchain[openai]` |
| Anthropic | `anthropic:claude-sonnet-4-6` | `uv add langchain[anthropic]` |
| Google | `google_genai:gemini-2.5-flash` | `uv add langchain[google-genai]` |
| Ollama（本地） | `ollama:llama3` | `uv add langchain[ollama]` |

### 三种模型配置方式

#### 方式1：模型字符串（最简）

```python
from deepagents import create_deep_agent

agent = create_deep_agent(model="moonshot:moonshot-v1-8k")
# 底层调用 init_chat_model("moonshot:moonshot-v1-8k")
```

#### 方式2：init_chat_model（推荐，灵活控制）

```python
from langchain.chat_models import init_chat_model
from deepagents import create_deep_agent

model = init_chat_model(
    model="moonshot:moonshot-v1-8k",
    temperature=0.1,      # 确定性输出
    max_tokens=4000,      # 最大输出长度
    timeout=120,          # 超时 120 秒
    max_retries=10,       # 网络不稳定时增加重试
)
agent = create_deep_agent(model=model)
```

#### 方式3：直接使用模型类（最大控制）

```python
from langchain_openai import ChatOpenAI
from deepagents import create_deep_agent

model = ChatOpenAI(
    model="moonshot-v1-8k",
    openai_api_base="https://api.moonshot.cn/v1",
    temperature=0.1,
)
agent = create_deep_agent(model=model)
```

### 参数参考

| 参数 | 含义 | 推荐值 |
|------|------|--------|
| `temperature=0` | 确定性输出 | 代码生成、翻译 |
| `temperature=0.7` | 平衡创意与准确 | 日常对话 |
| `temperature=1.5` | 高创意 | 创意写作 |
| `max_tokens` | 输出长度上限 | 按任务调整 |
| `timeout` | 请求超时秒数 | 30-120 秒 |
| `max_retries` | 失败重试次数 | 6（默认），不稳定时 10-15 |

### 动态模型选择

当应用需要让用户在运行时选择模型时，使用中间件动态切换：

```python
from dataclasses import dataclass
from langchain.chat_models import init_chat_model
from langchain.agents.middleware import wrap_model_call
from deepagents import create_deep_agent

@dataclass
class Context:
    model_name: str

@wrap_model_call
def dynamic_model(request, handler):
    model_name = request.runtime.context.model_name
    model = init_chat_model(model_name)
    return handler(request.override(model=model))

agent = create_deep_agent(
    model="moonshot:moonshot-v1-8k",  # 默认模型
    middleware=[dynamic_model],
    context_schema=Context,
)

# 调用时切换模型
result = agent.invoke(
    {"messages": [{"role": "user", "content": "你好"}]},
    context=Context(model_name="deepseek:deepseek-chat"),
)
```

**工作原理：**
1. `wrap_model_call` 拦截每次 LLM 调用
2. 从 `runtime.context` 读取用户选择的模型
3. 通过 `request.override(model=model)` 替换模型
4. `handler(request)` 继续执行

## 工具（Tools）

### @tool 装饰器

```python
from langchain.tools import tool

@tool
def get_weather(city: str) -> str:
    """获取城市天气信息
    
    Args:
        city: 城市名称
    """
    return f"{city} 天气晴朗，25°C"
```

**关键说明：**
- `"""..."""` 文档字符串是 LLM 选择工具的依据，必须清晰描述用途
- 类型注解（`city: str`）定义输入 schema，必须提供
- 返回值转为字符串

### 自定义工具名称

```python
@tool("web_search")
def search(query: str) -> str:
    """搜索网络信息"""
    return f"搜索结果: {query}"

print(search.name)  # web_search
```

### 带 Pydantic 参数验证

```python
from pydantic import BaseModel, Field
from typing import Literal
from langchain.tools import tool

class WeatherInput(BaseModel):
    """天气查询输入"""
    location: str = Field(description="城市名称或坐标")
    units: Literal["celsius", "fahrenheit"] = Field(
        default="celsius",
        description="温度单位"
    )

@tool(args_schema=WeatherInput)
def get_weather(
    location: str,
    units: str = "celsius",
) -> str:
    """获取当前天气"""
    temp = 22 if units == "celsius" else 72
    return f"{location} 当前温度: {temp}°{units[0].upper()}"
```

### 使用 ToolRuntime 访问上下文

工具可以通过 `ToolRuntime` 访问运行时信息：

```python
from langchain.tools import tool, ToolRuntime
from dataclasses import dataclass

@dataclass
class Context:
    user_id: str
    api_key: str

@tool
def get_user_data(query: str, runtime: ToolRuntime[Context]) -> str:
    """获取当前用户的数据"""
    user_id = runtime.context.user_id
    return f"用户 {user_id} 的数据: {query}"

@tool
def get_user_info(user_id: str, runtime: ToolRuntime[Context]) -> str:
    """查询用户信息"""
    store = runtime.store
    user_info = store.get(("users",), user_id)
    return str(user_info.value) if user_info else "未知用户"

@tool
def get_weather(city: str, runtime: ToolRuntime[Context]) -> str:
    """获取城市天气（带流式反馈）"""
    writer = runtime.stream_writer
    writer(f"正在查询: {city}")
    writer(f"获取数据成功")
    return f"{city} 天气晴朗"
```

### 工具返回值类型

| 返回类型 | 说明 | 示例 |
|---------|------|------|
| `str` | 简单字符串结果 | `f"{city}: 晴天"` |
| `dict` | 结构化数据 | `{"temp": 25, "humid": 60}` |
| `Command` | 更新 State + 路由 | 高级用法，见 LangGraph 篇 |

### 绑定工具到模型（底层原理）

```python
from langchain_openai import ChatOpenAI
from langchain.tools import tool

@tool
def get_weather(city: str) -> str:
    """获取城市天气"""
    return f"{city} 天气晴朗"

model = ChatOpenAI(
    model="moonshot-v1-8k",
    openai_api_base="https://api.moonshot.cn/v1",
)
model_with_tools = model.bind_tools([get_weather])

response = model_with_tools.invoke("北京天气怎么样？")

# 检查工具调用
for tool_call in response.tool_calls:
    print(f"工具: {tool_call['name']}")
    print(f"参数: {tool_call['args']}")
    print(f"ID: {tool_call['id']}")
```

### 强制工具调用

```python
# 强制使用任意工具
model.bind_tools([tool_1], tool_choice="any")

# 强制使用特定工具
model.bind_tools([tool_1], tool_choice="get_weather")
```

### 并行工具调用

模型可同时调用多个工具：

```python
response = model_with_tools.invoke("北京和上海的天气？")

# 可能返回多个工具调用
print(response.tool_calls)
# [
#   {'name': 'get_weather', 'args': {'city': '北京'}, 'id': 'call_1'},
#   {'name': 'get_weather', 'args': {'city': '上海'}, 'id': 'call_2'},
# ]
```

### 流式工具调用

```python
for chunk in model_with_tools.stream("北京天气？"):
    for tool_chunk in chunk.tool_call_chunks:
        if name := tool_chunk.get("name"):
            print(f"工具: {name}")
        if args := tool_chunk.get("args"):
            print(f"参数片段: {args}")

# 累积完整工具调用
gathered = None
for chunk in model_with_tools.stream("北京天气？"):
    gathered = chunk if gathered is None else gathered + chunk
print(gathered.tool_calls)  # 完整工具调用列表
```

## 系统提示词

### 自定义提示词

```python
from deepagents import create_deep_agent

research_prompt = """\
你是专业研究员。你的任务：
1. 彻底调查用户提出的问题
2. 使用工具收集信息
3. 将大型结果保存到文件系统
4. 撰写结构化研究报告
"""

agent = create_deep_agent(
    model="moonshot:moonshot-v1-8k",
    system_prompt=research_prompt,
)
```

### 完整系统提示词组成

Deep Agent 的最终系统提示词由以下部分组成：

| 顺序 | 组件 | 说明 |
|------|------|------|
| 1 | 自定义 `system_prompt` | 你传入的提示词 |
| 2 | 基础 Agent 提示 | 内置的指导说明 |
| 3 | 任务规划提示 | `write_todos` 使用说明 |
| 4 | Memory 提示 | 仅在配置 `memory` 时加入 |
| 5 | Skills 提示 | 仅在配置 `skills` 时加入 |
| 6 | 文件系统提示 | 文件工具使用说明 |
| 7 | 子代理提示 | `task` 工具使用说明 |
| 8 | 中间件提示 | 自定义中间件添加 |
| 9 | 人类审核提示 | 仅在配置 `interrupt_on` 时加入 |

### 动态提示词

当提示词需要根据运行时上下文变化时，使用 `@dynamic_prompt`：

```python
from langchain.agents.middleware import dynamic_prompt
from deepagents import create_deep_agent

@dynamic_prompt
def user_level_prompt(request) -> str:
    level = request.runtime.context.get("user_level", "beginner")
    
    if level == "expert":
        return "你是专家助手，提供详细技术分析。"
    elif level == "beginner":
        return "你是入门助手，用简单语言解释，避免术语。"
    
    return "你是友好的助手。"

agent = create_deep_agent(
    model="moonshot:moonshot-v1-8k",
    middleware=[user_level_prompt],
)
```

### 使用 SystemMessage（支持多模态）

```python
from langchain.messages import SystemMessage
from deepagents import create_deep_agent

agent = create_deep_agent(
    model="anthropic:claude-sonnet-4-6",
    system_prompt=SystemMessage(
        content=[
            {"type": "text", "text": "你是文学分析助手。"},
            {"type": "text", "text": "<小说全文>", "cache_control": {"type": "ephemeral"}},
        ]
    ),
)
```

`cache_control` 让 Anthropic 缓存内容块，降低成本。

### 自定义中间件

#### @wrap_model_call

```python
from langchain.tools import tool
from langchain.agents.middleware import wrap_model_call
from deepagents import create_deep_agent

@tool
def get_weather(city: str) -> str:
    """获取城市天气"""
    return f"{city} 天气晴朗"

call_count = [0]

@wrap_model_call
def log_model_calls(request, handler):
    """记录每次模型调用"""
    call_count[0] += 1
    print(f"[Middleware] 模型调用 #{call_count[0]}")
    result = handler(request)
    print(f"[Middleware] 响应: {result.response.content[:50]}...")
    return result

agent = create_deep_agent(
    model="moonshot:moonshot-v1-8k",
    tools=[get_weather],
    middleware=[log_model_calls],
)
```

#### @wrap_tool_call

```python
from langchain.agents.middleware import wrap_tool_call
from langchain.messages import ToolMessage

@wrap_tool_call
def handle_tool_errors(request, handler):
    """捕获工具错误并返回友好消息"""
    try:
        return handler(request)
    except Exception as e:
        return ToolMessage(
            content=f"工具错误: 请检查输入后重试。({str(e)})",
            tool_call_id=request.tool_call["id"],
        )

agent = create_deep_agent(
    model="moonshot:moonshot-v1-8k",
    tools=[get_weather],
    middleware=[handle_tool_errors],
)
```

#### @before_agent / @after_agent

```python
from langchain.agents.middleware import before_agent, after_agent

@before_agent
def input_guardrail(state, runtime):
    """Agent 执行前的输入检查"""
    user_input = state["messages"][-1].content
    
    banned_words = ["密码", "信用卡"]
    for word in banned_words:
        if word in user_input:
            return {"messages": [{"role": "assistant", "content": "请勿输入敏感信息"}]}
    
    return None  # 继续执行

@after_agent
def output_guardrail(state, runtime):
    """Agent 执行后的输出检查"""
    last = state["messages"][-1]
    if isinstance(last, dict) and "content" in last:
        if "敏感内容" in last["content"]:
            last["content"] = "无法提供该回复，请重新提问"
    return None
```

### 动态工具过滤

```python
from langchain.agents.middleware import wrap_model_call

@wrap_model_call
def filter_tools(request, handler):
    """根据认证状态过滤工具"""
    is_authenticated = request.state.get("authenticated", False)
    
    if not is_authenticated:
        tools = [t for t in request.tools if t.name.startswith("public_")]
        return handler(request.override(tools=tools))
    
    return handler(request)
```

### 自定义中间件类

```python
from langchain.agents.middleware import AgentMiddleware

class CustomMiddleware(AgentMiddleware):
    """自定义中间件类"""
    
    def wrap_model_call(self, request, handler):
        # 模型调用前
        print(f"模型调用: {request.state['messages'][-1].content}")
        result = handler(request)
        # 模型调用后
        print(f"模型响应: {result.response.content[:50]}...")
        return result
    
    def wrap_tool_call(self, request, handler):
        # 工具调用前
        print(f"工具调用: {request.tool_call['name']}")
        result = handler(request)
        # 工具调用后
        print(f"工具结果: {result.content[:50]}...")
        return result
    
    def before_agent(self, state, runtime):
        # Agent 执行前
        pass
    
    def after_agent(self, state, runtime):
        # Agent 执行后
        pass
```

### ⚠️ 中间件注意事项

> **不要在中间件中修改 `self.x` 属性！**
> 
> 子代理、并行工具、并发调用会同时运行，修改共享状态会导致竞态条件。
>
> **正确做法：** 使用图状态（Graph State）跟踪值：

```python
class CustomMiddleware(AgentMiddleware):
    def before_agent(self, state, runtime):
        return {"x": state.get("x", 0) + 1}  # 更新图状态，安全
```

## 错误处理

### 工具错误中间件

```python
from langchain.agents.middleware import wrap_tool_call
from langchain.messages import ToolMessage

@wrap_tool_call
def handle_tool_errors(request, handler):
    """捕获所有工具错误"""
    try:
        return handler(request)
    except Exception as e:
        return ToolMessage(
            content=f"工具错误: 请检查输入后重试。({str(e)})",
            tool_call_id=request.tool_call["id"],
        )

agent = create_deep_agent(
    model="moonshot:moonshot-v1-8k",
    tools=[get_weather],
    middleware=[handle_tool_errors],
)
```

### Agent 执行错误

```python
from langgraph.errors import GraphRecursionError

try:
    result = agent.invoke(
        {"messages": [{"role": "user", "content": "复杂问题"}]},
        config={"recursion_limit": 10},
    )
except GraphRecursionError:
    print("执行步数超限，请简化请求")
```

## 小结

| 要点 | 说明 |
|------|------|
| 模型配置 | 三种方式：字符串、init_chat_model、模型类 |
| 工具定义 | `@tool` 装饰器 + 类型注解 |
| ToolRuntime | 访问 context、store、stream_writer |
| 系统提示词 | 自定义 + 内置组件组合 |
| 中间件 | @wrap_model_call、@wrap_tool_call、@before_agent、@after_agent |
| 错误处理 | 中间件拦截工具错误，捕获 GraphRecursionError |
