## FastAPI 部署

### 完整服务

```python
from fastapi import FastAPI
from pydantic import BaseModel
from langgraph.checkpoint.memory import MemorySaver
from deepagents import create_deep_agent

app = FastAPI()
checkpointer = MemorySaver()

agent = create_deep_agent(
    model="moonshot:moonshot-v1-8k",
    checkpointer=checkpointer,
)

class ChatRequest(BaseModel):
    thread_id: str
    message: str

@app.post("/chat")
async def chat(request: ChatRequest):
    config = {"configurable": {"thread_id": request.thread_id}}
    result = agent.invoke(
        {"messages": [{"role": "user", "content": request.message}]},
        config,
    )
    return {"response": result["messages"][-1].content}

@app.post("/stream")
async def stream(request: ChatRequest):
    from fastapi.responses import StreamingResponse

    config = {"configurable": {"thread_id": request.thread_id}}

    async def generate():
        async for chunk in agent.astream(
            {"messages": [{"role": "user", "content": request.message}]},
            config,
        ):
            yield f"data: {chunk}\n\n"

    return StreamingResponse(generate(), media_type="text/event-stream")

@app.get("/history/{thread_id}")
async def history(thread_id: str):
    config = {"configurable": {"thread_id": thread_id}}
    history = list(agent.get_state_history(config))
    return {"history": [s.values for s in history]}
```

## 生产 Checkpointer

### PostgreSQL（推荐）

```python
from langgraph.checkpoint.postgres import PostgresSaver

conn_string = "postgresql://user:pass@host:5432/db"
checkpointer = PostgresSaver.from_conn_string(conn_string)
checkpointer.setup()

graph = builder.compile(checkpointer=checkpointer)
```

### 异步版本

```python
from langgraph.checkpoint.postgres.aio import AsyncPostgresSaver

checkpointer = AsyncPostgresSaver.from_conn_string(conn_string)
await checkpointer.setup()
```

### Checkpointer 对比

| 类型 | 安装 | 适用 |
|------|------|------|
| InMemorySaver | 内置 | 开发测试 |
| SqliteSaver | `langgraph-checkpoint-sqlite` | 本地应用 |
| PostgresSaver | `langgraph-checkpoint-postgres` | 生产环境 |

## 错误处理

```python
from langgraph.errors import GraphRecursionError

try:
    result = agent.invoke(
        {"messages": [{"role": "user", "content": "复杂问题"}]},
        config={"recursion_limit": 10},
    )
except GraphRecursionError:
    result = {"messages": [{"role": "assistant", "content": "执行超时，请简化请求"}]}
```

## 速率限制

```python
from langchain.agents.middleware import wrap_model_call
from datetime import datetime, timedelta

rate_limit_cache = {}

@wrap_model_call
def rate_limit(request, handler):
    user_id = request.runtime.context.user_id
    now = datetime.now()

    if user_id in rate_limit_cache:
        last_call = rate_limit_cache[user_id]
        if now - last_call < timedelta(seconds=1):
            raise Exception("请求过于频繁，请稍后再试")

    rate_limit_cache[user_id] = now
    return handler(request)
```

## Docker 部署

```dockerfile
FROM python:3.11

WORKDIR /app

COPY pyproject.toml uv.lock ./
RUN pip install uv && uv sync

COPY app ./app

CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000"]
```

docker build -t agent-server .
