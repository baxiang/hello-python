## 学习建议

| 建议 | 说明 |
|------|------|
| 按顺序学习 | 从第 01 章开始，建立完整的知识体系 |
| 动手实践 | 每章都有代码示例，建议运行并修改代码 |
| 做笔记 | 记录关键知识点和自己的理解 |
| 完成练习 | 每章后都有练习题，巩固所学知识 |
| 实战项目 | 学完基础后，尝试独立开发小项目 |
