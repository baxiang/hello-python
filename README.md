## 学习建议

1. **按顺序学习**：建议从第 01 章开始，按顺序学习，建立完整的知识体系
2. **实践为主**：每章都有代码示例，建议动手运行并修改代码
3. **做笔记**：记录关键知识点和自己的理解
4. **做练习**：每章后都有练习题，巩固所学知识
5. **做项目**：学完第 15 章的实战项目后，尝试自己独立开发小项目

## 配套工具

- **Python 版本**：推荐 Python 3.11+
- **编辑器**：VS Code + Python 插件
- **包管理器**：uv（快速、现代）或 pip
