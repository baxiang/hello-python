## 环境准备

本章推荐使用 `uv` 管理依赖。在开始网络相关项目前，在项目根目录执行：

```bash
# 安装 HTTP 请求库（项目一、四）
uv add requests

# 安装 HTML 解析库（项目四）
uv add beautifulsoup4
```

不需要联网的项目（项目二、三、五）只用 Python 标准库，无需额外安装。
