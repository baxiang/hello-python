## 学习建议

### 阶段一：基础入门（1-3 周）
1. 先学习 Web 基础和 HTTP 协议
2. 掌握 HTML/CSS 基础
3. 了解 JavaScript 语法

### 阶段二：Flask 框架（2-3 周）
1. Flask 入门：路由、模板
2. Flask 进阶：数据库、蓝图
3. Flask 高级：认证、缓存

### 阶段三：FastAPI 框架（1-2 周）
1. FastAPI 入门
2. FastAPI 进阶

### 阶段四：核心技能（1-2 周）
1. RESTful API 设计原则
2. WebSocket 实时通信

### 阶段五：项目实战（2-3 周）
1. 完整项目开发
2. 部署上线
