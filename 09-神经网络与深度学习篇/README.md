## 环境准备

开始前请确保已安装以下依赖：

```bash
# 创建项目
uv init deep-learning
cd deep-learning

# 安装 PyTorch (CPU 版本)
uv add torch torchvision torchaudio

# GPU 版本（需要 CUDA）
# uv add torch --index https://download.pytorch.org/whl/cu118

# 可视化工具
uv add matplotlib tensorboard notebook

# 其他工具
uv add numpy pandas scikit-learn pillow
```

**验证安装：**

```python
import torch
print(f"PyTorch 版本：{torch.__version__}")
print(f"CUDA 可用：{torch.cuda.is_available()}")
if torch.cuda.is_available():
    print(f"GPU 数量：{torch.cuda.device_count()}")
    print(f"当前 GPU: {torch.cuda.get_device_name(0)}")
```

## 学完本篇你能做什么？

完成本篇章学习后，你将能够：

✅ **理解神经网络核心原理**
   - 神经元与感知机模型
   - 激活函数（ReLU、Sigmoid、Softmax）
   - 反向传播与梯度下降
   - 损失函数与优化器

✅ **熟练使用 PyTorch 框架**
   - Tensor 创建与操作
   - 自动求导（autograd）
   - nn.Module 搭建网络
   - DataLoader 数据加载

✅ **掌握卷积神经网络（CNN）**
   - 卷积层、池化层原理
   - 经典架构（LeNet、AlexNet、ResNet）
   - 图像分类实战
   - 迁移学习技巧

✅ **掌握循环神经网络（RNN）**
   - RNN、LSTM、GRU 原理
   - 序列建模任务
   - 文本分类/情感分析
   - 词嵌入（Embedding）

✅ **独立完成深度学习项目**
   - 图像分类：猫狗识别
   - 文本生成：唐诗生成
   - 迁移学习：使用预训练模型
   - 模型部署基础
