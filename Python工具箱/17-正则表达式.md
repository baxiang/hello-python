## 第一部分：正则表达式基础

### 17.1 什么是正则表达式

#### 概念说明

```
┌─────────────────────────────────────────┐
│           正则表达式概念                │
├─────────────────────────────────────────┤
│                                         │
│  正则表达式（Regular Expression）       │
│  ─────────────────────────────          │
│  用于匹配字符串模式的强大工具           │
│                                         │
│  可以理解为：                           │
│  • 字符串的"模板"或"模式"               │
│  • 用于描述"什么样的字符串是我想要的"   │
│                                         │
│  ─────────────────────────────────      │
│                                         │
│  应用场景：                             │
│  📧 验证邮箱格式                        │
│  📱 验证手机号                          │
│  🔍 从文本提取特定信息                  │
│  🔄 批量替换文本                        │
│  ✂️ 分割复杂格式的字符串                │
│                                         │
└─────────────────────────────────────────┘
```

#### 正则表达式的思维方式

```
普通思维：
"我要找包含 abc 的字符串"
→ if "abc" in text:

正则思维：
"我要找由 3-5 个数字组成的字符串"
→ \d{3,5}

"我要找以 http 开头，com 结尾的网址"
→ ^http.*com$

"我要找标准的邮箱格式"
→ [\w\.-]+@[\w\.-]+\.\w+
```

### 17.2 re 模块基础

#### 导入和基本使用

```python
import re

# ─────────────────────────────────────
# 最常用的三个函数
# ─────────────────────────────────────

text = "Hello, World! My email is test@example.com"

# 1. search() - 查找第一个匹配
match = re.search(r"World", text)
if match:
    print("找到了 World")

# 2. findall() - 查找所有匹配
emails = re.findall(r"\w+@\w+\.\w+", text)
print(emails)  # ['test@example.com']

# 3. sub() - 替换
new_text = re.sub(r"World", "Python", text)
print(new_text)  # Hello, Python!
```

#### match 对象详解

```python
import re

match = re.search(r"(\d{4})-(\d{2})-(\d{2})", "今天是 2024-01-15")

if match:
    # ─────────────────────────────────
    # 获取匹配内容
    # ─────────────────────────────────
    print(match.group())    # 2024-01-15（完整匹配）
    print(match.group(0))   # 2024-01-15（同上）
    print(match.group(1))   # 2024（第 1 个分组）
    print(match.group(2))   # 01（第 2 个分组）
    print(match.group(3))   # 15（第 3 个分组）

    # ─────────────────────────────────
    # 获取位置信息
    # ─────────────────────────────────
    print(match.start())    # 3（匹配开始位置）
    print(match.end())      # 13（匹配结束位置）
    print(match.span())     # (3, 13)（开始和结束）

    # ─────────────────────────────────
    # 原始字符串和结束位置
    # ─────────────────────────────────
    print(match.string)     # 原始字符串
    print(match.re)         # 原始正则表达式
    print(match.regs)       # 所有分组的范围
```

**match 对象图解：**
```
text = "今天是 2024-01-15"
           │←  span() = (3, 13) →│
           │                     │
           ▼                     ▼
         2 0 2 4 - 0 1 - 1 5
         │     │ │   │ │   │ │
         │     │ └ group(1)     │

         │     │                │
         │     └── group(2)     │

              group(0) = "2024-01-15"
```

## 第二部分：元字符详解

### 17.3 基本元字符

#### 字符匹配元字符

```
┌─────────────────────────────────────────┐
│          基本元字符速查表               │
├─────────────────────────────────────────┤
│                                         │
│  .   匹配任意字符（除换行符）           │
│  ^   匹配字符串开头                     │
│  $   匹配字符串结尾                     │
│  *   匹配前一个字符 0 次或多次           │
│  +   匹配前一个字符 1 次或多次           │
│  ?   匹配前一个字符 0 次或 1 次          │
│  {n}   匹配前一个字符恰好 n 次          │
│  {n,m} 匹配前一个字符 n 到 m 次         │
│  []  字符类，匹配其中任意一个字符       │
│  |   或，匹配左边或右边                 │
│  ()  分组                               │
│  \   转义字符                           │
│                                         │
└─────────────────────────────────────────┘
```

#### 示例详解

```python
import re

# ─────────────────────────────────────
# . - 匹配任意字符（除换行符）
# ─────────────────────────────────────
# a.c 匹配 "a" + 任意字符 + "c"

print(re.findall(r"a.c", "abc"))           # ['abc']
print(re.findall(r"a.c", "aXc"))           # ['aXc']
print(re.findall(r"a.c", "a c"))           # ['a c']（空格也算）
print(re.findall(r"a.c", "ac"))            # []（必须有 1 个字符）

# ─────────────────────────────────────
# ^ - 匹配开头
# ─────────────────────────────────────
# ^Hello 匹配以 "Hello" 开头的字符串

print(re.findall(r"^Hello", "Hello World"))    # ['Hello']
print(re.findall(r"^Hello", "Say Hello"))      # []（不是开头）

# ─────────────────────────────────────
# $ - 匹配结尾
# ─────────────────────────────────────
# World$ 匹配以 "World" 结尾的字符串

print(re.findall(r"World$", "Hello World"))    # ['World']
print(re.findall(r"World$", "World War"))      # []（不是结尾）

# ─────────────────────────────────────
# ^ 和 $ 结合 - 精确匹配整个字符串
# ─────────────────────────────────────
# ^abc$ 只能匹配 "abc"，不能多也不能少

print(re.findall(r"^abc$", "abc"))         # ['abc']
print(re.findall(r"^abc$", "abcd"))        # []
print(re.findall(r"^abc$", "xabc"))        # []
```

### 17.4 量词详解

#### 量词类型

```
┌─────────────────────────────────────────┐
│             量词对比表                  │
├──────────────┬──────────────────────────┤
│    量词      │         说明             │
├──────────────┼──────────────────────────┤
│     *        │ 0 次或多次（贪婪）        │
│     *?       │ 0 次或多次（非贪婪）      │
│     +        │ 1 次或多次（贪婪）        │
│     +?       │ 1 次或多次（非贪婪）      │
│     ?        │ 0 次或 1 次（贪婪）       │
│     ??       │ 0 次或 1 次（非贪婪）     │
│     {n}      │ 恰好 n 次                 │
│     {n,}     │ 至少 n 次                 │
│     {n,m}    │ n 到 m 次                 │
└──────────────┴──────────────────────────┘
```

#### 贪婪 vs 非贪婪

```python
# ─────────────────────────────────────
# 贪婪匹配（默认）- 尽可能多匹配
# ─────────────────────────────────────
text = "<div>内容 1</div><div>内容 2</div>"

# 贪婪：.* 会匹配到最后一个 </div>
result = re.findall(r"<div>.*</div>", text)
print(result)
# ['<div>内容 1</div><div>内容 2</div>']
#   ↑ 整个字符串被当成一个匹配！

# ─────────────────────────────────────
# 非贪婪匹配 - 尽可能少匹配
# ─────────────────────────────────────

# 非贪婪：.*? 会在第一个 </div> 就停止
result = re.findall(r"<div>.*?</div>", text)
print(result)
# ['<div>内容 1</div>', '<div>内容 2</div>']
#   ↑ 两个独立的匹配 ✓

# ─────────────────────────────────────
# 图解贪婪匹配过程
# ─────────────────────────────────────
"""
文本：  <div>A</div><div>B</div>
        │                  │
        └────── .* ────────┘
        贪婪：尽可能多匹配

文本：  <div>A</div><div>B</div>
        │      │
        └─ .*? ─┘
        非贪婪：尽可能少匹配
"""
```

#### 量词示例

```python
import re

text = "a aa aaa aaaa aaaaa"

# * - 0 次或多次
print(re.findall(r"a*", text))
# ['', 'a', '', 'aa', '', 'aaa', '', 'aaaa', '', 'aaaaa', '']

# + - 1 次或多次
print(re.findall(r"a+", text))
# ['a', 'aa', 'aaa', 'aaaa', 'aaaaa']

# ? - 0 次或 1 次
print(re.findall(r"a?", text))
# ['a', '', 'a', 'a', '', 'a', 'a', 'a', '', 'a', 'a', 'a', 'a', '']

# {n} - 恰好 n 次
print(re.findall(r"a{3}", text))
# ['aaa', 'aaa']（从 aaaa 中取 3 个，从 aaaaa 中取 3 个）

# {n,} - 至少 n 次
print(re.findall(r"a{3,}", text))
# ['aaa', 'aaaa', 'aaaaa']

# {n,m} - n 到 m 次
print(re.findall(r"a{2,4}", text))
# ['aa', 'aaa', 'aaaa', 'aaaa']
```

**量词匹配次数图解：**
```
文本：a aa aaa aaaa aaaaa

r"a*"（0 次或多次）：
│ │ ││ │││ ││││ │││││
a  a a aa  a aa aaa ...

r"a+"（1 次或多次）：
│ │ ││ │││ ││││ │││││
a  aa  aaa  aaaa  aaaaa
↑  ↑   ↑    ↑     ↑
每个连续的 a 序列

r"a{2,4}"（2 到 4 次）：
     ││ │││ ││││ ││││
     aa aaa aaaa aaaa
              ↑      ↑
           取 4 个  取 4 个，剩 1 个
```

## 第三部分：字符类

### 17.5 预定义字符类

#### 常用字符类

```
┌─────────────────────────────────────────┐
│         预定义字符类速查                │
├─────────────────────────────────────────┤
│                                         │
│  \d  数字（digit）                      │
│      等价于 [0-9]                       │
│      \d+ 匹配 "1２３", "007"            │
│                                         │
│  \D  非数字（Non-digit）                │
│      等价于 [^0-9]                      │
│      \D+ 匹配 "abc", "Hello!"           │
│                                         │
│  \w  单词字符（word）                   │
│      等价于 [a-zA-Z0-9_]                │
│      \w+ 匹配 "hello", "test_123"       │
│                                         │
│  \W  非单词字符（Non-word）             │
│      等价于 [^a-zA-Z0-9_]               │
│      \W+ 匹配 "!@#", ", ."              │
│                                         │
│  \s  空白字符（space）                  │
│      空格、制表符、换行等               │
│      \s+ 匹配 " ", "  \t", "\n\n"       │
│                                         │
│  \S  非空白字符（Non-space）            │
│      \S+ 匹配 "abc", "123"              │
│                                         │
│  .   任意字符（除换行）                 │
│      .+ 匹配几乎所有字符串              │
│                                         │
└─────────────────────────────────────────┘
```

#### 示例详解

```python
import re

text = "Hello 123! 你好 456."

# ─────────────────────────────────────
# \d - 数字
# ─────────────────────────────────────
print(re.findall(r"\d+", text))
# ['123', '456']

# ─────────────────────────────────────
# \D - 非数字
# ─────────────────────────────────────
print(re.findall(r"\D+", text))
# ['Hello ', '! 你好 ', '.']

# ─────────────────────────────────────
# \w - 单词字符（字母、数字、下划线）
# ─────────────────────────────────────
print(re.findall(r"\w+", text))
# ['Hello', '123', '你好', '456']
# 注意：中文也被认为是\w（Unicode）

# ─────────────────────────────────────
# \W - 非单词字符
# ─────────────────────────────────────
print(re.findall(r"\W+", text))
# [' ', '! ', ' ', '.']

# ─────────────────────────────────────
# \s - 空白字符
# ─────────────────────────────────────
text2 = "a  b\tc\nd"
print(re.findall(r"\s+", text2))
# ['  ', '\t', '\n']

# ─────────────────────────────────────
# \S - 非空白字符
# ─────────────────────────────────────
print(re.findall(r"\S+", text2))
# ['a', 'b', 'c', 'd']
```

### 17.6 自定义字符类

#### 方括号语法

```python
import re

# ─────────────────────────────────────
# [abc] - 匹配 a、b 或 c 中的任意一个

print(re.findall(r"[aeiou]", "hello"))
# ['e', 'o']

# [a-z] - 匹配范围内的字符
# ─────────────────────────────────────

print(re.findall(r"[A-Z]+", "Hello World"))
# ['H', 'W']

# ─────────────────────────────────────
print(re.findall(r"[0-9]+", "电话 123"))

# [a-zA-Z] - 匹配任意字母
# ─────────────────────────────────────
print(re.findall(r"[a-zA-Z]+", "Hello123World"))

# [^abc] - 匹配非 abc 的字符（取反）
# ─────────────────────────────────────
print(re.findall(r"[^abc]+", "abcdef"))
# ['def']

# ─────────────────────────────────────
# 特殊字符在字符类中不需要转义
# ─────────────────────────────────────
print(re.findall(r"[.!?]+", "Hello! How are you?"))

# ─────────────────────────────────────
print(re.findall(r"[-abc]+", "abc-def"))

# ─────────────────────────────────────
# 多个范围组合
# ─────────────────────────────────────
# 匹配 16 进制数
print(re.findall(r"[0-9a-fA-F]+", "FF00AA 123 abc"))
```

## 第四部分：分组和捕获

### 17.7 圆括号分组

#### 基础分组

```python
import re

# ─────────────────────────────────────
# 分组：将多个字符视为一个整体
# ─────────────────────────────────────

# 量词作用于分组
print(re.findall(r"(ab)+", "ab abab ababab"))
# ['ab', 'ab', 'ab']

# 没有分组时，+ 只作用于 b
print(re.findall(r"ab+", "ab abb abbb"))
# ['ab', 'abb', 'abbb']

# ─────────────────────────────────────
# 捕获组：提取匹配的特定部分
# ─────────────────────────────────────

text = "我的电话是 010-12345678"
match = re.search(r"(\d{4})-(\d{8})", text)

if match:
    print(match.group(0))  # 010-12345678（完整匹配）
    print(match.group(1))  # 010（区号）
    print(match.group(2))  # 12345678（号码）
```

#### 分组引用

```python
# ─────────────────────────────────────
# 后向引用：在正则中引用之前的分组
# ─────────────────────────────────────

# 查找重复的单词
text = "hello hello world world"
print(re.findall(r"(\w+) \1", text))
# ['hello', 'world']
# \1 引用第一个分组 (\w+) 匹配的内容

# 查找重复字符
text = "aa bb cc dd"
print(re.findall(r"(\w)\1", text))
# ['a', 'b', 'c', 'd']

# 匹配成对的 HTML 标签
html = "<div>content</div><span>text</span>"
pattern = r"<(\w+)>(.*?)</\1>"
print(re.findall(pattern, html))
# [('div', 'content'), ('span', 'text')]
# \1 引用第一个分组，确保开始和结束标签一致
```

### 17.8 命名捕获组

#### 语法和使用

```python
import re

# ─────────────────────────────────────
# 命名捕获组语法
# ─────────────────────────────────────
# (?P<name>pattern)

text = "2024-01-15"
pattern = r"(?P<year>\d{4})-(?P<month>\d{2})-(?P<day>\d{2})"

match = re.search(pattern, text)

if match:
    # 通过组名访问
    print(match.group("year"))   # 2024
    print(match.group("month"))  # 01
    print(match.group("day"))    # 15

    # 获取所有命名组
    print(match.groupdict())
    # {'year': '2024', 'month': '01', 'day': '15'}

    # 也可以在替换中使用组名
    result = re.sub(pattern, r"\g<month>/\g<day>/\g<year>", text)
    print(result)  # 01/15/2024
```

#### 命名组的优势

```python
# ─────────────────────────────────────
# 普通分组（需要数括号位置）
# ─────────────────────────────────────
pattern1 = r"(\d{4})-(\d{2})-(\d{2})"
# group(1) = 年，group(2) = 月，group(3) = 日
# 容易搞混！

# ─────────────────────────────────────
# 命名分组（语义清晰）
# ─────────────────────────────────────
pattern2 = r"(?P<year>\d{4})-(?P<month>\d{2})-(?P<day>\d{2})"
# group("year") = 年，group("month") = 月，group("day") = 日
# 清晰易懂！
```

### 17.9 非捕获组和其他特殊组

#### 非捕获组

```python
# ─────────────────────────────────────
# (?:pattern) - 非捕获组
# ─────────────────────────────────────
# 只分组，不捕获（节省内存，提高性能）

text = "访问 https://example.com 或 http://test.org"

# 普通分组
print(re.findall(r"(https?|ftp)://\w+", text))
# ['https', 'http']（捕获了协议）

# 非捕获组
print(re.findall(r"(?:https?|ftp)://\w+", text))
# ['https://example', 'http://test']（整个匹配）

# ─────────────────────────────────────
# 使用场景：只需要分组，不需要捕获
# ─────────────────────────────────────

# 匹配 URL，只想要域名，不想要协议
text = "https://example.com/page"
# 使用非捕获组忽略协议
match = re.search(r"(?:https?|ftp)://(\w+\.\w+)", text)
print(match.group(1))  # example.com
```

#### 其他特殊组

```python
# ─────────────────────────────────────
# (?=pattern) - 正向先行断言
# ─────────────────────────────────────
# 匹配后面跟着 pattern 的位置（不消耗字符）

# 匹配后面跟着"元"的数字
text = "价格：100 元，运费：20 元"
print(re.findall(r"\d+(?=元)", text))
# ['100', '20']

# ─────────────────────────────────────
# (?!pattern) - 负向先行断言
# ─────────────────────────────────────
# 匹配后面不跟着 pattern 的位置

# 匹配后面不跟"元"的数字
text = "价格：100 元，编号：123"
print(re.findall(r"\d+(?!元)", text))
# ['123']

# ─────────────────────────────────────
# (?<=pattern) - 正向后行断言
# ─────────────────────────────────────
# 匹配前面是 pattern 的位置

# 匹配"元"前面的数字
text = "价格：100 元"
print(re.findall(r"(?<=元)\d+", text))  # []（元在后面）
print(re.findall(r"(?<=：)\d+", text))  # ['100']

# ─────────────────────────────────────
# (?<!pattern) - 负向后行断言
# ─────────────────────────────────────
# 匹配前面不是 pattern 的位置

# 匹配前面不是字母的数字
text = "a1 b2 c3 4"
print(re.findall(r"(?<![a-zA-Z])\d", text))  # ['4']
```

**断言图解：**
```
文本：  100 元 200 美元 300 元
        │       │       │
        ▼       ▼       ▼

r"\d+(?=元)" - 后面是"元"的数字
        │               │
        └─── 100, 300 ──┘

r"\d+(?!元)" - 后面不是"元"的数字
                │
                └── 200

r"(?<=：)\d+" - 冒号后面的数字
        │
        └── 100
```

## 第五部分：re 模块函数详解

### 17.10 查找函数

#### findall() - 查找所有

```python
import re
```

#### 标志示例

```python
import re

# ─────────────────────────────────────
# re.IGNORECASE - 忽略大小写
# ─────────────────────────────────────
text = "Hello WORLD hello"

print(re.findall(r"hello", text))
# ['hello']

print(re.findall(r"hello", text, re.IGNORECASE))
# ['Hello', 'hello']

print(re.findall(r"hello", text, re.I))  # 简写
# ['Hello', 'hello']

# ─────────────────────────────────────
# re.MULTILINE - 多行模式
# ─────────────────────────────────────
text = """line1
line2
line3"""

# 普通模式：^ 只匹配整个字符串开头
print(re.findall(r"^\w+", text))
# ['line1']

# 多行模式：^ 匹配每行开头
print(re.findall(r"^\w+", text, re.MULTILINE))
# ['line1', 'line2', 'line3']

print(re.findall(r"^\w+", text, re.M))  # 简写
# ['line1', 'line2', 'line3']

# ─────────────────────────────────────
# re.DOTALL - 点号匹配换行
# ─────────────────────────────────────
text = "a\nb\nc"

# 普通模式：. 不匹配换行
print(re.findall(r"a.c", text))
# []

# DOTALL 模式：. 匹配换行
print(re.findall(r"a.c", text, re.DOTALL))
# ['a\nb']

print(re.findall(r"a.c", text, re.S))  # 简写
# ['a\nb']

# ─────────────────────────────────────
# re.VERBOSE - 详细模式（可写注释）
# ─────────────────────────────────────
# 允许在正则中添加空格和注释，提高可读性

pattern = re.compile(r"""
    \d{4}      # 年
    -          # 分隔符
    \d{2}      # 月
    -          # 分隔符
    \d{2}      # 日
""", re.VERBOSE)

match = pattern.search("今天是 2024-01-15")
print(match.group())  # 2024-01-15

# ─────────────────────────────────────
# 组合使用多个标志
# ─────────────────────────────────────
# 使用 | 连接

text = """Hello
world"""

print(re.findall(r"^world", text, re.I | re.M))
# ['world']（忽略大小写 + 多行模式）
```

```python
import re

def validate_email(email):
    """
    验证邮箱格式

    规则：
    - 用户名：字母、数字、点、下划线、连字符
    - @ 符号
    - 域名：字母、数字、连字符
    - 顶级域名：2-6 个字母
    """
    pattern = r'^[\w\.-]+@[\w\.-]+\.\w{2,6}$'
    return bool(re.match(pattern, email))

# 测试

print(validate_email("user.name@company.cn"))  # True
print(validate_email("invalid.email"))         # False
print(validate_email("@example.com"))          # False
print(validate_email("test@.com"))             # False
```

```python
def validate_password(password):
    """
    验证密码强度

    规则：
    - 至少 8 位
    - 包含大写字母
    - 包含小写字母
    - 包含数字
    - 包含特殊字符
    """
    patterns = [
        (r'.{8,}', '长度至少 8 位'),
        (r'[A-Z]', '包含大写字母'),
        (r'[a-z]', '包含小写字母'),
        (r'\d', '包含数字'),
        (r'[!@#$%^&*(),.?":{}|<>]', '包含特殊字符'),
    ]

    errors = []
    for pattern, message in patterns:
        if not re.search(pattern, password):
            errors.append(message)

    if errors:
        return False, errors
    return True, "密码强度合格"

# 测试
valid, msg = validate_password("Weak")
print(f"密码：Weak → {msg}")

valid, msg = validate_password("Strong1!")
print(f"密码：Strong1! → {msg}")
```

```python
def format_phone(phone):
    """格式化电话号码为 138-1234-5678 格式"""
    # 去除所有非数字字符
    digits = re.sub(r'\D', '', phone)

    # 检查长度
    if len(digits) != 11:

    # 格式化
```

```python
def analyze_log(log_text):
    """分析日志，提取错误信息"""
    # 匹配 [ERROR] 开头的行
    pattern = r'\[(ERROR|WARNING|INFO)\]\s+(.+?)(?=\n\[|$)'
    matches = re.findall(pattern, log_text, re.DOTALL)
```

### 知识点回顾

| 知识点 | 说明 | 示例 |
|--------|------|------|
| `re.search()` | 查找第一个匹配 | `re.search(r"\d+", text)` |
| `re.findall()` | 查找所有匹配 | `re.findall(r"\w+", text)` |
| `re.sub()` | 替换 | `re.sub(r"a", "b", text)` |
| `re.split()` | 分割 | `re.split(r",", text)` |
| `re.compile()` | 编译正则 | `re.compile(r"\d+")` |
| `^` `$` | 开头结尾 | `^hello$` |
| `*` `+` `?` | 量词 | `a*`, `a+`, `a?` |
| `{n,m}` | 指定次数 | `a{2,4}` |
| `.` | 任意字符 | `a.c` |
| `[]` | 字符类 | `[aeiou]` |
| `\d` `\w` `\s` | 预定义类 | `\d+` |
| `()` | 分组 | `(abc)+` |
| `\1` `\2` | 后向引用 | `(\w+)\s+\1` |
| `(?P<n>...)` | 命名组 | `(?P<name>\w+)` |
| `(?=)` `(?!` | 断言 | `\d+(?=元)` |
