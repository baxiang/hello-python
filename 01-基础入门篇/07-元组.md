# 第 7 章 - 元组（详细版）

## 第一部分：什么是元组

### 5b.1 元组的概念

#### 概念说明

**元组（Tuple）** 与列表类似，但是**不可变**（创建后不能修改）。

```
┌─────────────────────────────────────────┐
│         列表 vs 元组                     │
├─────────────────────────────────────────┤
│                                         │
│  列表（list）：         元组（tuple）：  │
│  • 使用 []              • 使用 ()        │
│  • 可变（可修改）       • 不可变         │
│  • 方法较多             • 方法较少       │
│  • 稍慢                 • 稍快           │
│                                         │
│  使用场景：             使用场景：       │
│  • 需要修改的数据       • 固定不变的数据  │
│  • 一般数据集合         • 字典的键        │
│                         • 函数多返回值   │
│                                         │
└─────────────────────────────────────────┘
```

## 第二部分：创建元组

### 5b.2 创建方式

```python
# 方式 1：使用圆括号
colors = ("red", "green", "blue")
numbers = (1, 2, 3, 4, 5)

# 方式 2：不使用括号（也可以）
point = 10, 20
print(point)  # (10, 20)

# 方式 3：创建空元组
empty = ()
empty2 = tuple()

# 方式 4：创建单元素元组（必须有逗号！）
single = (1,)      # ✅ 正确
# single = (1)     # ❌ 这是整数，不是元组

# 方式 5：从其他可迭代对象创建
from_list = tuple([1, 2, 3])
print(from_list)  # (1, 2, 3)

from_string = tuple("abc")
print(from_string)  # ('a', 'b', 'c')
```

**重要提示：**
```
┌─────────────────────────────────────────┐
│           ⭐ 重要提示 ⭐                  │
├─────────────────────────────────────────┤
│                                         │
│  创建单元素元组时，必须有逗号！          │
│                                         │
│  (1)     →  整数 1，不是元组            │
│  (1,)    →  元组，包含一个元素          │
│                                         │
│  type((1))    →  <class 'int'>          │
│  type((1,))   →  <class 'tuple'>        │
│                                         │
└─────────────────────────────────────────┘
```

## 第三部分：访问元组元素

### 5b.3 索引与切片

```python
colors = ("red", "green", "blue")

# 索引访问（和列表一样）
print(colors[0])   # red
print(colors[1])   # green
print(colors[-1])  # blue

# 切片（和列表一样）
print(colors[0:2])  # ('red', 'green')
print(colors[::-1]) # ('blue', 'green', 'red')

# 统计
print(len(colors))            # 3
print(colors.count("red"))    # 1
print(colors.index("green"))  # 1
```

## 第四部分：元组的不可变性

### 5b.4 不可变的含义

```python
colors = ("red", "green", "blue")

# ❌ 错误：不能修改元组
# colors[0] = "yellow"  # TypeError: 'tuple' object does not support item assignment

# ❌ 错误：不能添加元素
# colors.append("purple")  # AttributeError

# ✅ 正确：可以重新赋值整个变量
colors = ("yellow", "green", "blue")
print(colors)  # ('yellow', 'green', 'blue')

# 变通方法：转换为列表，修改后再转回元组
colors = ("red", "green", "blue")
temp = list(colors)      # 转为列表
temp[0] = "yellow"       # 修改
colors = tuple(temp)     # 转回元组
print(colors)  # ('yellow', 'green', 'blue')
```

```
┌─────────────────────────────────────────┐
│         为什么需要不可变的元组？          │
├─────────────────────────────────────────┤
│                                         │
│  • 安全性：保护数据不被意外修改          │
│  • 性能：元组比列表占用更少内存          │
│  • 可哈希：可以作为字典的键              │
│  • 语义：表达"这组数据是固定的"          │
│                                         │
└─────────────────────────────────────────┘
```

## 第五部分：元组解包

### 5b.5 基本解包

```python
# 基本解包
person = ("Alice", 25, "Beijing")
name, age, city = person

print(name)  # Alice
print(age)   # 25
print(city)  # Beijing

# 交换变量值
a, b = 10, 20
a, b = b, a
print(a, b)  # 20 10

# 使用 * 收集剩余元素
numbers = (1, 2, 3, 4, 5)
first, *middle, last = numbers

print(first)   # 1
print(middle)  # [2, 3, 4]
print(last)    # 5
```

**解包图解：**
```
person = ("Alice", 25, "Beijing")
          │       │       │
          ▼       ▼       ▼
        name    age     city

结果：
name = "Alice"
age  = 25
city = "Beijing"
```

### 5b.6 函数返回多个值

元组最常见的应用场景之一是让函数返回多个值：

```python
def get_min_max(numbers):
    return min(numbers), max(numbers)  # 返回元组

data = [3, 1, 4, 1, 5, 9, 2, 6]
low, high = get_min_max(data)

print(f"最小值：{low}")   # 1
print(f"最大值：{high}")  # 9
```
