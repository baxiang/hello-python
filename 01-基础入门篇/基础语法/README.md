## 核心知识点

### 变量命名

- 字母/数字/下划线，不能以数字开头
- 使用 snake_case 命名风格

### 数据类型

- int（整数）、float（浮点数）
- str（字符串）、bool（布尔值）

### 运算符优先级

```
() > ** > * / // % > + -
```

### 流程控制

- if / if-else / if-elif-else
- for 循环 / while 循环
- break / continue
