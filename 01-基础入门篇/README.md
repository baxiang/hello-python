## 核心知识点

### Python入门
- Python 特点和应用领域
- 版本选择（Python 3）
- 环境搭建（Windows/Mac/Linux）
- 第一个程序

### 基础语法
- 变量命名规则
- 数据类型（int、float、str、bool）
- 运算符（算术、比较、逻辑）
- 流程控制（if、for、while）

### 字符串
- 创建、索引、切片
- 常用方法
- 格式化字符串
- 正则表达式

### 数据结构
- 列表（list）：有序可变
- 元组（tuple）：有序不可变
- 字典（dict）：键值对
- 集合（set）：无序不重复
- 推导式：简洁生成序列

## 学习建议

1. **先动手**：每学一个概念，都要在交互模式中尝试
2. **多练习**：完成每章的练习题
3. **写代码**：尝试写一些小程序巩固知识
4. **看文档**：学会查阅 Python 官方文档
