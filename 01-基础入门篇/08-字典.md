# 第 8 章 - 字典（详细版）

## 第一部分：什么是字典

### 5c.1 字典的概念

#### 概念说明

**字典（Dictionary）** 是存储键值对（key-value pairs）的数据结构。

```
┌─────────────────────────────────────────┐
│            字典的特点                   │
├─────────────────────────────────────────┤
│                                         │
│  • 存储键值对：{"name": "Alice"}         │
│  • 通过键快速查找值                      │
│  • 键必须唯一且不可变                    │
│  • 值可以重复，可以是任何类型            │
│  • Python 3.7+ 保持插入顺序             │
│                                         │
│  表示方法：使用花括号 {}                 │
│  例如：{"name": "Alice", "age": 25}      │
│                                         │
└─────────────────────────────────────────┘
```

#### 字典的视觉表示

```
字典：person = {"name": "Alice", "age": 25, "city": "Beijing"}

可视化表示：
┌─────────────────────────────────────────┐
│      键 (Key)   │    值 (Value)          │
├─────────────────┼────────────────────── ┤
│    "name"       │    "Alice"             │
│    "age"        │    25                  │
│    "city"       │    "Beijing"           │
└─────────────────────────────────────────┘

查找过程：
person["name"] → 找到键 "name" → 返回值 "Alice"
```

## 第二部分：创建字典

### 5c.2 创建方式

```python
# 方式 1：使用花括号
person = {"name": "Alice", "age": 25, "city": "Beijing"}

# 方式 2：创建空字典
empty_dict = {}
empty_dict2 = dict()

# 方式 3：使用 dict() 函数
person = dict(name="Alice", age=25, city="Beijing")

# 方式 4：从键值对列表创建
person = dict([("name", "Alice"), ("age", 25)])

# 方式 5：使用 zip() 组合两个列表
keys = ["name", "age", "city"]
values = ["Alice", 25, "Beijing"]
person = dict(zip(keys, values))
print(person)  # {'name': 'Alice', 'age': 25, 'city': 'Beijing'}

# 方式 6：字典推导式
squares = {x: x**2 for x in range(5)}
print(squares)  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
```

## 第三部分：访问字典元素

### 5c.3 读取值

```python
person = {"name": "Alice", "age": 25, "city": "Beijing"}

# 方式 1：使用方括号（键不存在会报错）
print(person["name"])  # Alice
# print(person["salary"])  # KeyError: 'salary'

# 方式 2：使用 get() 方法（推荐，更安全）
print(person.get("name"))           # Alice
print(person.get("salary"))         # None（不报错）
print(person.get("salary", 0))      # 0（指定默认值）

# 方式 3：使用 setdefault()（获取或设置）
age = person.setdefault("age", 30)  # 25（已存在，返回原值）
salary = person.setdefault("salary", 50000)  # 50000（不存在，设置并返回）
```

**方括号 vs get()：**
```
person = {"name": "Alice"}

person["name"]:
  └── 键存在 → 返回值
  └── 键不存在 → 报错 KeyError

person.get("name"):
  └── 键存在 → 返回值
  └── 键不存在 → 返回 None（不报错）

person.get("salary", 0):
  └── 键存在 → 返回值
  └── 键不存在 → 返回默认值 0
```

## 第四部分：修改字典

### 5c.4 添加和修改

```python
person = {"name": "Alice", "age": 25}

# 添加新键值对
person["city"] = "Beijing"
print(person)  # {'name': 'Alice', 'age': 25, 'city': 'Beijing'}

# 修改已有键值对
person["age"] = 26
print(person)  # {'name': 'Alice', 'age': 26, 'city': 'Beijing'}

# update() - 批量更新
person.update({"city": "Shanghai", "salary": 50000})
print(person)  # {'name': 'Alice', 'age': 26, 'city': 'Shanghai', 'salary': 50000}

# update() - 使用关键字参数
person.update(age=27, city="Guangzhou")
print(person)
```

### 5c.5 删除元素

```python
person = {"name": "Alice", "age": 25, "city": "Beijing"}

# pop() - 删除并返回值
age = person.pop("age")
print(f"删除的年龄：{age}")  # 25
print(person)  # {'name': 'Alice', 'city': 'Beijing'}

# popitem() - 删除并返回最后一个键值对（Python 3.7+）
last_item = person.popitem()
print(f"删除的项：{last_item}")  # ('city', 'Beijing')

# del - 删除语句
del person["name"]
print(person)  # {}

# clear() - 清空字典
person = {"name": "Alice", "age": 25}
person.clear()
print(person)  # {}
```

## 第五部分：遍历字典

### 5c.6 遍历方式

```python
person = {"name": "Alice", "age": 25, "city": "Beijing"}

# 遍历键（默认）
for key in person:
    print(key)
# 输出：name, age, city

# 或者明确使用 keys()
for key in person.keys():
    print(key)

# 遍历值
for value in person.values():
    print(value)
# 输出：Alice, 25, Beijing

# 遍历键值对（最常用）
for key, value in person.items():
    print(f"{key}: {value}")
# 输出：
# name: Alice
# age: 25
# city: Beijing
```

**遍历方法对比：**
```
person = {"name": "Alice", "age": 25}

for key in person:
  └── 只获取键
  → name, age

for value in person.values():
  └── 只获取值
  → Alice, 25

for key, value in person.items():
  └── 同时获取键和值
  → (name, Alice), (age, 25)
```
