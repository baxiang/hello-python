## 第五部分：第一个完整程序

### 1.10 编写交互式程序

```python
# 文件名：greeting.py

# 这是一个简单的问候程序

# 第 1 行：打印欢迎信息
print("=" * 40)
print("        欢迎学习 Python！")
print("=" * 40)

# 第 2 行：获取用户输入
# input() 函数会等待用户输入，并返回输入的文本
name = input("请问你叫什么名字？")

# 第 3 行：打印个性化问候
print()  # 打印空行
print(f"你好，{name}！很高兴见到你！")

# 第 4 行：获取更多信息
age = input("你今年多大了？")

# 第 5 行：计算并显示
birth_year = 2024 - int(age)  # int() 将字符串转为整数
print(f"你今年{age}岁，大约出生在{birth_year}年")

# 第 6 行：结束程序
print()
print("祝你学习愉快！")
print("=" * 40)
```

#### 运行程序

```bash
# 在终端运行
python greeting.py

# 示例输出
========================================
        欢迎学习 Python！
========================================
请问你叫什么名字？Alice

你好，Alice！很高兴见到你！
你今年多大了？25
你今年 25 岁，大约出生在 1999 年

祝你学习愉快！
========================================
```

#### 代码详解

```python
# 代码逐行解析

# ─────────────────────────────────────
# print("=" * 40)
# ─────────────────────────────────────
# " = " * 40  →  将 "=" 重复 40 次
# 结果："========================================"
# 用于打印分隔线

# ─────────────────────────────────────
# name = input("请问你叫什么名字？")
# ─────────────────────────────────────
# input() → 显示提示文字，等待用户输入
# 用户输入的内容被保存到变量 name 中
# name 的类型是字符串（str）

# ─────────────────────────────────────
# print(f"你好，{name}！")
# ─────────────────────────────────────
# f-string → 格式化字符串
# {name} → 将变量 name 的值插入此处

# ─────────────────────────────────────
# int(age)
# ─────────────────────────────────────
# int() → 将字符串转换为整数
# 因为 input() 返回的是字符串
# 要做数学运算，需要先转换为数字
```

## 本章总结

### 知识点回顾

| 知识点 | 要点 |
|--------|------|
| Python 版本 | 选择 Python 3，不要学 Python 2 |
| 安装 Windows | 勾选"Add Python to PATH" |
| 安装 Mac | 推荐用 Homebrew |
| 运行方式 | 交互式（学习）和脚本（项目） |
| 编辑器 | VS Code（推荐）或 PyCharm |
| 文件扩展名 | .py |

### 常用命令速查

```bash
# 查看 Python 版本
python --version      # Windows
python3 --version     # Mac/Linux

# 进入交互模式
python                # Windows
python3               # Mac/Linux

# 运行脚本文件
python 文件名.py       # Windows
python3 文件名.py      # Mac/Linux

# 退出交互模式
exit()                # 或 Ctrl+D (Mac/Linux)
                      # 或 Ctrl+Z 然后回车 (Windows)
```
