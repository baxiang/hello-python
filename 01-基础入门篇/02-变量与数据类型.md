# 第 2 章 - 变量与数据类型（详细版）

## 第一部分：变量详解

### 2.1 什么是变量

#### 变量的概念

**变量（Variable）** 是程序中用于存储数据的"容器"。你可以把它想象成一个有名字的盒子，盒子里面可以放各种数据。

```
┌─────────────────────────────────────────┐
│              变量示意图                  │
├─────────────────────────────────────────┤
│                                         │
│    变量名        变量值                  │
│      │            │                     │
│      ▼            ▼                     │
│   ┌─────┐     ┌───────┐                 │
│   │ name│────▶│"Alice"│                 │
│   └─────┘     └───────┘                 │
│                                         │
│   ┌─────┐     ┌───────┐                 │
│   │ age │────▶│  25   │                 │
│   └─────┘     └───────┘                 │
│                                         │
└─────────────────────────────────────────┘
```

#### 变量的特点

| 特点 | 说明 | 示例 |
|------|------|------|
| 有名字 | 每个变量都有一个名称 | `name`, `age` |
| 有值 | 变量存储具体的数据 | `"Alice"`, `25` |
| 可改变 | 变量的值可以被修改 | `age = 25` → `age = 26` |
| 有类型 | 变量存储的数据有类型 | 字符串、整数等 |

### 2.4 变量的赋值方式

#### 基本赋值

```python
# 方式 1：直接赋值
x = 10
y = 20
```

#### 链式赋值

```python
# 方式 2：多个变量赋相同的值
# 语法：变量 1 = 变量 2 = ... = 值

a = b = c = 0

# 等价于：
a = 0
b = 0
c = 0

print(a, b, c)  # 输出：0 0 0
```

**图解：**
```
a = b = c = 0
│   │   │   │
│   │   │   └── 值 0
│   │   └────── c 指向 0
│   └────────── b 指向 0
└────────────── a 指向 0

结果：三个变量都指向同一个值
```

#### 多重赋值（解包）

```python
# 方式 3：同时给多个变量赋不同的值
# 语法：变量 1, 变量 2, ... = 值 1, 值 2, ...

name, age, city = "Alice", 25, "Beijing"

# 等价于：
name = "Alice"
age = 25
city = "Beijing"

print(name)  # 输出：Alice
print(age)   # 输出：25
print(city)  # 输出：Beijing
```

**图解：**
```
name, age, city = "Alice", 25, "Beijing"
  │      │      │       │       │    │
  │      │      │       │       │    └── 第 3 个值
  │      │      │       │       └── 第 2 个值
  │      │      │       └── 第 1 个值
  │      │      └── 第 3 个变量
  │      └── 第 2 个变量
  └── 第 1 个变量

对应关系：
  name ← "Alice"
  age  ← 25
  city ← "Beijing"
```

#### 交换变量值

```python
# 传统方式（需要临时变量）
a = 10
b = 20

temp = a   # 先把 a 的值存到临时变量
a = b      # 把 b 的值给 a
b = temp   # 把临时变量的值给 b

print(a, b)  # 输出：20 10

# Python 方式（不需要临时变量）
a = 10
b = 20

a, b = b, a  # 一行搞定！

print(a, b)  # 输出：20 10
```

**为什么 Python 方式可以工作？**
```
a, b = b, a

执行过程：
1. 先计算右边的值：b, a → 20, 10
2. 然后同时赋值：a = 20, b = 10

这样就避免了需要临时变量的问题
```

## 第二部分：数据类型详解

### 2.5 什么是数据类型

#### 概念说明

**数据类型（Data Type）** 是数据的分类。不同类型的数据有不同的特点和操作方法。

```
┌─────────────────────────────────────────┐
│          Python 常见数据类型             │
├─────────────────────────────────────────┤
│                                         │
│  数值类型：                             │
│  • int（整数）：1, 2, 100, -5          │
│  • float（浮点数）：3.14, -0.5         │
│  • complex（复数）：1+2j               │
│                                         │
│  文本类型：                             │
│  • str（字符串）："Hello", "Python"    │
│                                         │
│  布尔类型：                             │
│  • bool（布尔）：True, False           │
│                                         │
│  容器类型：                             │
│  • list（列表）：[1, 2, 3]             │
│  • tuple（元组）：(1, 2, 3)            │
│  • dict（字典）：{"name": "Alice"}     │
│  • set（集合）：{1, 2, 3}              │
│                                         │
└─────────────────────────────────────────┘
```

#### 查看数据类型

```python
# 使用 type() 函数查看类型

name = "Alice"
age = 25
height = 1.68
is_student = True

print(type(name))   # 输出：<class 'str'>
print(type(age))    # 输出：<class 'int'>
print(type(height)) # 输出：<class 'float'>
print(type(is_student))  # 输出：<class 'bool'>
```

**type() 函数说明：**
```
type(变量)
  │    │
  │    └── 要查看的变量
  └── 返回该变量的类型

返回值格式：<class '类型名'>
```

### 2.6 字符串类型（str）

#### 创建字符串

```python
# 方式 1：单引号
name1 = 'Alice'

# 方式 2：双引号
name2 = "Alice"

# 单引号和双引号效果相同
print(name1 == name2)  # 输出：True
```

#### 包含引号的字符串

```python
# 当字符串中包含引号时，可以交替使用单双引号

# 字符串中有双引号，外面用单引号
quote1 = 'He said, "Hello!"'
print(quote1)  # 输出：He said, "Hello!"

# 字符串中有单引号，外面用双引号
quote2 = "It's a beautiful day"
print(quote2)  # 输出：It's a beautiful day

# 使用转义字符
quote3 = "He said, \"Hello!\""
quote4 = 'It\'s a beautiful day'
```

**转义字符说明：**
```
转义字符：以反斜杠 \ 开头的特殊字符

常见转义字符：
  \"  → 双引号
  \'  → 单引号
  \\  → 反斜杠
  \n  → 换行符
  \t  → 制表符（Tab）

示例：
  text = "第一行\n第二行"
  print(text)

  输出：
  第一行
  第二行
```

#### 多行字符串

### 2.7 整数类型（int）

#### 创建整数

```python
# 正整数
positive = 100

# 负整数
negative = -50

# 零
zero = 0

# 大整数（Python 整数没有大小限制）
big_number = 9999999999999999999999999999
```

#### 整数的进制表示

```python
# 十进制（默认）
decimal = 10
print(decimal)  # 输出：10

# 二进制（0b 或 0B 开头）
binary = 0b1010
print(binary)  # 输出：10

# 八进制（0o 或 0O 开头）
octal = 0o12
print(octal)  # 输出：10

# 十六进制（0x 或 0X 开头）
hexadecimal = 0xA
print(hexadecimal)  # 输出：10
```

**进制图解：**
```
不同进制表示同一个数（10）：

十进制：10    →  1×10 + 0×1 = 10
二进制：1010  →  1×8 + 0×4 + 1×2 + 0×1 = 10
八进制：012   →  1×8 + 2×1 = 10
十六进制：0A  →  10 = 10
```

### 2.8 浮点数类型（float）

#### 创建浮点数

```python
# 正浮点数
pi = 3.14159

# 负浮点数
negative_float = -0.5

# 零的浮点数
zero_float = 0.0
```

#### 浮点数精度问题

```python
# 注意：浮点数计算可能有精度误差

result = 0.1 + 0.2
print(result)  # 输出：0.30000000000000004

# 这不是 bug，是计算机表示浮点数的方式导致的
# 需要比较浮点数时，使用近似比较

# 正确做法
import math
print(math.isclose(0.1 + 0.2, 0.3))  # 输出：True
```

### 2.9 布尔类型（bool）

#### 布尔值

```python
# 布尔类型只有两个值
is_true = True
is_false = False

# 注意：首字母必须大写
# true 和 false 是错误的写法
```

#### 布尔值的来源

```python
# 1. 比较运算的结果
print(5 > 3)    # True
print(5 < 3)    # False
print(5 == 5)   # True
print(5 != 3)   # True

# 2. 逻辑运算的结果
print(True and False)  # False
print(True or False)   # True
print(not True)        # False

# 3. 类型转换的结果
print(bool(1))     # True
print(bool(0))     # False
print(bool(""))    # False
print(bool("Hi"))  # True
```

#### 真假值规则

```python
# 以下值在布尔上下文中为 False
print(bool(0))         # False
print(bool(0.0))       # False
print(bool(""))        # False
print(bool([]))        # False
print(bool({}))        # False
print(bool(None))      # False

# 其他所有值都为 True
print(bool(1))         # True
print(bool(-1))        # True
print(bool("Hello"))   # True
print(bool([1, 2]))    # True
```

**真假值规则总结：**
```
┌─────────────────────────────────────────┐
│            真假值判断规则                │
├─────────────────────────────────────────┤
│                                         │
│  False（假）：                          │
│  • 数字 0（包括 0, 0.0, 0j）            │
│  • 空字符串 ""                          │
│  • 空列表 []                            │
│  • 空字典 {}                            │
│  • 空集合 set()                         │
│  • None                                 │
│                                         │
│  True（真）：                           │
│  • 非零数字                             │
│  • 非空字符串                           │
│  • 非空容器                             │
│                                         │
└─────────────────────────────────────────┘
```

## 第三部分：类型转换

### 2.10 什么是类型转换

**类型转换（Type Conversion）** 是将一种数据类型的值转换为另一种数据类型。

```
┌─────────────────────────────────────────┐
│           常见类型转换函数               │
├─────────────────────────────────────────┤
│                                         │
│  int(x)     →  将 x 转换为整数           │
│  float(x)   →  将 x 转换为浮点数         │
│  str(x)     →  将 x 转换为字符串         │
│  bool(x)    →  将 x 转换为布尔值         │
│                                         │
└─────────────────────────────────────────┘
```

### 2.11 转换为整数（int）

```python
# 字符串转整数
num_str = "123"
num_int = int(num_str)
print(num_int)        # 输出：123
print(type(num_int))  # 输出：<class 'int'>

# 浮点数转整数（直接截断小数部分）
float_num = 3.9
int_num = int(float_num)
print(int_num)  # 输出：3（不是 4！）

# 布尔值转整数
print(int(True))   # 输出：1
print(int(False))  # 输出：0
```

**浮点数转整数的陷阱：**
```
int(3.9) = 3  ← 不是四舍五入，是直接去掉小数部分

如果需要四舍五入：
  round(3.9) = 4
  round(3.1) = 3
```

#### 常见错误

```python
# ❌ 错误：不能转换非数字字符串
# int("abc")  # ValueError: invalid literal

# ❌ 错误：不能转换带小数点的字符串
# int("3.14")  # ValueError

# ✅ 正确做法
int(float("3.14"))  # 先转浮点，再转整数 → 3
```

### 2.12 转换为浮点数（float）

```python
# 字符串转浮点数
num_str = "3.14"
num_float = float(num_str)
print(num_float)        # 输出：3.14
print(type(num_float))  # 输出：<class 'float'>

# 整数转浮点数
int_num = 10
float_num = float(int_num)
print(float_num)  # 输出：10.0

# 布尔值转浮点数
print(float(True))   # 输出：1.0
print(float(False))  # 输出：0.0
```

### 2.13 转换为字符串（str）

```python
# 整数转字符串
num = 123
text = str(num)
print(text)         # 输出："123"
print(type(text))   # 输出：<class 'str'>

# 浮点数转字符串
pi = 3.14
pi_str = str(pi)
print(pi_str)       # 输出："3.14"

# 布尔值转字符串
print(str(True))    # 输出："True"
print(str(False))   # 输出："False"
```

### 2.14 类型转换综合示例

```python
# 场景：用户输入的是字符串，需要转换为数字计算

# 用户输入
user_input = "25"

# 转换为整数
age = int(user_input)

# 计算
next_year_age = age + 1

# 转换回字符串用于输出
print("明年你的年龄是：" + str(next_year_age))
# 输出：明年你的年龄是：26
```

**为什么需要转换？**
```
用户输入 → "25" (字符串)
           ↓ int()
整数运算 → 25 (整数) + 1 = 26
           ↓ str()
输出显示 → "26" (字符串)
```

## 本章总结

| 知识点 | 要点 |
|--------|------|

```

❌ 使用关键字：class = "Math"
✅ 正确：class_name = "Math"

❌ 布尔值小写：true, false
```
