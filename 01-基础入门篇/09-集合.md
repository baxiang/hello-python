# 第 9 章 - 集合（详细版）

## 第一部分：什么是集合

### 5d.1 集合的概念

#### 概念说明

**集合（Set）** 是无序、不重复的元素集合。

```
┌─────────────────────────────────────────┐
│            集合的特点                   │
├─────────────────────────────────────────┤
│                                         │
│  • 无序：元素没有固定顺序              │
│  • 不重复：自动去重                    │
│  • 可变：可以添加、删除元素            │
│  • 支持集合运算：并集、交集、差集等    │
│                                         │
│  表示方法：使用花括号 {}               │
│  例如：{1, 2, 3, 4, 5}                 │
│                                         │
│  空集合必须用 set()，不能用 {}          │
│  （因为 {} 是空字典）                   │
│                                         │
└─────────────────────────────────────────┘
```

## 第二部分：创建集合

### 5d.2 创建方式

```python
# 方式 1：使用花括号
fruits = {"apple", "banana", "orange"}
numbers = {1, 2, 3, 4, 5}

# 方式 2：创建空集合（必须用 set()）
empty_set = set()
# empty_set = {}  # ❌ 这是空字典！

# 方式 3：从列表去重
numbers = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
unique_numbers = set(numbers)
print(unique_numbers)  # {1, 2, 3, 4}

# 方式 4：从字符串去重
chars = set("hello")
print(chars)  # {'h', 'e', 'l', 'o'}（注意：'l' 只保留一个）
```

## 第三部分：集合基本操作

### 5d.3 添加和删除元素

```python
fruits = {"apple", "banana", "orange"}

# add() - 添加单个元素
fruits.add("grape")
print(fruits)  # {'apple', 'banana', 'orange', 'grape'}

# update() - 添加多个元素
fruits.update(["mango", "peach"])
print(fruits)

# remove() - 删除元素（不存在会报错）
fruits.remove("banana")
# fruits.remove("xyz")  # KeyError: 'xyz'

# discard() - 删除元素（不存在不会报错）
fruits.discard("banana")  # 不会报错
fruits.discard("xyz")     # 同样不会报错

# pop() - 随机删除一个元素（因为集合无序）
removed = fruits.pop()
print(f"删除：{removed}")

# clear() - 清空集合
fruits.clear()
print(fruits)  # set()
```

**remove() vs discard()：**
```
fruits = {"apple", "banana"}

remove("banana"):
  └── 存在 → 删除成功
  └── 不存在 → 报错 KeyError ❌

discard("banana"):
  └── 存在 → 删除成功
  └── 不存在 → 静默忽略 ✅（推荐用于不确定元素是否存在的场景）
```

## 第四部分：集合运算

### 5d.4 数学集合运算

```python
A = {1, 2, 3, 4, 5}
B = {4, 5, 6, 7, 8}

# 并集（所有元素）
print(A | B)           # {1, 2, 3, 4, 5, 6, 7, 8}
print(A.union(B))

# 交集（共同元素）
print(A & B)           # {4, 5}
print(A.intersection(B))

# 差集（在 A 中但不在 B 中）
print(A - B)           # {1, 2, 3}
print(A.difference(B))

# 对称差集（只在一个集合中的元素）
print(A ^ B)           # {1, 2, 3, 6, 7, 8}
print(A.symmetric_difference(B))
```

**集合运算图解：**
```
A = {1, 2, 3, 4, 5}
B = {4, 5, 6, 7, 8}

文氏图表示：

    ┌───────────┐
    │ 1    2    │
    │   3   ┌───┼───┐
    │       │ 4 │ 5 │ 6   7   8
    │       └───┼───┘
    └───────────┘
        A           B

并集 (A | B): {1, 2, 3, 4, 5, 6, 7, 8}  → 所有区域
交集 (A & B): {4, 5}                    → 重叠区域
差集 (A - B): {1, 2, 3}                 → 只在 A 中
对称差 (A ^ B): {1, 2, 3, 6, 7, 8}      → 非重叠区域
```

### 5d.5 子集与超集判断

```python
A = {1, 2, 3}
B = {1, 2, 3, 4, 5}

# 子集：A 的所有元素都在 B 中
print(A.issubset(B))    # True
print(A <= B)           # True

# 超集：B 包含 A 的所有元素
print(B.issuperset(A))  # True
print(B >= A)           # True

# 不相交：两个集合没有共同元素
C = {6, 7, 8}
print(A.isdisjoint(C))  # True
```

## 第五部分：常见应用场景

### 5d.6 去重

```python
# 快速去除列表中的重复元素
data = [1, 2, 2, 3, 3, 3, 4, 5, 5]
unique = list(set(data))
print(unique)  # [1, 2, 3, 4, 5]（顺序不保证）
```

### 5d.7 成员检查

```python
# 集合的成员检查比列表快得多（O(1) vs O(n)）
valid_users = {"alice", "bob", "charlie", "david"}

user = "alice"
if user in valid_users:
    print(f"{user} 是有效用户")
```

## 第六部分：练习题

### 练习 1：集合运算

```python
# 两个班级选课情况
class_a = {"数学", "英语", "物理", "化学"}
class_b = {"英语", "物理", "历史", "地理"}

# 1. 两个班都选的课（交集）
both = class_a & class_b
print(f"都选的课：{both}")  # {'英语', '物理'}

# 2. 只在一个班选的课（对称差集）
only_one = class_a ^ class_b
print(f"只在一个班选的课：{only_one}")

# 3. 所有选的课程（并集）
all_courses = class_a | class_b
print(f"所有课程：{all_courses}")
```

### 练习 2：去重与统计

```python
# 找出两篇文章共同出现的单词
article1 = set("python is great for data science".split())
article2 = set("data science is amazing with python".split())

# 共同词汇
common = article1 & article2
print(f"共同词汇：{common}")

# 只在文章 1 中出现的词
only_in_1 = article1 - article2
print(f"只在文章1中：{only_in_1}")
```

## 本章总结

### 四种数据结构对比

| 特性 | 列表 | 元组 | 字典 | 集合 |
|------|------|------|------|------|
| 符号 | [] | () | {} | {} |
| 有序 | ✅ | ✅ | ✅(3.7+) | ❌ |
| 可变 | ✅ | ❌ | ✅ | ✅ |
| 可重复 | ✅ | ✅ | 键不可重复 | ❌ |
| 索引访问 | ✅ | ✅ | 按键访问 | ❌ |

### 如何选择数据结构

```
需要存储有序数据？
├─ 需要修改 → 列表（list）
└─ 不可修改 → 元组（tuple）

需要通过键查找？
└─ 字典（dict）

需要去重或集合运算？
└─ 集合（set）
```

```
集合常用方法速查：
  添加：add, update
  删除：remove（报错）, discard（不报错）, pop, clear
  运算：| 并集, & 交集, - 差集, ^ 对称差集
  判断：issubset, issuperset, isdisjoint
```
