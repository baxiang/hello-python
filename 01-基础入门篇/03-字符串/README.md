## 核心知识点

### 创建字符串

- 单引号、双引号、三引号
- f-string 格式化

### 索引和切片

- 正向索引从 0，反向索引从 -1
- 切片 `[start:end:step]`

### 常用方法

- `find/replace/split/join`
- `lower/upper/strip`

### 正则表达式

- `re.findall() / re.sub() / re.match()`
- 常用模式：`\d \w \s . * + ?`
