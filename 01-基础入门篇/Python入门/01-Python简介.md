## 什么是 Python

### 概念说明

**Python** 是一种高级、通用、解释型的编程语言，由荷兰程序员 **吉多·范罗苏姆（Guido van Rossum）** 于 1989 年圣诞节期间开始创建。

```
┌─────────────────────────────────────────┐
│          Python 名称的由来              │
├─────────────────────────────────────────┤
│                                         │
│  Python 的名字来源于英国喜剧团体        │
│  "Monty Python's Flying Circus"         │
│  （蒙提·派森的飞行马戏团）               │
│                                         │
│  与"蟒蛇"的意思无关！                   │
│  （虽然 Python 的 logo 是两条蛇）         │
│                                         │
└─────────────────────────────────────────┘
```

### Python 2 vs Python 3

```
┌─────────────────┬───────────┬───────────┐
```
