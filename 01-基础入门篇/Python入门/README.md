## 核心知识点

### Python 特点

- 简洁易读、开源免费、跨平台
- 丰富的库、多用途

### 应用领域

- Web 开发、数据分析、人工智能
- 自动化脚本、游戏开发、科学计算

### 版本选择

- **选择 Python 3**
- Python 2 已在 2020 年停止维护

### 安装要点

- Windows：勾选 "Add Python to PATH"
- Mac：推荐使用 Homebrew
- Linux：使用包管理器安装

### 运行方式

```bash
# 交互式模式
python          # Windows
python3         # Mac/Linux

# 脚本文件模式
python hello.py      # Windows
python3 hello.py     # Mac/Linux
```
