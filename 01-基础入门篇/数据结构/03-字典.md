# 字典

本章讲解 Python 字典的创建、访问、修改和遍历。

---

## 什么是字典

**字典（Dictionary）** 是存储键值对（key-value pairs）的数据结构。

```
┌─────────────────────────────────────────┐
│            字典的特点                   │
├─────────────────────────────────────────┤
│                                         │
│  • 存储键值对：{"name": "Alice"}         │
│  • 通过键快速查找值                      │
│  • 键必须唯一且不可变                    │
│  • 值可以重复，可以是任何类型            │
│  • Python 3.7+ 保持插入顺序             │
│                                         │
└─────────────────────────────────────────┘
```

---

## 创建字典

```python
# 使用花括号
person = {"name": "Alice", "age": 25, "city": "Beijing"}

# 创建空字典
empty_dict = {}
empty_dict2 = dict()

# 使用 dict() 函数
person = dict(name="Alice", age=25)

# 从键值对列表创建
person = dict([("name", "Alice"), ("age", 25)])

# 使用 zip() 组合两个列表
keys = ["name", "age", "city"]
values = ["Alice", 25, "Beijing"]
person = dict(zip(keys, values))
```

---

## 访问字典

```python
person = {"name": "Alice", "age": 25, "city": "Beijing"}

# 使用方括号（键不存在会报错）
print(person["name"])  # Alice

# 使用 get() 方法（推荐，更安全）
print(person.get("name"))        # Alice
print(person.get("salary"))      # None
print(person.get("salary", 0))   # 0（指定默认值）
```

---

## 修改字典

```python
person = {"name": "Alice", "age": 25}

# 添加新键值对
person["city"] = "Beijing"

# 修改已有键值对
person["age"] = 26

# update() - 批量更新
person.update({"city": "Shanghai", "salary": 50000})
```

---

## 删除元素

```python
person = {"name": "Alice", "age": 25, "city": "Beijing"}

# pop() - 删除并返回值
age = person.pop("age")

# popitem() - 删除最后一个键值对
last_item = person.popitem()

# del - 删除语句
del person["name"]

# clear() - 清空字典
person.clear()
```

---

## 遍历字典

```python
person = {"name": "Alice", "age": 25, "city": "Beijing"}

# 遍历键
for key in person:
    print(key)

# 遍历值
for value in person.values():
    print(value)

# 遍历键值对（最常用）
for key, value in person.items():
    print(f"{key}: {value}")
```
