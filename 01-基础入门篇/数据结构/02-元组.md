# 元组

本章讲解 Python 元组的创建、访问和不可变性。

---

## 什么是元组

**元组（Tuple）** 与列表类似，但是**不可变**（创建后不能修改）。

```
┌─────────────────────────────────────────┐
│         列表 vs 元组                     │
├─────────────────────────────────────────┤
│                                         │
│  列表（list）：         元组（tuple）：  │
│  • 使用 []              • 使用 ()        │
│  • 可变（可修改）       • 不可变         │
│  • 方法较多             • 方法较少       │
│                                         │
│  使用场景：             使用场景：       │
│  • 需要修改的数据       • 固定不变的数据  │
│  • 一般数据集合         • 字典的键        │
│                         • 函数多返回值   │
│                                         │
└─────────────────────────────────────────┘
```

---

## 创建元组

```python
# 使用圆括号
colors = ("red", "green", "blue")
numbers = (1, 2, 3, 4, 5)

# 不使用括号（也可以）
point = 10, 20

# 创建空元组
empty = ()
empty2 = tuple()

# 创建单元素元组（必须有逗号！）
single = (1,)      # ✅ 正确
# single = (1)     # ❌ 这是整数，不是元组

# 从其他可迭代对象创建
from_list = tuple([1, 2, 3])
```

---

## 访问元组

```python
colors = ("red", "green", "blue")

# 索引访问
print(colors[0])   # red
print(colors[-1])  # blue

# 切片
print(colors[0:2])  # ('red', 'green')
print(colors[::-1]) # ('blue', 'green', 'red')

# 统计
print(len(colors))            # 3
print(colors.count("red"))    # 1
print(colors.index("green"))  # 1
```

---

## 元组的不可变性

```python
colors = ("red", "green", "blue")

# ❌ 不能修改元组
# colors[0] = "yellow"  # TypeError

# ✅ 可以重新赋值整个变量
colors = ("yellow", "green", "blue")

# 变通方法：转换为列表，修改后再转回
temp = list(colors)
temp[0] = "yellow"
colors = tuple(temp)
```

## 元组解包

```python
# 基本解包
person = ("Alice", 25, "Beijing")
name, age, city = person

print(name)  # Alice
print(age)   # 25
print(city)  # Beijing

# 交换变量值
a, b = 10, 20
a, b = b, a
print(a, b)  # 20 10

# 使用 * 收集剩余元素
numbers = (1, 2, 3, 4, 5)
first, *middle, last = numbers

print(first)   # 1
print(middle)  # [2, 3, 4]
print(last)    # 5
```

---

## 函数返回多个值

```python
def get_min_max(numbers):
    return min(numbers), max(numbers)  # 返回元组

data = [3, 1, 4, 1, 5, 9, 2, 6]
low, high = get_min_max(data)

print(f"最小值：{low}")   # 1
print(f"最大值：{high}")  # 9
```
