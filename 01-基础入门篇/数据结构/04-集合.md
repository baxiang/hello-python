# 集合

本章讲解 Python 集合的创建、操作和集合运算。

---

## 什么是集合

**集合（Set）** 是无序、不重复的元素集合。

```
┌─────────────────────────────────────────┐
│            集合的特点                   │
├─────────────────────────────────────────┤
│                                         │
│  • 无序：元素没有固定顺序              │
│  • 不重复：自动去重                    │
│  • 可变：可以添加、删除元素            │
│  • 支持集合运算：并集、交集、差集等    │
│                                         │
│  空集合必须用 set()，不能用 {}          │
│  （因为 {} 是空字典）                   │
│                                         │
└─────────────────────────────────────────┘
```

---

## 创建集合

```python
# 使用花括号
fruits = {"apple", "banana", "orange"}

# 创建空集合（必须用 set()）
empty_set = set()
# empty_set = {}  # ❌ 这是空字典！

# 从列表去重
numbers = [1, 2, 2, 3, 3, 3, 4]
unique_numbers = set(numbers)
print(unique_numbers)  # {1, 2, 3, 4}

# 从字符串去重
chars = set("hello")
print(chars)  # {'h', 'e', 'l', 'o'}
```

---

## 集合操作

```python
fruits = {"apple", "banana", "orange"}

# add() - 添加元素
fruits.add("grape")

# update() - 添加多个元素
fruits.update(["mango", "peach"])

# remove() - 删除元素（不存在会报错）
fruits.remove("banana")

# discard() - 删除元素（不存在不会报错）
fruits.discard("xyz")  # 不会报错

# pop() - 随机删除一个元素
removed = fruits.pop()

# clear() - 清空集合
fruits.clear()
```

## 集合运算

```python
A = {1, 2, 3, 4, 5}
B = {4, 5, 6, 7, 8}

# 并集（所有元素）
print(A | B)           # {1, 2, 3, 4, 5, 6, 7, 8}
print(A.union(B))

# 交集（共同元素）
print(A & B)           # {4, 5}
print(A.intersection(B))

# 差集（在 A 中但不在 B 中）
print(A - B)           # {1, 2, 3}
print(A.difference(B))

# 对称差集（只在一个集合中的元素）
print(A ^ B)           # {1, 2, 3, 6, 7, 8}
```

---

## 子集与超集

```python
A = {1, 2, 3}
B = {1, 2, 3, 4, 5}

# 子集判断
print(A.issubset(B))    # True
print(A <= B)           # True

# 超集判断
print(B.issuperset(A))  # True
print(B >= A)           # True

# 不相交判断
C = {6, 7, 8}
print(A.isdisjoint(C))  # True
```

## 常见应用

```python
# 去重
data = [1, 2, 2, 3, 3, 3, 4]
unique = list(set(data))

# 成员检查（集合比列表快）
valid_users = {"alice", "bob", "charlie"}
if "alice" in valid_users:
    print("有效用户")
```
