# 推导式

本章讲解 Python 的列表推导式、字典推导式和集合推导式。

---

## 什么是推导式

**推导式（Comprehension）** 是 Python 特有的一种简洁的序列生成方式。

```
┌─────────────────────────────────────────┐
│          推导式的优势                   │
├─────────────────────────────────────────┤
│                                         │
│  • 代码简洁：一行代码完成循环           │
│  • 可读性好：逻辑集中                  │
│  • 性能更好：比普通循环更快             │
│                                         │
└─────────────────────────────────────────┘
```

---

## 列表推导式

### 基本语法

```python
# [表达式 for 变量 in 可迭代对象]

# 示例：创建平方列表
squares = [i ** 2 for i in range(1, 6)]
print(squares)  # [1, 4, 9, 16, 25]

# 对比传统写法
squares = []
for i in range(1, 6):
    squares.append(i ** 2)
```

### 带条件的推导式

```python
# [表达式 for 变量 in 可迭代对象 if 条件]

# 示例：只保留偶数
evens = [n for n in range(1, 11) if n % 2 == 0]
print(evens)  # [2, 4, 6, 8, 10]

# 示例：筛选并处理
even_squares = [n ** 2 for n in range(1, 11) if n % 2 == 0]
print(even_squares)  # [4, 16, 36, 64, 100]
```

### 嵌套循环

```python
# [表达式 for 变量 1 in 可迭代对象 1 for 变量 2 in 可迭代对象 2]

# 示例：生成所有组合
colors = ["red", "green"]
sizes = ["S", "M"]
products = [(color, size) for color in colors for size in sizes]
print(products)
# [('red', 'S'), ('red', 'M'), ('green', 'S'), ('green', 'M')]
```

## 字典推导式

### 基本语法

```python
# {键表达式: 值表达式 for 变量 in 可迭代对象}

# 示例：创建平方字典
squares = {i: i ** 2 for i in range(1, 6)}
print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# 示例：交换键值对
person = {"name": "Alice", "age": 25}
swapped = {v: k for k, v in person.items()}
print(swapped)  # {'Alice': 'name', 25: 'age'}
```

### 带条件的字典推导式

```python
# 示例：只保留值大于 60 的项
scores = {"Alice": 85, "Bob": 60, "Charlie": 45}
passed = {name: score for name, score in scores.items() if score >= 60}
print(passed)  # {'Alice': 85, 'Bob': 60}
```

---

## 集合推导式

### 基本语法

```python
# {表达式 for 变量 in 可迭代对象}

# 示例：创建平方集合
squares = {i ** 2 for i in range(1, 6)}
print(squares)  # {1, 4, 9, 16, 25}

# 示例：自动去重
numbers = [1, 2, 2, 3, 3, 3, 4]
unique = {n for n in numbers}
print(unique)  # {1, 2, 3, 4}
```
