# AI 聊天助手项目

使用 Python 调用大语言模型 API，构建一个智能聊天助手应用。

---

## 项目概述

```
┌─────────────────────────────────────────────────────────────┐
│                    AI 聊天助手架构                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   用户输入 → Python 应用 → LLM API → 响应输出               │
│                                                             │
│   核心功能：                                                 │
│   • 多轮对话管理                                            │
│   • 上下文记忆                                              │
│   • 流式响应输出                                            │
│   • 命令行/Web 界面                                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 技术栈

- **API 调用**: `openai` / `requests`
- **异步处理**: `asyncio` / `aiohttp`
- **Web 框架**: FastAPI + WebSocket
- **前端**: HTML + JavaScript（可选 Vue/React）

---

## 学习目标

1. 掌握 LLM API 调用方式
2. 理解对话上下文管理
3. 实现流式响应处理
4. 构建 CLI 和 Web 界面
