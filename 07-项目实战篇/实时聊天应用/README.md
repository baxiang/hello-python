# 实时聊天应用项目

使用 WebSocket 实现多人实时聊天应用，支持群聊和私聊功能。

---

## 项目概述

```
┌─────────────────────────────────────────────────────────────┐
│                    实时聊天应用架构                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   客户端 A ←──┐                                             │
│              │                                              │
│   客户端 B ←──┼── WebSocket Server ──→ 消息广播            │
│              │                                              │
│   客户端 C ←──┘                                             │
│                                                             │
│   核心功能：                                                 │
│   • 用户登录/注册                                           │
│   • 群聊消息                                                │
│   • 在线用户列表                                            │
│   • 消息历史记录                                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 技术栈

- **后端**: FastAPI + WebSocket
- **数据库**: SQLite
- **前端**: HTML + CSS + JavaScript
- **实时通信**: WebSocket

---

## 学习目标

1. 理解 WebSocket 协议原理
2. 实现多人实时聊天
3. 掌握连接状态管理
4. 实现消息持久化
