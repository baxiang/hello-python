# 任务管理系统

通过 5 个递进式项目，从简单 TODO 工具到完整项目管理系统。

---

## 项目导航

| 项目 | 文件 | 核心技能 |
|------|------|---------|
| 01 待办事项工具 | [01-待办事项工具.md](./01-待办事项工具.md) | CRUD、JSON 存储 |
| 02 任务管理服务 | [02-任务管理服务.md](./02-任务管理服务.md) | MySQL、ORM |
| 03 任务 API 服务 | [03-任务 API 服务.md](./03-任务 API 服务.md) | FastAPI、RESTful |
| 04 团队协作系统 | [04-团队协作系统.md](./04-团队协作系统.md) | 多用户、权限、标签 |
| 05 项目管理平台 | [05-项目管理平台.md](./05-项目管理平台.md) | 看板、甘特图、统计、Docker |

---

## 项目关系图

```
项目 01: 待办事项工具
    │
    ▼  (添加数据库存储)
项目 02: 任务管理服务
    │
    ▼  (添加 API 接口)
项目 03: 任务 API 服务
    │
    ▼  (添加多用户)
项目 04: 团队协作系统
    │
    ▼  (添加高级功能)
项目 05: 项目管理平台
```

---

## 技术栈总览

| 技术 | 项目 | 用途 |
|------|------|------|
| JSON | 01 | 本地存储 |
| SQLite | 01-02 | 轻量数据库 |
| MySQL | 02-05 | 主数据库 |
| SQLAlchemy | 02-05 | ORM |
| FastAPI | 03-05 | API 服务 |
| Vue.js | 05 | 前端界面 |
| Redis | 05 | 缓存 |
| Docker | 05 | 部署 |
