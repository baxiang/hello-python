# 文件管理系统

通过 4 个递进式项目，从本地文件管理到云存储服务。

---

## 项目导航

| 项目 | 文件 | 核心技能 |
|------|------|---------|
| 01 文件管理工具 | [01-文件管理工具.md](./01-文件管理工具.md) | os/shutil、路径操作 |
| 02 文件存储服务 | [02-文件存储服务.md](./02-文件存储服务.md) | MySQL、文件元数据 |
| 03 文件 API 服务 | [03-文件 API 服务.md](./03-文件 API 服务.md) | FastAPI、文件上传下载 |
| 04 文件管理系统 | [04-文件管理系统.md](./04-文件管理系统.md) | 权限管理、分享、回收站 |

---

## 项目关系图

```
项目 01: 文件管理工具
    │
    ▼  (添加数据库存储)
项目 02: 文件存储服务
    │
    ▼  (添加 API 接口)
项目 03: 文件 API 服务
    │
    ▼  (添加高级功能)
项目 04: 文件管理系统
```

---

## 技术栈总览

| 技术 | 项目 | 用途 |
|------|------|------|
| os/shutil | 01-04 | 文件操作 |
| MySQL | 02-04 | 元数据存储 |
| SQLAlchemy | 02-04 | ORM |
| FastAPI | 03-04 | API 服务 |
| Redis | 04 | 缓存 |
| hashlib | 01-04 | 文件哈希 |
