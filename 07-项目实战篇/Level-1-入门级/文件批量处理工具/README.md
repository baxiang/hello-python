# 文件批量处理工具

通过 5 个递进式项目，从简单脚本到完整工具集，系统掌握 Python 文件处理技能。

---

## 项目导航

| 项目 | 文件 | 核心技能 |
|------|------|---------|
| 01 文件查找工具 | [01-文件查找工具.md](./01-文件查找工具.md) | os、glob、pathlib |
| 02 批量重命名工具 | [02-批量重命名工具.md](./02-批量重命名工具.md) | 正则、批量操作 |
| 03 文件内容处理 | [03-文件内容处理.md](./03-文件内容处理.md) | 文本处理、编码 |
| 04 格式转换工具 | [04-格式转换工具.md](./04-格式转换工具.md) | 文件格式转换 |
| 05 自动化工具集 | [05-自动化工具集.md](./05-自动化工具集.md) | 脚本集成、CLI |

---

## 项目关系图

```
项目 01: 文件查找工具
    │
    ▼  (添加批量操作)
项目 02: 批量重命名工具
    │
    ▼  (添加内容处理)
项目 03: 文件内容处理
    │
    ▼  (添加格式转换)
项目 04: 格式转换工具
    │
    ▼  (整合所有功能)
项目 05: 自动化工具集
```

---

## 技术栈总览

| 技术 | 项目 | 用途 |
|------|------|------|
| os/pathlib | 01-05 | 文件系统操作 |
| glob | 01-05 | 文件匹配 |
| re | 02-05 | 正则表达式 |
| chardet | 03 | 编码检测 |
| Pillow | 04 | 图片处理 |
| click | 05 | CLI 命令行 |
