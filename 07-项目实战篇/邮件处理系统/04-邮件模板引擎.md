### 2.2 模板引擎基础

```python
# mail_template.py
from jinja2 import Environment, FileSystemLoader, select_autoescape
import os

class MailTemplate:
    """邮件模板引擎"""

    def __init__(self, template_dir: str = "./templates"):
        self.template_dir = template_dir
        os.makedirs(template_dir, exist_ok=True)

        self.env = Environment(
            loader=FileSystemLoader(template_dir),
            autoescape=select_autoescape(['html', 'xml']),
            trim_blocks=True,
            lstrip_blocks=True
        )

    def render(self, template_name: str, **kwargs) -> str:
        """渲染模板

        Args:
            template_name: 模板文件名
            **kwargs: 模板变量
        """
        template = self.env.get_template(template_name)
        return template.render(**kwargs)

    def render_string(self, template_string: str, **kwargs) -> str:
        """渲染模板字符串"""
        template = self.env.from_string(template_string)
        return template.render(**kwargs)
```

### 2.3 创建模板

```html
<!-- templates/welcome.html -->
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <style>
        body { font-family: Arial, sans-serif; line-height: 1.6; }
        .container { max-width: 600px; margin: 0 auto; padding: 20px; }
        .header { background: #4CAF50; color: white; padding: 20px; text-align: center; }
        .content { padding: 20px; background: #f9f9f9; }
        .footer { padding: 10px; text-align: center; color: #666; font-size: 12px; }
    </style>
</head>
<body>
    <div class="container">
        <div class="header">
            <h1>欢迎 {{ name }}！</h1>
        </div>
        <div class="content">
            <p>感谢您注册成为我们的会员！</p>
            <p>您的注册信息：</p>
            <ul>
                <li><strong>用户名：</strong>{{ username }}</li>
                <li><strong>注册时间：</strong>{{ register_date }}</li>
                <li><strong>会员等级：</strong>{{ level }}</li>
            </ul>

            {% if vip %}
            <p style="color: #ff9800;">🎉 恭喜您获得 VIP 会员资格！</p>
            {% endif %}
        </div>
        <div class="footer">
            <p>© 2024 公司名称 - 感谢您的支持</p>
        </div>
    </div>
</body>
</html>
```

### 2.4 使用模板发送邮件

```python
from mail_sender import MailSender
from mail_template import MailTemplate

# 初始化
sender = MailSender(
    smtp_server="smtp.qq.com",
    smtp_port=587,
    username="your_email@qq.com",
    password="your_auth_code"
)

template_engine = MailTemplate()

# 渲染模板
html_content = template_engine.render(
    "welcome.html",
    name="张三",
    username="zhangsan",
    register_date="2024-01-01",
    level="VIP",
    vip=True
)

# 发送邮件
sender.send_email(
    to_addrs=["user@example.com"],
    subject="欢迎注册",
    body=html_content,
    mail_type="html"
)
```

## 第三部分 - 高级功能

### 3.1 循环渲染

```html
<!-- templates/order_detail.html -->
<!DOCTYPE html>
<html>
<body>
    <h2>订单详情</h2>
    <p>订单号：{{ order_id }}</p>
    <p>下单时间：{{ order_date }}</p>

    <table border="1" cellpadding="10" style="border-collapse: collapse;">
        <tr>
            <th>商品名称</th>
            <th>数量</th>
            <th>单价</th>
            <th>小计</th>
        </tr>
        {% for item in items %}
        <tr>
            <td>{{ item.name }}</td>
            <td>{{ item.quantity }}</td>
            <td>¥{{ item.price }}</td>
            <td>¥{{ item.quantity * item.price }}</td>
        </tr>
        {% endfor %}
        <tr>
            <td colspan="3"><strong>总计</strong></td>
            <td><strong>¥{{ total }}</strong></td>
        </tr>
    </table>
</body>
</html>
```

### 3.2 模板继承

```html
<!-- templates/base.html -->
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <style>
        body { font-family: Arial, sans-serif; margin: 0; padding: 0; }
        .header { background: #333; color: white; padding: 20px; }
        .content { padding: 20px; }
        .footer { background: #f5f5f5; padding: 10px; text-align: center; }
    </style>
</head>
<body>
    <div class="header">
        <h1>{% block title %}邮件通知{% endblock %}</h1>
    </div>
    <div class="content">
        {% block content %}{% endblock %}
    </div>
    <div class="footer">
        {% block footer %}© 2024 公司名称{% endblock %}
    </div>
</body>
</html>
```

```html
<!-- templates/alert.html -->
{% extends "base.html" %}

{% block title %}价格提醒{% endblock %}

{% block content %}
<p>您好，{{ username }}：</p>
<p>{{ message }}</p>

{% if details %}
<h3>详细信息：</h3>
<ul>
{% for key, value in details.items() %}
    <li><strong>{{ key }}：</strong>{{ value }}</li>
{% endfor %}
</ul>
{% endif %}
{% endblock %}
```

### 3.3 宏定义

```html
<!-- templates/macros.html -->
{% macro button(text, url, color='#4CAF50') %}
<a href="{{ url }}" style="display: inline-block; padding: 10px 20px;
    background: {{ color }}; color: white; text-decoration: none;
    border-radius: 5px;">{{ text }}</a>
{% endmacro %}

{% macro badge(text, color='#2196F3') %}
<span style="display: inline-block; padding: 3px 8px;
    background: {{ color }}; color: white; border-radius: 3px;
    font-size: 12px;">{{ text }}</span>
{% endmacro %}
```

```html
<!-- 使用宏 -->
{% import "macros.html" as macros %}

{{ macros.button("查看详情", "https://example.com") }}
{{ macros.badge("新订单", "#ff9800") }}
```
