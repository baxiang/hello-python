# 邮件处理系统

通过 5 个递进式项目，从简单邮件发送到完整自动化工作流，系统掌握 Python 邮件处理技能。

---

## 项目导航

| 项目 | 文件 | 核心技能 |
|------|------|---------|
| 01 邮件发送工具 | [01-邮件发送工具.md](./01-邮件发送工具.md) | SMTP、邮件构建 |
| 02 邮件接收与解析 | [02-邮件接收与解析.md](./02-邮件接收与解析.md) | IMAP、邮件解析 |
| 03 附件处理系统 | [03-附件处理系统.md](./03-附件处理系统.md) | 文件处理、下载 |
| 04 邮件模板引擎 | [04-邮件模板引擎.md](./04-邮件模板引擎.md) | Jinja2、模板渲染 |
| 05 自动化工作流 | [05-自动化工作流.md](./05-自动化工作流.md) | 定时任务、规则引擎 |

---

## 项目关系图

```
项目 01: 邮件发送工具
    │
    ▼  (添加接收功能)
项目 02: 邮件接收与解析
    │
    ▼  (添加附件处理)
项目 03: 附件处理系统
    │
    ▼  (添加模板功能)
项目 04: 邮件模板引擎
    │
    ▼  (添加自动化)
项目 05: 自动化工作流
```

---

## 技术栈总览

| 技术 | 项目 | 用途 |
|------|------|------|
| smtplib | 01-05 | SMTP 邮件发送 |
| email | 01-05 | 邮件构建解析 |
| imaplib | 02-05 | IMAP 邮件接收 |
| Jinja2 | 04-05 | 模板引擎 |
| APScheduler | 05 | 定时任务 |
