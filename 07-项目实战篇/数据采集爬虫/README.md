# 数据采集爬虫项目

学习使用 Python 爬取网页数据，包括静态页面和动态页面。

---

## 项目概述

```
┌─────────────────────────────────────────────────────────────┐
│                    爬虫工作流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   目标网站 → 发送请求 → 获取响应 → 解析数据 → 存储数据      │
│                                                             │
│   核心技术：                                                 │
│   • requests - HTTP 请求                                    │
│   • BeautifulSoup - HTML 解析                               │
│   • playwright - 动态页面                                   │
│   • SQLite/CSV - 数据存储                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 技术栈

- **HTTP 请求**: `requests` / `httpx`
- **HTML 解析**: `beautifulsoup4` / `lxml`
- **动态页面**: `playwright`
- **数据存储**: `csv` / `json` / `sqlite3`

---

## 学习目标

1. 掌握 HTTP 请求方法
2. 学会 HTML 解析技巧
3. 处理动态加载页面
4. 实现数据持久化存储
