# 日志分析系统

通过 4 个递进式项目，从日志采集到分析告警。

---

## 项目导航

| 项目 | 文件 | 核心技能 |
|------|------|---------|
| 01 日志采集工具 | [01-日志采集工具.md](./01-日志采集工具.md) | 文件读取、正则解析、模拟数据 |
| 02 日志存储服务 | [02-日志存储服务.md](./02-日志存储服务.md) | MySQL、Elasticsearch |
| 03 日志 API 服务 | [03-日志 API 服务.md](./03-日志 API 服务.md) | FastAPI、搜索、过滤 |
| 04 日志分析系统 | [04-日志分析系统.md](./04-日志分析系统.md) | 统计、报表、告警 |

---

## 项目关系图

```
项目 01: 日志采集工具
    │
    ▼  (添加数据库存储)
项目 02: 日志存储服务
    │
    ▼  (添加 API 接口)
项目 03: 日志 API 服务
    │
    ▼  (添加分析功能)
项目 04: 日志分析系统
```

---

## 技术栈总览

| 技术 | 项目 | 用途 |
|------|------|------|
| 正则表达式 | 01-04 | 日志解析 |
| MySQL | 02-04 | 数据库存储 |
| Elasticsearch | 02-04 | 全文搜索 |
| SQLAlchemy | 02-04 | ORM |
| FastAPI | 03-04 | API 服务 |
| Redis | 04 | 缓存/告警 |

---

## 日志数据来源

项目内置模拟日志生成功能，学习时可直接生成测试数据。

```python
from log_generator import generate_access_log
generate_access_log(count=10000, output_file="access.log")
```
