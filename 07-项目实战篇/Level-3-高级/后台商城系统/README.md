# 后台商城系统

通过 5 个递进式项目，从数据模型到完整后台管理系统，打造工业级电商后端服务。

---

## 项目导航

| 项目 | 文件 | 核心技能 |
|------|------|---------|
| 01 商城基础架构与数据模型 | [01-商城基础架构与数据模型.md](./01-商城基础架构与数据模型.md) | SQLAlchemy、数据模型设计 |
| 02 商品管理与库存系统 | [02-商品管理与库存系统.md](./02-商品管理与库存系统.md) | 商品CRUD、库存管理、图片上传 |
| 03 订单处理与支付集成 | [03-订单处理与支付集成.md](./03-订单处理与支付集成.md) | 订单流程、支付集成、事务处理 |
| 04 用户系统与权限管理 | [04-用户系统与权限管理.md](./04-用户系统与权限管理.md) | JWT认证、RBAC权限、雪花ID |

## 项目特点

### 工业级特性

- **完整的数据模型**：用户、商品、订单、支付、库存等完整实体
- **事务支持**：订单创建、库存扣减、支付回调的完整事务处理
- **幂等性设计**：防止重复提交和重复支付
- **分布式锁**：库存扣减、支付回调等并发场景处理
- **缓存策略**：商品信息、用户会话的 Redis 缓存
- **异步任务**：订单超时取消、库存同步等后台任务
- **日志追踪**：完整的请求日志和业务日志
- **错误处理**：统一的异常处理和错误码体系
- **API 文档**：完整的 Swagger 文档
- **Docker 部署**：容器化部署方案
