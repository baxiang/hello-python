# 股票数据监控系统

通过 5 个递进式项目，从简单脚本到完整监控系统，系统掌握 Python 数据处理与实时监控技能。

---

## 项目导航

| 项目 | 文件 | 核心技能 |
|------|------|---------|
| 01 股票查询工具 | [01-股票查询工具.md](./01-股票查询工具.md) | HTTP 请求、JSON 解析 |
| 02 股票数据采集 | [02-股票数据采集.md](./02-股票数据采集.md) | 爬虫、数据清洗 |
| 03 数据存储与管理 | [03-数据存储与管理.md](./03-数据存储与管理.md) | MySQL、ORM |
| 04 Stock API 服务 | [04-Stock API 服务.md](./04-Stock API 服务.md) | FastAPI、RESTful |
| 05 股票监控平台 | [05-股票监控平台.md](./05-股票监控平台.md) | 实时监控、告警通知 |

---

## 项目关系图

```
项目 01: 股票查询工具
    │
    ▼  (添加数据持久化)
项目 02: 股票数据采集
    │
    ▼  (添加数据库存储)
项目 03: 数据存储与管理
    │
    ▼  (添加 API 接口)
项目 04: Stock API 服务
    │
    ▼  (添加监控和告警)
项目 05: 股票监控平台
```

---

## 技术栈总览

| 技术 | 项目 | 用途 |
|------|------|------|
| requests | 01-05 | HTTP 请求 |
| BeautifulSoup | 02 | 网页爬虫 |
| MySQL | 03-05 | 数据存储 |
| SQLAlchemy | 03-05 | ORM |
| FastAPI | 04-05 | API 服务 |
| APScheduler | 05 | 定时任务 |
| smtplib | 05 | 邮件告警 |
