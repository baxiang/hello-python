# 博客系统

通过 5 个递进式项目，从简单文章管理到完整 CMS 系统。

---

## 项目导航

| 项目 | 文件 | 核心技能 |
|------|------|---------|
| 01 文章管理工具 | [01-文章管理工具.md](./01-文章管理工具.md) | Markdown、文件存储 |
| 02 博客后端服务 | [02-博客后端服务.md](./02-博客后端服务.md) | MySQL、ORM |
| 03 博客 API 服务 | [03-博客 API 服务.md](./03-博客 API 服务.md) | FastAPI、搜索、分页 |
| 04 内容管理系统 | [04-内容管理系统.md](./04-内容管理系统.md) | 富文本编辑器、标签、分类 |
| 05 完整博客平台 | [05-完整博客平台.md](./05-完整博客平台.md) | 评论、点赞、主题、Docker |

---

## 项目关系图

```
项目 01: 文章管理工具
    │
    ▼  (添加数据库)
项目 02: 博客后端服务
    │
    ▼  (添加 API)
项目 03: 博客 API 服务
    │
    ▼  (添加管理功能)
项目 04: 内容管理系统
    │
    ▼  (添加社交功能)
项目 05: 完整博客平台
```

---

## 技术栈总览

| 技术 | 项目 | 用途 |
|------|------|------|
| Markdown | 01-05 | 文章格式 |
| MySQL | 02-05 | 数据库 |
| SQLAlchemy | 02-05 | ORM |
| FastAPI | 03-05 | API |
| Elasticsearch | 03-05 | 搜索 |
| Redis | 05 | 缓存 |
| Docker | 05 | 部署 |
