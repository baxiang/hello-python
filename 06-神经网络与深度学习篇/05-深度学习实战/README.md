## 核心技术

### 1. 迁移学习

**概念：** 使用在大型数据集上训练好的模型，迁移到自己的任务

```
在大数据集训练 → 学习通用特征
    ↓
在自己的数据集微调 → 适应特定任务
```

**优势：**
- ⚡ 训练速度快
- 💰 需要数据少
- 🎯 效果通常更好

### 2. 数据增强

```python
from torchvision import transforms

transform = transforms.Compose([
    transforms.RandomHorizontalFlip(),  # 随机水平翻转
    transforms.RandomRotation(10),      # 随机旋转
    transforms.ColorJitter(0.1, 0.1),   # 颜色抖动
    transforms.ToTensor(),
])
```

**作用：** 增加数据多样性，提高模型泛化能力

### 3. 模型优化技巧

| 技巧 | 作用 | 使用场景 |
|------|------|---------|
| **Dropout** | 防止过拟合 | 全连接层后 |
| **Batch Normalization** | 加速训练 | 卷积/全连接层后 |
| **学习率衰减** | 提高精度 | 训练后期 |
| **早停** | 防止过拟合 | 验证集loss不再下降 |

## 常见问题

### Q: 训练速度太慢怎么办？
**A:** 
- 减小 batch size
- 使用更小的模型
- 减少训练轮数
- 使用 GPU

### Q: 模型过拟合怎么办？
**A:** 
- 增加数据（数据增强）
- 使用 Dropout
- 减小模型复杂度
- 早停

### Q: 如何选择预训练模型？
**A:** 
- 图像分类：ResNet, EfficientNet
- 目标检测：YOLO, Faster R-CNN
- NLP：BERT, GPT
