## 核心概念预览

### 1. 神经元模型

```
输入 → 加权求和 → 激活函数 → 输出

x₁, x₂, x₃ → w₁·x₁ + w₂·x₂ + w₃·x₃ + b → f(z) → y
```

### 2. 网络结构

```
输入层    隐藏层    输出层
  ●         ●         ●
  ●    →    ●    →    ●
  ●         ●
```

### 3. 学习过程

```
1. 前向传播：计算预测值
2. 计算损失：衡量预测与真实值的差距
3. 反向传播：计算梯度
4. 更新权重：优化参数
```
