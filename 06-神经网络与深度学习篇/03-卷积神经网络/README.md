## CNN 核心概念

### 为什么需要 CNN？

**传统神经网络处理图像的问题：**

```
问题1：参数太多
- 输入：100x100 像素图像
- 展平：10,000 维向量
- 第一层 100 个神经元：10,000 x 100 = 100万参数！
❌ 参数爆炸，难以训练
```

**CNN 的解决方案：**

```
卷积操作：
- 使用小滤波器（如 3x3）
- 参数数量：3 x 3 = 9 个
✅ 参数大大减少！
```

### 卷积操作示意

```
输入图像         卷积核          输出特征图
┌───────┐      ┌─────┐         ┌─────┐
│1 2 3 4│      │1 0 1│         │? ? ?│
│2 3 4 5│  *   │0 1 0│    =    │? ? ?│
│3 4 5 6│      │1 0 1│         │? ? ?│
│4 5 6 7│      └─────┘         └─────┘
└───────┘

计算过程：
(1×1+2×0+3×1) + (2×0+3×1+4×0) + (3×1+4×0+5×1) = ?
```

### 经典架构

```
输入图像
    ↓
卷积层 1 → 池化层 1  (提取低级特征：边缘、纹理)
    ↓
卷积层 2 → 池化层 2  (提取中级特征：形状、部件)
    ↓
卷积层 3 → 池化层 3  (提取高级特征：物体)
    ↓
全连接层 → 输出     (分类结果)
```

## 实战项目

**图像分类任务：**

```python

class SimpleCNN(nn.Module):
    def __init__(self):
        super().__init__()

        self.pool = nn.MaxPool2d(2, 2)     # 最大池化
```
