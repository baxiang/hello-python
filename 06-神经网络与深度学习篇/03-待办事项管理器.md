# 项目三：待办事项管理器

本项目用纯 Python 标准库（`json`、`csv`、`pathlib`、`datetime`）实现一个命令行待办事项管理器。它覆盖了"增删改查"这四种最基础也最重要的数据操作，是学习面向对象设计和文件持久化的绝佳练习项目。

## 第三部分：核心概念讲解

### 3.1 数据持久化

#### 概念说明

"持久化"（Persistence）是一个专业术语，意思是：**把数据从容易消失的地方（内存）转移到不容易消失的地方（磁盘文件、数据库）**。

类比：你在白板上写的内容（内存），下班后白板会被擦掉。但如果你把内容拍照存到手机相册（文件），明天还能看到。

程序运行时，所有变量都在内存（RAM）里。程序一关闭，内存就被清空了。所以我们需要把数据写入磁盘文件。

**JSON vs 数据库的权衡**：

```
┌─────────────────┬───────────────────┬───────────────────────────┐
│  方案           │  优点             │  缺点                     │
├─────────────────┼───────────────────┼───────────────────────────┤
│  JSON 文件      │  简单，无需安装   │  并发写入有风险           │
│                 │  人类可读         │  数据量大时变慢           │
│                 │  适合小型应用     │                           │
├─────────────────┼───────────────────┼───────────────────────────┤
│  SQLite 数据库  │  支持 SQL 查询    │  需要学习 SQL             │
│                 │  并发安全         │  稍复杂                   │
│                 │  Python 内置      │                           │
├─────────────────┼───────────────────┼───────────────────────────┤
│  PostgreSQL等   │  高性能           │  需要安装服务器           │
│  专业数据库     │  多用户并发       │  运维成本高               │
│                 │  支持复杂查询     │                           │
└─────────────────┴───────────────────┴───────────────────────────┘
```

本项目用 JSON，因为待办事项数量不多，单用户使用，JSON 完全够用。

#### 示例代码

```python
import json
from pathlib import Path

# 演示：把列表保存到 JSON 文件，然后再读回来
def demo_persistence():
    data_file = Path("demo.json")

    # ① 保存到文件（持久化）
    todos = [
        {"id": 1, "title": "买牛奶", "completed": False},
        {"id": 2, "title": "跑步30分钟", "completed": True},
    ]

    with open(data_file, "w", encoding="utf-8") as f:
        # indent=2 让 JSON 格式化缩进，方便人类阅读
        # ensure_ascii=False 让中文正常显示，不转义为 \uXXXX
        json.dump(todos, f, indent=2, ensure_ascii=False)

    print(f"数据已保存到 {data_file}")

    # ② 从文件读取（加载）
    with open(data_file, "r", encoding="utf-8") as f:
        loaded = json.load(f)

    print(f"从文件读取了 {len(loaded)} 条数据")
    for item in loaded:
        status = "✓" if item["completed"] else "○"
        print(f"  [{status}] {item['id']}. {item['title']}")

demo_persistence()
# 输出：
# 数据已保存到 demo.json
# 从文件读取了 2 条数据
#   [○] 1. 买牛奶
#   [✓] 2. 跑步30分钟
```

### 3.2 @classmethod from_dict 模式

#### 概念说明

当我们从 JSON 文件读取数据时，得到的是 Python 字典（`dict`），而不是 `TodoItem` 对象。我们需要一种方法把字典"变回"对象。

`from_dict` 是一种常见的**类方法（classmethod）**模式，也叫**工厂方法**。

类比：想象 `TodoItem` 是一个工厂，`from_dict` 是工厂里的一条特殊流水线——你把一堆原材料（字典）送进去，流水线按照固定配方，组装出一个标准的 `TodoItem` 产品。

为什么用 `@classmethod` 而不是普通函数？因为这个方法逻辑上"属于"这个类——它知道如何构造这个类的实例，放在类里面代码更整洁，也更易找到。

```
普通函数（不推荐）：
  todo_from_dict(data)   ← 游离在外，不知道和哪个类有关

类方法（推荐）：
  TodoItem.from_dict(data)  ← 清晰表明"这是构造 TodoItem 的方法"
```

#### 示例代码

```python
from datetime import datetime

class TodoItem:
    def __init__(self, id: int, title: str, priority: str = "medium"):
        self.id = id
        self.title = title
        self.priority = priority
        self.completed = False
        self.tags = []
        self.created_at = datetime.now().isoformat(timespec="seconds")
        self.completed_at = None

    def to_dict(self) -> dict:
        """把对象转换为字典（用于保存到 JSON）"""
        return {
            "id": self.id,
            "title": self.title,
            "completed": self.completed,
            "priority": self.priority,
            "tags": self.tags,
            "created_at": self.created_at,
            "completed_at": self.completed_at,
        }

    @classmethod
    def from_dict(cls, data: dict) -> "TodoItem":
        """
        从字典构造 TodoItem 对象（用于从 JSON 文件加载）

        cls 是类本身（TodoItem），类似于 self 代表实例
        这样即使子类继承 TodoItem，from_dict 也能正确返回子类实例
        """
        # 先用正常方式创建一个对象
        item = cls(
            id=data["id"],
            title=data["title"],
            priority=data.get("priority", "medium"),  # 兼容旧数据，没有该字段时用默认值
        )
        # 再把字典中的其他字段逐一赋值
        item.completed = data.get("completed", False)
        item.tags = data.get("tags", [])
        item.created_at = data.get("created_at", item.created_at)
        item.completed_at = data.get("completed_at", None)
        return item


# 演示 from_dict 的用法
raw_data = {
    "id": 1,
    "title": "买牛奶",
    "completed": True,
    "priority": "high",
    "tags": ["购物"],
    "created_at": "2024-01-15T09:30:00",
    "completed_at": "2024-01-15T10:00:00",
}

todo = TodoItem.from_dict(raw_data)
print(todo.title)      # 买牛奶
print(todo.completed)  # True
print(todo.priority)   # high

# 再转回字典，应该和原始数据一样
print(todo.to_dict() == raw_data)  # True
```

### 3.3 排序的多键策略

#### 概念说明

我们希望列表的显示顺序是：**未完成的排在前面，已完成的排在后面；同等完成状态下，优先级高的排在前面**。

这叫**多键排序**——按照多个条件依次排序，类似 Excel 里的"先按A列排，A列相同再按B列排"。

Python 的 `sorted()` 函数接受一个 `key` 参数，当 key 函数返回一个**元组**时，Python 会按元组的元素**依次比较**：先比较第一个元素，相同时再比较第二个元素，以此类推。

```
元组比较规则示例：
  (False, 0)  vs  (False, 1)
   ↑ 相同        ↑ 不同，所以比较第二个元素
   → 0 < 1，所以 (False, 0) 排在前面

  (False, 0)  vs  (True, 0)
   ↑ 不同
   → False < True，所以 (False, 0) 排在前面
   （这正好让"未完成"排在"已完成"前面！）
```

#### 示例代码

```python
# 演示多键排序策略
todos_data = [
    {"title": "写报告",   "completed": False, "priority": "low"},
    {"title": "开会准备", "completed": False, "priority": "high"},
    {"title": "买牛奶",   "completed": True,  "priority": "medium"},
    {"title": "读文档",   "completed": False, "priority": "medium"},
    {"title": "回邮件",   "completed": True,  "priority": "high"},
]

# 优先级映射：数字越小排越前
priority_order = {"high": 0, "medium": 1, "low": 2}

# 排序 key 函数：返回一个元组
# 第一个元素：completed（False=0 排前面，True=1 排后面）
# 第二个元素：优先级对应的数字（0 最高优先级，排最前面）
sorted_todos = sorted(
    todos_data,
    key=lambda t: (t["completed"], priority_order[t["priority"]])
)

print("排序结果：")
for t in sorted_todos:
    status = "已完成" if t["completed"] else "未完成"
    print(f"  [{status}] [{t['priority']:6}] {t['title']}")

# 输出：
# 排序结果：
#   [未完成] [high  ] 开会准备
#   [未完成] [medium] 读文档
#   [未完成] [low   ] 写报告
#   [已完成] [high  ] 回邮件
#   [已完成] [medium] 买牛奶
```

### 3.4 命令行解析

#### 概念说明

用户在命令行输入的是一整行字符串，例如 `"add 买牛奶 --priority high"`。我们需要把它拆分成：
- 命令：`add`
- 参数：`买牛奶 --priority high`

`str.split(maxsplit=1)` 是一个非常实用的技巧：`maxsplit=1` 表示**最多切割一刀**，只把第一个空格之前的部分分离出来，后面的内容保持原样，不管里面有多少空格。

```
"add 买牛奶 --priority high".split(maxsplit=1)
   ↓ 切一刀
["add", "买牛奶 --priority high"]
 ↑命令   ↑剩余参数（原样保留）
```

这比 `.split()` 切成所有片段更安全——待办标题可能包含空格，全部切开就乱了。

#### 示例代码

```python
def parse_command(user_input: str):
    """
    解析用户输入的命令行
    返回 (命令, 参数字符串) 的元组
    """
    user_input = user_input.strip()  # 去掉首尾空格

    if not user_input:
        return "", ""

    # maxsplit=1：最多切一刀，命令和参数分开
    parts = user_input.split(maxsplit=1)
    command = parts[0].lower()         # 命令转小写，不区分大小写
    args_str = parts[1] if len(parts) > 1 else ""  # 没有参数时为空字符串

    return command, args_str


def parse_add_args(args_str: str):
    """
    解析 add 命令的参数
    例如："买牛奶 --priority high --tags 购物,生活"
    """
    # 提取 --priority 参数
    priority = "medium"  # 默认优先级
    tags = []

    # 简单的手动解析（真实项目可用 argparse 或 shlex）
    parts = args_str.split("--")
    title = parts[0].strip()  # "--" 之前的部分就是标题

    for part in parts[1:]:
        part = part.strip()
        if part.startswith("priority "):
            priority = part[len("priority "):].strip()
        elif part.startswith("tags "):
            tags_str = part[len("tags "):].strip()
            tags = [t.strip() for t in tags_str.split(",") if t.strip()]

    return title, priority, tags


# 演示
test_inputs = [
    "add 买牛奶 --priority high --tags 购物,生活",
    "done 3",
    "list --all",
    "search 学习",
    "QUIT",   # 大写也能识别
]

for inp in test_inputs:
    cmd, args = parse_command(inp)
    print(f"命令: {cmd!r:10}  参数: {args!r}")

# 输出：
# 命令: 'add'        参数: '买牛奶 --priority high --tags 购物,生活'
# 命令: 'done'       参数: '3'
# 命令: 'list'       参数: '--all'
# 命令: 'search'     参数: '学习'
# 命令: 'quit'       参数: ''
```

## 第四部分：完整代码实现

### 4.1 完整源代码

将以下代码保存为 `todo_manager.py`，直接运行即可使用。

```python
"""
待办事项管理器 - todo_manager.py
功能：增删改查 + 优先级 + 标签 + 统计 + CSV 导出
依赖：只用 Python 标准库，无需安装第三方库
用法：python todo_manager.py
"""

import json
import csv
from pathlib import Path
from datetime import datetime


# ─────────────────────────────────────────────
# 数据模型：TodoItem 类
# 负责：单条待办数据的存储和转换
# ─────────────────────────────────────────────

class TodoItem:
    """代表一条待办事项"""

    # 合法的优先级值（用类变量定义，方便统一校验）
    VALID_PRIORITIES = ("high", "medium", "low")

    def __init__(
        self,
        id: int,
        title: str,
        priority: str = "medium",
        tags: list[str] | None = None,
    ):
        self.id = id
        self.title = title
        self.completed = False
        self.priority = priority if priority in self.VALID_PRIORITIES else "medium"
        self.tags: list[str] = tags if tags is not None else []
        # 创建时间，精确到秒
        self.created_at: str = datetime.now().isoformat(timespec="seconds")
        self.completed_at: str | None = None  # 完成时才赋值

    # ── 序列化：对象 → 字典（用于写入 JSON 文件） ──

    def to_dict(self) -> dict:
        """把对象转换为可 JSON 序列化的字典"""
        return {
            "id": self.id,
            "title": self.title,
            "completed": self.completed,
            "priority": self.priority,
            "tags": self.tags,
            "created_at": self.created_at,
            "completed_at": self.completed_at,
        }

    # ── 反序列化：字典 → 对象（用于从 JSON 文件读取） ──

    @classmethod
    def from_dict(cls, data: dict) -> "TodoItem":
        """
        工厂方法：从字典构造 TodoItem 对象
        data.get(key, default) 处理旧数据中缺少某些字段的情况
        """
        item = cls(
            id=data["id"],
            title=data["title"],
            priority=data.get("priority", "medium"),
            tags=data.get("tags", []),
        )
        item.completed = data.get("completed", False)
        item.created_at = data.get("created_at", item.created_at)
        item.completed_at = data.get("completed_at", None)
        return item

    # ── 字符串表示 ──

    def __str__(self) -> str:
        """用于 print(todo) 时的友好显示"""
        # 状态图标
        status_icon = "✓" if self.completed else "○"

        # 优先级图标
        priority_icons = {"high": "🔴", "medium": "🟡", "low": "🟢"}
        priority_icon = priority_icons.get(self.priority, "  ")

        # 标签
        tags_str = f"  [{', '.join(self.tags)}]" if self.tags else ""

        # 优先级文字（对齐用）
        priority_label = {"high": "高", "medium": "中", "low": "低"}.get(
            self.priority, self.priority
        )

        return (
            f"[{status_icon}] #{self.id:3d}  "
            f"[{priority_label}] {self.title}{tags_str}"
        )

    def __repr__(self) -> str:
        """用于调试时的表示，显示更多细节"""
        return (
            f"TodoItem(id={self.id}, title={self.title!r}, "
            f"completed={self.completed}, priority={self.priority!r})"
        )


# ─────────────────────────────────────────────
# 业务逻辑：TodoManager 类
# 负责：管理全部待办，处理增删改查、持久化
# ─────────────────────────────────────────────

class TodoManager:
    """待办事项管理器，负责全部业务逻辑和数据持久化"""

    def __init__(self, data_file: str = "todos.json"):
        # 使用 pathlib.Path 管理路径，比字符串更安全
        self.data_file = Path(data_file)
        self.todos: list[TodoItem] = []
        self._load_todos()

    # ── 私有方法：持久化（以 _ 开头表示内部使用） ──

    def _load_todos(self) -> None:
        """从 JSON 文件读取待办列表到内存"""
        if not self.data_file.exists():
            # 文件不存在是正常情况（第一次运行），直接用空列表
            self.todos = []
            return

        try:
            with open(self.data_file, "r", encoding="utf-8") as f:
                raw_list = json.load(f)
            # 把每个字典转换为 TodoItem 对象
            self.todos = [TodoItem.from_dict(item) for item in raw_list]
        except (json.JSONDecodeError, KeyError) as e:
            # 文件损坏时，告知用户并使用空列表，不直接崩溃
            print(f"警告：数据文件损坏，已重置为空列表。原因：{e}")
            self.todos = []

    def _save_todos(self) -> None:
        """把内存中的待办列表写入 JSON 文件"""
        with open(self.data_file, "w", encoding="utf-8") as f:
            json.dump(
                [item.to_dict() for item in self.todos],
                f,
                indent=2,
                ensure_ascii=False,  # 中文不转义
            )

    def _next_id(self) -> int:
        """生成下一个唯一 ID（取当前最大 ID + 1）"""
        if not self.todos:
            return 1
        return max(item.id for item in self.todos) + 1

    def _find_by_id(self, todo_id: int) -> TodoItem | None:
        """按 ID 查找待办，找不到返回 None"""
        for item in self.todos:
            if item.id == todo_id:
                return item
        return None

    # ── 公开方法：增 ──

    def add(
        self,
        title: str,
        priority: str = "medium",
        tags: list[str] | None = None,
    ) -> TodoItem:
        """
        新增一条待办事项
        返回新建的 TodoItem 对象（方便调用方获取 ID）
        """
        title = title.strip()
        if not title:
            raise ValueError("待办标题不能为空")

        new_item = TodoItem(
            id=self._next_id(),
            title=title,
            priority=priority,
            tags=tags or [],
        )
        self.todos.append(new_item)
        self._save_todos()  # 立即持久化
        return new_item

    # ── 公开方法：删 ──

    def delete(self, todo_id: int) -> TodoItem:
        """
        删除指定 ID 的待办
        返回被删除的对象（方便显示"已删除：xxx"）
        """
        item = self._find_by_id(todo_id)
        if item is None:
            raise ValueError(f"找不到 ID 为 {todo_id} 的待办")
        self.todos.remove(item)
        self._save_todos()
        return item

    def clear_completed(self) -> int:
        """删除所有已完成的待办，返回删除的数量"""
        before = len(self.todos)
        self.todos = [item for item in self.todos if not item.completed]
        deleted = before - len(self.todos)
        if deleted > 0:
            self._save_todos()
        return deleted

    # ── 公开方法：改 ──

    def complete(self, todo_id: int) -> TodoItem:
        """标记指定待办为已完成"""
        item = self._find_by_id(todo_id)
        if item is None:
            raise ValueError(f"找不到 ID 为 {todo_id} 的待办")
        if item.completed:
            raise ValueError(f"#{todo_id} 已经是完成状态")
        item.completed = True
        item.completed_at = datetime.now().isoformat(timespec="seconds")
        self._save_todos()
        return item

    def uncomplete(self, todo_id: int) -> TodoItem:
        """取消指定待办的完成状态"""
        item = self._find_by_id(todo_id)
        if item is None:
            raise ValueError(f"找不到 ID 为 {todo_id} 的待办")
        if not item.completed:
            raise ValueError(f"#{todo_id} 尚未完成，无法取消")
        item.completed = False
        item.completed_at = None
        self._save_todos()
        return item

    def edit(
        self,
        todo_id: int,
        new_title: str | None = None,
        new_priority: str | None = None,
        new_tags: list[str] | None = None,
    ) -> TodoItem:
        """
        修改待办的标题、优先级或标签
        只传入需要修改的字段，不传的字段保持不变
        这种"只改传入的字段"的设计叫做 partial update（局部更新）
        """
        item = self._find_by_id(todo_id)
        if item is None:
            raise ValueError(f"找不到 ID 为 {todo_id} 的待办")

        if new_title is not None:
            new_title = new_title.strip()
            if not new_title:
                raise ValueError("标题不能为空")
            item.title = new_title

        if new_priority is not None:
            if new_priority not in TodoItem.VALID_PRIORITIES:
                raise ValueError(
                    f"无效优先级 {new_priority!r}，"
                    f"可选值：{', '.join(TodoItem.VALID_PRIORITIES)}"
                )
            item.priority = new_priority

        if new_tags is not None:
            item.tags = new_tags

        self._save_todos()
        return item

    # ── 公开方法：查 ──

    def list_all(self, include_completed: bool = True) -> list[TodoItem]:
        """
        获取排序后的待办列表
        排序规则：未完成排前面，同等状态下高优先级排前面
        """
        priority_order = {"high": 0, "medium": 1, "low": 2}

        items = self.todos if include_completed else [
            item for item in self.todos if not item.completed
        ]

        return sorted(
            items,
            key=lambda t: (t.completed, priority_order.get(t.priority, 99))
        )

    def search(self, keyword: str) -> list[TodoItem]:
        """
        搜索标题中包含关键词的待办（不区分大小写）
        也会搜索标签
        """
        keyword_lower = keyword.lower()
        results = []
        for item in self.todos:
            # 搜索标题
            if keyword_lower in item.title.lower():
                results.append(item)
                continue
            # 搜索标签
            if any(keyword_lower in tag.lower() for tag in item.tags):
                results.append(item)
        return results

    def get_statistics(self) -> dict:
        """计算并返回统计数据"""
        total = len(self.todos)
        completed = sum(1 for item in self.todos if item.completed)
        pending = total - completed

        # 按优先级统计（只统计未完成的）
        priority_count = {"high": 0, "medium": 0, "low": 0}
        for item in self.todos:
            if not item.completed:
                priority_count[item.priority] = priority_count.get(item.priority, 0) + 1

        # 完成率
        completion_rate = (completed / total * 100) if total > 0 else 0.0

        return {
            "total": total,
            "completed": completed,
            "pending": pending,
            "completion_rate": completion_rate,
            "priority_count": priority_count,
        }

    # ── 显示方法 ──

    def display(self, items: list[TodoItem] | None = None) -> None:
        """
        在终端打印待办列表
        如果不传 items，默认显示全部（已排序）
        """
        if items is None:
            items = self.list_all()

        if not items:
            print("  （暂无待办事项）")
            return

        print()
        print("─" * 55)
        print(f"  待办事项列表（共 {len(items)} 条）")
        print("─" * 55)
        for item in items:
            print(f"  {item}")
        print("─" * 55)
        print()

    def show_statistics(self) -> None:
        """在终端打印统计信息"""
        stats = self.get_statistics()
        bar_length = 20  # 进度条长度（字符数）
        filled = int(stats["completion_rate"] / 100 * bar_length)
        bar = "█" * filled + "░" * (bar_length - filled)

        print()
        print("─" * 45)
        print("  待办事项统计")
        print("─" * 45)
        print(f"  总计：{stats['total']} 条")
        print(f"  已完成：{stats['completed']} 条")
        print(f"  待完成：{stats['pending']} 条")
        print(f"  完成率：[{bar}] {stats['completion_rate']:.1f}%")
        print()
        print("  未完成优先级分布：")
        pc = stats["priority_count"]
        print(f"    高优先级：{pc.get('high', 0)} 条")
        print(f"    中优先级：{pc.get('medium', 0)} 条")
        print(f"    低优先级：{pc.get('low', 0)} 条")
        print("─" * 45)
        print()

    # ── 导出方法 ──

    def export_to_csv(self, filename: str) -> Path:
        """
        把全部待办导出为 CSV 文件
        CSV 是通用格式，可以用 Excel、Numbers 等打开

        参数：
            filename: 导出的文件名，例如 "todos_backup.csv"
        返回：
            导出文件的 Path 对象
        """
        output_path = Path(filename)

        # CSV 的列名（表头）
        fieldnames = [
            "id", "title", "completed", "priority",
            "tags", "created_at", "completed_at"
        ]

        with open(output_path, "w", newline="", encoding="utf-8-sig") as f:
            # utf-8-sig 带 BOM，让 Excel 正确显示中文
            writer = csv.DictWriter(f, fieldnames=fieldnames)
            writer.writeheader()  # 写入表头行

            for item in self.todos:
                row = item.to_dict()
                # CSV 里标签存为逗号分隔的字符串
                row["tags"] = ",".join(item.tags)
                # bool 转为中文，更易读
                row["completed"] = "是" if item.completed else "否"
                writer.writerow(row)

        return output_path

HELP_TEXT = """

def parse_add_args(args_str: str) -> tuple[str, str, list[str]]:
    """
    解析 add 命令的参数字符串
    例如："买牛奶 --priority high --tags 购物,生活"
    返回：(标题, 优先级, 标签列表)
    """
    priority = "medium"
    tags: list[str] = []

    # 以 "--" 为分隔符切分，第一段是标题，后面是各个选项
    parts = args_str.split("--")
    title = parts[0].strip()

    for part in parts[1:]:
        part = part.strip()
        if part.lower().startswith("priority "):
            priority = part[len("priority "):].strip().lower()
        elif part.lower().startswith("tags "):
            tags_str = part[len("tags "):].strip()
            tags = [t.strip() for t in tags_str.split(",") if t.strip()]

    return title, priority, tags


def main() -> None:
    """程序主入口：读取用户输入，分发到对应的处理逻辑"""
    manager = TodoManager("todos.json")

    print("╔══════════════════════════════════╗")
    print("║    待办事项管理器  v1.0           ║")
    print("║    输入 help 查看命令列表         ║")
    print("╚══════════════════════════════════╝")
    print()

    # 启动时显示未完成的待办
    manager.display(manager.list_all(include_completed=False))

    while True:
        try:
            # input() 等待用户输入，prompt 提示符
            raw = input("todo> ").strip()
        except (EOFError, KeyboardInterrupt):
            # Ctrl+D 或 Ctrl+C 优雅退出
            print("\n再见！")
            break

        if not raw:
            continue  # 用户只按了 Enter，跳过空输入

        # 最多切一刀：命令 和 剩余参数
        parts = raw.split(maxsplit=1)
        command = parts[0].lower()
        args = parts[1] if len(parts) > 1 else ""

        # ── 分发到对应处理逻辑 ──
        try:
            if command in ("quit", "exit", "q"):
                print("再见！")
                break

            elif command == "help":
                print(HELP_TEXT)

            elif command == "add":
                if not args:
                    print("用法：add <待办内容> [--priority high/medium/low] [--tags 标签1,标签2]")
                    continue
                title, priority, tags = parse_add_args(args)
                if not title:
                    print("错误：标题不能为空")
                    continue
                item = manager.add(title, priority=priority, tags=tags)
                tag_info = f"  标签：{', '.join(item.tags)}" if item.tags else ""
                print(f"已添加：#{item.id} {item.title}  [{item.priority}]{tag_info}")

            elif command == "list":
                include_all = args.strip() == "--all"
                items = manager.list_all(include_completed=include_all)
                manager.display(items)

            elif command == "search":
                if not args:
                    print("用法：search <关键词>")
                    continue
                results = manager.search(args)
                if results:
                    print(f"搜索 {args!r}，找到 {len(results)} 条结果：")
                    manager.display(results)
                else:
                    print(f"未找到包含 {args!r} 的待办")

            elif command == "done":
                if not args:
                    print("用法：done <ID>")
                    continue
                todo_id = int(args.strip())
                item = manager.complete(todo_id)
                print(f"已完成：#{item.id} {item.title}")

            elif command == "undone":
                if not args:
                    print("用法：undone <ID>")
                    continue
                todo_id = int(args.strip())
                item = manager.uncomplete(todo_id)
                print(f"已取消完成：#{item.id} {item.title}")

            elif command == "edit":
                # 用法：edit <ID> <新标题> 或 edit <ID> --priority high
                if not args:
                    print("用法：edit <ID> <新标题>")
                    continue
                edit_parts = args.split(maxsplit=1)
                todo_id = int(edit_parts[0])
                rest = edit_parts[1] if len(edit_parts) > 1 else ""

                # 判断是修改标题还是优先级
                if rest.startswith("--priority "):
                    new_priority = rest[len("--priority "):].strip()
                    item = manager.edit(todo_id, new_priority=new_priority)
                    print(f"已更新：#{item.id} 优先级改为 {item.priority}")
                elif rest.startswith("--tags "):
                    tags_str = rest[len("--tags "):].strip()
                    new_tags = [t.strip() for t in tags_str.split(",") if t.strip()]
                    item = manager.edit(todo_id, new_tags=new_tags)
                    print(f"已更新：#{item.id} 标签改为 {item.tags}")
                else:
                    item = manager.edit(todo_id, new_title=rest)
                    print(f"已更新：#{item.id} 标题改为 {item.title!r}")

            elif command == "delete":
                if not args:
                    print("用法：delete <ID>")
                    continue
                todo_id = int(args.strip())
                item = manager.delete(todo_id)
                print(f"已删除：#{item.id} {item.title}")

            elif command == "clear":
                count = manager.clear_completed()
                if count > 0:
                    print(f"已清空 {count} 条已完成的待办")
                else:
                    print("没有已完成的待办可以清空")

            elif command == "stats":
                manager.show_statistics()

            elif command == "export":
                filename = args.strip() if args else "todos_export.csv"
                if not filename.endswith(".csv"):
                    filename += ".csv"
                path = manager.export_to_csv(filename)
                print(f"已导出 {len(manager.todos)} 条记录到：{path.resolve()}")

            else:
                print(f"未知命令：{command!r}，输入 help 查看可用命令")

        except ValueError as e:
            # ValueError 通常是业务逻辑错误（ID不存在、格式错误等）
            print(f"错误：{e}")
        except Exception as e:
            # 其他意外错误，显示但不崩溃
            print(f"发生错误：{e}")


if __name__ == "__main__":
    main()
```

### 6.1 截止日期（due_date 字段 + 过期提醒）

**目标**：给每条待办添加可选的截止日期，列表显示时若已过期则高亮提示。

**提示**：
- 在 `TodoItem.__init__` 增加 `due_date: str | None = None` 字段
- 在 `__str__` 里判断 `datetime.fromisoformat(self.due_date) < datetime.now()` 来显示过期提示
- `add` 命令支持 `--due 2024-02-01` 参数

```python
# 扩展示意（不完整，需要自己补全）
def __str__(self) -> str:
    overdue_str = ""
    if self.due_date and not self.completed:
        due = datetime.fromisoformat(self.due_date)
        if due < datetime.now():
            days = (datetime.now() - due).days
            overdue_str = f"  ⚠ 已超期 {days} 天"
    return f"[{'✓' if self.completed else '○'}] #{self.id:3d}  {self.title}{overdue_str}"
```

### 6.2 重复事项（每天/每周自动创建）

**目标**：标记某条待办为"每天重复"，程序启动时自动为今天创建一条。

**提示**：

- `_check_recurrence()` 遍历所有有 `recurrence` 的待办，检查今天是否已经存在，若不存在则自动 `add`

### 6.3 子任务（在待办下挂子项）

**目标**：支持为一条待办添加子任务，只有所有子任务完成才能完成父任务。

**提示**：
- 在 `TodoItem` 增加 `subtasks: list[dict]` 字段
- 子任务结构：`{"text": "买全脂牛奶", "done": False}`
- `complete()` 方法里检查 `all(s["done"] for s in item.subtasks)`

```python
# 扩展示意
def add_subtask(self, todo_id: int, subtask_text: str) -> TodoItem:
    item = self._find_by_id(todo_id)
    if item is None:
        raise ValueError(f"找不到 ID {todo_id}")
    item.subtasks.append({"text": subtask_text, "done": False})
```

### 6.5 Web 界面（用 Flask 包一层 HTTP API）

**目标**：把命令行版本改造为 Web 应用，用浏览器访问。

**提示**：
- 安装 Flask：`uv add flask`
- `TodoManager` 类无需修改，直接复用
- 用 Flask 路由把 HTTP 请求映射到 `manager` 的方法

```python
# web_todo.py 示意（需要安装 flask）
from flask import Flask, jsonify, request
from todo_manager import TodoManager  # 复用原来的类

app = Flask(__name__)
manager = TodoManager("todos.json")

@app.get("/todos")
def get_todos():
    """GET /todos → 返回所有待办的 JSON"""
    items = manager.list_all()
    return jsonify([item.to_dict() for item in items])

@app.post("/todos")
def create_todo():
    """POST /todos {"title": "买牛奶"} → 新增待办"""
    data = request.json
    item = manager.add(data["title"], priority=data.get("priority", "medium"))
    return jsonify(item.to_dict()), 201

@app.patch("/todos/<int:todo_id>/complete")
def complete_todo(todo_id: int):
    """PATCH /todos/1/complete → 标记完成"""
    item = manager.complete(todo_id)
    return jsonify(item.to_dict())

if __name__ == "__main__":
    app.run(debug=True)  # 访问 http://localhost:5000/todos
```
