## 前置知识要求

开始本章前，你需要掌握：
- Python 编程（函数、类、面向对象）
- NumPy 基础（数组操作、矩阵运算）
- 线性代数基础（向量、矩阵、点积）
- 微积分基础（导数、梯度概念）
- 机器学习概念（监督学习、损失函数、优化）

## 学习路径

```
第1步：理解基础概念
    ↓  什么是神经网络？为什么需要它？
第2步：学习 PyTorch 工具
    ↓  如何用代码实现神经网络？
第3步：掌握经典模型
    ↓  CNN、RNN 等经典架构
第4步：完成实战项目
    ↓  真实场景应用
```
