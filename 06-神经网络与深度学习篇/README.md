# 神经网络与深度学习篇

从零开始学习深度学习，通过实战项目掌握神经网络核心技术。

> **重要：** 深度学习是高级主题，需要先掌握前面章节的基础知识。

---

## 📚 学习路径

```
第1步：理解基础概念
    ↓  什么是神经网络？为什么需要它？
第2步：学习 PyTorch 工具
    ↓  如何用代码实现神经网络？
第3步：掌握经典模型
    ↓  CNN、RNN 等经典架构
第4步：完成实战项目
    ↓  真实场景应用
```

## 💡 学习建议

### 初学者建议

1. **不要跳过基础**
   - 先理解机器学习概念（第5篇）
   - 补充必要的数学知识
   - 不要直接看代码，先理解原理

2. **动手实践**
   - 每学一个概念，就写代码验证
   - 从简单的例子开始
   - 不要急于做复杂项目

3. **循序渐进**
   ```
   简单模型 → 复杂模型 → 实战项目
   
   线性回归 → 多层感知机 → CNN/RNN → 实战
   ```

4. **遇到困难怎么办？**
   - 先检查前置知识是否掌握
   - 看不懂原理就先看应用
   - 多看几个不同的解释
   - 亲手画图、写公式

### 学习顺序

**推荐路径：**
```
01-神经网络基础 (理解原理)
    ↓
02-PyTorch 入门 (掌握工具)
    ↓
03-卷积神经网络 CNN (图像处理)
    ↓
04-循环神经网络 RNN (文本处理)
    ↓
05-深度学习实战 (综合应用)
```

**时间安排：**
- 每章 1-2 周
- 每天学习 2-3 小时
- 总计 6-8 周完成

## ❓ 常见问题

### Q: 需要很强的数学基础吗？
**A:** 需要基础数学（线性代数、微积分），但不需太高深。能理解向量和导数就够用。

### Q: 没有 GPU 能学吗？
**A:** 可以！用 Google Colab 免费版，或用 CPU 跑小模型（虽然慢）。

### Q: 为什么推荐 PyTorch 而不是 TensorFlow？
**A:** PyTorch 更符合 Python 编程习惯，代码更易读，学习曲线更平缓。

### Q: 学完能找工作吗？
**A:** 学完后还需要大量实战项目经验。建议完成 3-5 个完整项目。

### Q: 卡在某个概念怎么办？
**A:** 
1. 检查前置知识是否掌握
2. 看不同的教程解释
3. 找可视化演示（如 3Blue1Brown）
4. 在社区提问
