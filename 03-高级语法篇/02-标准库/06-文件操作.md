## 第一部分：文件读写基础

### 12.1 open() 函数

#### 概念说明

`open()` 函数用于打开文件，返回文件对象。

**基本语法：**

```python
file = open(filename, mode, encoding)
```

**打开模式：**

| 模式 | 说明 |
|------|------|
| `r` | 读取（默认），文件必须存在 |
| `w` | 写入，文件不存在则创建，存在则覆盖 |
| `a` | 追加，文件不存在则创建 |
| `x` | 创建写入，文件已存在则报错 |
| `b` | 二进制模式 |
| `t` | 文本模式（默认） |
| `+` | 读写模式 |

**组合模式：**

```python
# 读写模式
f = open('file.txt', 'r+')   # 读写，文件必须存在

# 写读模式
f = open('file.txt', 'w+')   # 写读，覆盖原文件

# 二进制读取
f = open('image.png', 'rb')  # 二进制读

# 二进制写入
f = open('data.bin', 'wb')   # 二进制写
```

### 12.2 with 语句

#### 概念说明

使用 `with` 语句可以自动管理文件资源，确保文件正确关闭。

**示例代码：**

```python
# 推荐：使用 with 语句
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()
# 文件自动关闭

# 不推荐：手动关闭
f = open('file.txt', 'r', encoding='utf-8')
content = f.read()
f.close()  # 忘记关闭可能导致资源泄漏
```

**with 语句的优势：**

```
┌─────────────────────────────────────────────────────────────┐
│              with 语句的优势                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   1. 自动关闭                                               │
│      代码块结束后自动调用 f.close()                         │
│                                                             │
│   2. 异常安全                                               │
│      即使发生异常，文件也会正确关闭                         │
│                                                             │
│   3. 代码简洁                                               │
│      不需要手动管理资源                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 12.3 读取文件

#### 概念说明

文件对象提供多种读取方法。

**读取方法：**

```python
# read() - 读取全部内容
with open('file.txt', 'r') as f:
    content = f.read()  # 返回字符串

# read(size) - 读取指定字节数
with open('file.txt', 'r') as f:
    chunk = f.read(1024)  # 读取 1024 字节

# readline() - 读取一行
with open('file.txt', 'r') as f:
    line = f.readline()  # 返回一行字符串

# readlines() - 读取所有行
with open('file.txt', 'r') as f:
    lines = f.readlines()  # 返回列表，每行一个元素

# 迭代读取（推荐处理大文件）
with open('file.txt', 'r') as f:
    for line in f:  # 每次读取一行，内存友好
        print(line.strip())
```

**读取示例：**

```python
# 读取配置文件
def read_config(filename):
    config = {}
    with open(filename, 'r', encoding='utf-8') as f:
        for line in f:
            line = line.strip()
            if line and not line.startswith('#'):
                key, value = line.split('=')
                config[key.strip()] = value.strip()
    return config

# config.txt
# host=localhost
# port=8080

config = read_config('config.txt')
print(config)  # {'host': 'localhost', 'port': '8080'}
```

### 12.4 写入文件

#### 概念说明

使用 `write()` 方法写入内容。

**写入方法：**

```python
# write() - 写入字符串
with open('file.txt', 'w', encoding='utf-8') as f:
    f.write('Hello, World!\n')
    f.write('Python 文件操作')

# writelines() - 写入多行
lines = ['第一行\n', '第二行\n', '第三行\n']
with open('file.txt', 'w', encoding='utf-8') as f:
    f.writelines(lines)

# 追加写入
with open('file.txt', 'a', encoding='utf-8') as f:
    f.write('\n追加的内容')
```

**写入示例：**

```python
# 写入日志
import datetime

def write_log(filename, message):
    timestamp = datetime.datetime.now().strftime('%Y-%m-%d %H:%M:%S')
    with open(filename, 'a', encoding='utf-8') as f:
        f.write(f'[{timestamp}] {message}\n')

write_log('app.log', '用户登录成功')
write_log('app.log', '订单创建完成')
```

### 12.5 文件指针

#### 概念说明

文件指针指示当前读写位置，可以使用 `seek()` 移动。

**指针操作：**

```python
with open('file.txt', 'r') as f:
    # 获取当前位置
    pos = f.tell()
    print(f"当前位置: {pos}")

    # 读取 10 字节
    data = f.read(10)
    print(f"读取后位置: {f.tell()}")

    # 移动到文件开头
    f.seek(0)

    # 移动到指定位置
    f.seek(20)  # 从开头移动 20 字节

    # 相对移动
    f.seek(10, 1)  # 从当前位置移动 10 字节

    # 移动到文件末尾
    f.seek(0, 2)  # 从末尾移动 0 字节（定位到末尾）
```

**seek() 参数：**

| 参数值 | 含义 |
|--------|------|
| `0` | 从文件开头计算 |
| `1` | 从当前位置计算 |
| `2` | 从文件末尾计算 |

## 第二部分：pathlib 路径处理

### 12.6 Path 对象

#### 概念说明

`pathlib.Path` 提供面向对象的路径操作方式。

**基本用法：**

```python
from pathlib import Path

# 创建 Path 对象
p = Path('data/file.txt')
p = Path('/home/user/documents')

# 当前目录
p = Path.cwd()

# 用户主目录
p = Path.home()

# 路径拼接（使用 / 操作符）
p = Path('data') / 'subdir' / 'file.txt'
```

**路径属性：**

```python
p = Path('/home/user/documents/report.pdf')

# 路径各部分
print(p.name)      # report.pdf（文件名）
print(p.stem)      # report（不含扩展名）
print(p.suffix)    # .pdf（扩展名）
print(p.parent)    # /home/user/documents（父目录）
print(p.anchor)    # /（根路径）

# 路径信息
print(p.exists())  # 是否存在
print(p.is_file()) # 是否是文件
print(p.is_dir())  # 是否是目录
print(p.is_absolute())  # 是否是绝对路径
```

### 12.7 目录操作

#### 概念说明

Path 对象提供目录创建、遍历等方法。

**创建目录：**

```python
from pathlib import Path

# 创建目录
p = Path('data/output')
p.mkdir()  # 创建目录

# 递归创建目录
p = Path('data/subdir/output')
p.mkdir(parents=True, exist_ok=True)

# exist_ok=True：目录已存在时不报错
```

**遍历目录：**

```python
# iterdir() - 遍历目录内容
p = Path('data')
for item in p.iterdir():
    print(item.name, item.is_file(), item.is_dir())

# glob() - 模式匹配
p = Path('data')
for txt_file in p.glob('*.txt'):
    print(txt_file)

# rglob() - 递归匹配
p = Path('data')
for py_file in p.rglob('*.py'):  # 递归查找所有 .py 文件
    print(py_file)
```

**删除目录：**

```python
# 删除空目录
p = Path('empty_dir')
p.rmdir()

# 删除目录及其内容（需要 shutil）
import shutil
shutil.rmtree('directory')
```

### 12.8 文件操作

#### 概念说明

Path 对象提供文件读写、复制、删除等方法。

**文件读写：**

```python
from pathlib import Path

p = Path('data/file.txt')

# 读取文本
content = p.read_text(encoding='utf-8')

# 写入文本
p.write_text('Hello, World!', encoding='utf-8')

# 读取二进制
data = p.read_bytes()

# 写入二进制
p.write_bytes(b'\x00\x01\x02')
```

**文件操作：**

```python
from pathlib import Path
import shutil

# 重命名
p = Path('old_name.txt')
p.rename('new_name.txt')

# 移动
p = Path('file.txt')
p.rename(Path('new_dir/file.txt'))

# 复制（需要 shutil）
shutil.copy('source.txt', 'dest.txt')
shutil.copy2('source.txt', 'dest.txt')  # 保留元数据

# 删除文件
p = Path('file.txt')
p.unlink()  # 删除文件
```

## 第三部分：异常处理

### 12.9 文件异常

#### 概念说明

文件操作可能抛出多种异常，需要正确处理。

**常见异常：**

```python
from pathlib import Path

def safe_read(filename):
    """安全读取文件"""
    try:
        p = Path(filename)
        return p.read_text(encoding='utf-8')
    
    except FileNotFoundError:
        print(f"文件不存在: {filename}")
        return None
    
    except PermissionError:
        print(f"无权限访问: {filename}")
        return None
    
    except UnicodeDecodeError:
        print(f"编码错误: {filename}")
        # 尝试其他编码
        try:
            return p.read_text(encoding='gbk')
        except:
            return None
    
    except IOError as e:
        print(f"IO 错误: {e}")
        return None
```

**异常类型：**

| 异常 | 说明 |
|------|------|
| `FileNotFoundError` | 文件不存在 |
| `PermissionError` | 权限不足 |
| `IsADirectoryError` | 期望文件但路径是目录 |
| `NotADirectoryError` | 期望目录但路径是文件 |
| `UnicodeDecodeError` | 编码错误 |
| `FileExistsError` | 文件已存在（x 模式） |
