## 核心知识点

### 三种并发模型

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| 多线程 | 受 GIL 限制，共享内存 | I/O 密集型 |
| 多进程 | 不受 GIL 限制，独立内存 | CPU 密集型 |
| asyncio | 单线程，高并发 | I/O 密集型 + 高并发 |

### 选择建议

```
CPU 密集型 → 多进程
I/O 密集型 + 高并发 → asyncio
I/O 密集型 + 同步库 → 多线程
```
