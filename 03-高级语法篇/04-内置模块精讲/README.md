## 学习重点

### 1. 为什么学这些？
原生 Python 数据结构（list, dict）虽然强大，但在特定场景下（如频繁查找、复杂计算）效率较低或代码冗长。使用标准库往往能实现 **O(1) 复杂度** 或 **C 语言级** 的性能优化。

### 2. 核心原则
- **不要重复造轮子**：能用内置模块解决的，优先使用内置模块。
- **类型安全**：配合 `typing` 模块，标准库能提供很好的类型提示支持。
- **性能优先**：`itertools` 和 `collections` 底层由 C 实现，效率极高。
