## 核心知识点

### 模块

- **定义**：包含 Python 代码的 `.py` 文件
- **作用**：代码组织、复用、命名空间隔离
- **导入**：`import`、`from ... import`、`import ... as`

### 包

- **定义**：包含模块的目录
- **标识**：`__init__.py` 文件
- **结构**：可以有子包，形成层级结构

### 导入机制

- **绝对导入**：使用完整路径
- **相对导入**：使用 `.` 和 `..`
- **搜索顺序**：`sys.path` 列表

### 包管理

- **pip**：Python 包管理工具
- **虚拟环境**：隔离项目依赖
- **requirements.txt**：记录项目依赖
