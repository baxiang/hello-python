## 第二部分：多线程编程

### 14.3 创建线程

#### 基础用法

```python
import threading
import time

# ─────────────────────────────────────
# 方式 1：使用 Thread 类
# ─────────────────────────────────────

def worker(name, delay):
    """工作函数"""
    for i in range(3):
        time.sleep(delay)
        print(f"{name} 执行第 {i+1} 次")

# 创建线程对象
t1 = threading.Thread(target=worker, args=("线程 1", 1))
t2 = threading.Thread(target=worker, args=("线程 2", 1.5))

# 启动线程
t1.start()
t2.start()

# 等待线程完成
t1.join()
t2.join()

print("所有线程完成")
```

**线程生命周期图解：**
```
主线程
  │
  ├── 创建 Thread 对象
  │      │
  │      ▼
  │   线程（未启动）
  │
  ├── start()
  │      │
  │      ▼
  │   线程（运行中）← 操作系统调度
  │
  └── join()
         │
         ▼
      等待线程结束
         │
         ▼
      线程（已结束）
```

#### 线程参数详解

```python
import threading

def greet(name, greeting="Hello", times=1):
    for _ in range(times):
        print(f"{greeting}, {name}!")

# ─────────────────────────────────────
# args - 位置参数
# ─────────────────────────────────────
t1 = threading.Thread(target=greet, args=("Alice",))
# 注意：单个参数也要加逗号，形成元组

# ─────────────────────────────────────
# kwargs - 关键字参数
# ─────────────────────────────────────
t2 = threading.Thread(target=greet, kwargs={
    "name": "Bob",
    "greeting": "Hi",
    "times": 3
})

# ─────────────────────────────────────
# 混合使用
# ─────────────────────────────────────
t3 = threading.Thread(
    target=greet,
    args=("Carol",),           # 位置参数
    kwargs={"greeting": "Hey"}  # 关键字参数
)

# 启动
t1.start()
t2.start()
t3.start()
```

## 第二部分 - asyncio 异步编程



### 20.2 事件循环的类型

```python
import asyncio

# ─────────────────────────────────────
# 1. 获取当前事件循环
# ─────────────────────────────────────

# 在协程内部获取
async def inside_coroutine():
    loop = asyncio.get_running_loop()
    print(f"当前运行中的循环：{loop}")

# 在协程外部获取（Python 3.10+）
loop = asyncio.get_event_loop()
print(f"当前线程的循环：{loop}")

# ─────────────────────────────────────
# 2. 创建新的事件循环
# ─────────────────────────────────────

new_loop = asyncio.new_event_loop()
asyncio.set_event_loop(new_loop)

# ─────────────────────────────────────
# 3. 检查循环状态
# ─────────────────────────────────────

print(f"循环是否运行中：{loop.is_running()}")
print(f"循环是否已关闭：{loop.is_closed()}")
```
