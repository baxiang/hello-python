# 代码规范

本章讲解 Python 代码风格指南和格式化工具。

## PEP 8 风格指南

### 代码布局

```python
# ✅ 使用 4 个空格缩进
def hello():
    print("Hello")
    if True:
        print("World")

# ✅ 每行不超过 79 字符
result = (
    some_function(arg1, arg2, arg3)
    + another_function(arg4, arg5)
)

# ✅ 顶层函数/类之间空两行
def function_one():
    pass


def function_two():
    pass
```

### 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 变量/函数 | snake_case | `user_name`, `calculate_total()` |
| 类名 | PascalCase | `UserProfile`, `DataProcessor` |
| 常量 | UPPER_CASE | `MAX_SIZE`, `API_KEY` |
| 私有变量 | _prefix | `_internal_value` |

### 导入规范

```python
# ✅ 导入顺序：标准库 → 第三方库 → 本地模块
import os
import sys

import requests
```

## 类型提示

### 基本类型注解

```python
from typing import List, Dict, Optional, Union

# 基本类型
def greet(name: str) -> str:
    return f"Hello, {name}"

# 容器类型
def process_items(items: List[str]) -> Dict[str, int]:
    return {item: len(item) for item in items}

# 可选类型
def find_user(user_id: int) -> Optional[dict]:
    pass

# Python 3.10+ 新语法
def parse_value(value: str | int) -> str:
    return str(value)
```

### 类类型注解

```python
class Person:
    def __init__(self, name: str, age: int) -> None:
        self.name = name
        self.age = age

    def greet(self) -> str:
        return f"Hi, I'm {self.name}"
```

## 代码格式化工具

### Black 格式化器

```bash
# 安装
uv add --dev black

# 格式化整个项目
black .

# 检查格式（不修改）
black --check .
```

### Ruff Linter

```bash
# 安装
uv add --dev ruff

# 检查代码
ruff check .

# 自动修复
ruff check --fix .

# 格式化
ruff format .
```

### pyproject.toml 配置

```toml
[tool.ruff]
line-length = 88
target-version = "py311"

[tool.ruff.lint]
select = ["E", "W", "F", "I", "B"]
ignore = ["E501"]
```
