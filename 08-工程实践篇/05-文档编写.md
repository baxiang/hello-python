### 1.1 函数文档字符串

```python
def calculate_compound_interest(
    principal: float,
    rate: float,
    time: int,
    compound_frequency: int = 12
) -> float:
    """计算复利。
    
    根据本金、利率和时间计算复利收益。
    
    Args:
        principal: 本金金额
        rate: 年利率（小数形式，如 0.05 表示 5%）
        time: 投资时间（年）
        compound_frequency: 每年复利次数，默认 12（月复利）
    
    Returns:
        最终收益金额（包含本金）
    
    Raises:
        ValueError: 当本金或利率为负数时
        TypeError: 当参数类型不正确时
    
    Examples:
        >>> calculate_compound_interest(1000, 0.05, 10)
        1647.009...
        
        >>> calculate_compound_interest(1000, 0.05, 10, 1)  # 年复利
        1628.894...
    
    Note:
        此函数使用标准复利公式：A = P(1 + r/n)^(nt)
    
    See Also:
        calculate_simple_interest: 计算单利
        calculate_continuous_compound: 计算连续复利
    """
    if principal < 0 or rate < 0:
        raise ValueError("本金和利率必须为非负数")
    
    amount = principal * (1 + rate / compound_frequency) ** (compound_frequency * time)
    return amount
```

## 第二部分 - Sphinx 文档

### 2.1 安装与配置

```bash
# 安装 Sphinx
uv add --dev sphinx sphinx-rtd-theme

# 初始化文档
cd docs
sphinx-quickstart

# 目录结构
docs/
├── conf.py           # Sphinx 配置
├── index.rst         # 主文档
├── _static/          # 静态文件
├── _templates/       # 模板文件
└── source/           # 源代码（可选）
```

### 2.2 conf.py 配置

```python
# docs/conf.py
import sys
from pathlib import Path

# 添加源代码路径
sys.path.insert(0, str(Path('../src').resolve()))

# 项目信息
project = 'My Package'
copyright = '2024, Your Name'
author = 'Your Name'
release = '1.0.0'

# 扩展
extensions = [
    'sphinx.ext.autodoc',      # 自动生成 API 文档
    'sphinx.ext.napoleon',     # 支持 Google/NumPy 风格文档字符串
    'sphinx.ext.viewcode',     # 添加源代码链接
    'sphinx.ext.intersphinx',  # 链接到其他项目文档
    'sphinx.ext.todo',         # 支持 todo 指令
    'myst_parser',             # 支持 Markdown
]

# 主题
html_theme = 'sphinx_rtd_theme'

# autodoc 配置
autodoc_default_options = {
    'members': True,
    'member-order': 'bysource',
    'special-members': '__init__',
    'undoc-members': True,
    'exclude-members': '__weakref__',
}

# intersphinx 配置
intersphinx_mapping = {
    'python': ('https://docs.python.org/3', None),
    'requests': ('https://requests.readthedocs.io/en/latest/', None),
}
```

## 第三部分 - MkDocs 文档

### 3.1 安装与配置

```bash
# 安装 MkDocs
uv add --dev mkdocs mkdocs-material mkdocstrings

# 初始化
mkdocs new my-docs
cd my-docs
```

### 3.2 mkdocs.yml 配置

```yaml
# mkdocs.yml
site_name: My Package 文档
site_description: Python 包文档
site_author: Your Name
site_url: https://your-package.readthedocs.io

repo_name: your-username/my-package
repo_url: https://github.com/your-username/my-package

theme:
  name: material
  language: zh
  features:
    - navigation.tabs
    - navigation.sections
    - navigation.expand
    - search.suggest
    - search.highlight
  palette:
    - scheme: default
      primary: indigo
      toggle:
        icon: material/toggle-switch-off-outline
        name: 切换到深色模式
    - scheme: slate
      toggle:
        icon: material/toggle-switch
        name: 切换到浅色模式

plugins:
  - search
  - mkdocstrings:
      handlers:
        python:
          options:
            docstring_style: google

nav:
  - 首页: index.md
  - 安装: install.md
  - 使用指南:
    - 快速开始: guide/quickstart.md
    - 基础用法: guide/basic.md
    - 高级用法: guide/advanced.md
  - API 参考:
    - 核心模块: api/core.md
    - 客户端: api/client.md
  - 关于: about.md

markdown_extensions:
  - admonition
  - codehilite
  - toc:
      permalink: true
```

# docs/index.md

## 第四部分 - README 最佳实践

### 4.1 README 模板

[![PyPI version](https://badge.fury.io/py/my-package.svg)](https://badge.fury.io/py/my-package)

> 一句话描述你的项目

## 安装

```bash
pip install my-package
```

## 快速开始

```python
from my_package import Client

client = Client(api_key="your-key")
result = client.do_something()
print(result)
```

## 功能特点

- **功能一**: 描述
- **功能二**: 描述

pytest

## 最佳实践清单

### ✅ 必须遵守

1. 所有公共 API 必须有文档字符串
2. README 必须包含安装和使用示例
3. 复杂逻辑必须有注释说明

### ✅ 推荐做法

1. 使用 Sphinx 或 MkDocs 生成文档
2. 文档字符串遵循 Google 或 NumPy 风格
3. 在 CI 中构建和部署文档
4. 保持文档与代码同步更新
