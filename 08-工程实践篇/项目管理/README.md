## 核心知识点

### 包管理工具

- pip：Python 官方包管理器
- uv：新一代高速包管理器
- conda：跨平台包管理器

### 项目结构

```
my_project/
├── pyproject.toml
├── src/
│   └── my_package/
├── tests/
└── docs/
```

### 最佳实践

- 使用 src 布局
- 始终使用虚拟环境
- 锁定依赖版本
