# Python 工具箱 - 第 11、12、17 章合集

本目录将第 11 章（模块与包）、第 12 章（文件操作）、第 17 章（正则表达式）整合在一起，同时将第 11 章的标准库模块拆分为独立的参考文档，便于按需查阅。

---

## 目录结构

```
Python工具箱/
├── README.md               ← 本文件（总索引）
│
├── 11-模块与包.md           ← 模块机制、自定义模块、包、pip
│
├── 标准库/                  ← 标准库各模块独立参考文档
│   ├── math.md             ← 数学运算（sqrt/pi/ceil/log...）
│   ├── random.md           ← 随机数（randint/choice/shuffle...）
│   ├── datetime.md         ← 日期时间（now/strftime/timedelta...）
│   ├── os-pathlib.md       ← 系统接口与路径（os/pathlib 合并）
│   └── json.md             ← JSON 处理（dumps/loads/dump/load）
│
├── 12-文件操作.md           ← 文件读写、路径处理、异常处理
│
└── 17-正则表达式.md         ← 正则语法、re 模块、实际应用
```
